# 媒体对象

## 课程简介

媒体对象（Media Object）是 Bootstrap 提供的一种用于构建图文混排布局的组件，常见于博客评论、文章列表等场景。它可以将图片与文字内容并排展示，并支持嵌套、对齐方式控制和列表形式。

---

## 关键知识点

- 媒体对象基础结构：`media` > 图片 + `media-body`
- 嵌套媒体对象：在 `media-body` 内再嵌套 `media`
- 对齐方式：`align-self-start`（顶部）、`align-self-center`（居中）、`align-self-end`（底部）
- 列表形式的媒体对象：使用 `ul` + `li class="media"`
- 图片间距：`mr-3` 为图片添加右侧间距

---

## 关键代码示例

### 1. 基本媒体对象

```html
<div class="media">
  <a href="#"><img class="mr-3" src="images/img_05.jpg" alt="" width="150"></a>
  <div class="media-body">
    <h5 class="mt-0">每天运动能改善记忆能力吗？</h5>
    <p>媒体对象的正文内容...</p>
  </div>
</div>
```

**要点**：`.media` 类创建弹性盒容器，图片和 `.media-body` 并排显示。`mr-3` 为图片添加右侧间距。

### 2. 嵌套媒体对象

在 `media-body` 内部再添加一个 `media` 结构，实现评论回复效果。

```html
<div class="media">
  <a href="#"><img src="images/img_05.jpg" width="150" class="mr-3" alt=""></a>
  <div class="media-body">
    <h5>每天运动能改善记忆能力吗？</h5>
    <p>正文内容...</p>

    <!-- 嵌套的媒体对象 -->
    <div class="media mt-4">
      <a href="#"><img src="images/img_05.jpg" width="150" class="mr-3" alt=""></a>
      <div class="media-body">
        <h5>每天运动能改善记忆能力吗？</h5>
        <p>回复内容...</p>
      </div>
    </div>
  </div>
</div>
```

### 3. 对齐方式 - 顶部对齐

在图片的父元素（如 `<a>`）上添加 `align-self-start`，实现图片顶部对齐。

```html
<div class="media">
  <a href="#" class="align-self-start">
    <img src="images/img_05.jpg" width="150" class="mr-3" alt="">
  </a>
  <div class="media-body">
    <h5>每天运动能改善记忆能力吗？</h5>
    <p>内容...</p>
  </div>
</div>
```

### 4. 对齐方式 - 垂直居中对齐

```html
<div class="media">
  <a href="#" class="align-self-center">
    <img src="images/img_05.jpg" width="150" class="mr-3" alt="">
  </a>
  <div class="media-body">
    <h5>每天运动能改善记忆能力吗？</h5>
    <p>内容...</p>
  </div>
</div>
```

### 5. 对齐方式 - 底部对齐

```html
<div class="media">
  <a href="#" class="align-self-end">
    <img src="images/img_05.jpg" width="150" class="mr-3" alt="">
  </a>
  <div class="media-body">
    <h5>每天运动能改善记忆能力吗？</h5>
    <p>内容...</p>
  </div>
</div>
```

### 6. 列表形式的媒体对象

使用无序列表展示多条媒体对象，适合文章列表或评论列表场景。

```html
<ul class="list-unstyle">
  <li class="media">
    <a href="#"><img src="images/img_05.jpg" width="150" class="mr-3" alt=""></a>
    <div class="media-body">
      <h5>每天运动能改善记忆能力吗？</h5>
      <p>内容...</p>
    </div>
  </li>
  <li class="media my-4">
    <a href="#"><img src="images/img_05.jpg" width="150" class="mr-3" alt=""></a>
    <div class="media-body">
      <h5>每天运动能改善记忆能力吗？</h5>
      <p>内容...</p>
    </div>
  </li>
  <li class="media">
    <a href="#"><img src="images/img_05.jpg" width="150" class="mr-3" alt=""></a>
    <div class="media-body">
      <h5>每天运动能改善记忆能力吗？</h5>
      <p>内容...</p>
    </div>
  </li>
</ul>
```

---

## 效果说明

- 基本媒体对象实现图片在左、文字在右的图文混排布局
- 嵌套媒体对象可以模拟评论回复的层级结构，内层媒体对象通过 `mt-4` 添加上间距
- 三种对齐方式分别对应图片与文字的顶部、居中、底部对齐，默认为顶部对齐
- 列表形式的媒体对象适用于多条图文信息的展示，列表项之间可用 `my-4` 添加垂直间距
