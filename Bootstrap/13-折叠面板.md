# 第13课：折叠面板（Collapse）

## 课程简介

本课讲解 Bootstrap 4 中折叠面板（Collapse）组件的使用方法，包括基础折叠、多目标控制、手风琴效果以及通过 JavaScript 方法与事件控制折叠行为。

## 关键知识点

1. **折叠基础**：通过 `data-toggle="collapse"` 和 `data-target`（或 `href`）指定目标元素，实现点击切换显示/隐藏
2. **默认展开**：为目标元素添加 `show` 类，使其默认为展开状态
3. **多目标控制**：`data-target` 可以使用类选择器（如 `.all`），一个按钮同时控制多个折叠区域
4. **手风琴**：通过 `data-parent` 属性将多个折叠面板关联到同一个父容器，实现同时只展开一个的效果
5. **JavaScript 方法**：`.collapse('toggle')` / `.collapse('show')` / `.collapse('hide')`
6. **JavaScript 事件**：`show.bs.collapse` / `shown.bs.collapse` / `hide.bs.collapse` / `hidden.bs.collapse`

---

## 一、基础折叠

使用 `<a>` 标签或 `<button>` 标签触发折叠。`<a>` 标签用 `href` 指向目标，`<button>` 用 `data-target` 指向目标。

```html
<p>
  <a href="#kengdie" class="btn btn-primary" data-toggle="collapse">坑爹</a>
  <button class="btn btn-primary" data-toggle="collapse" data-target="#kengdie">坑爹</button>
</p>
<div class="collapse show" id="kengdie">
  <div class="card card-body">
    折叠面板的内容区域。
  </div>
</div>
```

**效果说明**：点击按钮或链接，折叠区域在显示与隐藏之间切换。默认展开（因为添加了 `show` 类）。

---

## 二、多个目标

一个按钮可以通过类选择器控制多个折叠区域。

```html
<p>
  <a href="#diaosi" class="btn btn-primary" data-toggle="collapse">屌丝</a>
  <button class="btn btn-primary" data-toggle="collapse" data-target="#tuhao">土豪</button>
  <button class="btn btn-primary" data-toggle="collapse" data-target="#nvshen">女神</button>
  <button class="btn btn-primary" data-toggle="collapse" data-target=".all">控制全部</button>
</p>

<div class="col-4 collapse all" id="diaosi">
  <div class="card card-body">屌丝内容...</div>
</div>
<div class="col-4 collapse all" id="tuhao">
  <div class="card card-body">土豪内容...</div>
</div>
<div class="col-4 collapse all" id="nvshen">
  <div class="card card-body">女神内容...</div>
</div>
```

**效果说明**：前三个按钮分别控制各自的折叠区域，"控制全部"按钮通过 `.all` 类选择器同时控制所有折叠区域的展开与收起。

## 三、手风琴效果

通过 `data-parent` 属性实现同一时间只能展开一个面板的效果。

```html
<div class="accordion" id="accordion">
  <!-- 第一项 -->
  <div class="card">
    <div class="card-header">
      <button class="btn btn-link" data-toggle="collapse" data-target="#item1">屌丝</button>
    </div>
    <div class="collapse show" id="item1" data-parent="#accordion">
      <div class="card-body">屌丝内容...</div>
    </div>
  </div>
  <!-- 第二项 -->
  <div class="card">
    <div class="card-header">
      <button class="btn btn-link" data-toggle="collapse" data-target="#item2">土豪</button>
    </div>
    <div class="collapse" id="item2" data-parent="#accordion">
      <div class="card-body">土豪内容...</div>
    </div>
  </div>
  <!-- 第三项 -->
  <div class="card">
    <div class="card-header">
      <button class="btn btn-link" data-toggle="collapse" data-target="#item3">女神</button>
    </div>
    <div class="collapse" id="item3" data-parent="#accordion">
      <div class="card-body">女神内容...</div>
    </div>
  </div>
</div>
```

**效果说明**：点击任意一项展开时，其余已展开的项会自动收起。`data-parent="#accordion"` 指定了共用的父容器。

## 四、JavaScript 方法与事件

通过 JS 代码手动控制折叠行为，并监听折叠过程中的事件。

```javascript
// 方法：切换折叠状态
$('.myBtn').click(function () {
  $('#myCollapse').collapse('toggle');  // 切换
  // $('#myCollapse').collapse('show'); // 显示
  // $('#myCollapse').collapse('hide'); // 隐藏
});

// 事件
$('#myCollapse').on('show.bs.collapse', function () {
  console.log('显示事件发生了');    // 调用 show 时触发
});
$('#myCollapse').on('shown.bs.collapse', function () {
  console.log('折叠块完全显示了！');  // 完全展开后触发
});
$('#myCollapse').on('hide.bs.collapse', function () {
  console.log('隐藏事件发生了');    // 调用 hide 时触发
});
$('#myCollapse').on('hidden.bs.collapse', function () {
  console.log('折叠块完全隐藏了！');  // 完全收起后触发
});
```

**效果说明**：`show`/`hide` 事件在动作开始时触发，`shown`/`hidden` 事件在动画完成后触发。
