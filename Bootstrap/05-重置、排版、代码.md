# 重置、排版、代码

## 课程简介

本节课介绍 Bootstrap 对 HTML 原生元素的重置样式、排版（Typography）相关工具类，以及代码展示相关的样式。Bootstrap 会对浏览器默认样式进行统一重置，并提供丰富的排版工具类来美化文本内容。

---

## 小节一：重置样式（01-重置.html）

### 课程简介

Bootstrap 对 HTML 原生元素进行了样式重置，确保跨浏览器的一致性表现。

### 关键知识点

- Bootstrap 使用 CSS 变量（Custom Properties）定义主题色
- 重置了表格、列表、表单等元素的默认样式
- `<table>` 添加 `.table` `.table-bordered` 类获得美化效果
- 标题、段落、列表等元素都有统一的默认样式

### 关键代码示例

#### CSS 变量使用

```css
/* Bootstrap 定义了 --blue 等主题色变量，可直接使用 */
.testVal {
  width: 100px;
  height: 100px;
  background: var(--blue);
}
```

#### 表格基础样式

```html
<table class="table table-bordered">
  <tr>
    <td>周一</td>
    <td>周二</td>
    <td>周三</td>
  </tr>
</table>
```

- `.table`：Bootstrap 表格基础类
- `.table-bordered`：添加边框

#### 原生元素在 Bootstrap 下的默认渲染

Bootstrap 对以下原生元素进行了样式重置，使其在不同浏览器下表现一致：

- 标题：`<h1>` ~ `<h6>`
- 段落：`<p>`
- 列表：`<ul>`、`<ol>`、`<dl>`
- 预格式化文本：`<pre>`
- 表格：`<table>`、`<caption>`
- 表单：`<form>`、`<fieldset>`、`<legend>`、`<input>`、`<select>`、`<textarea>`、`<button>`

### 效果说明

- CSS 变量让主题色的管理更加方便
- 表格添加 `.table` 类后获得统一的间距和排版
- 所有表单控件在 Bootstrap 下有一致的样式基准

## 小节二：排版（02-排版.html）

### 课程简介

Bootstrap 提供了丰富的排版工具类，用于控制标题、段落、文本装饰、列表等的展示效果。

### 关键知识点

#### 1. 标题样式

```html
<!-- 原生标题 -->
<h1>陈学辉</h1>
<h2>陈学辉</h2>
<h3>陈学辉</h3>

<!-- 使用 .h* 类让非标题元素拥有标题样式 -->
<p class="h1">陈学辉</p>
<p class="h2">陈学辉</p>
<p class="h3">陈学辉</p>
```

#### 2. 副标题

使用 `.text-muted` 配合 `<small>` 标签创建副标题：

```html
<h3>
  这是一个标题
  <small class="text-muted">这是一个副标题</small>
</h3>
```

#### 3. 更大的标题（Display 标题）

`.display-*` 用于创建更大、更醒目的标题，适用于页面主视觉区域：

```html
<h1>普通 h1 标题</h1>
<h1 class="display-1">Display 1</h1>
<h1 class="display-2">Display 2</h1>
<h1 class="display-3">Display 3</h1>
<h1 class="display-4">Display 4</h1>
```

#### 4. 引言段落

`.lead` 让段落文字更大、更突出：

```html
<p class="lead">这是一段引言文字</p>
<p>这是普通段落文字</p>
```

#### 5. 内联文本元素

```html
<!-- 标记重点 -->
<p>你可以使用 mark 标签来标记<mark>重点</mark>词汇</p>

<!-- 删除与插入 -->
<p><del>被删除的文本</del>，<ins>新插入的文本</ins></p>

<!-- 小号文本 -->
<p><small>这段文本的字体会小一些</small></p>

<!-- 加粗与倾斜 -->
<p><strong>加粗的文本</strong></p>
<p><em>倾斜的文本</em></p>

<!-- 使用类替代标签 -->
<p><span class="mark">用 .mark 代替 mark 标签</span></p>
<p><span class="small">用 .small 代替 small 标签</span></p>
```

#### 6. 缩写

```html
<!-- 基本缩写，鼠标悬停显示完整内容 -->
<p><abbr title="attribute">attr</abbr></p>

<!-- .initialism 让缩写字体更小 -->
<p><abbr title="HyperText Markup Language" class="initialism">HTML</abbr></p>
```

#### 7. 引用与署名

```html
<blockquote class="blockquote">
  <p>时间就像海绵里的水，只要愿挤，总还是有的</p>
  <footer class="blockquote-footer">来自<cite>鲁迅</cite></footer>
</blockquote>
```

#### 8. 文本对齐

```html
<p class="text-left">左对齐</p>
<p class="text-center">居中对齐</p>
<p class="text-right">右对齐</p>
```

#### 9. 列表样式

```html
<!-- 无样式列表（去掉默认的项目符号和缩进） -->
<ul class="list-unstyled">
  <li>red</li>
  <li>
    <ul class="list-unstyled">
      <li>blue</li>
    </ul>
  </li>
</ul>

<!-- 内联列表（水平排列） -->
<ul class="list-inline">
  <li class="list-inline-item">red</li>
  <li class="list-inline-item">blue</li>
  <li class="list-inline-item">green</li>
</ul>
```

> 注意：3.x 版本不需要在 `<li>` 上添加 `.list-inline-item` 类，4.x 需要。

#### 10. 描述列表与文字截断

```html
<dl class="row">
  <dt class="col-sm-3">关键词</dt>
  <dd class="col-sm-9">描述内容</dd>

  <!-- .text-truncate 让超出部分省略显示 -->
  <dt class="col-sm-3 text-truncate">很长很长很长很长的关键词...</dt>
  <dd class="col-sm-9">描述内容</dd>
</dl>
```

> `.text-truncate` 在 3.x 版本中对应的类是 `.text-overflow`。

### 效果说明

- `.display-*` 提供了 4 种超大标题尺寸，适合首页主标题
- `.lead` 让段落文字更突出，适合引言或首段
- `.list-unstyled` 和 `.list-inline` 可以快速改变列表的展示方式
- `.text-truncate` 实现单行文本溢出省略

## 小节三：代码（03-代码.html）

### 课程简介

Bootstrap 提供了代码展示相关的样式类，用于在页面中优雅地展示代码片段、键盘按键和变量。

### 关键知识点

#### 1. 内联代码

使用 `<code>` 标签在文本中嵌入代码片段：

```html
<p>我要在这段文本里写一个<code>&lt;section&gt;</code>的 HTML 标签</p>
```

> 注意：`<code>` 中的 `<` 和 `>` 需要转义为 `&lt;` 和 `&gt;`。

#### 2. 代码块

使用 `<pre>` 包裹 `<code>` 展示多行代码：

```html
<pre class="pre-scrollable">
  <code>
    &lt;h1&gt;这是一个标题&lt;/h1&gt;
    &lt;p&gt;这是一段文字&lt;/p&gt;
  </code>
</pre>
```

- `.pre-scrollable`：设置最大高度 340px，超出后显示滚动条

#### 3. 变量

使用 `<var>` 标签标记变量：

```html
<p><var>y</var> = <var>m</var><var>x</var> + <var>b</var></p>
```

#### 4. 键盘输入

使用 `<kbd>` 标签标记键盘按键：

```html
<p>想要保存的话，请按下 <kbd>ctrl</kbd> + <kbd>s</kbd></p>
```

### 效果说明

- `<code>` 显示为浅色背景的等宽字体，适合行内代码
- `.pre-scrollable` 限制代码块高度为 340px，超长代码自动出现滚动条
- `<var>` 以斜体显示变量名
- `<kbd>` 以深色背景的等宽字体显示键盘按键，模拟物理按键外观
