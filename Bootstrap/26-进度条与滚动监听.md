# 进度条与滚动监听

## 课程简介

本节课学习 Bootstrap 4 中的 **进度条（Progress）** 和 **滚动监听（ScrollSpy）** 组件。进度条用于展示任务完成进度，支持条纹和动画效果；滚动监听用于在页面滚动时自动更新导航的高亮状态。

---

## 一、进度条（Progress）

### 关键知识点

- `.progress` 是进度条的容器类（flex布局，高度1rem，带背景和圆角）
- `.progress-bar` 是进度条填充部分，需指定宽度（行间样式或预定义宽度类）
- 可通过 `bg-*` 类修改颜色：`bg-success`、`bg-danger`、`bg-info`、`bg-warning`
- 父级设置高度后子级自动继承
- 支持在一个进度条中放置多个 `.progress-bar`
- `.progress-bar-striped` 添加条纹效果
- `.progress-bar-animated` 添加动画条纹效果

### 关键代码示例

**基础进度条：**

```html
<!-- 通过行间样式指定宽度 -->
<div class="progress">
  <div class="progress-bar" style="width: 25%"></div>
</div>

<!-- 通过预定义类指定宽度 -->
<div class="progress">
  <div class="progress-bar w-50"></div>
</div>
```

**自定义高度：**

```html
<div class="progress" style="height: 50px;">
  <div class="progress-bar" style="width: 75%"></div>
</div>
```

**不同颜色：**

```html
<div class="progress">
  <div class="progress-bar bg-success" style="width: 90%"></div>
</div>
<div class="progress">
  <div class="progress-bar bg-danger" style="width: 90%"></div>
</div>
```

**多个进度条：**

```html
<div class="progress" style="height: 50px;">
  <div class="progress-bar" style="width: 15%"></div>
  <div class="progress-bar bg-warning" style="width: 30%"></div>
  <div class="progress-bar bg-info" style="width: 20%"></div>
</div>
```

**条纹进度条：**

```html
<div class="progress">
  <div class="progress-bar progress-bar-striped" style="width: 50%"></div>
</div>
<div class="progress">
  <div class="progress-bar progress-bar-striped bg-success" style="width: 25%"></div>
</div>
```

**动画条纹进度条：**

```html
<div class="progress">
  <div class="progress-bar progress-bar-striped progress-bar-animated" style="width: 75%"></div>
</div>
```

### 效果说明

进度条以水平条形式展示完成百分比，支持多种颜色、条纹和动画效果。可在同一个进度条内放置多段不同颜色的进度。

---

## 二、滚动监听（ScrollSpy）

### 关键知识点

- 滚动监听只能用在 **导航（nav）** 和 **列表组（list-group）** 中
- 如果滚动的是 body 以外的元素，需添加 `height` 和 `overflow-y: scroll` 样式
- 需要引入 JS
- 关键属性：
  - `data-spy="scroll"` -- 让元素具有滚动监听功能
  - `data-target="#id"` -- 指向被监听的导航/列表组的 id
  - `data-offset="0"` -- 计算滚动位置时的偏移量
- 内容区域中的标题 id 需与导航链接的 href 值一一对应
- 支持 **导航中嵌套** 的形式
- 事件：`activate.bs.scrollspy` -- 每当滚动到新项时触发

### 关键代码示例

**导航中的滚动监听：**

```html
<!-- 导航栏 -->
<nav class="navbar navbar-light bg-light" id="scroll1">
  <a href="#" class="navbar-brand">Navbar</a>
  <ul class="nav nav-pills">
    <li class="nav-item">
      <a class="nav-link" href="#fat">@fat</a>
    </li>
    <li class="nav-item">
      <a class="nav-link" href="#mdo">@mdo</a>
    </li>
    <li class="nav-item dropdown">
      <a href="#" class="nav-link dropdown-toggle" data-toggle="dropdown">Dropdown</a>
      <div class="dropdown-menu">
        <a class="dropdown-item" href="#one">one</a>
        <a class="dropdown-item" href="#two">two</a>
        <a class="dropdown-item" href="#three">three</a>
      </div>
    </li>
  </ul>
</nav>

<!-- 滚动内容区域 -->
<div class="scrollBody" data-spy="scroll" data-target="#scroll1" data-offset="0">
  <h4 id="fat">@fat</h4>
  <p>内容...</p>
  <h4 id="mdo">@mdo</h4>
  <p>内容...</p>
  <h4 id="one">one</h4>
  <p>内容...</p>
</div>
```

**嵌套导航的滚动监听：**

```html
<nav class="navbar navbar-light bg-light flex-column" id="scroll2">
  <a class="navbar-brand" href="#">Navbar</a>
  <nav class="nav nav-pills flex-column">
    <a class="nav-link" href="#item-1">Item 1</a>
    <nav class="nav nav-pills flex-column">
      <a class="nav-link ml-3 my-1" href="#item-1-1">Item 1-1</a>
      <a class="nav-link ml-3 my-1" href="#item-1-2">Item 1-2</a>
    </nav>
    <a class="nav-link" href="#item-2">Item 2</a>
    <a class="nav-link" href="#item-3">Item 3</a>
  </nav>
</nav>
```

**列表组中的滚动监听：**

```html
<!-- 列表组导航 -->
<div class="list-group" id="list-example">
  <a class="list-group-item list-group-item-action" href="#list-item-1">Item 1</a>
  <a class="list-group-item list-group-item-action" href="#list-item-2">Item 2</a>
  <a class="list-group-item list-group-item-action" href="#list-item-3">Item 3</a>
  <a class="list-group-item list-group-item-action" href="#list-item-4">Item 4</a>
</div>

<!-- 滚动内容 -->
<div class="scrollBody" data-spy="scroll" data-target="#list-example" data-offset="0">
  <h4 id="list-item-1">Item 1</h4>
  <p>内容...</p>
  <h4 id="list-item-2">Item 2</h4>
  <p>内容...</p>
</div>
```

**滚动监听事件：**

```javascript
$('[data-spy="scroll"]').on('activate.bs.scrollspy', function() {
  console.log('滚动到新项了');
});
```

### 效果说明

当内容区域滚动时，对应的导航项会自动高亮。点击导航项时内容平滑滚动到对应位置。支持导航栏、嵌套导航和列表组三种形式。
