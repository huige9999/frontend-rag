# 11 - 卡片

## 课程简介

本节课学习 Bootstrap 4 中的 **卡片（Cards）** 组件。卡片是一个灵活的内容容器，可以包含页眉、页脚、图片、列表、链接等多种内容类型。本课涵盖卡片的基础结构、图片、列表组、对齐、导航、图像覆盖、样式配色以及三种排版方式（卡片组、卡片 deck、瀑布流）。

---

## 一、卡片基础结构

### 关键知识点

- `.card`：卡片容器
- `.card-body`：卡片内容区域
- `.card-title`：卡片标题
- `.card-subtitle`：卡片副标题
- `.card-text`：卡片文本内容
- `.card-link`：卡片中的链接

### 关键代码示例

```html
<div class="card">
    <div class="card-body">
        <h3 class="card-title">这是卡片的标题</h3>
        <h4 class="card-subtitle mb-3">这是卡片的副标题</h4>
        <p class="card-text">这是卡片的文本内容</p>
        <a href="#" class="card-link">链接1</a>
        <a href="#" class="card-link">链接2</a>
    </div>
</div>
```

---

## 二、图片卡片

### 关键知识点

- `.card-img-top`：图片在卡片顶部
- `.card-img-bottom`：图片在卡片底部
- `.card-img-overlay`：文字覆盖在图片上方

### 关键代码示例

**图片在上：**
```html
<div class="card">
    <img class="card-img-top" src="图片.jpg" alt="">
    <div class="card-body">
        <p class="card-text">图片的说明文字</p>
    </div>
</div>
```

**图片在下：**
```html
<div class="card">
    <div class="card-body">
        <h5 class="card-title">标题</h5>
        <p class="card-text">文本内容</p>
    </div>
    <img class="card-img-bottom" src="图片.jpg" alt="">
</div>
```

**文字覆盖图片：**
```html
<div class="card">
    <img class="card-img-top" src="图片.jpg" alt="">
    <div class="card-img-overlay text-white">
        <h5 class="card-title">标题</h5>
        <p class="card-text">覆盖在图片上的文字</p>
    </div>
</div>
```

## 三、列表组、页眉、页脚

### 关键代码示例

```html
<div class="card">
    <div class="card-header">这是卡片的头部（页眉）</div>
    <ul class="list-group list-group-flush">
        <li class="list-group-item">列表项1</li>
        <li class="list-group-item">列表项2</li>
        <li class="list-group-item">列表项3</li>
    </ul>
    <div class="card-footer">这是卡片的底部（页脚）</div>
</div>
```

### 效果说明

`.list-group-flush` 使列表项与卡片无缝贴合（去除列表项的外边框）。页眉和页脚自动带有背景色和边框线。

## 四、组合模式

### 关键代码示例

```html
<div class="card">
    <div class="card-header">标题</div>
    <img class="card-img-top" src="图片.jpg" alt="">
    <div class="card-body">
        <h5 class="card-title">卡片标题</h5>
        <p class="card-text">文本内容...</p>
        <a href="#" class="card-link btn btn-primary">按钮1</a>
        <a href="#" class="card-link btn btn-info">按钮2</a>
    </div>
    <ul class="list-group list-group-flush">
        <li class="list-group-item">列表项</li>
    </ul>
    <div class="card-footer">页脚</div>
</div>
```

### 效果说明

卡片内部可以灵活组合页眉、图片、内容体、列表组、页脚等各个部分，实现丰富的内容展示。

## 七、卡片样式

### 关键知识点

- **背景色**：`.bg-primary`、`.bg-secondary`、`.bg-success`、`.bg-danger`、`.bg-warning`、`.bg-info`、`.bg-light`、`.bg-dark`
- **边框色**：`.border-primary`、`.border-secondary` 等配合 `.card` 使用
- **页眉/页脚透明边框**：`.bg-transparent` + `.border-*`

### 关键代码示例

**背景色卡片：**
```html
<div class="card text-white bg-primary">
    <div class="card-header">头部</div>
    <div class="card-body">
        <h5 class="card-title">标题</h5>
        <p class="card-text">内容...</p>
    </div>
</div>
```

**边框色卡片：**
```html
<div class="card border-primary">
    <div class="card-header">头部</div>
    <div class="card-body text-primary">
        <h5 class="card-title">标题</h5>
        <p class="card-text">内容...</p>
    </div>
</div>
```

**透明页眉/页脚：**
```html
<div class="card border-success">
    <div class="card-header bg-transparent border-success">头部</div>
    <div class="card-body text-primary">内容...</div>
    <div class="card-footer bg-transparent border-success">底部</div>
</div>
```

## 八、卡片排版

### 关键知识点

- **卡片组（`.card-group`）**：卡片之间紧密相连，无间距，等高
- **卡片 Deck（`.card-deck`）**：卡片之间有间距，互不相连，但等高
- **瀑布流（`.card-columns`）**：卡片以瀑布流方式排列，高度不等

### 关键代码示例

**卡片组：**
```html
<div class="card-group">
    <div class="card">
        <img class="card-img-top" src="图片.jpg" alt="">
        <div class="card-body">
            <h5 class="card-title">标题</h5>
            <p class="card-text">内容...</p>
        </div>
    </div>
    <!-- 更多卡片... -->
</div>
```

**卡片 Deck：**
```html
<div class="card-deck">
    <div class="card">
        <img class="card-img-top" src="图片.jpg" alt="">
        <div class="card-body">
            <h5 class="card-title">标题</h5>
            <p class="card-text">内容...</p>
        </div>
    </div>
    <!-- 更多卡片... -->
</div>
```

**瀑布流：**
```html
<div class="card-columns">
    <div class="card">
        <img class="card-img-top" src="图片.jpg" alt="">
        <div class="card-body">
            <h5 class="card-title">标题1</h5>
            <p class="card-text">内容...</p>
        </div>
    </div>
    <div class="card">
        <img class="card-img-top" src="图片.jpg" alt="">
        <div class="card-body">
            <h5 class="card-title">标题2</h5>
            <p class="card-text">更长的内容...</p>
        </div>
    </div>
    <!-- 更多卡片，自动瀑布流排列 -->
</div>
```

### 效果说明

- `.card-group`：卡片紧贴排列，底边对齐，适合统一风格的展示
- `.card-deck`：卡片之间有间距但保持等高，适合产品列表
- `.card-columns`：自动多列瀑布流排列，卡片高度可以不同，适合博客或新闻流
