# 实战项目-2

## 课程简介

本节课是 Bootstrap 实战项目的第二部分，在上一节课基础上完成**右侧内容区域**的开发，包括轮播图、三栏图片展示、星享俱乐部、星巴克精选、咖啡星讲堂等模块，并添加底部移动端菜单栏和菜单弹出时的滚动条控制。

---

## 关键知识点

- 使用 Carousel 组件实现图片轮播（控制器 + 指示器）
- 使用 `img-fluid` + `img-thumbnail` 实现响应式缩略图
- 使用 Flex 布局实现内容区域的灵活排列
- 使用 `fixed-bottom` + `d-sm-none` 实现移动端固定底部菜单
- 使用 jQuery 监听 `window.resize` 处理响应式逻辑
- 通过控制 `body` 的 `overflow` 属性管理菜单弹出时的滚动行为

### JavaScript 完整交互逻辑

```javascript
// 菜单显示/隐藏
$('#showBtn').on('click', function(){
    $('.menu').removeClass('hide').addClass('show');
    shadowMenu();
});
$('#closeBtn').on('click', function(){
    $('.menu').removeClass('show').addClass('hide');
    shadowMenu();
});

// 窗口大小改变时重新计算
$(window).resize(shadowMenu);

// 控制菜单弹出时的滚动条
function shadowMenu(){
    // 屏幕宽度<=992 且 菜单处于显示状态时，隐藏body滚动条
    if($(window).innerWidth() <= 992 && $('.menu').hasClass('show')){
        $('body').css('overflow', 'hidden');
    } else {
        $('body').css('overflow', 'auto');
    }
}
```

## 效果说明

- 右侧内容区包含轮播图、三栏图片、星享俱乐部、星巴克精选、咖啡星讲堂等完整模块
- 轮播图支持左右切换和底部指示器，使用 iconfont 图标替代默认箭头
- 星巴克精选使用四栏卡片布局，响应式在移动端变为两栏（`col-6 col-md-3`）
- 移动端（sm以下）显示固定底部菜单栏，桌面端隐藏（`d-sm-none`）
- 菜单弹出时自动隐藏 body 滚动条，关闭后恢复，提升移动端体验
- 窗口尺寸变化时自动检测并调整滚动条状态
