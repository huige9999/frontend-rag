# 第17课：输入框组（Input Group）

## 课程简介

本课讲解 Bootstrap 4 输入框组（Input Group）的使用方法，通过在输入框前后添加文本、按钮、下拉菜单等额外组件，扩展表单输入的功能和视觉效果。

## 关键知识点

1. **输入框组结构**：`.input-group` 容器 + `.input-group-prepend`（前置） / `.input-group-append`（后置） + `.input-group-text`（文本内容）
2. **前后置添加**：可单独前置、单独后置或同时前后置
3. **支持的控件**：`<input>`、`<textarea>`、`<select>`、自定义文件上传
4. **尺寸控制**：`.input-group-lg` / `.input-group-sm`
5. **嵌入复选框/单选框**：在 `.input-group-text` 内嵌 `<input type="checkbox/radio">`
6. **多个输入框**：一个输入框组内可放置多个 `<input>`
7. **多个额外组件**：`.input-group-prepend/append` 内可放置多个 `.input-group-text`
8. **按钮组合**：在 prepend/append 中直接放置 `<button>`
9. **下拉菜单组合**：在 prepend/append 中放置带 `dropdown-toggle` 的按钮
10. **分离式按钮组合**：使用 `.dropdown-toggle-split` 的分离式下拉按钮
11. **自定义选择框**：`.custom-select` 与输入框组配合
12. **自定义文件上传**：`.custom-file` 与输入框组配合

---

## 一、基础用法（前后置文本）

```html
<!-- 前置文本 -->
<div class="input-group">
  <div class="input-group-prepend">
    <span class="input-group-text">@</span>
  </div>
  <input type="text" class="form-control" placeholder="用户名">
</div>

<!-- 后置文本 -->
<div class="input-group">
  <input type="text" class="form-control" placeholder="邮箱名">
  <div class="input-group-append">
    <span class="input-group-text">@example.com</span>
  </div>
</div>

<!-- 前后都有文本 -->
<div class="input-group">
  <div class="input-group-prepend">
    <span class="input-group-text">$</span>
  </div>
  <input type="text" class="form-control">
  <div class="input-group-append">
    <span class="input-group-text">.00</span>
  </div>
</div>
```

**效果说明**：`.input-group-prepend` 在输入框左侧添加前缀，`.input-group-append` 在右侧添加后缀。

## 二、textarea 与输入框组

```html
<div class="input-group">
  <div class="input-group-prepend">
    <span class="input-group-text">文本输入框</span>
  </div>
  <textarea class="form-control"></textarea>
</div>
```

---

## 三、尺寸控制

```html
<div class="input-group input-group-lg">
  <div class="input-group-prepend">
    <span class="input-group-text">大尺寸</span>
  </div>
  <input type="text" class="form-control">
</div>

<div class="input-group">
  <div class="input-group-prepend">
    <span class="input-group-text">正常尺寸</span>
  </div>
  <input type="text" class="form-control">
</div>

<div class="input-group input-group-sm">
  <div class="input-group-prepend">
    <span class="input-group-text">小尺寸</span>
  </div>
  <input type="text" class="form-control">
</div>
```

**效果说明**：`.input-group-lg` 和 `.input-group-sm` 添加在 `.input-group` 上，会同时控制输入框和附加组件的大小。

---

## 四、嵌入复选框与单选框

```html
<!-- 单选框 -->
<div class="input-group">
  <div class="input-group-prepend">
    <span class="input-group-text">
      <input type="radio">
    </span>
  </div>
  <input type="text" class="form-control">
</div>

<!-- 复选框 -->
<div class="input-group">
  <div class="input-group-prepend">
    <span class="input-group-text">
      <input type="checkbox">
    </span>
  </div>
  <input type="text" class="form-control">
</div>
```

**效果说明**：在 `.input-group-text` 内嵌入 `<input type="radio/checkbox">`，使输入框前显示一个可选中的按钮。

---

## 五、多个输入框与多个额外组件

```html
<!-- 多个输入框 -->
<div class="input-group">
  <div class="input-group-prepend">
    <span class="input-group-text">多个输入框</span>
  </div>
  <input type="text" class="form-control" placeholder="请输入姓">
  <input type="text" class="form-control" placeholder="请输入名">
  <input type="text" class="form-control">
</div>

<!-- 多个额外组件 -->
<div class="input-group">
  <input type="text" class="form-control">
  <div class="input-group-append">
    <span class="input-group-text">$</span>
    <span class="input-group-text">0.00</span>
  </div>
</div>
```

---

## 六、按钮组合

```html
<!-- 前置按钮 -->
<div class="input-group">
  <div class="input-group-prepend">
    <button class="btn btn-outline-secondary">左按钮</button>
  </div>
  <input type="text" class="form-control">
</div>

<!-- 后置按钮 -->
<div class="input-group">
  <input type="text" class="form-control">
  <div class="input-group-append">
    <button class="btn btn-outline-secondary">右按钮</button>
  </div>
</div>

<!-- 多个按钮 -->
<div class="input-group">
  <div class="input-group-prepend">
    <button class="btn btn-outline-secondary">左按钮1</button>
    <button class="btn btn-outline-secondary">左按钮2</button>
  </div>
  <input type="text" class="form-control">
</div>
```

---

## 七、带下拉菜单的按钮组合

```html
<div class="input-group">
  <div class="input-group-prepend">
    <button class="btn btn-outline-secondary dropdown-toggle" data-toggle="dropdown">左侧下拉菜单</button>
    <div class="dropdown-menu">
      <a href="#" class="dropdown-item">北京</a>
      <a href="#" class="dropdown-item">上海</a>
      <a href="#" class="dropdown-item">广州</a>
      <div class="dropdown-divider"></div>
      <a href="#" class="dropdown-item">深圳</a>
    </div>
  </div>
  <input type="text" class="form-control">
</div>
```

### 分离式按钮组合

```html
<div class="input-group">
  <div class="input-group-prepend">
    <button class="btn btn-outline-success">分离式左侧下拉菜单</button>
    <button class="btn btn-outline-success dropdown-toggle dropdown-toggle-split"
      data-toggle="dropdown"></button>
    <div class="dropdown-menu">...</div>
  </div>
  <input type="text" class="form-control">
</div>
```

## 八、自定义选择框组合

```html
<!-- label 标签在左边 -->
<div class="input-group">
  <div class="input-group-prepend">
    <label for="select1" class="input-group-text">label标签在左边</label>
  </div>
  <select class="custom-select" id="select1">
    <option>城市</option>
    <option>北京</option>
    <option>上海</option>
  </select>
</div>

<!-- 按钮在右边 -->
<div class="input-group">
  <select class="custom-select">
    <option>城市</option>
    <option>北京</option>
  </select>
  <div class="input-group-append">
    <button class="btn btn-outline-danger">button标签在右边</button>
  </div>
</div>
```

## 九、自定义文件上传组合

```html
<!-- 上传在左边 -->
<div class="input-group">
  <div class="input-group-prepend">
    <span class="input-group-text">上传在左边</span>
  </div>
  <div class="custom-file">
    <input type="file" class="custom-file-input" id="file1">
    <label for="file1" class="custom-file-label"></label>
  </div>
</div>

<!-- 按钮在右边 -->
<div class="input-group">
  <div class="custom-file">
    <input type="file" class="custom-file-input" id="file2">
    <label for="file2" class="custom-file-label"></label>
  </div>
  <div class="input-group-append">
    <button class="btn btn-outline-secondary">按钮在右边</button>
  </div>
</div>
```

**效果说明**：`.custom-file` 组件可以与 `.input-group` 配合，在文件选择器前后添加提示文本或操作按钮。
