# 07 - 边框、颜色、显示、嵌入

## 课程简介

本节课学习 Bootstrap 4 中的四个工具类：**边框（Borders）**、**颜色（Colors）**、**显示（Display）** 和 **嵌入（Embeds）**。这些工具类可以帮助开发者快速设置元素的边框样式、文字与背景颜色、显示类型以及响应式嵌入多媒体内容。

---

## 一、边框（Borders）

### 关键知识点

- **添加边框**：使用 `.border`、`.border-top`、`.border-right`、`.border-bottom`、`.border-left` 为元素添加各方向的边框
- **删除边框**：使用 `.border-0`、`.border-top-0` 等移除指定方向的边框
- **边框颜色**：使用 `.border-primary`、`.border-secondary`、`.border-success`、`.border-danger`、`.border-warning`、`.border-info`、`.border-light`、`.border-dark`、`.border-white` 设置边框颜色
- **边框圆角**：使用 `.rounded`、`.rounded-top`、`.rounded-right`、`.rounded-bottom`、`.rounded-left` 设置圆角

### 关键代码示例

**添加边框：**
```html
<span class="border">四边边框</span>
<span class="border-top">上边框</span>
<span class="border-right">右边框</span>
<span class="border-bottom">下边框</span>
<span class="border-left">左边框</span>
```

**删除边框：**
```html
<span class="border border-0">移除全部边框</span>
<span class="border-top border-top-0">移除上边框</span>
```

**边框颜色：**
```html
<span class="border border-primary">主色边框</span>
<span class="border border-success">成功色边框</span>
<span class="border border-danger">危险色边框</span>
```

**边框圆角：**
```html
<span class="rounded">四边圆角</span>
<span class="rounded-top">上方圆角</span>
<span class="rounded-circle">圆形</span>
```

### 效果说明

边框类可以为任何行内或块级元素添加指定方向的 1px 实线边框，并可配合颜色类和圆角类组合使用，实现丰富的边框效果。

---

## 二、颜色（Colors）

### 关键知识点

- **文字颜色**：使用 `.text-primary`、`.text-secondary`、`.text-success`、`.text-danger`、`.text-warning`、`.text-info`、`.text-light`、`.text-dark`、`.text-body`、`.text-muted`、`.text-white`、`.text-black-50`、`.text-white-50` 设置文字颜色
- **链接颜色**：同样适用于 `<a>` 标签，使用 `.text-*` 类名
- **背景颜色**：使用 `.bg-primary`、`.bg-secondary`、`.bg-success`、`.bg-danger`、`.bg-warning`、`.bg-info`、`.bg-light`、`.bg-dark`、`.bg-white`、`.bg-transparent` 设置背景色

### 关键代码示例

**文字颜色：**
```html
<p class="text-primary">主色文字</p>
<p class="text-success">成功色文字</p>
<p class="text-danger">危险色文字</p>
<p class="text-muted">灰色弱化文字</p>
<p class="text-white-50 bg-dark">50%透明白色文字</p>
```

**链接颜色：**
```html
<a href="#" class="text-primary">主色链接</a>
<a href="#" class="text-danger">危险色链接</a>
```

**背景颜色：**
```html
<div class="text-white bg-primary">主色背景</div>
<div class="text-white bg-success">成功色背景</div>
<div class="text-dark bg-transparent">透明背景</div>
```

### 效果说明

颜色类提供了统一的语义化配色方案，文字颜色和背景颜色可以自由组合。`.text-white-50` 和 `.text-black-50` 分别表示 50% 透明度的白色和黑色。浅色文字建议搭配深色背景使用（如 `.text-light` 搭配 `.bg-dark`）。

## 三、显示（Display）

### 关键知识点

- **显示类型**：使用 `.d-inline`、`.d-block`、`.d-inline-block`、`.d-flex`、`.d-table` 等改变元素的显示类型
- **响应式显示/隐藏**：使用 `.d-{breakpoint}-block`、`.d-none` 配合断点控制元素在不同屏幕尺寸下的显示与隐藏
- **仅在指定尺寸显示**：通过 `.d-none` 配合 `.d-{breakpoint}-block` 实现只在某个断点下显示
- **在指定尺寸隐藏**：使用 `.d-{breakpoint}-none` 实现从某个断点开始隐藏
- **打印显示**：使用 `.d-print-block` 控制打印时的显示，`.d-none` 配合隐藏屏幕显示

### 关键代码示例

**改变显示类型：**
```html
<div class="d-inline" style="width: 150px;">d-inline 行内元素</div>
<span class="d-block">d-block 块级元素</span>
<span class="d-inline-block" style="width: 200px;">d-inline-block 行内块元素</span>
<div class="d-flex">d-flex 弹性盒</div>
<div class="d-table">d-table 表格显示</div>
```

**仅在指定屏幕尺寸下显示：**
```html
<div class="d-xl-block d-none">只在超大屏幕下显示</div>
<div class="d-none d-lg-block d-xl-none">只在大屏幕下显示</div>
<div class="d-none d-md-block d-lg-none">只在中等屏幕下显示</div>
<div class="d-none d-sm-block d-md-none">只在小屏幕下显示</div>
<div class="d-sm-none">只在超小屏幕下显示</div>
```

**在指定尺寸以上隐藏：**
```html
<div class="d-xl-none">超大屏幕以外显示</div>
<div class="d-lg-none d-xl-block">大屏以外显示</div>
<div class="d-md-none d-lg-block">中等屏以外显示</div>
```

**打印时显示：**
```html
<div class="d-none d-print-block">仅在打印时显示此内容</div>
```

### 效果说明

Bootstrap 4 的显示工具类替代了 3.x 版本的 `.visible-*-block` 和 `.hidden-*` 写法，语法更加统一。通过 `d-{breakpoint}-{value}` 的组合，可以精确控制元素在各断点下的显示行为。

## 四、嵌入（Embeds）

### 关键知识点

- **响应式嵌入**：使用 `.embed-responsive` 配合比例类创建响应式的嵌入容器
- **宽高比**：支持 `.embed-responsive-21by9`（21:9）、`.embed-responsive-16by9`（16:9）、`.embed-responsive-4by3`（4:3）、`.embed-responsive-1by1`（1:1）
- 适用于 `<iframe>`、`<embed>`、`<video>` 等多媒体元素

### 关键代码示例

```html
<!-- 21:9 比例 -->
<div class="embed-responsive embed-responsive-21by9">
    <iframe src="视频地址" allowfullscreen></iframe>
</div>

<!-- 16:9 比例 -->
<div class="embed-responsive embed-responsive-16by9">
    <iframe src="视频地址" allowfullscreen></iframe>
</div>

<!-- 4:3 比例 -->
<div class="embed-responsive embed-responsive-4by3">
    <iframe src="视频地址" allowfullscreen></iframe>
</div>

<!-- 1:1 比例 -->
<div class="embed-responsive embed-responsive-1by1">
    <iframe src="视频地址" allowfullscreen></iframe>
</div>
```

### 效果说明

嵌入组件会根据父容器的宽度自动调整内部多媒体元素的大小，保持设定的宽高比不变。这在嵌入视频（如优酷、YouTube）时特别有用，确保视频在任何屏幕尺寸下都能正确显示。
