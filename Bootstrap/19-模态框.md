# 模态框

## 课程简介

模态框（Modal）是 Bootstrap 中最常用的交互组件之一，用于在页面上弹出对话框，展示重要内容、表单或提示信息，用户必须与之交互后才能返回主页面。本节课讲解模态框的结构、居中、栅格嵌套、尺寸控制、data属性以及方法与事件。

---

## 关键知识点

- 模态框基本结构：`modal` > `modal-dialog` > `modal-content` > `modal-header` / `modal-body` / `modal-footer`
- 垂直居中：`modal-dialog-centered`
- 模态框内嵌套栅格系统：`container-fluid`
- 尺寸控制：`modal-xl`、`modal-lg`、`modal-sm`
- 通过按钮触发模态框：`data-toggle="modal"` + `data-target="#id"`
- 关闭模态框：`data-dismiss="modal"`
- data属性控制行为：`data-backdrop`、`data-keyboard`、`data-focus`、`data-show`
- JavaScript方法：`.modal('show')`、`.modal('hide')`
- JavaScript事件：`show.bs.modal`、`shown.bs.modal`、`hide.bs.modal`、`hidden.bs.modal`
- 动态修改模态框内容：通过 `show.bs.modal` 事件获取 `relatedTarget`

## 关键代码示例

### 1. 基本模态框结构

```html
<!-- 触发按钮 -->
<button class="btn btn-primary" data-toggle="modal" data-target="#modal1">普通模态框</button>

<!-- 模态框 -->
<div class="modal fade" id="modal1">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title">标题</h5>
        <button class="close" data-dismiss="modal">
          <span>&times;</span>
        </button>
      </div>
      <div class="modal-body">
        模态框主体内容...
      </div>
      <div class="modal-footer">
        <button class="btn btn-secondary" data-dismiss="modal">关闭</button>
        <button class="btn btn-primary">保存</button>
      </div>
    </div>
  </div>
</div>
```

### 2. 垂直居中的模态框

在 `modal-dialog` 上添加 `modal-dialog-centered` 类。

```html
<div class="modal fade" id="modal2">
  <div class="modal-dialog modal-dialog-centered">
    <div class="modal-content">
      <!-- header / body / footer 同上 -->
    </div>
  </div>
</div>
```

### 3. 模态框内使用栅格系统

在 `modal-body` 内使用 `container-fluid` 包裹栅格。

```html
<div class="modal-body">
  <div class="container-fluid">
    <div class="row">
      <div class="col-md-4 bg-info border">第1列</div>
      <div class="col-md-4 bg-info border">第2列</div>
      <div class="col-md-4 bg-info border">第3列</div>
    </div>
  </div>
</div>
```

### 4. 包含表单的模态框

```html
<div class="modal-body">
  <form action="#">
    <div class="form-group">
      <label for="username" class="col-form-label">用户名：</label>
      <input type="text" class="form-control" id="username">
    </div>
    <div class="form-group">
      <label for="comment" class="col-form-label">评论：</label>
      <textarea class="form-control" id="comment"></textarea>
    </div>
  </form>
</div>
```

### 5. 不同尺寸的模态框

通过在 `modal-dialog` 上添加尺寸类控制大小。

```html
<!-- 超大尺寸 -->
<div class="modal-dialog modal-xl">...</div>

<!-- 大尺寸 -->
<div class="modal-dialog modal-lg">...</div>

<!-- 小尺寸 -->
<div class="modal-dialog modal-sm">...</div>
```

### 6. data属性

在触发按钮上添加 data 属性控制模态框行为。

```html
<button class="btn btn-dark"
  data-toggle="modal"
  data-target="#modal8"
  data-backdrop="true"    <!-- 是否显示遮罩层 -->
  data-keyboard="false"   <!-- 按ESC是否关闭模态框 -->
  data-focus="true"       <!-- 让模态框获取焦点 -->
  data-show="false">      <!-- 初始化时是否显示 -->
  data属性
</button>
```

### 7. 动态修改模态框内容

多个按钮触发同一个模态框，但传递不同数据。

```html
<button class="btn btn-success" data-toggle="modal" data-target="#modal4" data-whatever="kaivon">按钮1</button>
<button class="btn btn-success" data-toggle="modal" data-target="#modal4" data-whatever="陈学辉">按钮2</button>
```

```javascript
$('#modal4').on('show.bs.modal', function(ev) {
  // ev.relatedTarget 为点击的那个 button
  var value = $(ev.relatedTarget).data('whatever');
  $(this).find('.modal-title').text('新评论来自于：' + value);
  $(this).find('.modal-body input').val(value);
});
```

### 8. 方法与事件

```javascript
// 方法：手动显示/隐藏模态框
$('#myBtn').click(function() {
  $('#myModal').modal('show');
});
// $('#myModal').modal('hide');

// 事件
$('#myModal').on('show.bs.modal', function(e) {
  console.log('显示前');
});
$('#myModal').on('shown.bs.modal', function(e) {
  console.log('完全显示了');
});
$('#myModal').on('hide.bs.modal', function(e) {
  console.log('隐藏前');
});
$('#myModal').on('hidden.bs.modal', function(e) {
  console.log('完全隐藏了');
});
```

## 效果说明

- 普通模态框从页面上方滑入，带有淡入淡出动画（`fade` 类）
- 垂直居中模态框在页面正中央显示
- 模态框内的栅格系统遵循标准的 Bootstrap 栅格规则
- 不同尺寸模态框适配不同内容量：`modal-xl` 最宽，`modal-sm` 最窄
- 同一模态框可以根据触发按钮的不同动态显示不同内容
- 通过 JavaScript 事件可以精确控制模态框显示/隐藏各阶段的行为
