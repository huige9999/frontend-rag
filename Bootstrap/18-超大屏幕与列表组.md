# 第18课：超大屏幕与列表组（Jumbotron & List Group）

## 课程简介

本课讲解 Bootstrap 4 中两个实用组件：超大屏幕（Jumbotron）用于展示大号英雄区域内容，列表组（List Group）用于展示一系列内容项。包含基础用法、样式变体、徽章、自定义内容、选项卡功能以及 JavaScript 方法与事件。

---

# 第一部分：超大屏幕（Jumbotron）

## 关键知识点

1. **基础超大屏幕**：`.jumbotron` 类创建一个灰色背景的大号展示区域
2. **内部组件**：`.display-4`（大号标题）、`.lead`（引导文字）、`<hr class="my-4">`（分割线）
3. **全屏超大屏幕**：`.jumbotron-fluid` + `.container-fluid` 占满父容器宽度

---

## 一、基础超大屏幕

```html
<div class="jumbotron">
  <h1 class="display-4">Hello, world!</h1>
  <p class="lead">This is a simple hero unit, a simple jumbotron-style component.</p>
  <hr class="my-4">
  <p>It uses utility classes for typography and spacing.</p>
  <a href="#" class="btn btn-primary btn-lg">Learn more</a>
</div>
```

**效果说明**：`.jumbotron` 创建一个带圆角、灰色背景的展示区域。`.display-4` 提供超大号标题，`.lead` 提供引导性文字样式。`<hr class="my-4">` 添加水平分割线。

---

## 二、全屏超大屏幕

```html
<div class="jumbotron jumbotron-fluid">
  <div class="container-fluid">
    <h1 class="display-4">Fluid jumbotron</h1>
    <p class="lead">This is a modified jumbotron that occupies the entire horizontal space.</p>
  </div>
</div>
```

**效果说明**：`.jumbotron-fluid` 移除圆角和内边距限制，使超大屏幕占满父容器的全部宽度。内部使用 `.container-fluid` 控制内容宽度。

---

# 第二部分：列表组（List Group）

## 关键知识点

1. **基础列表组**：`.list-group` + `.list-group-item`
2. **激活与禁用**：`.active`（选中状态）、`.disabled`（禁用状态）
3. **可点击项**：`.list-group-item-action` 添加 hover 效果（支持 `<a>` 和 `<button>`）
4. **Flush 样式**：`.list-group-flush` 移除外边框和圆角
5. **背景色**：`.list-group-item-primary` / `success` / `danger` / `warning` / `info` / `light` / `dark`
6. **带背景色的交互项**：`.list-group-item-action` + 背景色类
7. **徽章**：在列表项内使用 `.badge` + `.badge-pill`
8. **自定义内容**：列表项内可放置标题、段落、小字等丰富内容
9. **选项卡**：`data-toggle="list"` 实现列表组与内容面板的联动
10. **JavaScript 方法**：`.tab('show')` 切换激活项
11. **JavaScript 事件**：`show.bs.tab` / `shown.bs.tab` / `hide.bs.tab` / `hidden.bs.tab`

---

## 一、基础列表组

```html
<!-- 无序列表 -->
<ul class="list-group">
  <li class="list-group-item">red</li>
  <li class="list-group-item active">green</li>
  <li class="list-group-item">blue</li>
  <li class="list-group-item disabled">yellow</li>
  <li class="list-group-item">pink</li>
</ul>

<!-- 可点击的链接/按钮列表 -->
<div class="list-group">
  <a href="#" class="list-group-item list-group-item-action">red</a>
  <a href="#" class="list-group-item list-group-item-action active">green</a>
  <button class="list-group-item list-group-item-action" disabled>yellow</button>
  <button class="list-group-item list-group-item-action">pink</button>
</div>

<!-- Flush 样式（移除外边框） -->
<ul class="list-group list-group-flush">
  <li class="list-group-item">red</li>
  <li class="list-group-item">green</li>
</ul>
```

**效果说明**：`.active` 标记当前选中项（蓝色高亮），`.disabled` 禁用项（灰色，不可点击），`.list-group-item-action` 添加 hover 效果，`.list-group-flush` 移除外边框和圆角。

---

## 二、添加背景色

```html
<!-- 纯背景色 -->
<ul class="list-group">
  <li class="list-group-item list-group-item-primary">添加背景色</li>
  <li class="list-group-item list-group-item-secondary">添加背景色</li>
  <li class="list-group-item list-group-item-success">添加背景色</li>
  <li class="list-group-item list-group-item-danger">添加背景色</li>
  <li class="list-group-item list-group-item-warning">添加背景色</li>
  <li class="list-group-item list-group-item-info">添加背景色</li>
  <li class="list-group-item list-group-item-light">添加背景色</li>
  <li class="list-group-item list-group-item-dark">添加背景色</li>
</ul>

<!-- 带交互的背景色 -->
<ul class="list-group">
  <li class="list-group-item list-group-item-action list-group-item-success">有hover状态的</li>
  <li class="list-group-item list-group-item-action list-group-item-danger">有hover状态的</li>
</ul>
```

**效果说明**：8种主题背景色。加上 `.list-group-item-action` 后，鼠标悬停时背景色会加深。

## 三、使用徽章

```html
<ul class="list-group">
  <li class="list-group-item d-flex justify-content-between align-items-center">
    red
    <span class="badge badge-primary badge-pill">14</span>
  </li>
  <li class="list-group-item d-flex justify-content-between align-items-center">
    green
    <span class="badge badge-success badge-pill">5</span>
  </li>
  <li class="list-group-item d-flex justify-content-between align-items-center">
    blue
    <span class="badge badge-danger badge-pill">1</span>
  </li>
</ul>
```

**效果说明**：通过 `.d-flex` + `.justify-content-between` + `.align-items-center` 实现文字与徽章两端对齐且垂直居中。`.badge-pill` 使徽章显示为圆角胶囊样式。

## 五、选项卡（列表组 + 内容面板）

列表组与 `tab-content` 配合，实现点击切换内容面板的效果。

```html
<div class="row">
  <div class="col-4">
    <div class="list-group">
      <a href="#list-home" class="list-group-item list-group-item-action active" data-toggle="list">Home</a>
      <a href="#list-profile" class="list-group-item list-group-item-action" data-toggle="list">Profile</a>
      <a href="#list-messages" class="list-group-item list-group-item-action" data-toggle="list">Messages</a>
      <span data-target="#list-settings" class="list-group-item list-group-item-action" data-toggle="list">Settings</span>
    </div>
  </div>
  <div class="col-8">
    <div class="tab-content">
      <div class="tab-pane fade show active" id="list-home">Home 内容...</div>
      <div class="tab-pane fade" id="list-profile">Profile 内容...</div>
      <div class="tab-pane fade" id="list-messages">Messages 内容...</div>
      <div class="tab-pane fade" id="list-settings">Settings 内容...</div>
    </div>
  </div>
</div>
```

**效果说明**：列表项添加 `data-toggle="list"` 启用选项卡功能。`href="#id"` 指向对应的内容面板。`.tab-pane` 为面板项，`.fade` 添加淡入淡出动画，`.show.active` 标记当前显示的面板。

---

## 六、JavaScript 方法与事件

```javascript
// 方法：选中第一个选项
$('#myList a:first-child').tab('show');

// 点击切换
$('#myList a').on('click', function () {
  $(this).tab('show');
});

// 事件
$('#myList a:nth-child(2)').on('show.bs.tab', function (e) {
  console.log('该内容要显示了');
});
$('#myList a:nth-child(2)').on('shown.bs.tab', function (e) {
  console.log('该内容已经显示了');
});
$('#myList a:nth-child(2)').on('hide.bs.tab', function (e) {
  console.log('该内容要隐藏了');
});
$('#myList a:nth-child(2)').on('hidden.bs.tab', function (e) {
  console.log('该内容已经隐藏了');
});
```

**效果说明**：
- `.tab('show')` 方法手动切换到指定选项
- `show.bs.tab` / `hide.bs.tab` 在切换动作开始时触发
- `shown.bs.tab` / `hidden.bs.tab` 在动画完成后触发
