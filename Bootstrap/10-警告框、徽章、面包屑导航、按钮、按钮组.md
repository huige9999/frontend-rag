# 10 - 警告框、徽章、面包屑导航、按钮、按钮组

## 课程简介

本节课学习 Bootstrap 4 中五个常用的 UI 组件：**警告框（Alerts）**、**徽章（Badges）**、**面包屑导航（Breadcrumb）**、**按钮（Buttons）** 和 **按钮组（Button Group）**。这些组件是构建交互式页面时使用最频繁的基础元素。

---

## 一、警告框（Alerts）

### 关键知识点

- **基础样式**：使用 `.alert` 配合颜色修饰类创建警告框
- **颜色类型**：`.alert-primary`、`.alert-secondary`、`.alert-success`、`.alert-danger`、`.alert-warning`、`.alert-info`、`.alert-light`、`.alert-dark`
- **链接样式**：在警告框内的链接使用 `.alert-link` 类，自动匹配警告框颜色
- **额外内容**：可使用 `.alert-heading` 添加标题，`<hr>` 分隔，添加更多段落
- **可关闭警告框**：添加 `.alert-dismissible`、`.fade`、`.show`，配合关闭按钮实现
- **JS 方法**：`$('#alert').alert('close')` 通过代码关闭警告框
- **JS 事件**：`close.bs.alert`（关闭前触发）、`closed.bs.alert`（关闭后触发）

### 关键代码示例

**基础警告框：**
```html
<div class="alert alert-primary">alert-primary</div>
<div class="alert alert-success">alert-success</div>
<div class="alert alert-danger">alert-danger</div>
<div class="alert alert-warning">alert-warning</div>
```

**带链接的警告框：**
```html
<div class="alert alert-primary">
    这是一个<a href="#" class="alert-link">链接</a>
</div>
```

**带额外内容的警告框：**
```html
<div class="alert alert-success">
    <h4 class="alert-heading">标题</h4>
    <p>详细内容...</p>
    <hr>
    <p>附加说明</p>
</div>
```

**可关闭的警告框：**
```html
<div class="alert alert-warning alert-dismissible fade show">
    这是一个可以关闭的警告框
    <button class="close" data-dismiss="alert">&times;</button>
</div>
```

**JS 控制关闭：**
```javascript
// 方法：关闭警告框
$('.closeBtn').click(function () {
    $('.myAlert').alert('close');
});

// 事件
$('.myAlert').on('close.bs.alert', function () { /* 关闭前 */ });
$('.myAlert').on('closed.bs.alert', function () { /* 关闭后 */ });
```

### 效果说明

警告框会以带背景色的圆角矩形展示，可包含链接、标题、分隔线等丰富内容。可关闭的警告框需要引入 jQuery 和 Bootstrap JS。点击关闭按钮后，警告框会以淡出动画消失。

## 二、徽章（Badges）

### 关键知识点

- **基础徽章**：使用 `.badge` 类
- **颜色徽章**：`.badge-primary`、`.badge-secondary`、`.badge-success`、`.badge-danger`、`.badge-warning`、`.badge-info`、`.badge-light`、`.badge-dark`
- **胶囊徽章**：添加 `.badge-pill` 类实现圆角胶囊效果
- **自适应大小**：徽章会根据父元素的字号自动调整大小
- **在按钮中使用**：常用于显示消息数量
- **链接徽章**：可在 `<a>` 标签上使用，具有悬停和焦点状态

### 关键代码示例

**基础颜色徽章：**
```html
<span class="badge badge-primary">主色徽章</span>
<span class="badge badge-success">成功徽章</span>
<span class="badge badge-danger">危险徽章</span>
```

**随标题大小自适应：**
```html
<h1><span class="badge badge-primary">徽章</span></h1>
<h2><span class="badge badge-secondary">徽章</span></h2>
<h3><span class="badge badge-success">徽章</span></h3>
```

**按钮中的徽章：**
```html
<button class="btn btn-primary">
    消息 <span class="badge badge-light">5</span>
</button>
```

**胶囊徽章：**
```html
<span class="badge badge-pill badge-primary">胶囊徽章</span>
<span class="badge badge-pill badge-success">胶囊徽章</span>
```

**链接徽章：**
```html
<a href="#" class="badge badge-pill badge-primary">可点击徽章</a>
```

## 三、面包屑导航（Breadcrumb）

### 关键知识点

- 使用 `<nav>` 标签包裹，内部使用 `<ol class="breadcrumb">` 列表
- 每个 `<li class="breadcrumb-item">` 代表一个层级
- 当前页面项添加 `.active` 类，不需要 `<a>` 标签

### 关键代码示例

```html
<nav>
    <ol class="breadcrumb">
        <li class="breadcrumb-item"><a href="#">首页</a></li>
        <li class="breadcrumb-item"><a href="#">关于我们</a></li>
        <li class="breadcrumb-item active">联系我们</li>
    </ol>
</nav>
```

### 效果说明

面包屑导航以水平排列展示层级路径，项之间以 `/` 分隔。最后一项（当前页）以深色文字显示，不可点击。

## 四、按钮（Buttons）

### 关键知识点

- **基础按钮**：使用 `.btn` 配合颜色类 `.btn-primary`、`.btn-secondary`、`.btn-success`、`.btn-danger`、`.btn-warning`、`.btn-info`、`.btn-light`、`.btn-dark`、`.btn-link`
- **支持的标签**：`<button>`、`<a>`、`<input type="button/submit/reset">`
- **边框按钮**：使用 `.btn-outline-*` 系列类
- **按钮尺寸**：`.btn-lg`（大按钮）、默认（正常）、`.btn-sm`（小按钮）
- **块级按钮**：`.btn-block` 使按钮占满整行
- **激活/禁用状态**：`.active`（激活）、`disabled` 属性或 `.disabled` 类（禁用）
- **切换状态**：`data-toggle="button"` 实现点击切换激活状态
- **单选按钮组**：`.btn-group-toggle` + `data-toggle="buttons"` 实现选项卡效果
- **JS 方法**：`$('#btn').button('toggle')` 通过代码切换按钮状态

### 关键代码示例

**基础按钮：**
```html
<button class="btn btn-primary">主色按钮</button>
<button class="btn btn-success">成功按钮</button>
<button class="btn btn-danger">危险按钮</button>
<button class="btn btn-link">链接按钮</button>
```

**不同标签的按钮：**
```html
<a href="#" class="btn btn-primary">a标签按钮</a>
<input type="button" value="input按钮" class="btn btn-success">
<input type="submit" value="提交" class="btn btn-danger">
```

**边框按钮：**
```html
<button class="btn btn-outline-primary">边框按钮</button>
<button class="btn btn-outline-success">边框按钮</button>
```

**按钮尺寸：**
```html
<button class="btn btn-primary btn-lg">大按钮</button>
<button class="btn btn-secondary">正常按钮</button>
<button class="btn btn-success btn-sm">小按钮</button>
```

**块级按钮：**
```html
<button class="btn btn-primary btn-lg btn-block">块级按钮</button>
```

**启用与禁用：**
```html
<button class="btn btn-primary active">启用状态</button>
<button class="btn btn-primary" disabled>禁用状态</button>
<a href="#" class="btn btn-secondary disabled">禁用链接按钮</a>
```

**切换按钮：**
```html
<button class="btn btn-primary" data-toggle="button">点击切换状态</button>
```

**选项卡效果：**
```html
<div class="btn-group btn-group-toggle" data-toggle="buttons">
    <label class="btn btn-secondary active">
        <input type="radio" name="options" checked>Active
    </label>
    <label class="btn btn-secondary">
        <input type="radio" name="options">Option 2
    </label>
    <label class="btn btn-secondary">
        <input type="radio" name="options">Option 3
    </label>
</div>
```

## 五、按钮组（Button Group）

### 关键知识点

- **基础按钮组**：使用 `.btn-group` 包裹一组按钮，使按钮紧密排列
- **按钮工具栏**：`.btn-toolbar` 包裹多个 `.btn-group`，形成工具栏布局
- **混合输入框**：工具栏中可以混合 `.input-group` 输入框组件
- **按钮组尺寸**：`.btn-group-lg`（大）、默认（正常）、`.btn-group-sm`（小）
- **嵌套下拉菜单**：在按钮组内嵌套 `.btn-group` + `.dropdown-toggle` 实现下拉按钮
- **垂直排列**：`.btn-group-vertical` 使按钮组垂直排列

### 关键代码示例

**基础按钮组：**
```html
<div class="btn-group">
    <button class="btn btn-primary">Left</button>
    <button class="btn btn-primary">Center</button>
    <button class="btn btn-primary">Right</button>
</div>
```

**按钮工具栏：**
```html
<div class="btn-toolbar">
    <div class="btn-group mr-2">
        <button class="btn btn-success">1</button>
        <button class="btn btn-success">2</button>
        <button class="btn btn-success">3</button>
    </div>
    <div class="btn-group mr-2">
        <button class="btn btn-success">5</button>
        <button class="btn btn-success">6</button>
    </div>
</div>
```

**混合输入框：**
```html
<div class="btn-toolbar">
    <div class="btn-group mr-2">
        <button class="btn btn-secondary">1</button>
        <button class="btn btn-secondary">2</button>
    </div>
    <div class="input-group">
        <div class="input-group-prepend">
            <div class="input-group-text">@</div>
        </div>
        <input type="text" class="form-control" placeholder="输入内容">
    </div>
</div>
```

**按钮组尺寸：**
```html
<div class="btn-group btn-group-lg">...</div>
<div class="btn-group">...</div>
<div class="btn-group btn-group-sm">...</div>
```

**嵌套下拉菜单：**
```html
<div class="btn-group">
    <button class="btn btn-danger">1</button>
    <button class="btn btn-danger">2</button>
    <div class="btn-group">
        <button class="btn btn-secondary dropdown-toggle" data-toggle="dropdown">城市</button>
        <div class="dropdown-menu">
            <a href="#" class="dropdown-item">北京</a>
            <a href="#" class="dropdown-item">上海</a>
        </div>
    </div>
</div>
```

**垂直排列：**
```html
<div class="btn-group-vertical">
    <button class="btn btn-warning">北京</button>
    <button class="btn btn-warning">上海</button>
    <button class="btn btn-warning">广州</button>
</div>
```

### 效果说明

按钮组将多个按钮合并为一个整体，按钮之间无间隙。工具栏可以组合多个按钮组和输入框。嵌套下拉菜单需要引入 Popper.js。垂直排列的按钮组适合侧边栏导航场景。
