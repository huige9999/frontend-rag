# 第14课：下拉菜单（Dropdown）

## 课程简介

本课讲解 Bootstrap 4 中下拉菜单（Dropdown）组件的完整用法，包括基础下拉菜单、分离式按钮、展开方向、菜单项类型、非交互式菜单、偏移属性以及 JavaScript 方法与事件。

## 关键知识点

1. **基础结构**：外层 `.dropdown` 容器 + 按钮 `.dropdown-toggle`（`data-toggle="dropdown"`）+ 菜单 `.dropdown-menu` + 菜单项 `.dropdown-item`
2. **支持标签**：`<button>` 和 `<a>` 均可作为触发按钮
3. **分离式按钮**：使用 `.btn-group` + `.dropdown-toggle-split` 将主按钮和下拉箭头分离
4. **展开方向**：`.dropdown`（向下）、`.dropup`（向上）、`.dropright`（向右）、`.dropleft`（向左）
5. **菜单对齐**：`.dropdown-menu-right` 实现右对齐
6. **分割线**：`.dropdown-divider`
7. **菜单尺寸**：通过按钮的 `.btn-lg` / `.btn-sm` 控制大小
8. **非交互式菜单项**：`.dropdown-item-text`（纯文本）、`.dropdown-header`（标题）
9. **菜单项状态**：`.active`（当前选中）、`.disabled`（禁用）
10. **偏移属性**：`data-offset="10,20"` 设置下拉菜单的偏移量
11. **JavaScript 方法**：`.dropdown('toggle')` / `.dropdown('show')` / `.dropdown('hide')`
12. **JavaScript 事件**：`show.bs.dropdown` / `shown.bs.dropdown` / `hide.bs.dropdown` / `hidden.bs.dropdown`

---

## 一、基础下拉菜单

使用 `<button>` 或 `<a>` 标签创建触发按钮。

```html
<!-- button 标签下拉 -->
<div class="dropdown">
  <button class="btn btn-secondary dropdown-toggle" data-toggle="dropdown">button标签下拉</button>
  <div class="dropdown-menu">
    <a href="#" class="dropdown-item">北京</a>
    <a href="#" class="dropdown-item">上海</a>
    <a href="#" class="dropdown-item">广州</a>
  </div>
</div>

<!-- a 标签下拉（带分割线） -->
<div class="dropdown">
  <a href="javascript:;" class="btn btn-primary dropdown-toggle" data-toggle="dropdown">a标签下拉</a>
  <div class="dropdown-menu">
    <a href="#" class="dropdown-item">北京</a>
    <a href="#" class="dropdown-item">上海</a>
    <a href="#" class="dropdown-item">广州</a>
    <div class="dropdown-divider"></div>
    <a href="#" class="dropdown-item">深圳</a>
  </div>
</div>
```

**效果说明**：点击按钮弹出下拉菜单，`<a>` 标签示例中使用 `.dropdown-divider` 添加了一条分割线。

---

## 二、分离式按钮下拉菜单

主按钮与下拉箭头按钮分离，主按钮可独立绑定点击事件。

```html
<div class="btn-group dropdown">
  <button class="btn btn-success btn-lg">分离式按钮下拉菜单</button>
  <button class="btn btn-success btn-lg dropdown-toggle dropdown-toggle-split"
    data-toggle="dropdown"></button>
  <div class="dropdown-menu">
    <a href="#" class="dropdown-item">北京</a>
    <a href="#" class="dropdown-item">上海</a>
    <a href="#" class="dropdown-item">广州</a>
    <div class="dropdown-divider"></div>
    <a href="#" class="dropdown-item">深圳</a>
  </div>
</div>
```

**效果说明**：左侧主按钮显示文字，右侧小箭头按钮触发下拉菜单。

---

## 三、下拉菜单尺寸

通过按钮的 `.btn-lg` 或 `.btn-sm` 类控制下拉菜单按钮的大小。

```html
<div class="dropdown">
  <button class="btn btn-danger btn-lg dropdown-toggle" data-toggle="dropdown">大按钮</button>
  <div class="dropdown-menu">...</div>
</div>
<div class="dropdown">
  <button class="btn btn-danger btn-sm dropdown-toggle" data-toggle="dropdown">小按钮</button>
  <div class="dropdown-menu">...</div>
</div>
```

---

## 四、展开方向

通过外层容器的类名控制下拉菜单的展开方向。

```html
<div class="dropup">
  <button class="btn btn-info dropdown-toggle" data-toggle="dropdown">向上展开</button>
  <div class="dropdown-menu">...</div>
</div>
<div class="dropright">
  <button class="btn btn-info dropdown-toggle" data-toggle="dropdown">向右展开</button>
  <div class="dropdown-menu">...</div>
</div>
<div class="dropdown">
  <button class="btn btn-info dropdown-toggle" data-toggle="dropdown">向下展开</button>
  <div class="dropdown-menu">...</div>
</div>
<div class="dropleft">
  <button class="btn btn-info dropdown-toggle" data-toggle="dropdown">向左展开</button>
  <div class="dropdown-menu">...</div>
</div>
```

**效果说明**：四个方向分别是 `.dropup`（上）、`.dropright`（右）、`.dropdown`（下，默认）、`.dropleft`（左）。

---

## 五、菜单项与右对齐

菜单项可使用 `<a>` 或 `<button>` 标签，`.dropdown-menu-right` 实现右对齐。

```html
<div class="dropdown">
  <button class="btn btn-info dropdown-toggle" data-toggle="dropdown">菜单项</button>
  <div class="dropdown-menu dropdown-menu-right">
    <a href="#" class="dropdown-item">北京</a>
    <a href="#" class="dropdown-item">上海</a>
    <div class="dropdown-divider"></div>
    <button class="dropdown-item">深圳</button>
  </div>
</div>
```

---

## 六、非交互式下拉菜单

直接显示菜单内容，不通过按钮触发。

```html
<div class="dropdown-menu show">
  <span class="dropdown-item-text">非点击的项</span>
  <h4 class="dropdown-header">下拉菜单的标题</h4>
  <a href="#" class="dropdown-item">北京</a>
  <a href="#" class="dropdown-item active">上海</a>     <!-- 当前选中 -->
  <a href="#" class="dropdown-item disabled">广州</a>   <!-- 禁用状态 -->
  <div class="dropdown-divider"></div>
  <button class="dropdown-item">深圳</button>
</div>
```

**效果说明**：`.dropdown-item-text` 为纯文本项，`.dropdown-header` 为标题，`.active` 为选中状态，`.disabled` 为禁用状态。添加 `show` 类使菜单默认可见。

---

## 七、偏移属性

通过 `data-offset` 设置下拉菜单相对于触发按钮的偏移量。

```html
<div class="dropdown">
  <button class="btn btn-info dropdown-toggle" data-toggle="dropdown" data-offset="10,20">偏移</button>
  <div class="dropdown-menu">...</div>
</div>
```

**效果说明**：`data-offset="10,20"` 表示水平偏移10px、垂直偏移20px。

---

## 八、JavaScript 方法与事件

```javascript
// 方法
$('#myBtn').click(function () {
  // $('#myDropdown').dropdown('show');
  // $('#myDropdown').dropdown('hide');
  // $('#myDropdown').dropdown('toggle');
});

// 事件（注意：事件绑定在 .dropdown 容器上，而非 .dropdown-menu 上）
$('#myDropdown1').on('show.bs.dropdown', function () {
  console.log('调用了show方法');
});
$('#myDropdown1').on('shown.bs.dropdown', function () {
  console.log('下拉列表完全显示了');
});
$('#myDropdown1').on('hide.bs.dropdown', function () {
  console.log('调用了hide方法');
});
$('#myDropdown1').on('hidden.bs.dropdown', function () {
  console.log('下拉列表完全隐藏了');
});
```

**效果说明**：下拉菜单事件需绑定在 `.dropdown` 容器元素上。`show`/`hide` 在动作开始时触发，`shown`/`hidden` 在动画完成后触发。
