# 栅格系统（二）

## 课程简介

本节课深入讲解 Bootstrap 栅格系统的高级用法，包括混合排列、列对齐、列排序、列偏移、间距控制和嵌套布局。这些功能使得栅格系统可以应对各种复杂的布局需求。

## 关键知识点

### 核心内容概览

| 功能 | 类名格式 | 说明 |
|------|---------|------|
| 混合排列 | `col-{breakpoint}-*` 组合 | 同一元素在不同断点下列数不同 |
| 垂直对齐（行） | `align-items-*` | start / center / end |
| 垂直对齐（列） | `align-self-*` | start / center / end |
| 水平对齐 | `justify-content-*` | start / center / end / around / between |
| 列排序 | `order-{breakpoint}-*` | first / last / 数字 |
| 列偏移 | `offset-{breakpoint}-*` | 向右偏移指定列数 |
| 间距 | `ml-{breakpoint}-auto` / `mr-{breakpoint}-auto` | 自动间距 |
| 嵌套 | 列内再放 `.row` | 子级以父列宽度为基准再分 12 列 |

---

## 小节一：混合排列与组合模式

同一个元素可以设置不同断点下的列数，实现跨屏幕的响应式布局：

```html
<!--
  超大屏：一行 6 个，每个占 2 列 (col-xl-2)
  大屏：  一行 4 个，每个占 3 列 (col-lg-3)
  中屏：  一行 3 个，每个占 4 列 (col-md-4)
  小屏：  一行 2 个，每个占 6 列 (col-sm-6)
  超小屏：一行 1 个，每个占 12 列 (col-12)
-->
<div class="row mt-5">
  <div class="col-xl-2 col-lg-3 col-md-4 col-sm-6 col-12">内容</div>
  <div class="col-xl-2 col-lg-3 col-md-4 col-sm-6 col-12">内容</div>
  <div class="col-xl-2 col-lg-3 col-md-4 col-sm-6 col-12">内容</div>
  <div class="col-xl-2 col-lg-3 col-md-4 col-sm-6 col-12">内容</div>
  <div class="col-xl-2 col-lg-3 col-md-4 col-sm-6 col-12">内容</div>
  <div class="col-xl-2 col-lg-3 col-md-4 col-sm-6 col-12">内容</div>
</div>
```

---

## 小节二：设置一个列宽，其余自动平分

指定一列的宽度后，其余使用 `.col` 的列会自动平分剩余空间：

```html
<div class="row mt-5">
  <div class="col-sm-7">在小屏幕下占 7 列</div>
  <div class="col">自动平分剩余的宽度</div>
  <div class="col">自动平分剩余的宽度</div>
  <div class="col">自动平分剩余的宽度</div>
</div>
```

---

## 小节三：根据内容调整列宽

使用 `.col-{breakpoint}-auto` 让列宽由内容撑开：

```html
<div class="row mt-5">
  <div class="col-md-auto">在中等屏幕下由内容撑开宽度</div>
  <div class="col">自动平分剩余的宽度</div>
  <div class="col-lg-2">在大屏下占 2 列</div>
</div>
```

---

## 小节四：所有尺寸下占同样的列数

使用 `.col-*`（不带断点前缀）使所有尺寸下列数一致：

```html
<div class="row mt-5">
  <div class="col-8">所有尺寸下都占 8 列</div>
  <div class="col-4">所有尺寸下都占 4 列</div>
</div>
```

---

## 小节五：垂直对齐

### 行级对齐

通过在 `.row` 上添加类来控制所有列的垂直对齐方式：

```html
<!-- 顶部对齐 -->
<div class="row align-items-start">
  <div class="col">顶部对齐</div>
  <div class="col">顶部对齐</div>
</div>

<!-- 居中对齐 -->
<div class="row align-items-center">
  <div class="col">居中对齐</div>
  <div class="col">居中对齐</div>
</div>

<!-- 底部对齐 -->
<div class="row align-items-end">
  <div class="col">底部对齐</div>
  <div class="col">底部对齐</div>
</div>
```

### 列级单独对齐

通过在列上添加类来单独控制某一列的垂直对齐：

```html
<div class="row">
  <div class="col align-self-start">顶对齐</div>
  <div class="col align-self-center">中间对齐</div>
  <div class="col align-self-end">底部对齐</div>
</div>
```

---

## 小节六：水平对齐

通过 `justify-content-*` 控制列的水平排列方式：

```html
<!-- 左对齐（默认） -->
<div class="row justify-content-start">
  <div class="col-4">左对齐</div>
</div>

<!-- 居中对齐 -->
<div class="row justify-content-center">
  <div class="col-4">居中对齐</div>
</div>

<!-- 右对齐 -->
<div class="row justify-content-end">
  <div class="col-4">右对齐</div>
</div>

<!-- 分散居中对齐（每个元素两侧间距相等） -->
<div class="row justify-content-around">
  <div class="col-4">分散居中</div>
</div>

<!-- 左右两端对齐（元素间距自动平分） -->
<div class="row justify-content-between">
  <div class="col-4">两端对齐</div>
</div>
```

---

## 小节七：列排序

使用 `.order-*` 或 `.order-{breakpoint}-*` 改变列的显示顺序：

```html
<!-- 基本排序 -->
<div class="row">
  <div class="col">第1列</div>
  <div class="col order-5">第2列（排到第5位）</div>
  <div class="col order-6">第3列（排到第6位）</div>
</div>

<!-- 指定断点的排序（仅在 >=1200px 时排序生效） -->
<div class="row">
  <div class="col">第1列</div>
  <div class="col order-xl-5">第2列</div>
  <div class="col order-xl-2">第3列</div>
</div>

<!-- order-first 排第一位，order-last 排最后一位 -->
<div class="row">
  <div class="col">第1列</div>
  <div class="col order-first">第2列（排到最前）</div>
  <div class="col order-last">第3列（排到最后）</div>
  <div class="col">第4列</div>
</div>
```

> 注意：3.x 版本使用的是 `.col-{breakpoint}-push-*` 和 `.col-{breakpoint}-pull-*`，4.x 改为 `.order-*`。

---

## 小节八：列偏移

使用 `.offset-{breakpoint}-*` 让列向右偏移指定列数：

```html
<!-- 基本偏移 -->
<div class="row">
  <div class="col-md-4">第1列</div>
  <div class="col-md-4 offset-md-4">往右偏移 4 列</div>
</div>

<!-- 居中布局 -->
<div class="row">
  <div class="col-3 offset-md-3">第1列</div>
  <div class="col-3 offset-md-3">第2列</div>
</div>

<!-- 不同断点不同偏移 -->
<div class="row">
  <div class="col-sm-5 col-md-6">小屏占 5 列，中屏占 6 列</div>
  <div class="col-sm-5 offset-sm-3 col-md-6 offset-md-5">小屏偏移 3 列，中屏偏移 5 列</div>
</div>
```

---

## 小节九：间距控制

使用 margin 工具类让列之间产生间距：

```html
<!-- ml-auto 使右侧列远离到最右边 -->
<div class="row">
  <div class="col-md-4">第1列</div>
  <div class="col-md-4 ml-auto">第2列，跑到最右边</div>
</div>

<!-- ml-md-auto 在中屏下自动计算左间距 -->
<div class="row">
  <div class="col-md-3 ml-md-auto">中屏下自动左间距</div>
  <div class="col-md-3 ml-md-auto">中屏下自动左间距</div>
</div>

<!-- col-auto 宽度由内容撑开 -->
<div class="row">
  <div class="col-auto mr-auto">宽度由内容撑开</div>
  <div class="col-auto">宽度由内容撑开</div>
</div>
```

| 类名 | 说明 |
|------|------|
| `mr-{breakpoint}-auto` | 使右侧的列远离到最右边 |
| `ml-{breakpoint}-auto` | 使左侧的列远离到最左边 |

---

## 小节十：嵌套布局

每一个列里面可以再放 `.row`，嵌套的元素以父列的宽度为基准再分 12 列：

```html
<div class="row mt-5">
  <div class="col-sm-9">
    父级的第 1 列
    <div class="row">
      <div class="col-sm-8 col-6">子级第 1 列，小屏占 8 列，超小屏占 6 列</div>
      <div class="col-sm-4 col-6">子级第 2 列，小屏占 4 列，超小屏占 6 列</div>
    </div>
  </div>
  <div class="col-sm-3">父级的第 2 列</div>
</div>
```

## 效果说明

- 混合排列可以实现在不同屏幕下展示不同的列数
- 垂直/水平对齐提供了灵活的布局控制
- 列排序不需要改变 DOM 结构就能调整视觉顺序
- 列偏移常用于实现居中或特定间距的布局
- 间距工具类 `ml-auto` / `mr-auto` 可以快速实现弹性间距
- 嵌套布局让复杂的多层级页面结构变得简单
