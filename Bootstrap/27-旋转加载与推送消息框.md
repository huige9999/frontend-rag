# 旋转加载与推送消息框

## 课程简介

本节课学习 Bootstrap 4 中的 **旋转加载（Spinners）** 和 **推送消息框（Toasts）** 组件。旋转加载用于表示页面或操作的加载状态；推送消息框用于向用户展示轻量级的通知消息。

---

## 一、旋转加载（Spinners）

### 关键知识点

- `.spinner-border` -- 边框旋转加载（圆形旋转边框）
- `.spinner-grow` -- 渐变旋转加载（闪烁圆点）
- 颜色通过 `text-*` 类控制：`text-primary`、`text-success`、`text-danger`、`text-warning`、`text-info`、`text-light`、`text-dark`
- `.spinner-border-sm` / `.spinner-grow-sm` -- 小尺寸
- 大尺寸通过行间样式 `width` 和 `height` 设置
- 可嵌入按钮中，配合 `disabled` 属性表示加载状态

### 关键代码示例

**边框旋转加载（多种颜色）：**

```html
<div class="spinner-border"></div>
<div class="spinner-border text-primary"></div>
<div class="spinner-border text-success"></div>
<div class="spinner-border text-danger"></div>
<div class="spinner-border text-warning"></div>
<div class="spinner-border text-info"></div>
<div class="spinner-border text-dark"></div>
```

**渐变旋转加载（多种颜色）：**

```html
<div class="spinner-grow"></div>
<div class="spinner-grow text-primary"></div>
<div class="spinner-grow text-success"></div>
<div class="spinner-grow text-danger"></div>
<div class="spinner-grow text-warning"></div>
<div class="spinner-grow text-info"></div>
```

**尺寸控制：**

```html
<!-- 小尺寸 -->
<div class="spinner-border spinner-border-sm"></div>
<div class="spinner-grow spinner-grow-sm"></div>

<!-- 大尺寸（行间样式） -->
<div class="spinner-border text-info" style="width: 3rem; height: 3rem;"></div>
<div class="spinner-grow text-info" style="width: 3rem; height: 3rem;"></div>
```

**嵌入按钮中：**

```html
<!-- 只有加载图标 -->
<button class="btn btn-primary" disabled>
  <span class="spinner-border spinner-border-sm"></span>
</button>

<!-- 加载图标 + 文字 -->
<button class="btn btn-primary ml-2" disabled>
  <span class="spinner-border spinner-border-sm"></span>
  loading...
</button>

<!-- 渐变加载图标 -->
<button class="btn btn-primary" disabled>
  <span class="spinner-grow spinner-grow-sm"></span>
  loading...
</button>
```

### 效果说明

边框旋转加载显示一个不断旋转的圆环，渐变加载显示一个闪烁的圆点。可灵活调整颜色和大小，常用于按钮的加载状态提示。

## 二、推送消息框（Toasts）

### 关键知识点

- `.toast` -- 推送消息框容器
- `.toast-header` -- 消息头部（包含标题、时间、关闭按钮）
- `.toast-body` -- 消息正文
- `data-autohide="false"` -- 禁止自动隐藏
- `data-dismiss="toast"` -- 关闭按钮触发属性
- 消息框可以堆叠显示
- 需要通过 JS 初始化并显示：`$('.toast').toast('show')`

### 关键代码示例

**基本推送消息框：**

```html
<div class="toast" data-autohide="false">
  <div class="toast-header">
    <strong class="mr-auto">Bootstrap</strong>
    <small>11 mins ago</small>
    <button class="close ml-2 mb-1" data-dismiss="toast">
      <span>&times;</span>
    </button>
  </div>
  <div class="toast-body">
    Hello, world! This is a toast message.
  </div>
</div>
```

**居中显示（利用flex布局定位）：**

```html
<div class="d-flex bg-dark p-5 justify-content-center">
  <div class="toast" data-autohide="false">
    <div class="toast-header">
      <strong class="mr-auto">Bootstrap</strong>
      <small>11 mins ago</small>
      <button class="close ml-2 mb-1" data-dismiss="toast">
        <span>&times;</span>
      </button>
    </div>
    <div class="toast-body">
      Hello, world! This is a toast message.
    </div>
  </div>
</div>
```

**data-属性配置：**

```html
<div class="toast" data-autohide="true" data-animation="true" data-delay="5000">
  ...
</div>
```

**JS 初始化、方法与事件：**

```javascript
// 初始化并显示
$('.toast').toast('show');

// 方法
$('#myToast').toast('show');    // 显示
$('#myToast').toast('hide');    // 隐藏

// 事件
$('#myToast').on('show.bs.toast', function() {
  console.log('消息框要显示了');
});
$('#myToast').on('shown.bs.toast', function() {
  console.log('消息框已经显示了');
});
$('#myToast').on('hide.bs.toast', function() {
  console.log('消息框要隐藏了');
});
$('#myToast').on('hidden.bs.toast', function() {
  console.log('消息框已经隐藏了');
});
```

### 效果说明

推送消息框是一个轻量级的通知组件，类似于移动端的推送通知。包含标题栏和内容区，可手动关闭或自动隐藏，支持动画和延迟配置。
