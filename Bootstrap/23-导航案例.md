# 导航案例

## 课程简介

本节课通过一个综合案例，模拟 Bootstrap 官方网站的导航栏布局，将导航栏、下拉菜单、搜索框、侧边栏折叠等功能整合在一起，展示导航组件在实际项目中的应用方式。

---

## 关键知识点

- 导航栏综合应用：`navbar` + `navbar-expand` + 自定义样式
- 导航栏弹性布局方向控制：`flex-column flex-md-row` 实现响应式排列
- 版本下拉菜单：`dropdown-menu-right` 右对齐下拉菜单
- 图标按钮：使用 SVG 图标作为导航链接
- 响应式显示控制：`d-none d-md-flex` 控制元素在不同断点的显示
- 侧边栏折叠：`collapse` 组件用于侧边导航的展开/收起
- 搜索框与折叠按钮结合：`d-md-none` 在中等屏幕以上隐藏折叠按钮

---

## 关键代码示例

### 1. 顶部导航栏（模拟 Bootstrap 官网）

自定义导航栏样式，包含 Logo、导航链接、版本下拉菜单和图标链接。

```html
<header class="navbar navbar-expand navbar-dark navbarBg flex-column flex-md-row">
  <a href="#" class="navbar-brand mr-md-2">
    <img src="bootstrap-solid.svg" alt="" width="36" class="border rounded">
  </a>
  <ul class="navbar-nav">
    <li class="nav-item"><a href="#" class="nav-link">Home</a></li>
    <li class="nav-item"><a href="#" class="nav-link">Documentation</a></li>
    <li class="nav-item"><a href="#" class="nav-link">Examples</a></li>
    <li class="nav-item"><a href="#" class="nav-link">Themes</a></li>
    <li class="nav-item"><a href="#" class="nav-link">Expo</a></li>
    <li class="nav-item"><a href="#" class="nav-link">Blog</a></li>
  </ul>

  <!-- 右侧链接，中等屏幕以上显示 -->
  <ul class="navbar-nav ml-md-auto d-none d-md-flex">
    <!-- 版本下拉菜单 -->
    <li class="nav-item dropdown">
      <a href="#" class="nav-link dropdown-toggle" data-toggle="dropdown">v4.3</a>
      <div class="dropdown-menu dropdown-menu-right">
        <a href="#" class="dropdown-item active">Latest (4.3.x)</a>
        <a href="#" class="dropdown-item">v4.2.1</a>
        <a href="#" class="dropdown-item">v4.0.0</a>
        <div class="dropdown-divider"></div>
        <a href="#" class="dropdown-item">v3.4.1</a>
      </div>
    </li>
    <!-- SVG 图标链接 -->
    <li class="nav-item">
      <a href="#" class="nav-link">
        <svg class="svgSize" viewBox="0 0 512 499.36">...</svg>
      </a>
    </li>
  </ul>

  <!-- 下载按钮，大屏以上显示 -->
  <a href="#" class="btn btn-outline-warning ml-lg-3 d-none d-lg-block">Download</a>
</header>
```

### 2. 自定义导航栏样式

通过自定义 CSS 实现与 Bootstrap 官网一致的外观。

```css
.navbarBg {
  min-height: 4rem;
  background: #563d7c;
  box-shadow: 0 0.5rem 1rem rgba(0,0,0,.05), inset 0 -1px 0 rgba(0,0,0,.1);
}
.navbarBg .navbar-nav .nav-link {
  color: #cbbde2;
}
```

### 3. SVG 图标尺寸控制

```css
.svgSize {
  width: 1rem;
  height: 1rem;
  vertical-align: text-top;
}
```

### 4. 侧边栏搜索与折叠菜单

搜索框与折叠按钮组合，中等屏幕以上隐藏折叠按钮，小屏幕点击折叠按钮展开侧边导航。

```html
<div class="container-fluid">
  <div class="row">
    <div class="col-md-3 col-xl-2 border-right border-bottom">
      <!-- 搜索框 + 折叠按钮 -->
      <form action="#" class="d-flex py-3 px-1">
        <input type="search" class="form-control" placeholder="Search...">
        <!-- 折叠按钮仅在小屏幕显示 -->
        <button type="button" class="btn p-0 ml-3 d-md-none"
          data-toggle="collapse" data-target="#docsNav">
          <svg width="30" height="30" viewBox="0 0 30 30">
            <path stroke="currentColor" stroke-linecap="round"
              stroke-width="2" d="M4 7h22M4 15h22M4 23h22"></path>
          </svg>
        </button>
      </form>

      <!-- 可折叠的侧边导航 -->
      <div class="collapse" id="docsNav">
        <div>
          <h6 class="px-4 pt-2"><a href="#" class="text-dark">Components</a></h6>
          <ul class="nav">
            <li><a href="#">Alerts</a></li>
            <li><a href="#">Badge</a></li>
            <li><a href="#">Buttons</a></li>
            <li><a href="#">Card</a></li>
          </ul>
        </div>
      </div>
    </div>
    <div class="col-md-9 col-xl-10">
      <!-- 主内容区域 -->
    </div>
  </div>
</div>
```

### 5. 侧边栏导航样式

```css
#docsNav .nav {
  display: block;
}
#docsNav .nav li {
  padding: 0.25rem 1.5rem;
}
#docsNav .nav li a {
  color: rgba(0, 0, 0, 0.65);
}
```

---

## 效果说明

- 顶部导航栏使用自定义紫色背景（`#563d7c`），包含 Logo、多个导航链接、版本下拉菜单和 SVG 图标按钮
- `flex-column flex-md-row` 让导航栏在小屏竖排、中屏以上横排
- `d-none d-md-flex` 控制右侧元素仅在中屏以上显示，`d-none d-lg-block` 控制 Download 按钮仅在大屏显示
- 侧边栏使用 `col-md-3 col-xl-2` 实现响应式宽度，搜索框始终可见，折叠按钮仅在小屏显示
- 侧边导航使用 `collapse` 组件，点击菜单图标按钮后展开/收起
