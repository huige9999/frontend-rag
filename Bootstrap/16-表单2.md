# 第16课：表单2（Form Validation & Custom Forms）

## 课程简介

本课讲解 Bootstrap 4 表单进阶用法，包括表单验证（浏览器原生验证 + Bootstrap 样式反馈）和自定义表单控件（自定义复选框、单选框、下拉框、滑块、文件选择）。

## 关键知识点

1. **表单验证**：`needs-validation` 类 + `novalidate` 属性 + JavaScript 手动触发验证
2. **验证反馈**：`.valid-feedback`（验证成功提示）、`.invalid-feedback`（验证失败提示）
3. **验证状态类**：提交后需给 form 添加 `was-validated` 类以显示验证样式
4. **必填验证**：`required` 属性
5. **自定义复选框**：`.custom-control` + `.custom-checkbox` + `.custom-control-input` + `.custom-control-label`
6. **自定义单选框**：`.custom-control` + `.custom-radio`，`.custom-control-inline` 行内排列
7. **自定义下拉框**：`.custom-select`
8. **自定义滑块**：`.custom-range`，支持 `min`/`max`/`step` 属性
9. **自定义文件选择**：`.custom-file` + `.custom-file-input` + `.custom-file-label`

## 一、表单验证

### HTML 结构

表单需要添加 `needs-validation` 类和 `novalidate` 属性（禁用浏览器默认验证提示），然后通过 JavaScript 控制显示 Bootstrap 的验证样式。

```html
<form action="#" class="needs-validation" novalidate>
  <div class="form-row">
    <div class="col-md-4 form-group">
      <label for="surname">姓</label>
      <input type="text" class="form-control" id="surname" placeholder="请输入姓" required>
      <div class="valid-feedback">验证通过！</div>
      <div class="invalid-feedback">验证失败！</div>
    </div>
    <div class="col-md-4 form-group">
      <label for="name">名</label>
      <input type="text" class="form-control" id="name" placeholder="请输入名" required>
      <div class="valid-feedback">验证通过！</div>
    </div>
    <div class="col-md-4 form-group">
      <label for="usename">用户名</label>
      <input type="text" class="form-control" id="usename" placeholder="请输入用户名" required>
      <div class="invalid-feedback">请输入正确的用户名！</div>
    </div>
  </div>
  <div class="form-group">
    <div class="form-check">
      <input type="checkbox" class="form-check-input" id="grid" required>
      <label for="grid" class="form-check-label">同意所读内容</label>
      <div class="invalid-feedback">你必需同意此内容后才能继续操作！</div>
    </div>
  </div>
  <button type="submit" class="btn btn-primary">提交</button>
</form>
```

### 验证 JavaScript

```javascript
(function () {
  var forms = document.querySelectorAll('.needs-validation');
  forms.forEach(function (form) {
    form.addEventListener('submit', function (ev) {
      if (form.checkValidity() === false) {
        ev.preventDefault();
        ev.stopPropagation();
      }
      form.classList.add('was-validated');
    });
  });
})();
```

**效果说明**：
- `novalidate` 禁用浏览器默认验证，由 Bootstrap 接管显示验证样式
- 提交时通过 `checkValidity()` 判断是否验证通过
- 无论通过与否，都给 form 添加 `was-validated` 类，触发 Bootstrap 的 `:valid` / `:invalid` 伪类样式
- `.valid-feedback` 在验证通过时显示，`.invalid-feedback` 在验证失败时显示

## 二、自定义表单控件

### 自定义复选框

```html
<div class="form-group custom-control custom-checkbox">
  <input type="checkbox" id="eat" class="custom-control-input">
  <label for="eat" class="custom-control-label">吃饭</label>
</div>
```

### 自定义单选框

```html
<!-- 默认垂直排列 -->
<div class="form-group custom-control custom-radio">
  <input type="radio" id="male" class="custom-control-input" name="sex">
  <label for="male" class="custom-control-label">男</label>
</div>
<div class="form-group custom-control custom-radio">
  <input type="radio" id="female" class="custom-control-input" name="sex">
  <label for="female" class="custom-control-label">女</label>
</div>

<!-- 行内排列 -->
<div class="form-group custom-control custom-radio custom-control-inline">
  <input type="radio" id="male1" class="custom-control-input" name="sex1">
  <label for="male1" class="custom-control-label">男</label>
</div>
<div class="form-group custom-control custom-radio custom-control-inline">
  <input type="radio" id="female1" class="custom-control-input" name="sex1" disabled>
  <label for="female1" class="custom-control-label">女</label>
</div>
```

**效果说明**：自定义控件比默认控件更美观。`.custom-control-inline` 使多个单选框在同一行显示。`disabled` 属性可禁用单个选项。

### 自定义下拉框

```html
<select class="form-group custom-select">
  <option value="">城市</option>
  <option value="">北京</option>
  <option value="">上海</option>
  <option value="">广州</option>
</select>
```

### 自定义滑块

```html
<!-- 基础滑块 -->
<div class="form-group">
  <label for="range1">滑块</label>
  <input type="range" class="custom-range" id="range1">
</div>

<!-- 有范围的滑块 -->
<div class="form-group">
  <label for="range2">有范围的滑块</label>
  <input type="range" class="custom-range" min="0" max="5" id="range2">
</div>

<!-- 规定步长的滑块 -->
<div class="form-group">
  <label for="range3">规定步长的滑块</label>
  <input type="range" class="custom-range" min="0" max="5" step="0.5" id="range3">
</div>
```

### 自定义文件选择

```html
<div class="form-group custom-file">
  <input type="file" class="custom-file-input" id="file">
  <label for="file" class="custom-file-label">Choose file</label>
</div>
```

**效果说明**：`.custom-file` 将原生文件选择器包装为统一美观的样式。`.custom-file-label` 显示文件选择提示文字。
