# 09 - 剩余工具集合

## 课程简介

本节课学习 Bootstrap 4 中剩余的工具类集合，涵盖**浮动**、**关闭图标**、**图像替换**、**内容溢出**、**定位**、**阴影**、**尺寸**、**间距**、**文本处理**、**垂直对齐**和**可见性**等内容。这些工具类在日常开发中使用频率极高，是构建页面的重要辅助手段。

---

## 一、浮动（Float）

### 关键知识点

- `.float-left`：左浮动；`.float-right`：右浮动；`.float-none`：不浮动
- 响应式写法：`.float-{breakpoint}-left/right/none`
- `.clearfix`：清除浮动（解决父元素高度塌陷问题）

### 关键代码示例

```html
<!-- 浮动 -->
<div class="float-left">左浮动</div>
<div class="float-right">右浮动</div>
<div class="float-none">不浮动</div>

<!-- 响应式浮动 -->
<div class="float-md-left">中等屏幕以上左浮动</div>
<div class="float-sm-right">小屏幕以上右浮动</div>

<!-- 清除浮动 -->
<div class="clearfix">
    <div class="float-left">左浮动</div>
    <div class="float-right">右浮动</div>
</div>
```

## 三、图像替换（Image Replacement）

### 关键代码示例

```html
<h1 class="text-hide" style="background-image: url('logo.svg'); width: 50px; height: 50px;">
    Bootstrap
</h1>
```

### 效果说明

`.text-hide` 类将文字隐藏但保留背景图片，用于在保留 SEO 语义的同时展示图片 Logo。

## 四、内容溢出（Overflow）

### 关键知识点

- `.overflow-auto`：内容溢出时显示滚动条
- `.overflow-hidden`：隐藏溢出内容

### 关键代码示例

```html
<div style="width: 200px; height: 100px;" class="overflow-auto">
    内容溢出时自动显示滚动条...
</div>
<div style="width: 200px; height: 100px;" class="overflow-hidden">
    溢出的内容会被隐藏...
</div>
```

## 五、定位（Position）

### 关键知识点

- `.position-static`：默认值，无定位
- `.position-relative`：相对定位
- `.position-absolute`：绝对定位
- `.position-fixed`：固定定位
- `.position-sticky`：粘性定位
- `.fixed-top`：固定在顶部；`.fixed-bottom`：固定在底部
- `.sticky-top`：粘性置顶（需放在 body 下的第一层级才有效果）

### 关键代码示例

```html
<div class="position-static">默认定位</div>
<div class="position-relative">相对定位</div>
<div class="position-absolute">绝对定位</div>
<div class="position-fixed">固定定位</div>
<div class="position-sticky">粘性定位</div>

<div class="fixed-top">固定在页面顶部</div>
<div class="sticky-top">滚动到一定位置后固定在顶部</div>
```

### 关键知识点

## 七、尺寸（Sizing）

### 关键知识点

- **宽度**：`.w-25`、`.w-50`、`.w-75`、`.w-100`、`.w-auto`
- **高度**：`.h-25`、`.h-50`、`.h-75`、`.h-100`、`.h-auto`
- **最大宽度**：`.mw-100`
- **最大高度**：`.mh-100`

### 关键代码示例

```html
<!-- 宽度 -->
<div class="w-25">Width 25%</div>
<div class="w-50">Width 50%</div>
<div class="w-75">Width 75%</div>
<div class="w-100">Width 100%</div>

<!-- 高度 -->
<div class="h-25">Height 25%</div>
<div class="h-50">Height 50%</div>
<div class="h-100">Height 100%</div>

<!-- 最大尺寸 -->
<div class="mw-100">最大宽度100%</div>
<div class="mh-100">最大高度100%</div>
```

## 八、间距（Spacing）

### 关键知识点

- 格式：`{property}{sides}-{size}` 或 `{property}{sides}-{breakpoint}-{size}`
- **property**：`m`（margin）、`p`（padding）
- **sides**：`t`（上）、`b`（下）、`l`（左）、`r`（右）、`x`（左右）、`y`（上下）、空（四边）
- **size**：`0`~`5`、`auto`
- 响应式居中：`.mx-auto` 实现水平居中

### 关键代码示例

```html
<div class="p-3 m-3">四边内边距和外边距</div>
<div class="py-3 my-3">上下内边距和外边距</div>
<div class="px-3 mx-3">左右内边距和外边距</div>
<div class="px-3 mx-auto">水平居中</div>
<div class="p-3 my-lg-3">响应式间距（大屏生效）</div>
```

## 九、文本处理（Text）

### 关键知识点

- **对齐**：`.text-justify`（两端对齐）、`.text-center`（居中）
- **响应式对齐**：`.text-{breakpoint}-center/left/right`
- **换行处理**：`.text-nowrap`（不换行）、`.text-truncate`（截断并显示省略号）
- **大小写转换**：`.text-lowercase`（小写）、`.text-uppercase`（大写）、`.text-capitalize`（首字母大写）
- **字体粗细**：`.font-weight-bold`（加粗）、`.font-weight-normal`（正常）、`.font-weight-light`（细体）
- **斜体**：`.font-italic`
- **等宽字体**：`.text-monospace`

### 关键代码示例

```html
<p class="text-justify">两端对齐的文字...</p>
<p class="text-lg-center">大屏幕下居中对齐</p>

<div class="text-nowrap text-truncate" style="width: 8rem;">
    超出部分显示省略号...
</div>

<p class="text-lowercase">ALL LOWERCASE</p>
<p class="text-uppercase">all uppercase</p>
<p class="text-capitalize">capitalize first letter</p>

<p class="font-weight-bold">加粗字体</p>
<p class="font-weight-light">更细的字体</p>
<p class="font-italic">倾斜字体</p>
<p class="text-monospace">等宽字体</p>
```

## 十一、可见性（Visibility）

### 关键知识点

- `.visible`：元素可见
- `.invisible`：元素不可见，但**仍然占据空间**
- `.d-none`：元素不可见且**不占据空间**

### 关键代码示例

```html
<div class="visible">可以看见</div>
<div class="invisible">看不见，但占据空间</div>
<div class="d-none">看不见，不占据空间</div>
```
