# 图片、表格、图文

## 课程简介

本节课介绍 Bootstrap 中图片、表格和图文组件的使用方法。包括响应式图片、图片样式、响应式表格、以及图文排版（Figure）组件。

---

## 小节一：图片（01-图片.html）

### 课程简介

Bootstrap 提供了多种图片相关的工具类，包括响应式图片、缩略图、圆角、对齐方式以及响应式图片方案（picture）。

### 关键知识点

#### 1. 响应式图片

使用 `.img-fluid` 让图片自适应容器宽度：

```html
<div class="container">
  <div class="row">
    <img src="images/img_01.jpg" class="img-fluid" alt="">
  </div>
</div>
```

> `.img-fluid` 等价于 `max-width: 100%; height: auto;`

#### 2. 缩略图

使用 `.img-thumbnail` 给图片添加圆角边框：

```html
<div class="col-md-4">
  <img src="images/img_01.jpg" class="img-thumbnail" alt="">
</div>
```

#### 3. 圆角图片

使用 `.rounded` 给图片添加圆角：

```html
<div class="col-md-4">
  <img src="images/img_01.jpg" class="rounded img-fluid" alt="">
</div>
```

#### 4. 图片对齐 -- 通过浮动

```html
<!-- 左浮动 -->
<img src="images/img_01.jpg" class="float-left" alt="">
<!-- 右浮动 -->
<img src="images/img_01.jpg" class="float-right" alt="">
```

#### 5. 图片对齐 -- 通过父级文本对齐

```html
<div class="col text-right">
  <img src="images/img_01.jpg" alt="">
  <img src="images/img_01.jpg" alt="">
</div>
```

#### 6. 图片居中

图片居中需要先变为块级元素（`d-block`），再使用 `mx-auto`（margin: 0 auto）：

```html
<div class="col">
  <img src="images/img_01.jpg" class="d-block mx-auto" alt="">
</div>
```

#### 7. 响应式图片方案（picture 元素）

使用 `<picture>` 和 `<source>` 根据屏幕宽度加载不同尺寸的图片：

```html
<picture>
  <source media="(min-width: 1200px)" srcset="images/1140.jpg">
  <source media="(min-width: 992px)" srcset="images/960.jpg">
  <source media="(min-width: 768px)" srcset="images/720.jpg">
  <source media="(min-width: 576px)" srcset="images/540.jpg">
  <img src="images/img_01.jpg" alt="">
</picture>
```

- 当屏幕宽度 >= 1200px 时加载 1140.jpg
- 当屏幕宽度 >= 992px 时加载 960.jpg
- 依此类推，< 576px 时加载默认的 img_01.jpg

### 效果说明

- `.img-fluid` 是响应式图片的基础类，确保图片不会超出容器
- `.img-thumbnail` 添加圆角边框效果，适合相册展示
- 图片居中需要组合使用 `.d-block` 和 `.mx-auto`
- `<picture>` 方案可以为不同屏幕加载不同分辨率的图片，优化加载性能

## 小节二：表格（02-表格.html）

### 课程简介

Bootstrap 对表格提供了丰富的样式类，包括基础样式、紧凑样式和响应式表格。

### 关键知识点

#### 响应式表格

使用 `.table-responsive` 包裹表格，当屏幕宽度不足时出现水平滚动条：

```html
<div class="container">
  <div class="row">
    <div class="table-responsive">
      <table class="table table-sm">
        <caption>课程表</caption>
        <thead>
          <tr>
            <th>heading</th>
            <th>heading</th>
            <!-- ... 更多列 ... -->
          </tr>
        </thead>
        <tbody>
          <tr>
            <td>语文</td>
            <td>数学</td>
            <td>英语</td>
            <td>生物</td>
            <td>化学</td>
          </tr>
          <!-- ... 更多行 ... -->
        </tbody>
      </table>
    </div>
  </div>
</div>
```

### 常用表格类

| 类名 | 说明 |
|------|------|
| `.table` | 表格基础样式（添加内边距和底部边框） |
| `.table-sm` | 紧凑表格（减少单元格内边距） |
| `.table-bordered` | 添加所有边框 |
| `.table-striped` | 斑马纹（奇数行添加背景色） |
| `.table-hover` | 鼠标悬停行高亮 |
| `.table-dark` | 深色背景表格 |
| `.table-responsive` | 响应式包裹层，水平滚动 |

> 注意：`.table-responsive` 是加在包裹表格的 `<div>` 上的，不是加在 `<table>` 上。

### 效果说明

- `.table-sm` 让表格更紧凑，适合数据密集的表格
- `.table-responsive` 确保表格在窄屏下不会被截断，而是出现水平滚动条
- `<caption>` 用于描述表格内容，有助于无障碍访问

## 小节三：图文区（03-图文区.html）

### 课程简介

Bootstrap 提供了 `<figure>` 图文组件，用于展示带有标题说明的图片内容。

### 关键知识点

#### Figure 图文组件

```html
<div class="container">
  <div class="row">
    <figure class="figure">
      <img src="images/1140.jpg" class="img-fluid rounded figure-img" alt="">
      <figcaption class="figure-caption text-center">一叶知秋</figcaption>
    </figure>
  </div>
</div>
```

### 相关类名

| 类名 | 说明 |
|------|------|
| `.figure` | figure 容器类 |
| `.figure-img` | 图片类（添加底部间距） |
| `.figure-caption` | 图片说明文字类（浅色小字） |
| `.img-fluid` | 响应式图片 |
| `.rounded` | 圆角 |

### 效果说明

- `<figure>` + `<figcaption>` 是 HTML5 语义化的图文组合标签
- `.figure-caption` 让说明文字以较小、较浅的颜色显示
- 可以配合 `.text-center` / `.text-right` 等类控制说明文字的对齐方式
- `.figure-img` 自动为图片底部添加合适的间距，与说明文字分离
