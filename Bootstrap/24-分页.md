# 分页

## 课程简介

分页（Pagination）是 Bootstrap 提供的用于构建页码导航的组件，常用于列表、搜索结果等需要翻页的场景。本节课讲解分页的基本结构、激活与禁用状态、尺寸控制以及对齐方式。

---

## 关键知识点

- 分页基本结构：`pagination` > `page-item` > `page-link`
- 激活状态：`active`
- 禁用状态：`disabled`
- 使用 `<span>` 替代 `<a>` 实现不可点击的分页项
- 尺寸控制：`pagination-lg`（大）、默认（中）、`pagination-sm`（小）
- 对齐方式：`justify-content-center` / `justify-content-end`

---

## 关键代码示例

### 1. 基本分页

```html
<ul class="pagination">
  <li class="page-item"><a href="#" class="page-link"><span>&laquo;</span></a></li>
  <li class="page-item"><a href="#" class="page-link">1</a></li>
  <li class="page-item"><a href="#" class="page-link">2</a></li>
  <li class="page-item"><a href="#" class="page-link">3</a></li>
  <li class="page-item"><a href="#" class="page-link"><span>&raquo;</span></a></li>
</ul>
```

### 2. 激活与禁用状态

使用 `<span>` 替代 `<a>` 标签，配合 `active` 和 `disabled` 类。

```html
<ul class="pagination">
  <li class="page-item disabled"><span class="page-link">上一页</span></li>
  <li class="page-item"><span class="page-link">1</span></li>
  <li class="page-item active"><span class="page-link">2</span></li>
  <li class="page-item"><span class="page-link">3</span></li>
  <li class="page-item"><span class="page-link">下一页</span></li>
</ul>
```

**要点**：
- `disabled` 禁用分页项，使其不可点击，显示为灰色
- `active` 高亮当前页码
- 使用 `<span>` 替代 `<a>` 可以移除链接的点击效果，更适合不可点击的分页项

### 3. 分页尺寸

通过在 `pagination` 上添加尺寸类控制分页组件的大小。

```html
<!-- 大尺寸 -->
<ul class="pagination pagination-lg">
  <li class="page-item"><a href="#" class="page-link">上一页</a></li>
  <li class="page-item"><a href="#" class="page-link">1</a></li>
  <li class="page-item"><a href="#" class="page-link">2</a></li>
  <li class="page-item"><a href="#" class="page-link">3</a></li>
  <li class="page-item"><a href="#" class="page-link">下一页</a></li>
</ul>

<!-- 默认尺寸 -->
<ul class="pagination">...</ul>

<!-- 小尺寸 -->
<ul class="pagination pagination-sm">...</ul>
```

### 4. 对齐方式

利用弹性盒对齐类控制分页的水平位置。

```html
<!-- 靠右对齐 -->
<ul class="pagination justify-content-end">
  <li class="page-item"><a href="#" class="page-link">上一页</a></li>
  <li class="page-item"><a href="#" class="page-link">1</a></li>
  <li class="page-item"><a href="#" class="page-link">2</a></li>
  <li class="page-item"><a href="#" class="page-link">3</a></li>
  <li class="page-item"><a href="#" class="page-link">下一页</a></li>
</ul>
<!-- 居中对齐：justify-content-center -->
<!-- 靠左对齐：justify-content-start（默认） -->
```

---

## 效果说明

- 基本分页水平排列页码按钮，使用 `&laquo;` 和 `&raquo;` 字符作为上一页/下一页箭头
- 激活项（`active`）以蓝色高亮显示当前页码
- 禁用项（`disabled`）以灰色显示且不可点击，适合"上一页"在第一页时的状态
- 三种尺寸 `pagination-lg`、默认、`pagination-sm` 适配不同场景
- 分页默认靠左对齐，可通过 `justify-content-center` 或 `justify-content-end` 调整为居中或靠右
