# 响应式原理

## 课程简介

本节课深入讲解 CSS 媒体查询（Media Query）的实现原理，包括媒体类型、媒体特性、逻辑运算符以及响应式布局的实际应用。媒体查询是 Bootstrap 响应式栅格系统的底层基础。

## 关键知识点

### 响应式设计的特点

1. 网页的宽度自动调整
2. 尽量少用绝对宽度
3. 字体要使用 `rem`、`em` 作为单位
4. 布局要使用浮动、弹性布局

### CSS @规则

| 规则 | 说明 |
|------|------|
| `@charset` | 定义编码 |
| `@import` | 引入 CSS 文件（CSS 模块化） |
| `@font-face` | 自定义字体 |
| `@keyframes` | animation 里的关键帧 |
| `@media` | 媒体查询 |

### 媒体查询的三要素

1. 媒体类型
2. 媒体特性
3. 逻辑运算符

---

## 小节一：响应式原理（01-响应式原理.html）

媒体查询根据一个或多个基于设备类型、具体特点和环境来应用样式。

```css
/* 基本语法 */
@media all and (min-width: 400px) {
  div {
    font-size: 100px;
  }
}
```

---

## 小节二：媒体类型（02-媒体类型.html）

### 媒体类型列表

| 类型 | 说明 |
|------|------|
| `all` | 所有设备 |
| `print` | 打印机设备 |
| `screen` | 彩色的电脑屏幕 |
| `speech` | 听觉设备（针对有视力障碍的人士） |

> 注意：`tty`、`tv`、`projection`、`handheld`、`braille`、`embossed`、`aural` 等类型在媒体查询 4 中已废弃。

### 代码示例

```css
/* 打印机设备 */
@media print {
  div { font-size: 80px; }
}

/* 电脑屏幕 */
@media screen {
  div { width: 300px; height: 300px; background: pink; }
}

/* 所有设备 */
@media all {
  div { width: 400px; height: 400px; background: grey; }
}
```

## 小节三：媒体特性（03-媒体特性.html）

### 常用媒体特性

| 特性 | 说明 |
|------|------|
| `width` / `min-width` / `max-width` | 宽度相关 |
| `height` / `min-height` / `max-height` | 高度相关 |
| `orientation` | 方向：`landscape`（横屏）/ `portrait`（竖屏） |
| `aspect-ratio` | 宽高比 |
| `-webkit-device-pixel-ratio` | 像素比（webkit 内核私有属性） |

### 代码示例

```css
/* 屏幕宽度 > 500px 时生效 */
@media (min-width: 500px) {
  div { background: green; }
}

/* 屏幕宽度 < 400px 时生效 */
@media (max-width: 400px) {
  div { background: blue; }
}

/* 横屏时生效（宽度 > 高度） */
@media (orientation: landscape) {
  div { width: 400px; height: 100px; }
}

/* 竖屏时生效（高度 > 宽度） */
@media (orientation: portrait) {
  div { width: 100px; height: 400px; }
}

/* 宽高比为 4:3 时生效 */
@media (aspect-ratio: 4/3) {
  div { border: 10px solid #000; }
}

/* 像素比为 2 时生效（如 Retina 屏） */
@media (-webkit-device-pixel-ratio: 2) {
  div:after { content: '高像素密度屏幕'; font-size: 50px; }
}
```

## 小节四：逻辑运算符（04-逻辑运算符.html）

### 逻辑运算符列表

| 运算符 | 说明 |
|--------|------|
| `and` | 合并多个媒体查询（并且） |
| `,` | 匹配某个媒体查询（或者） |
| `not` | 对媒体查询结果取反 |
| `only` | 仅在媒体查询匹配成功后应用样式（防范老旧浏览器） |

### 代码示例

```css
/* and：屏幕 >= 700px 并且横屏 */
@media all and (min-width: 700px) and (orientation: landscape) {
  div { background: green; }
}

/* 逗号（或）：屏幕 < 800px 或者横屏 */
@media (max-width: 800px), (orientation: landscape) {
  div { background: pink; }
}

/* not：对条件取反，即屏幕 > 800px */
@media not all and (max-width: 800px) {
  div { background: blue; }
}

/* only：仅在匹配成功后应用样式 */
@media only screen and (min-width: 1000px) {
  div { background: grey; }
}
```

## 小节五：媒体查询应用 -- 房子示例（05-媒体查询应用.html）

通过媒体查询实现不同屏幕尺寸下的内容自适应变化。

```css
/* 默认样式 */
div { padding: 50px 0; border: 1px solid #000800; }
div:after { content: '这是一所房子'; }

/* < 1000px */
@media all and (max-width: 1000px) {
  div { background: green; }
  div:after { content: '房子好大，可以放下坑了'; }
}

/* < 800px */
@media all and (max-width: 800px) {
  div { background: khaki; }
  div:after { content: '房子变小了，只能放一张双人床'; }
}

/* < 500px */
@media all and (max-width: 500px) {
  div { background: hotpink; }
  div:after { content: '房子更小了，只能放一张单人床'; }
}

/* < 300px */
@media all and (max-width: 300px) {
  div { background: turquoise; }
  div:after { content: '房子也忒小了，床都放不下'; }
}
```

## 小节六：响应式布局实战（06-响应式例子.html）

使用媒体查询实现不同屏幕宽度下的列数自适应布局。

```css
body { margin: 0; }
div {
  width: 100px; height: 100px;
  background: green; float: left;
  border: 1px solid #000;
  box-sizing: border-box;
}

/* < 400px：显示 1 列 */
@media all and (max-width: 400px) {
  div { width: 100%; }
}

/* >= 400px：显示 2 列 */
@media all and (min-width: 400px) {
  div { width: 50%; }
}

/* >= 600px：显示 3 列 */
@media all and (min-width: 600px) {
  div { width: 33.3%; }
}

/* >= 800px：显示 4 列 */
@media all and (min-width: 800px) {
  div { width: 25%; }
}

/* >= 1000px：显示 5 列 */
@media all and (min-width: 1000px) {
  div { width: 20%; }
}
```

## 效果说明

- 媒体类型允许针对不同设备（屏幕、打印机等）应用不同样式
- 媒体特性可以检测屏幕宽度、高度、方向、像素比等设备特征
- 逻辑运算符用于组合多个媒体查询条件
- 通过 `max-width` 实现桌面优先（从大到小），通过 `min-width` 实现移动端优先（从小到大）
- 响应式布局的核心思想：根据屏幕宽度动态调整元素尺寸和布局方式
