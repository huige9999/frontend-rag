# 提示框与提示工具

## 课程简介

本节课学习 Bootstrap 4 中的两个交互提示组件：**提示框（Popover）** 和 **提示工具（Tooltip）**。提示框可以承载标题和内容，点击时弹出；提示工具则更加轻量，鼠标悬停时显示简短提示文字。

---

## 一、提示框（Popover）

### 关键知识点

- 提示框通过 `data-toggle="popover"` 属性触发
- 必须通过 JS 初始化：`$('[data-toggle="popover"]').popover()`
- 支持四个弹出方向：`top`、`right`、`bottom`、`left`
- 支持 `data-` 属性进行配置
- 禁用元素需在外层包裹容器来实现弹出效果
- 提供方法：`show`、`hide`、`toggle`、`enable`、`disable`、`toggleEnabled`
- 提供事件：`show.bs.popover`、`shown.bs.popover`、`hide.bs.popover`、`hidden.bs.popover`、`inserted.bs.popover`

### 关键代码示例

**基本提示框：**

```html
<button class="btn btn-danger"
  data-toggle="popover"
  title="这里放的是提示框的标题"
  data-content="这里放的是提示框的内容">
  提示框
</button>
```

**四个弹出方向：**

```html
<button class="btn btn-secondary" data-toggle="popover" title="上面显示" data-placement="top">上面</button>
<button class="btn btn-info" data-toggle="popover" title="右侧显示" data-placement="right">右侧</button>
<button class="btn btn-primary" data-toggle="popover" title="下面显示" data-placement="bottom">下面</button>
<button class="btn btn-warning" data-toggle="popover" title="左侧显示" data-placement="left">左侧</button>
```

**禁用元素弹出提示框（外层包裹方案）：**

```html
<span class="d-inline-block" data-toggle="popover" data-trigger="hover"
  data-content="禁用的元素想要弹出提示框，需要在外面添加一层">
  <button class="btn btn-primary" style="pointer-events: none;" disabled>禁用的按钮</button>
</span>
```

**data-属性配置：**

```html
<button class="btn btn-success" data-toggle="popover"
  title="这是<b>标题</b>"
  data-content="这是内容"
  data-animation="true"
  data-container="body"
  data-delay='{"show": 1000, "hide": 2000}'
  data-html="true"
  data-placement="top"
  data-trigger="hover"
  data-offset="120">
  data-属性
</button>
```

**JS 初始化与方法事件：**

```javascript
// 初始化（必须）
$('[data-toggle="popover"]').popover();

// 方法
$('#myPopover').popover('show');      // 显示
$('#myPopover').popover('hide');      // 隐藏
$('#myPopover').popover('toggle');    // 切换
$('#myPopover').popover('enable');    // 启用
$('#myPopover').popover('disable');   // 禁用
$('#myPopover').popover('toggleEnabled'); // 切换启用/禁用

// 事件
$('#myPopover').on('show.bs.popover', function() { /* 要显示了 */ });
$('#myPopover').on('shown.bs.popover', function() { /* 完全显示了 */ });
$('#myPopover').on('hide.bs.popover', function() { /* 要隐藏了 */ });
$('#myPopover').on('hidden.bs.popover', function() { /* 完全隐藏了 */ });
$('#myPopover').on('inserted.bs.popover', function() { /* DOM已插入 */ });
```

### 效果说明

点击按钮后弹出带标题和内容的提示框气泡，可控制弹出方向。失去焦点可自动隐藏，支持动画、延迟、HTML内容等配置。

---

## 二、提示工具（Tooltip）

### 关键知识点

- 提示工具通过 `data-toggle="tooltip"` 属性触发
- 必须通过 JS 初始化：`$('[data-toggle="tooltip"]').tooltip()`
- 支持四个方向：`data-placement="top/right/bottom/left"`
- 可用于任何元素，如按钮、链接等
- 支持 `data-html="true"` 渲染 HTML 内容
- 禁用元素同样需要外层包裹方案
- 方法与事件和提示框类似，事件名为 `*.bs.tooltip`

### 关键代码示例

**基本用法与四个方向：**

```html
<button class="btn btn-secondary" data-toggle="tooltip"
  title="这个提示工具在上面显示" data-placement="top">上面弹出</button>
<button class="btn btn-info" data-toggle="tooltip"
  title="右侧显示" data-placement="right">右侧弹出</button>
<button class="btn btn-primary" data-toggle="tooltip"
  title="下面显示" data-placement="bottom">下面弹出</button>
<button class="btn btn-warning" data-toggle="tooltip"
  title="左侧显示" data-placement="left">左侧弹出</button>
```

**链接上使用提示工具：**

```html
<a href="#" data-toggle="tooltip" title="Default tooltip">you probably</a>
```

**自定义 HTML 内容：**

```html
<button class="btn btn-success" data-toggle="tooltip" data-html="true"
  title="自<em>定</em>义<b>html</b>">自定义HTML</button>
```

**禁用元素弹出提示工具：**

```html
<span class="d-inline-block" data-toggle="tooltip"
  title="禁用的元素弹出提示工具，需要在外面添加一个层">
  <button class="btn btn-primary" style="pointer-events: none;" disabled>禁用按钮</button>
</span>
```

**JS 初始化与方法事件：**

```javascript
// 初始化（必须）
$('[data-toggle="tooltip"]').tooltip();

// 方法
$('#myTooltip').tooltip('show');
$('#myTooltip').tooltip('hide');
$('#myTooltip').tooltip('toggle');
$('#myTooltip').tooltip('enable');
$('#myTooltip').tooltip('disable');
$('#myTooltip').tooltip('toggleEnabled');

// 事件
$('#myTooltip').on('show.bs.tooltip', function() { /* 要显示了 */ });
$('#myTooltip').on('shown.bs.tooltip', function() { /* 完全显示了 */ });
$('#myTooltip').on('hide.bs.tooltip', function() { /* 要隐藏了 */ });
$('#myTooltip').on('hidden.bs.tooltip', function() { /* 完全隐藏了 */ });
$('#myTooltip').on('inserted.bs.tooltip', function() { /* DOM已插入 */ });
```

### 效果说明

鼠标悬停在元素上时显示简短的文字提示，比提示框更轻量。可控制方向、支持HTML内容渲染，适用于按钮、链接等多种元素。
