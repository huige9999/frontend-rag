# 12 - 轮播图

## 课程简介

本节课学习 Bootstrap 4 中的 **轮播图（Carousel）** 组件。轮播图用于循环展示多张图片或内容，支持自动播放、手动控制（前后箭头）、指示器、字幕说明以及多种配置选项和 JS 事件。

---

## 一、基础轮播图

### 关键知识点

- `.carousel`：轮播图容器
- `.slide`：添加滑动动画效果
- `.carousel-fade`：淡入淡出效果（替代滑动）
- `.carousel-inner`：轮播内容包裹层
- `.carousel-item`：每个轮播项，第一项需添加 `.active` 类
- `data-ride="carousel"`：页面加载后自动开始播放

### 关键代码示例

```html
<div class="carousel slide carousel-fade" data-ride="carousel">
    <div class="carousel-inner">
        <div class="carousel-item active">
            <img src="pic_01.jpg" class="img-fluid">
        </div>
        <div class="carousel-item">
            <img src="pic_02.jpg" class="img-fluid">
        </div>
        <div class="carousel-item">
            <img src="pic_03.jpg" class="img-fluid">
        </div>
    </div>
</div>
```

## 二、带控制箭头的轮播图

### 关键知识点

- `.carousel-control-prev`：上一张箭头容器
- `.carousel-control-next`：下一张箭头容器
- `.carousel-control-prev-icon` / `.carousel-control-next-icon`：箭头图标
- `data-slide="prev"` / `data-slide="next"`：控制方向
- `href="#id"` 指向轮播图的 `id`

### 关键代码示例

```html
<div class="carousel slide" data-ride="carousel" id="control">
    <div class="carousel-inner">
        <div class="carousel-item active">
            <img src="pic_01.jpg" class="img-fluid">
        </div>
        <div class="carousel-item">
            <img src="pic_02.jpg" class="img-fluid">
        </div>
        <div class="carousel-item">
            <img src="pic_03.jpg" class="img-fluid">
        </div>
    </div>

    <!-- 左右箭头 -->
    <a href="#control" class="carousel-control-prev" data-slide="prev">
        <span class="carousel-control-prev-icon"></span>
    </a>
    <a href="#control" class="carousel-control-next" data-slide="next">
        <span class="carousel-control-next-icon"></span>
    </a>
</div>
```

## 三、带指示器的轮播图

### 关键知识点

- `.carousel-indicators`：指示器容器（有序列表 `<ol>`）
- `data-target="#id"`：指向轮播图的 `id`
- `data-slide-to="0"`：指定跳转到第几张（从 0 开始）
- `.active`：当前激活的指示器

### 关键代码示例

```html
<ol class="carousel-indicators">
    <li data-target="#indicators" data-slide-to="0" class="active"></li>
    <li data-target="#indicators" data-slide-to="1"></li>
    <li data-target="#indicators" data-slide-to="2"></li>
</ol>
```

---

## 四、带字幕的轮播图

### 关键知识点

- `.carousel-caption`：字幕容器，放在 `.carousel-item` 内部
- 可包含标题 `<h5>` 和说明文字 `<p>`

### 关键代码示例

```html
<div class="carousel-item active">
    <img src="pic_01.jpg" class="img-fluid">
    <div class="carousel-caption text-danger">
        <h5>这是第1张图</h5>
        <p>这是第1张图的说明文字...</p>
    </div>
</div>
```

## 五、轮播图配置选项

### 关键知识点

通过 `data-*` 属性在轮播图容器上配置：

| 属性 | 说明 |
|------|------|
| `data-interval` | 自动切换的时间间隔（毫秒），如 `2000` |
| `data-keyboard` | 是否响应键盘事件，`true`/`false` |
| `data-pause` | 鼠标悬停时是否暂停，`"hover"`/`false` |
| `data-wrap` | 是否循环播放，`true`/`false` |
| `data-ride` | 页面加载后是否自动播放，`"carousel"` |

### 关键代码示例

```html
<div class="carousel slide" id="options" data-ride="carousel"
     data-interval="2000"
     data-keyboard="true"
     data-pause="false"
     data-wrap="true">
    <!-- 轮播内容、箭头、指示器 -->
</div>
```

## 六、JS 方法与事件

### 关键知识点

**方法：**
- `.carousel(options)`：初始化轮播图并设置选项
- `.carousel('cycle')`：开始循环播放
- `.carousel('pause')`：暂停播放
- `.carousel('prev')`：切换到上一张
- `.carousel('next')`：切换到下一张
- `.carousel(number)`：跳转到指定索引的图片

**事件：**
- `slide.bs.carousel`：开始切换时触发，事件对象包含：
  - `direction`：切换方向
  - `relatedTarget`：目标 DOM 元素
  - `from`：当前图片索引
  - `to`：目标图片索引
- `slid.bs.carousel`：切换完成后触发

### 关键代码示例

```javascript
// 初始化并开始播放
$('.play').click(function () {
    $('#methods').carousel({
        interval: 2000
    });
    $('#methods').carousel('cycle');
});

// 暂停播放
$('.pause').click(function () {
    $('#methods').carousel('pause');
});

// 上一张
$('.prevBtn').click(function () {
    $('#methods').carousel('prev');
});

// 下一张
$('.nextBtn').click(function () {
    $('#methods').carousel('next');
});

// 指示器点击跳转
$('.indicatorsBtn li').each(function (index, element) {
    $(element).click(function () {
        $('#methods').carousel(index);
    });
});

// 开始切换事件
$('#methods').on('slide.bs.carousel', function (ev) {
    console.log(ev.direction);       // 方向
    console.log(ev.relatedTarget);   // 目标DOM
    console.log(ev.from);            // 当前索引
    console.log(ev.to);              // 目标索引
});

// 切换完成事件
$('#methods').on('slid.bs.carousel', function () {
    console.log('切换完成');
});
```

### 效果说明

轮播图是一个功能丰富的组件，默认通过 `data-ride="carousel"` 自动播放。通过 JS 方法可以精确控制播放、暂停、切换等行为。两个事件分别在不同时机触发，适合添加自定义动画或数据统计逻辑。使用 JS 功能需要引入 jQuery 和 Bootstrap JS 文件。
