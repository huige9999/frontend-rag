# 实战项目-1

## 课程简介

本节课是 Bootstrap 实战项目的第一部分，使用 Bootstrap 4 制作一个**星巴克风格**的咖啡网站页面。本节主要完成左侧固定栏（导航、标题、登录注册）和菜单弹出层的效果，以及响应式布局的搭建。

---

## 关键知识点

- 使用 `position-fixed` 实现左侧固定栏布局
- 使用 `container-fluid` + `row` + `col-lg-4` + `col-lg-8` 实现左右两栏
- 使用 `d-none d-sm-flex` 实现响应式显示/隐藏
- 使用 `d-flex` + `align-items-center` 实现垂直居中
- 使用 jQuery 的 `addClass` / `removeClass` 实现菜单的显示与隐藏
- 结合 iconfont 自定义图标增强视觉效果
- 使用 `position-absolute` 实现菜单的覆盖层效果

### JavaScript 交互逻辑

```javascript
// 点击菜单按钮显示菜单
$('#showBtn').on('click', function(){
    $('.menu').removeClass('hide').addClass('show');
});

// 点击关闭按钮隐藏菜单
$('#closeBtn').on('click', function(){
    $('.menu').removeClass('show').addClass('hide');
});
```

## 效果说明

- 页面分为左右两栏：左侧固定展示品牌信息和导航，右侧为可滚动的内容区域
- 导航栏在小屏幕（sm以下）隐藏，只在大屏幕显示
- 点击菜单图标弹出全屏覆盖层菜单，点击关闭按钮收回
- 登录注册按钮使用 Bootstrap 的 `btn-outline-success` 样式
- 使用 iconfont 图标替代文字按钮，提升视觉效果
