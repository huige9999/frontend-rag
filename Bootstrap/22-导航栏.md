# 导航栏

## 课程简介

导航栏（Navbar）是 Bootstrap 中用于构建网站顶部或底部导航的组件，相比基础导航（Nav），导航栏提供了更丰富的功能，包括品牌Logo、导航链接、表单、文本、颜色主题、定位方式以及响应式折叠等。本节课全面讲解导航栏的用法。

---

## 关键知识点

- 导航栏基础结构：`navbar` + `navbar-expand-{breakpoint}` + 颜色主题
- 品牌Logo：`navbar-brand`
- 导航链接列表：`navbar-nav` > `nav-item` > `nav-link`
- 导航栏中的表单：`form-inline`
- 导航栏中的文本：`navbar-text`
- 颜色方案：`bg-light`/`bg-dark` + `navbar-light`/`navbar-dark`
- 居中容器：将 `navbar` 放在 `container` 内
- 固定定位：`fixed-top`、`fixed-bottom`
- 粘性定位：`sticky-top`
- 响应式折叠：`navbar-toggler` + `collapse navbar-collapse`
- 下拉菜单集成：`dropdown`

---

## 关键代码示例

### 1. 基本导航栏

```html
<nav class="navbar navbar-expand-lg bg-light">
  <a href="#" class="navbar-brand">
    <img src="bootstrap-solid.svg" alt="" width="30" class="align-top"> Bootstrap
  </a>
  <ul class="navbar-nav">
    <li class="nav-item"><a href="#" class="nav-link">入门</a></li>
    <li class="nav-item"><a href="#" class="nav-link">布局</a></li>
    <li class="nav-item"><a href="#" class="nav-link">内容</a></li>
  </ul>
</nav>
```

### 2. 导航栏中的表单

```html
<nav class="navbar bg-light">
  <a href="#" class="navbar-brand">Bootstrap</a>
  <form action="#" class="form-inline">
    <div class="input-group">
      <div class="input-group-prepend">
        <span class="input-group-text">@</span>
      </div>
      <input type="search" class="form-control mr-sm-2" placeholder="Search">
    </div>
    <button type="submit" class="btn btn-outline-success mt-2 mt-sm-0">Search</button>
  </form>
</nav>
```

### 3. 导航栏中的文本

使用 `navbar-text` 在导航栏中插入纯文本内容。

```html
<nav class="navbar navbar-expand-lg bg-light">
  <a href="#" class="navbar-brand">Bootstrap</a>
  <ul class="navbar-nav mr-auto">
    <li class="nav-item"><a href="#" class="nav-link">入门</a></li>
  </ul>
  <span class="navbar-text">这是导航当中的文本</span>
</nav>
```

### 4. 颜色方案

`bg-*` 控制背景色，`navbar-dark`/`navbar-light` 控制文字颜色。

```html
<nav class="navbar navbar-expand-lg bg-dark navbar-dark">
  <a href="#" class="navbar-brand">Bootstrap</a>
  <ul class="navbar-nav mr-auto">
    <li class="nav-item"><a href="#" class="nav-link">入门</a></li>
  </ul>
  <form action="#" class="form-inline">
    <input type="search" class="form-control mr-sm-2" placeholder="Search">
    <button type="submit" class="btn btn-outline-success mt-2 mt-sm-0">Search</button>
  </form>
</nav>
```

### 5. 居中容器

将导航栏放在 `container` 内，使导航栏居中显示并与页面内容对齐。

```html
<div class="container">
  <nav class="navbar navbar-expand-lg bg-dark navbar-dark mt-5">
    <a href="#" class="navbar-brand">Bootstrap</a>
    <ul class="navbar-nav mr-auto">
      <li class="nav-item"><a href="#" class="nav-link">入门</a></li>
    </ul>
  </nav>
</div>
```

### 6. 固定定位与粘性定位

```html
<!-- 固定在顶部 -->
<nav class="navbar navbar-light bg-light fixed-top">
  <a href="#" class="navbar-brand">固定在顶部</a>
</nav>

<!-- 固定在底部 -->
<nav class="navbar navbar-light bg-light fixed-bottom">
  <a href="#" class="navbar-brand">固定在底部</a>
</nav>

<!-- 粘性定位在浏览器顶部（滚动时固定） -->
<nav class="navbar navbar-light bg-light sticky-top">
  <a href="#" class="navbar-brand">粘性定位在浏览器的顶部</a>
</nav>
```

### 7. 响应式折叠导航栏

小屏幕下显示折叠按钮，大屏幕下展开导航。

```html
<nav class="navbar navbar-expand-md bg-success navbar-dark">
  <!-- 左侧是Logo，右侧是按钮 -->
  <a href="#" class="navbar-brand">Bootstrap</a>
  <button class="navbar-toggler" data-toggle="collapse" data-target="#navbar">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbar">
    <ul class="navbar-nav mr-auto">
      <li class="nav-item"><a href="#" class="nav-link">入门</a></li>
      <li class="nav-item"><a href="#" class="nav-link">布局</a></li>
    </ul>
    <form action="#" class="form-inline">
      <input type="search" class="form-control mr-sm-2" placeholder="Search">
      <button type="submit" class="btn btn-outline-light mt-2 mt-sm-0">Search</button>
    </form>
  </div>
</nav>
```

**三种折叠布局方式**：

- Logo在左，折叠按钮在右（标准方式）
- 折叠按钮在左，Logo在右

```html
<!-- 左侧是按钮，右侧是Logo -->
<nav class="navbar navbar-expand-md bg-success navbar-dark">
  <button class="navbar-toggler" data-toggle="collapse" data-target="#navbar2">
    <span class="navbar-toggler-icon"></span>
  </button>
  <a href="#" class="navbar-brand">Bootstrap</a>
  <div class="collapse navbar-collapse" id="navbar2">
    <!-- 导航内容 -->
  </div>
</nav>
```

### 8. 导航栏中的下拉菜单

```html
<li class="nav-item dropdown">
  <a href="#" class="nav-link dropdown-toggle" data-toggle="dropdown">城市</a>
  <div class="dropdown-menu">
    <a href="#" class="dropdown-item">北京</a>
    <a href="#" class="dropdown-item">上海</a>
    <a href="#" class="dropdown-item">广州</a>
    <div class="dropdown-divider"></div>
    <a href="#" class="dropdown-item">深圳</a>
  </div>
</li>
```

---

## 效果说明

- 基本导航栏水平排列Logo和导航链接，`navbar-expand-lg` 控制在大屏以上展开
- 深色背景需配合 `navbar-dark` 使文字变为浅色，浅色背景配合 `navbar-light`
- 表单可以直接嵌入导航栏，使用 `form-inline` 保持水平排列
- `fixed-top`/`fixed-bottom` 始终固定在视口顶部/底部，`sticky-top` 在滚动到顶部时固定
- 响应式折叠在指定断点以下显示汉堡按钮，点击展开/收起导航内容
