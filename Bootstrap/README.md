## 📖 课件格式说明

每个课件文件统一包含以下结构：

- **标题** — 课程名称
- **课程简介** — 本节课核心内容概述
- **关键知识点** — 从源文件提取的概念性说明（表格或列表形式）
- **关键代码示例** — 仅提取核心 Bootstrap 用法片段，附带注释说明
- **效果说明** — 代码运行效果和注意事项
