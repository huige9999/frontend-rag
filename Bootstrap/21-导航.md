# 导航

## 课程简介

导航（Nav）是 Bootstrap 提供的基础导航组件，可用于构建页面内的导航菜单、标签页等。本节课从基础导航类出发，逐步讲解对齐方式、垂直排列、Tabs 标签、胶囊式标签、宽度分配、弹性布局结合、下拉菜单以及选项卡功能，最后介绍导航的 JavaScript 方法与事件。

---

## 关键知识点

- 基础导航类：`nav`（弹性盒模型）、`nav-link`（块元素 + padding）
- 使用 `<ul>` 或 `<nav>` 标签构建导航
- 水平对齐：`justify-content-start` / `justify-content-center` / `justify-content-end`
- 垂直排列：`flex-column`、`flex-sm-column`
- Tabs 标签：`nav-tabs`，配合 `nav-item`、`active`、`disabled`
- 胶囊式标签：`nav-pills`
- 宽度分配：`nav-fill`（按比例分配，宽度由内容定）、`nav-justified`（等宽分配）
- 弹性布局结合：`flex-column flex-sm-row` 实现响应式排列
- 结合下拉菜单：`dropdown`、`dropdown-toggle`、`dropdown-menu`
- 选项卡：`data-toggle="tab"` + `tab-content` + `tab-pane`
- 垂直选项卡：`nav-pills flex-column` + 栅格布局
- JavaScript方法：`.tab('show')`
- JavaScript事件：`show.bs.tab`、`shown.bs.tab`、`hide.bs.tab`、`hidden.bs.tab`

## 关键代码示例

### 1. 基础导航（ul 标签）

```html
<ul class="nav">
  <li><a class="nav-link" href="#">入门</a></li>
  <li><a class="nav-link" href="#">布局</a></li>
  <li><a class="nav-link" href="#">内容</a></li>
  <li><a class="nav-link" href="#">组件</a></li>
</ul>
```

### 2. 使用 nav 标签

```html
<nav class="nav">
  <a class="nav-link" href="#">使用nav标签</a>
  <a class="nav-link" href="#">布局</a>
  <a class="nav-link" href="#">内容</a>
  <a class="nav-link" href="#">组件</a>
</nav>
```

### 3. 水平对齐

利用弹性盒模型的主轴对齐 class。

```html
<ul class="nav justify-content-center">
  <li><a class="nav-link" href="#">入门</a></li>
  <li><a class="nav-link" href="#">布局</a></li>
</ul>
<!-- justify-content-start（默认）/ justify-content-center / justify-content-end -->
```

### 4. 垂直排列

```html
<ul class="nav flex-sm-column">
  <li><a class="nav-link" href="#">垂直排列</a></li>
  <li><a class="nav-link" href="#">布局</a></li>
</ul>

<!-- 也可以使用 nav 标签 -->
<nav class="nav flex-column">
  <a class="nav-link" href="#">垂直排列</a>
  <a class="nav-link" href="#">布局</a>
</nav>
```

### 5. Tabs 标签样式

```html
<ul class="nav nav-tabs">
  <li class="nav-item">
    <a class="nav-link active" href="#">tabs标签</a>   <!-- 当前激活项 -->
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">布局</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" href="#">组件</a>     <!-- 禁用，没有hover样式 -->
  </li>
</ul>
```

### 6. 胶囊式标签

```html
<ul class="nav nav-pills">
  <li class="nav-item">
    <a class="nav-link active" href="#">胶囊式标签</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">布局</a>
  </li>
</ul>
```

### 7. 宽度分配 - 按比例填充（nav-fill）

`nav-fill` 会将所有空间按比例分配给导航项，但每项宽度由内容决定。

```html
<ul class="nav nav-pills nav-fill">
  <li class="nav-item">
    <a class="nav-link active" href="#">宽度由内容而定</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">布局</a>
  </li>
</ul>
```

**注意**：使用 `<nav>` 标签时，`<a>` 上需要额外添加 `nav-item` 类。

```html
<nav class="nav nav-pills nav-fill">
  <a class="nav-link nav-item active" href="#">宽度由内容而定</a>
  <a class="nav-link nav-item" href="#">布局</a>
</nav>
```

### 8. 宽度分配 - 等宽（nav-justified）

`nav-justified` 将所有空间均分给每个导航项。

```html
<ul class="nav nav-pills nav-justified">
  <li class="nav-item">
    <a class="nav-link active" href="#">宽度为等宽</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">布局</a>
  </li>
</ul>
```

### 9. 响应式弹性布局

默认竖排，小屏以上横排。

```html
<nav class="nav nav-pills nav-justified flex-column flex-sm-row">
  <a class="nav-item nav-link active" href="#">弹性布局</a>
  <a class="nav-item nav-link" href="#">布局</a>
  <a class="nav-item nav-link disabled" href="#">组件</a>
</nav>
```

### 10. 结合下拉菜单

```html
<ul class="nav nav-tabs">
  <li class="nav-item">
    <a class="nav-link active" href="#">下拉菜单</a>
  </li>
  <li class="nav-item dropdown">
    <a class="nav-link dropdown-toggle" data-toggle="dropdown" href="#">城市</a>
    <div class="dropdown-menu">
      <a class="dropdown-item" href="#">北京</a>
      <a class="dropdown-item" href="#">上海</a>
      <a class="dropdown-item" href="#">广州</a>
      <div class="dropdown-divider"></div>
      <a class="dropdown-item" href="#">深圳</a>
    </div>
  </li>
</ul>
```

### 11. 选项卡（Tabs with Content）

导航链接与内容面板联动。

```html
<nav class="nav nav-tabs">
  <a class="nav-item nav-link active" data-toggle="tab" href="#home">Home</a>
  <a class="nav-item nav-link" data-toggle="tab" href="#profile">Profile</a>
  <a class="nav-item nav-link" data-toggle="tab" href="#contact">Contact</a>
</nav>

<div class="tab-content">
  <div class="tab-pane fade show active" id="home">Home 内容...</div>
  <div class="tab-pane fade" id="profile">Profile 内容...</div>
  <div class="tab-pane fade" id="contact">Contact 内容...</div>
</div>
```

### 12. 垂直形式的选项卡

利用栅格系统实现左侧导航、右侧内容的布局。

```html
<div class="row">
  <div class="col-3">
    <nav class="nav nav-pills flex-column">
      <a class="nav-item nav-link active" data-toggle="tab" href="#home1">Home</a>
      <a class="nav-item nav-link" data-toggle="tab" href="#profile1">Profile</a>
      <a class="nav-item nav-link" data-toggle="tab" href="#contact1">Contact</a>
    </nav>
  </div>
  <div class="col-9">
    <div class="tab-content">
      <div class="tab-pane fade show active" id="home1">Home 内容...</div>
      <div class="tab-pane fade" id="profile1">Profile 内容...</div>
      <div class="tab-pane fade" id="contact1">Contact 内容...</div>
    </div>
  </div>
</div>
```

### 13. 方法与事件

```javascript
// 方法：手动切换选项卡
$('#myTab a:last-child').tab('show');

$('#myTab a').on('click', function(e) {
  e.preventDefault();
  $(this).tab('show');
});

// 事件
$('#myTab a:nth-child(2)').on('show.bs.tab', function(e) {
  console.log('该内容要显示了');
});
$('#myTab a:nth-child(2)').on('shown.bs.tab', function(e) {
  console.log('该内容已经显示了');
});
$('#myTab a:nth-child(2)').on('hide.bs.tab', function(e) {
  console.log('该内容要隐藏了');
});
$('#myTab a:nth-child(2)').on('hidden.bs.tab', function(e) {
  console.log('该内容已经隐藏了');
});
```

---

## 效果说明

- 基础导航将列表项变为横向弹性盒布局，`nav-link` 使链接变为块元素并添加内边距
- Tabs 样式为导航添加标签页外观，配合 `active` 高亮当前项
- 胶囊式标签为激活项添加圆角背景色
- `nav-fill` 按内容比例分配宽度，`nav-justified` 实现严格等宽
- 选项卡功能通过 `data-toggle="tab"` 和 `tab-content`/`tab-pane` 实现导航与内容面板联动
- 垂直选项卡适合左侧菜单 + 右侧内容的经典布局
