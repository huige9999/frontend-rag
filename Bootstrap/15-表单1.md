# 第15课：表单1（Form Basics）

## 课程简介

本课讲解 Bootstrap 4 表单的基础用法，包括表单控件（文本框、密码框、下拉框、文件、滑块、复选框、单选框）、表单尺寸控制、只读属性、栅格排列布局、水平表单和内联表单。

## 关键知识点

1. **表单组**：`.form-group` 将 `label` 与表单控件配对组合
2. **表单控件**：`.form-control` 适用于 `<input>`、`<select>`、`<textarea>`
3. **文件输入**：`.form-control-file`
4. **滑块**：`.form-control-range`
5. **复选框与单选框**：`.form-check` + `.form-check-input` + `.form-check-label`，`.form-check-inline` 水平排列
6. **帮助文本**：`.form-text` + `.text-muted`
7. **禁用状态**：`disabled` 属性
8. **表单尺寸**：`.form-control-lg` / `.form-control-sm`、`.col-form-label-lg` / `.col-form-label-sm`
9. **只读属性**：`readonly` 属性；`.form-control-plaintext` 去掉边框的只读样式
10. **栅格排列**：`.form-row` 比 `.row` 间距更小，适合表单场景
11. **水平表单**：`.form-group.row` + `.col-form-label` 实现标签与控件水平排列
12. **内联表单**：`.form-inline` 使所有表单元素在同一行显示

## 一、基础表单控件

包含文本框、密码框、下拉框、文件、滑块、复选框、单选框的完整示例。

```html
<form action="#">
  <!-- 文本框 -->
  <div class="form-group">
    <label for="usename">用户名</label>
    <input type="text" id="usename" class="form-control" placeholder="请输入用户名">
    <small class="form-text text-muted">用户名需要用英文、符号组成</small>
  </div>

  <!-- 密码框 -->
  <div class="form-group">
    <label for="password">密码</label>
    <input type="password" class="form-control" id="password" placeholder="请输入密码">
  </div>

  <!-- 禁用状态 -->
  <div class="form-group">
    <label for="passwordAgain">确认密码</label>
    <input type="password" class="form-control" id="passwordAgain" placeholder="请再次输入密码" disabled>
  </div>

  <!-- 下拉框 -->
  <div class="form-group">
    <label for="area">所在地</label>
    <select id="area" class="form-control">
      <option value="">北京</option>
      <option value="">上海</option>
      <option value="">广州</option>
      <option value="">深圳</option>
    </select>
  </div>

  <!-- 文件选择 -->
  <div class="form-group">
    <label for="file">选择文件</label>
    <input type="file" class="form-control-file" id="file">
  </div>

  <!-- 滑块 -->
  <div class="form-group">
    <label for="range">滑块</label>
    <input type="range" class="form-control-range" id="range">
  </div>
</form>
```

## 二、复选框与单选框

```html
<!-- 复选框（行内排列） -->
<div class="form-group">
  <label>兴趣爱好：</label>
  <div class="form-check form-check-inline">
    <input type="checkbox" id="eat" class="form-check-input">
    <label for="eat" class="form-check-label">吃饭</label>
  </div>
  <div class="form-check form-check-inline">
    <input type="checkbox" id="sleep" class="form-check-input">
    <label for="sleep" class="form-check-label">睡觉</label>
  </div>
  <div class="form-check form-check-inline">
    <input type="checkbox" id="play" class="form-check-input">
    <label for="play" class="form-check-label">打豆豆</label>
  </div>
</div>

<!-- 单选框（行内排列） -->
<div class="form-group">
  <label>性别：</label>
  <div class="form-check form-check-inline">
    <input type="radio" id="male" class="form-check-input" name="sex">
    <label for="male" class="form-check-label">男</label>
  </div>
  <div class="form-check form-check-inline">
    <input type="radio" id="female" class="form-check-input" name="sex">
    <label for="female" class="form-check-label">女</label>
  </div>
</div>
```

**效果说明**：`.form-check-inline` 使复选框/单选框在同一行水平排列。单选框通过相同的 `name` 属性实现互斥。

## 三、表单尺寸

```html
<input type="text" class="form-control form-control-lg" placeholder="大尺寸的表单">
<input type="text" class="form-control" placeholder="正常尺寸的表单">
<input type="text" class="form-control form-control-sm" placeholder="小尺寸的表单">

<!-- 下拉框尺寸 -->
<select class="form-control col-form-label-lg">
  <option>大尺寸的下拉框</option>
</select>
<select class="form-control col-form-label-sm">
  <option>小尺寸的下拉框</option>
</select>
```

**效果说明**：`.form-control-lg` / `.form-control-sm` 控制输入框大小，`.col-form-label-lg` / `.col-form-label-sm` 控制下拉框大小。

## 四、只读属性

```html
<input type="text" class="form-control" placeholder="只读文本" readonly>
<input type="text" class="form-control-plaintext" placeholder="没有边框的只读文本" readonly>
```

**效果说明**：`readonly` 使输入框不可编辑。`.form-control-plaintext` 移除边框样式，只显示纯文本。

## 五、栅格排列表单

使用 `.row` 或 `.form-row` 配合 `.col` 实现表单的栅格布局。`.form-row` 比 `.row` 的间距更紧凑。

```html
<!-- 使用 row -->
<form action="#">
  <div class="row">
    <div class="col">
      <input type="text" class="form-control" placeholder="栅格排列">
    </div>
    <div class="col">
      <input type="text" class="form-control" placeholder="栅格排列">
    </div>
  </div>
</form>

<!-- 使用 form-row（间距更小） -->
<form action="#">
  <div class="form-row">
    <div class="col">
      <input type="text" class="form-control" placeholder="栅格排列">
    </div>
    <div class="col">
      <input type="text" class="form-control" placeholder="栅格排列">
    </div>
  </div>
</form>
```

### 综合栅格表单示例

```html
<form action="#">
  <div class="form-row">
    <div class="col-md-6 form-group">
      <label for="email1">Email</label>
      <input type="email" class="form-control" id="email1" placeholder="请输入邮箱账号">
    </div>
    <div class="col-md-6 form-group">
      <label for="password">Password</label>
      <input type="password" class="form-control" id="password" placeholder="请输入密码">
    </div>
  </div>
  <div class="form-row">
    <div class="col-md-6 form-group">
      <label for="city">城市</label>
      <input type="text" class="form-control" id="city" placeholder="请输入您所在的城市">
    </div>
    <div class="col-md-4 form-group">
      <label for="marriage">婚姻状况</label>
      <select class="form-control" id="marriage">
        <option>已婚</option>
        <option>未婚</option>
      </select>
    </div>
    <div class="col-md-2 form-group">
      <label for="education">学历</label>
      <select class="form-control" id="education">
        <option>博士</option>
        <option>硕士</option>
        <option>本科</option>
      </select>
    </div>
  </div>
  <button type="submit" class="btn btn-primary">提交</button>
</form>
```

**效果说明**：通过 `.col-md-*` 类实现不同宽度的列布局，`.form-row` 使表单项之间的间距更紧凑。

## 六、水平排列表单

使用 `.form-group.row` + `.col-form-label` 实现标签与输入框水平排列。

```html
<form action="#">
  <div class="form-group row">
    <label for="email2" class="col-sm-2 col-form-label">Email</label>
    <div class="col-sm-10">
      <input type="email" class="form-control" id="email2" placeholder="请输入email">
    </div>
  </div>
  <div class="form-group row">
    <label for="password3" class="col-sm-2 col-form-label">Password</label>
    <div class="col-sm-10">
      <input type="password" class="form-control" id="password3" placeholder="请输入密码">
    </div>
  </div>
  <div class="form-group row">
    <label class="col-sm-2">兴趣爱好：</label>
    <div class="col-sm-10">
      <div class="form-check">
        <input type="checkbox" id="eat" class="form-check-input">
        <label for="eat" class="form-check-label">吃饭</label>
      </div>
    </div>
  </div>
</form>
```

**效果说明**：`.col-form-label` 使 label 标签与输入框垂直居中对齐。

---

## 七、内联表单

使用 `.form-inline` 使所有表单元素在同一行水平排列。

```html
<form action="#" class="form-inline">
  <div class="form-group mr-5">
    <label>用户名：</label>
    <input type="email" class="form-control" value="kaivon@qq.com">
  </div>
  <div class="form-group mr-5">
    <label>密码：</label>
    <input type="password" class="form-control">
  </div>
  <div class="form-group mr-5">
    <input type="checkbox" class="form-control-input">
    <label class="form-control-label">记住我</label>
  </div>
  <div class="form-group mr-5">
    <button type="submit" class="btn btn-primary">提交</button>
  </div>
</form>
```

**效果说明**：所有表单元素排列在同一行中，使用 `mr-5` 设置右边距来分隔各表单组。
