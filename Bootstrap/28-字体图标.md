# 字体图标

## 课程简介

本节课学习如何在 Bootstrap 项目中使用 **字体图标**。主要介绍三种方式：Bootstrap 3 内置的 Glyphicons 图标、阿里巴巴 iconfont 自定义图标（Unicode 引用和 class 引用）、以及 SVG Symbol 引用方式。

---

## 关键知识点

- Bootstrap 3 自带 Glyphicons 图标，Bootstrap 4 已移除
- 阿里巴巴 iconfont（https://www.iconfont.cn/）是常用的图标管理平台
- 图标引用方式有三种：
  1. **Unicode 引用** -- 通过 `&#xe642;` 等编码引用
  2. **Font class 引用** -- 通过 class 名如 `icon-icon-test3` 引用
  3. **SVG Symbol 引用** -- 通过 `<svg><use xlink:href="#icon-xxx"></use></svg>` 引用
- 需要先通过 `@font-face` 定义字体，再通过 `.iconfont` 类应用字体

---

## 关键代码示例

**@font-face 字体声明：**

```css
@font-face {
  font-family: 'iconfont';
  src: url('fonts/iconfont.eot');
  src: url('fonts/iconfont.eot?#iefix') format('embedded-opentype'),
    url('fonts/iconfont.woff2') format('woff2'),
    url('fonts/iconfont.woff') format('woff'),
    url('fonts/iconfont.ttf') format('truetype'),
    url('fonts/iconfont.svg#iconfont') format('svg');
}

.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}
```

**Bootstrap 3 Glyphicons 图标：**

```html
<!-- 需要引入 Bootstrap 3 的 CSS -->
<span class="glyphicon glyphicon-heart" style="font-size: 5rem; color: #f00;"></span>

<button class="btn btn-success" style="font-size: 3rem;">
  <span class="glyphicon glyphicon-star"></span> 陈学辉
</button>
```

**iconfont Unicode 引用方式：**

```html
<!-- 需引入 css/iconfont.css -->
<span class="iconfont" style="font-size: 5rem; color: red;">&#xe642;</span>
<span class="iconfont" style="font-size: 5rem; color: red;">&#xe648;</span>
<span class="iconfont" style="font-size: 5rem; color: red;">&#xe651;</span>
```

**iconfont Font class 引用方式：**

```html
<span class="iconfont icon-icon-test3" style="font-size: 5rem; color: green;"></span>
<span class="iconfont icon-icon-test2" style="font-size: 5rem; color: green;"></span>
<span class="iconfont icon-icon-test1" style="font-size: 5rem; color: green;"></span>
```

**iconfont SVG Symbol 引用方式：**

```html
<!-- 需引入 js/iconfont.js -->
<style>
  .icon {
    width: 1em;
    height: 1em;
    vertical-align: -0.15em;
    fill: currentColor;
    overflow: hidden;
  }
</style>

<svg class="icon" aria-hidden="true" style="font-size: 5rem; color: blue;">
  <use xlink:href="#icon-icon-test"></use>
</svg>
<svg class="icon" aria-hidden="true" style="font-size: 5rem; color: blue;">
  <use xlink:href="#icon-icon-test1"></use>
</svg>
```

---

## 效果说明

- Glyphicons 是 Bootstrap 3 内置图标，直接通过 class 使用
- iconfont 的 Unicode 方式最通用但不够直观
- Font class 方式语义更清晰，推荐使用
- SVG Symbol 方式支持多色图标，是现代推荐的方案
