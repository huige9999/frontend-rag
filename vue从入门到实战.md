## 2. 章节逻辑说明

这是这门体系里**最长、最核心的一门课**，主线是**「学框架概念（01~04）→ 搭工程（05）→ 补关键能力（06~14）→ 用 my-site 博客项目实战落地（15~29）→ 工程化与状态管理进阶（30~34）→ 收尾剩余页面与优化（35~42）」**。一条「先理论后实战、理论在实战中反复巩固」的强主线，贯穿一个真实的个人博客项目。

### 第一段：Vue2 概念入门（01~04）

- **01 前端框架的由来**：为什么要有框架，原生 JS 的痛点。
- **02 第一个vue应用**：跑通第一个 Vue 实例。
- **03 核心概念**：数据响应式、模板、指令（配 vue 知识体系 xmind + 笔记）。
- **04 组件**：组件化思想，组件的注册与使用。

**为什么先讲概念**：建立「数据驱动视图」的心智模型是一切的前提。

### 第二段：搭工程 + 补关键能力（05~14）

- **05 搭建工程**：用脚手架起 `my-site` 项目（项目主线从此开始）。
- **06 计算属性 / 07 组件事件 / 08 优化工程结构 / 09 组件练习**：补齐组件与工程基础。
- **10 插槽 / 11 路由 / 12 弹出消息 / 13 获取远程数据 / 14 组件生命周期**：插槽、vue-router、全局消息、请求接口、生命周期——这几章是写真实项目天天要用的能力。

**为什么这么安排**：概念懂了之后，先搭好工程壳子，再补「写业务必用」的能力（计算属性、插槽、路由、远程数据、生命周期），为后面真正开发页面铺路。

### 第三段：my-site 博客项目实战（15~29）

把前面所有能力用到一个真实博客项目上，按页面推进：

- **15~17 完成首页 part1/2/3**：博客首页全流程。
- **18~20 自定义指令 / 组件混入 / 组件递归**：穿插讲进阶复用机制（递归组件正是后面详情页/菜单要用的）。
- **21~22 开发文章列表页 part1/2**。
- **23~29 开发文章详情页 part1~5**：详情页是组件通信的大练兵，期间穿插 **25 $listeners 和 v-model（组件通信）、27 事件总线（跨组件通信）**——这两章就是为解决详情页开发中遇到的通信问题而插讲的。

**为什么这么安排**：这是「理论 → 实战」的核心段，能力点（自定义指令、组件通信、事件总线）都是「项目做到那儿了才需要、需要了就讲」的按需教学，非常贴近真实开发节奏。

### 第四段：工程化与状态管理进阶（30~34）

- **30 代码优化 / 31 图片懒加载 / 32 数据共享 / 33 vuex经典案例 / 34 在项目中应用vuex**：从「功能实现」走向「工程优化 + 集中状态管理（vuex）」。

**为什么这么安排**：项目主体跑通后，再讲 vuex 这种「跨组件状态管理」才有真实场景支撑，避免空讲。

### 第五段：收尾与优化（35~42）

- **35 页面标题统一处理 / 36 关于我 / 37 项目&效果 / 38 留言板**：完成剩余页面。
- **39 打包结果分析 / 40 异步组件 / 41 无数据显示 / 42 404页面**：发布前的优化与细节收尾。

### 主线一句话总结

**Vue2 概念（01~04）→ 搭工程补能力（05~14）→ my-site 博客项目实战（15~29）→ 优化与 vuex（30~34）→ 收尾页面与打包优化（35~42）**，目标是边学边做一个完整的 Vue2 博客项目，把概念、组件、路由、通信、状态管理、打包优化全链路打通。

## 3. 使用场景等元信息沉淀

- **学完能用在哪**：独立用 Vue2 + vue-router + vuex 从零开发并打包一个多页面 SPA（课程产物 my-site 就是直接可用的个人博客模板）。
- **和实际项目的关系**：my-site 是一套完整的 Vue2 工程脚手式项目，涵盖了组件设计、路由、请求封装、全局消息、状态管理、懒加载、打包优化等真实项目全部要素。
- **和其他课程的关系**：上承《工程化》《网络》《第三方库》（axios）；是《vue 组件库从入门到实战》的前置（后者基于 element-admin 做中后台，需要先会本项目级的 Vue）。Vue2 之后另有独立的《Vue3》课程做版本升级。
- **面试/教学高频场景**：组件通信（含 `$listeners`、事件总线，25/27）、计算属性 vs watch（06）、插槽（10）、生命周期（14）、vuex（33~34）、路由（11）。

## 4. 临时补充

- 这门课最大的特点：**以 my-site 个人博客项目贯穿始终**，从第 05 章搭工程起，几乎所有「课堂代码」都是 my-site 的阶段性提交——复习时直接对着最终项目代码 + 按章节回看即可。
- **按需插讲**是本课节奏的关键：18~20、25、27 这些进阶能力章都是在项目开发到对应场景时才讲的，复习时建议把每章放回它所在的页面开发流程里理解，而不是孤立背诵。
- 详情页（23~29）是组件通信的集中练兵场，`$listeners`（25）和事件总线（27）两章务必和详情页代码对照看。
- 仓库里「解决项目无法搭建问题」是环境排错辅助目录，搭不起来时先去这里查。
- 注意这是 **Vue2** 课（渡一大纲明确把 Vue2 与 Vue3 分开），别和独立的《Vue3》课程混淆。
