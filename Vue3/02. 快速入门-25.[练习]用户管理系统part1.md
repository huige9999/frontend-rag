# [练习]用户管理系统part1

本练习实现了一个用户管理系统的基础版本，综合运用了 Vue3 前端路由（Vue Router）、组件化开发、Props、组件 v-model、插槽（作用域插槽）、Axios 请求封装、生命周期钩子等知识点。

主要功能包括：
- 使用 Vue Router 实现多页面导航（首页、关于我们、添加用户、用户详情）
- 嵌套路由（关于我们页面下包含邮箱和电话子页面）
- 动态路由（用户详情页通过路由参数传递用户 ID）
- 用户列表展示与搜索过滤
- Axios 二次封装和 API 函数封装
- 组件通信（Props、v-model、自定义事件、插槽）

## src/router/index.js

前端路由配置，包含首页、关于我们（嵌套路由）、添加用户、用户详情（动态路由）。

```js
import { createRouter, createWebHistory } from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'
import AddOrEdit from '../views/AddOrEdit.vue'
import Detail from '../views/Detail.vue'
import Email from '../views/Email.vue'
import Tel from '../views/Tel.vue'

const router = createRouter({
  history: createWebHistory(),
  routes: [
    {
      path: '/home',
      name: 'Home',
      component: Home
    },
    {
      path: '/about',
      name: 'About',
      component: About,
      children: [
        {
          path: 'email',
          component: Email
        },
        {
          path: 'tel',
          component: Tel
        },
        {
          path: '',
          redirect: '/about/email'
        }
      ]
    },
    {
      path: '/add',
      name: 'AddOrEdit',
      component: AddOrEdit
    },
    {
      path: '/detail/:id',
      component: Detail
    }
  ]
})
export default router
```

## src/api/request.js

Axios 请求的二次封装，设置了基础 URL、超时时间、请求拦截器和响应拦截器。

```js
import axios from 'axios'

const request = axios.create({
  baseURL: 'http://localhost:3000',
  timeout: 5000
})

request.interceptors.request.use((config) => {
  return config
})

request.interceptors.response.use(
  (response) => {
    return response
  },
  (error) => {
    return Promise.reject(error)
  }
)

export default request
```

## src/views/Home.vue

首页视图，展示用户列表并支持搜索过滤功能。在 onMounted 生命周期中请求用户数据。

```vue
<template>
  <div>
    <h1>用户列表</h1>
    <input
      type="text"
      class="form-control"
      placeholder="搜索内容"
      v-model="searchItem"
      @input="changeHandle"
    />
    <table class="table table-striped table-bordered">
      <thead>
        <tr>
          <th>姓名</th>
          <th>年龄</th>
          <th>联系方式</th>
          <th>操作</th>
        </tr>
      </thead>
      <tbody>
        <tr v-for="item in list" :key="item.id">
          <td>{{ item.name }}</td>
          <td>{{ item.age }}</td>
          <td>{{ item.phone }}</td>
          <td>
            <router-link :to="`/detail/${item.id}`">详情</router-link>
          </td>
        </tr>
      </tbody>
    </table>
  </div>
</template>

<script setup>
import { onMounted, ref, computed } from 'vue'
import { getUserListApi } from '../api/userApi.js'

const userList = ref([])
const searchItem = ref('')
const searchList = ref([])

onMounted(() => {
  getUserListApi().then(({ data }) => {
    userList.value = data
  })
})

function changeHandle() {
  searchList.value = userList.value.filter((item) => {
    return item.name.includes(searchItem.value)
  })
}

const list = computed(() => (searchItem.value ? searchList.value : userList.value))
</script>
```

## src/components/UserCard.vue

用户卡片组件，通过 Props 接收用户数据并展示，包含自定义校验规则。

```vue
<template>
  <div class="user-card">
    <img :src="user.avatarUrl" alt="用户头像" class="avatar" />
    <div class="user-info">
      <h2>{{ user.name }}</h2>
      <p>{{ user.email }}</p>
      <p>{{ age }}</p>
      <p>{{ typeof age }}</p>
    </div>
  </div>
</template>

<script setup>
defineProps({
  user: {
    type: Object,
    required: true,
    validator: (value) => {
      return value.name && value.email && value.avatarUrl
    }
  },
  age: {
    type: [Number, String],
    default: 18
  }
})
</script>
```

## src/components/Rating.vue

星级评分组件，使用 defineModel 实现组件 v-model 双向绑定，支持修饰符自定义 setter。

```vue
<template>
  <div class="rating-container">
    <span v-for="star in 5" :key="star" class="star" @click="setStar(star)">
      {{ model >= star ? '★' : '☆' }}
    </span>
    <input type="text" v-model="model" />
  </div>
</template>

<script setup>
const [model, modifiers] = defineModel({
  required: true,
  set(value) {
    console.log(value)
    if (modifiers.number) {
      if (isNaN(value)) {
        value = 0
      } else {
        value = Number(value)
      }
      if (value < 0) {
        value = 0
      } else if (value > 5) {
        value = 5
      }
      return value
    }
  }
})

function setStar(newStar) {
  model.value = newStar
}
</script>
```
