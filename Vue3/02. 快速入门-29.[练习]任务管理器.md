# [练习]任务管理器

本练习实现了两个项目：一个简单的计数器（counter）和一个任务管理器（task-manager），综合运用了 Pinia 状态管理库和 Vue3 的组件化开发。

### counter 项目
简单的计数器应用，演示 Pinia 状态管理库的基本使用，包括 options 风格和 composition 风格（setup store）两种创建 Store 的方式。

### task-manager 项目
任务管理器应用，综合运用了 Pinia 状态管理、组件化开发、异步操作（模拟 API 请求）等知识点。

主要功能包括：
- 添加新任务（Ctrl+Enter 快捷键）
- 切换任务完成状态
- 删除任务
- 按状态分类展示（待完成/已完成）
- 任务统计（总数和已完成数）
- 加载状态提示（loading spinner）

### src/stores/useCounterStore.js

计数器状态仓库，演示了 Pinia 的 Composition API 风格（Setup Store），使用 ref 定义状态，函数定义 actions。

```js
import { defineStore } from 'pinia'
import { ref } from 'vue'

export const useCounterStore = defineStore('counter', () => {
  const count = ref(0)

  function increment() {
    count.value++
  }

  function decrement() {
    count.value--
  }

  return {
    count,
    increment,
    decrement
  }
})
```

```vue
<template>
  <div class="container">
    <h1>任务管理器</h1>

import { ref, onMounted, computed } from 'vue'

const taskStore = useTaskStore()

const pendingTasks = computed(() => taskStore.pendingTasks)
const taskCount = computed(() => taskStore.taskCount)
const completedTaskCount = computed(() => taskStore.completedTaskCount)

  await taskStore.fetchTasks()
})

}
```

### src/stores/useTaskStore.js

任务状态仓库，使用 Pinia Options 风格定义 state、getters 和 actions。actions 中包含异步操作，模拟与服务端的交互。

```js
import { defineStore } from 'pinia'
import {
  fetchTasksFromServer,
  addTaskToServer,
  deleteTaskFromServer,
  toggleTaskStatusOnServer
} from '../api/taskApi.js'

export const useTaskStore = defineStore('taskStore', {
  state: () => ({
    tasks: [],
    loading: false
  }),
  getters: {
    completedTasks: (state) => state.tasks.filter((task) => task.completed),
    pendingTasks: (state) => state.tasks.filter((task) => !task.completed),
    taskCount: (state) => state.tasks.length,
    completedTaskCount: (state) => state.tasks.filter((task) => task.completed).length
  },
  actions: {
    async fetchTasks() {
      this.loading = true
      const tasks = await fetchTasksFromServer()
      this.tasks = tasks
      this.loading = false
    },
    async addTask(task) {
      this.loading = true
      const newTask = await addTaskToServer(task)
      this.tasks.push(newTask)
      this.loading = false
    },
    async deleteTask(taskId) {
      this.loading = true
      await deleteTaskFromServer(taskId)
      this.tasks = this.tasks.filter((task) => task.id !== taskId)
      this.loading = false
    },
    async toggleTaskStatus(taskId) {
      this.loading = true
      await toggleTaskStatusOnServer(taskId)
      const task = this.tasks.find((task) => task.id === taskId)
      if (task) {
        task.completed = !task.completed
      }
      this.loading = false
    }
  }
})
```

### src/api/taskApi.js

模拟异步 API 请求，使用 Promise 和 setTimeout 模拟服务端延迟。

```js
export const fetchTasksFromServer = () => {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve([
        { id: 1, title: '学习Vue3', completed: false },
        { id: 2, title: '学习Pinia', completed: true }
      ])
    }, 1000)
  })
}

export const addTaskToServer = (task) => {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve({ ...task, id: Date.now() })
    }, 500)
  })
}

export const toggleTaskStatusOnServer = (taskId) => {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve(taskId)
    }, 500)
  })
}

export const deleteTaskFromServer = (taskId) => {
  return new Promise((resolve) => {
    setTimeout(() => {
      resolve(taskId)
    }, 500)
  })
}
```

### src/components/TaskItem.vue

单个任务项组件，展示任务标题，支持点击切换完成状态和删除操作。直接通过 Pinia Store 调用异步 actions。

```vue
<template>
  <li :class="[task.completed ? 'completed' : 'pending']">
    <span @click="toggleStatus">{{ task.title }}</span>
    <button @click="deleteTask">删除</button>
  </li>
</template>

<script setup>
import { useTaskStore } from '../stores/useTaskStore'
const props = defineProps({
  task: {
    type: Object,
    required: true
  }
})
const taskStore = useTaskStore()

async function deleteTask() {
  await taskStore.deleteTask(props.task.id)
}

async function toggleStatus() {
  await taskStore.toggleTaskStatus(props.task.id)
}
</script>
```
