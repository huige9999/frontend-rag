# 微信API—分享功能

本节课程以一个 TodoList 待办事项小程序为例，演示微信小程序的分享功能，包括转发给朋友和分享到朋友圈。核心使用了 `onShareAppMessage` 生命周期和 `wx.showShareMenu` API。

## 核心知识点

- `onShareAppMessage`：监听用户点击页面内转发按钮的事件，自定义转发内容
- `wx.showShareMenu`：显示当前页面的分享按钮
- `button` 组件的 `open-type="share"` 触发分享
- 全局数据管理（`globalData`）在多页面间共享数据

## 全局数据（app.js）

```javascript
// app.js
App({
  globalData: {
    // 存储所有的待办事项
    list : [{
      content : '看电影',
      isComplete : true
    },{
      content : '学习小程序',
      isComplete : false
    }]
  }
})
```

## 首页逻辑（pages/index/index.js）

```javascript
const app = getApp()

Page({
  data: {
    newContent : "", // 用户输入新的待办事项
    list: app.globalData.list // 从全局 list 中获取所有任务
  },
  onLoad(){
    // 开启分享功能
    wx.showShareMenu({
      withShareTicket : true,
      menus : ["shareAppMessage", "shareTimeline"]
    })
  },
  add(){
    // 添加新的待办事项
    if(this.data.newContent){
      app.globalData.list.unshift({
        content : this.data.newContent,
        isComplete: false
      });
      this.setData({
        newContent: '',
        list: app.globalData.list
      });
      wx.showToast({
        title: '新增任务成功',
        icon:'success'
      })
    } else {
      wx.showToast({
        title: '请输入内容',
        icon:'error'
      })
    }
  },
  fresh(){
    this.setData({
      list : app.globalData.list
    })
  },
  onShow(){
    this.fresh();
  },
  // 分享事件处理函数，需要 return 一个 Object
  onShareAppMessage(){
    return {
      title : "这是一个待办事项的小程序",
      path : "/pages/index/index",
      imageUrl : "/imgs/todolist.jpeg"
    }
  }
})
```

## 首页结构（pages/index/index.wxml）

```xml
<view class="inputContainer">

    class="input"

<view class="btnContainer">

</view>
```

## 自定义组件 item（components/item/item.js）

```javascript
const app = getApp();
Component({
  properties: {
    item: Object
  },
  methods: {
    checkHandle() {
      const item = this.properties.item;
      const listItem = app.globalData.list.filter(i => i.content === item.content)[0];
      listItem.isComplete = !listItem.isComplete;
      if (listItem.isComplete) {
        wx.showToast({ title: '标记【完成】', icon: 'success' })
      } else {
        wx.showToast({ title: '标记【未完成】', icon: 'success' })
      }
      this.setData({
        item: { isComplete: !item.isComplete }
      })
      this.triggerEvent('fresh');
    },
    deleteHandle() {
      wx.showModal({
        title: '确认删除',
        content: "是否要删除此项任务？",
        success: res => {
          if (res.confirm) {
            const item = this.properties.item;
            const index = app.globalData.list.findIndex(i => i.content === item.content);
            app.globalData.list.splice(index, 1);
            wx.showToast({ title: '删除成功' });
            this.triggerEvent("fresh");
          }
        }
      })
    }
  }
})
```

## 组件模板（components/item/item.wxml）

```xml
<view class="item">
  <view class="content">
    <checkbox checked="{{item.isComplete}}" class="checkbox" bindtap="checkHandle"></checkbox>
    <text class="{{item.isComplete ? 'finished' : ''}}">{{item.content}}</text>
  </view>
  <image class="delete" src="/imgs/delete.png" bindtap="deleteHandle"></image>
</view>
```

## 小结

本节通过 TodoList 项目演示了微信小程序的分享功能。关键点在于：
1. 使用 `button` 组件的 `open-type="share"` 触发分享
2. 在页面的 `onShareAppMessage` 方法中自定义分享标题、路径和图片
3. 使用 `wx.showShareMenu` 开启分享菜单，支持分享到朋友圈
