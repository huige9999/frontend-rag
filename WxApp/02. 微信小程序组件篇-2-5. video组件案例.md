# video组件案例

本节课通过一个视频播放案例，演示了 `video` 组件的使用，包括弹幕功能、通过 `VideoContext` 控制视频的播放和暂停、发送弹幕等操作。

---

## 页面结构（index.wxml）

```xml
<view class="container">
  <video
    id="myVideo"
    src="http://wxsnsdy.tc.qq.com/105/20210/snsdyvideodownload?filekey=30280201010421301f0201690402534804102ca905ce620b1241b726bc41dcff44e00204012882540400&bizid=1023&hy=SH&fileparam=302c020101042530230204136ffd93020457e3c4ff02024ef202031e8d7f02030f42400204045a320a0201000400"
    danmu-list="{{damuList}}"
    danmu-btn="{{true}}"
  />
</view>
<!-- 发送弹幕 -->
<view class="inputContainer">
  <input
    type="text"
    model:value="{{newDanmu}}"
    bindinput="inputHandle"
  />
</view>
<button bindtap="sendDanmu">发送弹幕</button>
<button bindtap="playHandle">播放</button>
<button bindtap="pauseHandle">暂停</button>
```

### 要点说明

- **video** 组件核心属性：
  - `id`：视频的唯一标识，用于通过 `wx.createVideoContext` 获取视频上下文
  - `src`：视频资源的地址
  - `danmu-list`：弹幕列表，传入一个数组，每项包含 `text`（弹幕文字）、`time`（出现时间，单位秒）、`color`（弹幕颜色）
  - `danmu-btn`：是否显示弹幕按钮，让用户控制弹幕的显示/隐藏
- `model:value` 是双向绑定语法，将输入框的值与 `newDanmu` 数据双向绑定
- 三个按钮分别对应：发送弹幕、播放、暂停

## 页面逻辑（index.js）

```javascript
function getRandomColor () {
  const rgb = []
  for (let i = 0 ; i < 3; ++i){
    let color = Math.floor(Math.random() * 256).toString(16)
    color = color.length == 1 ? '0' + color : color
    rgb.push(color)
  }
  return '#' + rgb.join('')
}

Page({
  data: {
    damuList : [{
      text : "第1s出现的弹幕",
      time : 1,
      color : "white"
    },{
      text : "asasasad",
      time : 3,
      color : "white"
    }],
    newDanmu : "", // 用户输入的新弹幕的内容
  },
  inputHandle(){},
  // 无论是发送弹幕，还是控制整个视频的播放或者暂停
  // 我们都需要拿到这个 video 的上下文对象
  // 之后调用上下文对象的方法来进行相应的操作
  onReady(){
    this.context = wx.createVideoContext('myVideo');
  },
  // 发送弹幕
  sendDanmu(){
    this.context.sendDanmu({
      text : this.data.newDanmu,
      color : getRandomColor()
    })
  },
  // 播放视频
  playHandle(){
    this.context.play();
  },
  // 暂停视频
  pauseHandle(){
    this.context.pause();
  }
})
```

### 要点说明

- **getRandomColor**：生成随机颜色的辅助函数，将 RGB 各通道转为16进制拼接成 `#RRGGBB` 格式
- **damuList**：预设的弹幕数据，每条弹幕包含 `text`（文字）、`time`（出现秒数）、`color`（颜色）
- **onReady**：页面初次渲染完成时，通过 `wx.createVideoContext('myVideo')` 获取视频上下文对象
- **VideoContext** 的常用方法：
  - `sendDanmu({text, color})`：发送弹幕
  - `play()`：播放视频
  - `pause()`：暂停视频
