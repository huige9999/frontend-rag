# 微信API—动画案例

本节课程演示微信小程序的动画 API（`wx.createAnimation`），通过创建动画对象并调用动画方法来实现元素的动画效果。案例包含一个基础的位移动画和一个城市交换动画。

## 核心知识点

- `wx.createAnimation`：创建一个动画实例
- 动画配置：`duration`（持续时间）、`timingFunction`（动画效果）
- 动画方法：`translateX`、`opacity`、`width` 等
- `step()`：表示一组动画完成
- `export()`：导出动画队列，绑定到元素的 `animation` 属性

## 页面逻辑（index.js）

```javascript
Page({
  data: {
    viewAnimate: "", // 保存一系列的动画队列
    start: "北京",   // 起始地
    end: "上海",     // 目的地
    lAnimate: "",    // 存储左边的动画序列
    rAnimate: ""     // 存储右边的动画序列
  },
  // 基础动画示例
  startMove() {
    // 1. 创建一个动画对象
    const animationObj = wx.createAnimation({
      duration: 3000,
      timingFunction: 'ease'
    });
    // 2. 安排动画，step() 表示一组动画结束
    animationObj.translateX(100).step().opacity(0.5).width(200).step();
    // 3. 导出动画队列并设置到数据中
    this.setData({
      viewAnimate: animationObj.export()
    })
  },
  // 城市交换动画
  trigger() {
    const option = {
      duration: 100,
      timingFunction: 'ease-in'
    };
    const lanimation = wx.createAnimation(option);
    const ranimation = wx.createAnimation(option);
    // 起始地动画：向右移动并透明
    lanimation.translateX(100).opacity(.1).step();
    // 目的地动画：向左移动并透明
    ranimation.translateX(-100).opacity(.1).step();
    this.setData({
      lAnimate: lanimation.export(),
      rAnimate: ranimation.export(),
    });
    // 100ms后将两个盒子移回原位，并交换文字
    setTimeout(() => {
      lanimation.translateX(0).opacity(1).step();
      ranimation.translateX(0).opacity(1).step();
      const temp = this.data.start;
      this.setData({
        lAnimate: lanimation.export(),
        rAnimate: ranimation.export(),
        end : temp,
        start : this.data.end,
      });
    }, 100)
  }
})
```

## 页面结构（index.wxml）

```xml
<view>小程序动画快速入门示例</view>
<view animation="{{viewAnimate}}" class="container1"></view>
<button type="primary" bindtap="startMove">开始动画</button>

<view>动画实例</view>
<view class="animation">
  <view class="dis-flex">
    <!-- 起始地 -->
    <view class='flex3' animation="{{lAnimate}}">{{start}}</view>
    <!-- 交换按钮 -->
    <view class='flex1' bindtap="trigger">
      <em class='iconfont switch-icon'></em>
    </view>
    <!-- 目的地 -->
    <view class='flex3' animation="{{rAnimate}}">{{end}}</view>
  </view>
</view>
```

## 小结

本节介绍了微信小程序动画 API 的使用方法。关键流程为：
1. 使用 `wx.createAnimation` 创建动画对象，配置 `duration` 和 `timingFunction`
2. 调用动画方法（如 `translateX`、`opacity`、`width`）安排动画
3. 调用 `step()` 标记一组动画结束，之后可以继续安排下一组动画
4. 调用 `export()` 导出动画队列，赋值给元素的 `animation` 属性
5. 在 `setTimeout` 中可以实现连续的动画序列效果
