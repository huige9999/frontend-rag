# 微信API—指南针案例

本节课程使用微信小程序的罗盘 API（`wx.onCompassChange`）实现一个手机指南针应用。通过监听设备方向变化，实时更新罗盘指针的方向和角度。

## 核心知识点

- `wx.onCompassChange`：监听罗盘数据变化事件
- 罗盘返回的 `direction` 字段表示设备面对的方向角度（0-360）
- CSS `transform: rotate()` 实现罗盘图片旋转
- 根据角度判断方向（东南西北等）

## 指南针页面逻辑（pages/compass/compass.js）

```javascript
Page({
  data: {
    direction : "--", // 方向 【西北】【西南】
    angle : "--",     // 角度 【269.72】
    rotate : "",      // 罗盘旋转的角度
  },
  onLoad(){
    // 监听罗盘变化
    wx.onCompassChange((res) => {
      // res ---> {accuracy: "unreliable", direction: 202.69806}
      let newAngle = res.direction.toFixed(2); // 保留两位小数
      let radios = res.direction.toFixed(0);   // 取整数
      this.setData({
        angle : newAngle,
        direction : this.check(radios),
        rotate: 360 - radios
      })
    });
    // 3秒后检测是否有罗盘
    setTimeout(()=>{
      if(this.data.direction === "--" && this.data.angle === "--"){
        wx.showToast({
          title: '没有电子罗盘',
          icon : "loading",
          duration : 3000,
          mask : true
        })
      }
    },3000);
  },
  // 根据角度来判断方向
  check(i){
    if (15 <= i && i <= 75) {
      return '东北'
    } else if (75 < i && i < 105) {
      return '正东'
    } else if (105 <= i && i <= 165) {
      return '东南'
    } else if (165 < i && i < 195) {
      return '正南'
    } else if (195 <= i && i <= 255) {
      return '西南'
    } else if (255 < i && i < 285) {
      return '正西'
    } else if (285 <= i && i <= 345) {
      return '西北'
    } else {
      return '正北'
    }
  }
})
```

## 小结

本节通过罗盘 API 实现了一个指南针应用。关键点包括：
1. `wx.onCompassChange` 是一个持续监听的事件，设备方向变化时会不断触发回调
2. `direction` 返回的角度范围是 0-360，根据角度区间可以判断出东南西北等方向
3. 使用 CSS `transform: rotate()` 配合动态数据实现罗盘图片的实时旋转效果
4. 注意部分设备可能没有电子罗盘，需要做兼容处理
