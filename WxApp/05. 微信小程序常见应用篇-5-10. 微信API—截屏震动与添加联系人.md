# 微信API—截屏震动与添加联系人

本节课程演示了多个微信小程序设备相关的 API，包括截屏监听、设备振动、屏幕常亮以及添加手机联系人。

## 核心知识点

- `wx.onUserCaptureScreen`：监听用户截屏事件
- `wx.vibrateLong` / `wx.vibrateShort`：触发设备长/短振动
- `wx.setKeepScreenOn`：设置屏幕是否常亮
- `wx.addPhoneContact`：添加手机通讯录联系人

## 页面逻辑（index.js）

```javascript
Page({
  data: {
    userCaptureScreen : "用户没有截屏"
  },
  onShow(){
    // 监听用户截屏事件
    wx.onUserCaptureScreen(() => {
      this.setData({
        userCaptureScreen : "截屏了！！！"
      })
    })
  },
  // 长震动
  virateLong(){
    wx.vibrateLong({})
  },
  // 短震动
  vibrateShort(){
    wx.vibrateShort({
      type : "heavy"
    })
  },
  // 屏幕常亮
  keepScreenOn(){
    wx.setKeepScreenOn({
      keepScreenOn: true,
    })
  },
  // 添加手机联系人
  addPhoneContact(){
    wx.addPhoneContact({
      firstName: '张三',
      nickName: 'zangsan',
      remark: '程序员',
      mobilePhoneNumber: '13938271434',
      email: '154543234@qq.com'
    })
  }
})
```

## 小结

本节介绍了四个设备交互相关的 API：
1. `wx.onUserCaptureScreen`：可以监听用户的截屏行为，适合用于截屏提醒或记录
2. `wx.vibrateLong` / `wx.vibrateShort`：触发设备振动，`vibrateShort` 支持 `type` 参数（heavy/medium/light）
3. `wx.setKeepScreenOn`：防止屏幕自动息屏，适用于阅读、视频等场景
4. `wx.addPhoneContact`：调用系统通讯录，可以直接向手机添加联系人信息
