# 综合案例—扫码综合演示

本节课程是一个扫码综合案例，包含三个主要功能：扫码识别、二维码生成、扫描历史记录。通过 `wx.scanCode`、`wx.downloadFile`、`wx.getStorage`/`wx.setStorage` 等 API 综合运用，实现一个完整的扫码工具应用。

## 核心知识点

- `wx.scanCode`：扫码识别，获取扫码类型和内容
- `wx.downloadFile`：下载文件（用于生成二维码图片）
- `wx.getStorage` / `wx.setStorage` / `wx.getStorageSync`：本地缓存管理
- `wx.setClipboardData`：复制内容到剪贴板
- WeUI 样式库的使用

### 1. 扫码页面（pages/index/index.js）

扫码后保存结果到本地存储，支持复制扫码内容。

```javascript
// 扫描类型和中文意思的映射
const scanType = {
  'WX_CODE': '微信小程序',
  'QR_CODE': '二维码',
  'EAN_8': '条形码（EAN_8）',
  'EAN_13': '条形码（EAN_13）',
  'UPC_A': '条形码（UPC_A）',
  'UPC_E': '条形码（UPC_E）',
  'CODE_25': '条形码（CODE_25）',
  'CODE_39': '条形码（CODE_39）',
  'CODE_128': '条形码（CODE_128）',
};

Page({
  data: {
    scanResult : {
      isShow : false,
      type : "",
      text : ""
    }
  },
  // 扫描事件处理
  onScan(){
    wx.scanCode({
      success : res => {
        this.setData({
          scanResult : {
            isShow : true,
            type : scanType[res.scanType],
            text : res.result
          }
        });
        // 将扫描结果存储到本地
        if(this.data.scanResult.text !== ""){
          wx.getStorage({
            key : "scanLogs",
            complete : res => {
              let scanLogs = res.data?.length > 0 ? res.data : [];
              this.data.scanResult.date = Date.now();
              scanLogs.unshift(this.data.scanResult);
              wx.setStorageSync('scanLogs', scanLogs);
            }
          })
        }
      }
    })
  },
  // 复制扫码内容
  onCopy(){
    wx.setClipboardData({
      data: this.data.scanResult.text,
    })
  }
})
```

          imgTempFilePath : res.tempFilePath

    })

})

### 3. 历史记录页面（pages/logs/logs.js）

从本地存储读取扫描历史，展示列表并可点击查看详情。

```javascript
const {formatTime} = require("../../utils/util");
Page({
  data: {
    scanLogs : []
  },
  onShow(){
    wx.getStorage({
      key : "scanLogs",
      success : res => {
        this.setData({
          scanLogs : (res.data || []).map(n => {
            n.date = formatTime(new Date(n.date));
            n.text = n.text.length > 10 ? n.text.slice(0,10) + "..." : n.text;
            return n;
          })
        })
      }
    })
  }
})
```

### 4. 详情页面（pages/detail/detail.js）

查看单条扫描记录的详情，支持删除操作。

```javascript
Page({
  data: {
    id : null,
    scanResult : {}
  },
  onLoad(options){
    // 从缓存中获取对应 id 的扫描内容
    wx.getStorage({
      key : "scanLogs",
      success : res => {
        this.setData({
          scanResult : res.data[options.id],
          id : options.id
        })
      }
    })
  },
  onDelete(){
    wx.getStorage({
      key : "scanLogs",
      success : res => {
        let scanLogs = res.data;
        scanLogs.splice(this.data.id, 1);
        wx.setStorageSync('scanLogs', scanLogs);
        wx.navigateBack();
      }
    })
  }
})
```

## 小结

本节综合案例串联了扫码、文件下载、本地存储、剪贴板等多个 API，实现了一个完整的扫码工具应用。关键点包括：
1. `wx.scanCode` 返回的 `scanType` 可判断扫码类型（二维码、条形码等）
2. 使用本地存储（`wx.setStorageSync`）持久化扫描历史
3. 通过 `wx.downloadFile` 配合第三方 API 实现二维码生成
4. `wx.setClipboardData` 实现内容复制功能
