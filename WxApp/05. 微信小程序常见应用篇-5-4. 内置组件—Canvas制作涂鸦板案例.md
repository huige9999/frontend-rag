# 内置组件—Canvas制作涂鸦板案例

本节课程使用微信小程序的 Canvas 内置组件，实现一个简易的涂鸦板应用。通过监听触摸事件，实现在画布上自由绘画的功能，支持切换画笔粗细、颜色以及橡皮擦。

## 核心知识点

- Canvas 2D 新版接口的使用（`type="2d"`）
- `wx.createSelectorQuery` 获取 Canvas 节点
- 触摸事件：`bindtouchstart`、`bindtouchmove`、`bindtouchend`
- Canvas 绑定上下文的绘图方法：`moveTo`、`lineTo`、`stroke`
- 设备像素比 `dpr` 的处理，防止画布被拉伸

## 页面逻辑（index.js）

```javascript
Page({
  data: {
    ctx: "", // 保存 canvas 上下文
    pen : 5, // 画笔默认的宽度
    color : "#000", // 画笔默认的颜色
  },
  startX : 0, // 保存 X 坐标
  startY : 0, // 保存 Y 坐标
  onLoad() {
    // 获取到 canvas 的上下文
    const query = wx.createSelectorQuery();
    query.select("#myCanvas")
      .fields({
        node: true,
        size: true
      })
      .exec(res => {
        const canvas = res[0].node;
        const ctx = canvas.getContext('2d');

        // 关键步骤：新版 canvas 需要显式设置画布的宽高
        // 根据设备像素比设置，否则画布会被拉伸
        const dpr = wx.getSystemInfoSync().pixelRatio
        canvas.width = res[0].width * dpr
        canvas.height = res[0].height * dpr
        ctx.scale(dpr, dpr);

        this.setData({ ctx });
      })
  },
  // 手指按下
  touchstart(e) {
    this.startX = e.changedTouches[0].x;
    this.startY = e.changedTouches[0].y;
    // 绘制准备工作
    this.data.ctx.lineWidth = this.data.pen;   // 笔粗细
    this.data.ctx.strokeStyle = this.data.color; // 画笔颜色
    this.data.ctx.lineCap = 'round';            // 设置笔头为圆形
    this.data.ctx.beginPath();
  },
  // 手指移动
  touchmove(e) {
    var startX1 = e.changedTouches[0].x;
    var startY1 = e.changedTouches[0].y;
    // 将画笔移动到手指按下的地方
    this.data.ctx.moveTo(this.startX, this.startY);
    // 根据实时坐标进行描线
    this.data.ctx.lineTo(startX1, startY1);
    this.data.ctx.stroke();
    // 更新起始坐标
    this.startX = startX1;
    this.startY = startY1;
  },
  // 手指抬起
  touchend() {
    this.data.ctx.closePath();
  },
  // 选择画笔的粗细
  penSelect(e){
    if(this.data.color === "#ddd"){
      // 从橡皮擦切换过来的，需要还原颜色
      this.setData({
        pen : parseInt(e.currentTarget.dataset.param),
        color : "#000"
      })
    } else {
      this.setData({
        pen : parseInt(e.currentTarget.dataset.param),
      })
    }
  },
  // 设置颜色
  colorSelect(e){
    this.setData({
      color : e.currentTarget.dataset.param
    })
  },
  // 橡皮擦（实际上就是用背景色覆盖）
  clearCanvas(){
    this.setData({
      color : '#ddd',
      pen : 20
    })
  }
})
```

## 页面结构（index.wxml）

```xml
<!-- 最外层的容器 -->
<view class="container">
  <!-- 画布区域 -->
  <canvas type="2d" id="myCanvas" class="canvas-area" disable-scroll="false"
    bindtouchstart="touchstart"
    bindtouchmove="touchmove"
    bindtouchend="touchend">
  </canvas>
  <!-- 画布工具区域 -->
  <view class='canvas-tools'>
    <!-- 细笔绘制 -->
    <view class='box box1' bindtap='penSelect' data-param='5'></view>
    <!-- 粗笔绘制 -->
    <view class='box box2' bindtap='penSelect' data-param='15'></view>
    <!-- 红色笔 -->
    <view class='box box3' bindtap='colorSelect' data-param='#c03'></view>
    <!-- 黄色笔 -->
    <view class='box box4' bindtap='colorSelect' data-param='#f90'></view>
    <!-- 橡皮 -->
    <view class='box box5' bindtap='clearCanvas'></view>
  </view>
</view>
```

## 小结

本节通过 Canvas 组件实现了一个涂鸦板。关键技术点包括：
1. 使用新版 Canvas 2D 接口，通过 `SelectorQuery` 获取 Canvas 节点
2. 必须根据设备像素比（`pixelRatio`）设置画布实际宽高，防止画面拉伸
3. 通过 `touchstart`、`touchmove`、`touchend` 三个触摸事件配合 `moveTo/lineTo/stroke` 实现自由绘画
4. 橡皮擦的本质是用背景色（与画布背景相同）进行覆盖绘制
