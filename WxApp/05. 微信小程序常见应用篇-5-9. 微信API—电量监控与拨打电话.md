# 微信API—电量监控与拨打电话

本节课程演示两个实用的微信 API：获取设备电量信息（`wx.getBatteryInfo`）和拨打电话（`wx.makePhoneCall`）。

## 核心知识点

- `wx.getBatteryInfo`：获取设备电量信息，包括电量值和是否正在充电
- `wx.makePhoneCall`：拨打电话
- `<progress>` 组件：展示电量进度条

## 页面逻辑（index.js）

```javascript
Page({
  data: {
    level : "",       // 电量值
    isCharging : "",  // 是否在充电
    phone : "",       // 电话号码
  },
  onShow(){
    // 每次页面显示时获取最新的电量信息
    wx.getBatteryInfo({
      success : res => {
        console.log(res);
        this.setData({
          level : res.level,
          isCharging : res.isCharging
        })
      }
    })
  },
  callPhone(){
    // 拨打电话
    if(this.data.phone){
      wx.makePhoneCall({
        phoneNumber : this.data.phone
      })
    } else {
      wx.showModal({
        title: '提示',
        content: '电话号码没有填写',
        showCancel : false
      })
    }
  }
})
```

## 页面结构（index.wxml）

```xml
<view class='container'>
  <!-- 电量监控 -->
  <view class='page-section'>
    <view class='page-section-title'>电量监控</view>
    <view class='page-section-area'>
      <view>当前电量值：{{level}}</view>
      <view>是否在充电：{{isCharging}}</view>
      <progress percent='{{level}}' show-info active></progress>
    </view>
  </view>
  <!-- 拨打电话 -->
  <view class='page-section'>
    <view class='page-section-title'>系统电话</view>
    <view class='page-section-area'>
      <form bindsubmit='formCallPhone'>
        <input placeholder='电话号码' name='phoneNumber' model:value="{{phone}}"></input>
        <button bindtap='callPhone' form-type='submit'>拨打电话</button>
      </form>
    </view>
  </view>
</view>
```

## 小结

本节介绍了两个实用的设备相关 API：
1. `wx.getBatteryInfo` 获取设备电量，返回 `level`（电量百分比）和 `isCharging`（是否在充电），配合 `<progress>` 组件可以直观展示电量
2. `wx.makePhoneCall` 可以直接拨打指定号码，调用后会弹出系统拨号确认框
