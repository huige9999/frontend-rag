# 轮播图案例

本节课通过一个轮播图案例，演示了微信小程序中 `swiper` 和 `swiper-item` 组件的使用，实现了带指示器、前后边距、自动切换的轮播效果，并通过缩放动画突出当前展示的图片。

---

## 页面结构（index.wxml）

```xml
<swiper
  class="container"
  indicator-dots
  indicator-color="#a39f99"
  indicator-active-color="#f49641"
  interval="3000"
  previous-margin="50rpx"
  next-margin="50rpx"
  bindchange="swiperchange"
  circular
>
  <block wx:for="{{imgUrls}}" wx:key="this">
    <swiper-item>
      <image src="{{item}}" class="image {{swiperIndex === index ? 'active' : 'quiet'}}" mode="aspectFill"></image>
    </swiper-item>
  </block>
</swiper>
```

### 要点说明

- **swiper**：轮播容器组件，常用属性如下：
  - `indicator-dots`：是否显示面板指示点
  - `indicator-color` / `indicator-active-color`：指示点颜色和当前选中颜色
  - `interval`：自动切换时间间隔（毫秒）
  - `previous-margin` / `next-margin`：前边距和后边距，露出前后图片的一部分，形成"卡片"效果
  - `circular`：是否采用衔接滑动（无限循环）
  - `bindchange`：current 改变时触发 change 事件
- **swiper-item**：仅可放置在 swiper 中，代表每一项
- 使用 `wx:for` 遍历图片数组，根据当前轮播下标 `swiperIndex` 动态切换 `active` / `quiet` 样式类，实现缩放动画

## 页面逻辑（index.js）

```javascript
Page({
  data: {
    imgUrls: [
      'https://file.moyublog.com/d/file/2021-02-17/e50db8f1b1ab841c863a49250b79d367.jpg',
      'https://img.tt98.com/d/file/pic/2018081420174325/5b63b5ccdad61.jpg',
      'https://img2.baidu.com/it/u=2344850090,2589644779&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=889',
      'https://img1.baidu.com/it/u=536270855,1477431967&fm=253&fmt=auto&app=138&f=JPEG?w=500&h=889',
    ],
    swiperIndex : 0, // 轮播的下标
  },
  swiperchange(e){
    this.setData({
      swiperIndex : e.detail.current
    })
  }
})
```

### 要点说明

- `imgUrls`：存储轮播图片的远程 URL 数组
- `swiperIndex`：记录当前轮播的下标，初始为 0
- `swiperchange`：监听轮播变化事件，通过 `e.detail.current` 获取当前索引并更新数据，从而驱动视图层的样式切换

## 页面样式（index.wxss）

```css
.container{
  height: 1100rpx;
}

.image{
  width: 100%;
  height: 90%;
  border-radius: 15rpx;
}

/* 当前图片 */
image.active{
  transform: none;
  transition: all .2s ease-in;
}

/* 不是当前图片 */
image.quiet{
  transform: scale(0.9);
  transition: all .2s ease-in;
}
```

### 要点说明

- 当前图片（`active`）保持原始大小，非当前图片（`quiet`）缩小到 0.9 倍，形成视觉上的"聚焦"效果
- 使用 `transition` 实现平滑的缩放过渡动画
- `border-radius` 给图片添加圆角效果
