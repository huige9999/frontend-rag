## 📦 技术栈

- **前端框架**：Vue 3 + TypeScript
- **国际化库**：vue-i18n@next（v9）
- **构建工具**：Vite + @intlify/unplugin-vue-i18n
- **UI 框架**：Element Plus
- **后端**：Node.js + Express + MongoDB
- **核心 API**：Intl（PluralRules / DateTimeFormat / NumberFormat）
