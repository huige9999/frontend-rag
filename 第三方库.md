## 2. 章节逻辑说明

这门课是**「前端常用第三方库速览」**，没有强理论主线，而是按「前端开发里最常碰到的几类场景」逐个过一遍对应的明星库：**DOM 操作 → 工具函数 → CSS 动画 → 网络请求 → 接口模拟 → 日期处理 → 图表可视化**。每个库解决一类问题，彼此相对独立。

- **01 jQuery**：让 DOM 操作更容易。是「数据驱动框架」出现前的事实标准，理解它有助于看老项目和理解「操作 DOM」本身。
- **02 Lodash**：你能想到的工具函数它基本都写了（防抖节流、深拷贝、集合操作等）。
- **03 Animate.css**：常见 CSS 动画效果现成集合，引入即用。
- **04 Axios**：让网络请求更简单——是《网络》里原生 AJAX 的工程化替代，现代项目几乎标配。
- **05 MockJS**：拦截 ajax + 生成模拟数据，后端接口没好的时候前端自己造数据。
- **06 Moment**：让日期处理更容易（格式化、计算、比较）。
- **07 ECharts**：搞定所有你能想到的图表，数据可视化首选。

**为什么这个顺序**：从最基础的 DOM（jQuery）→ 通用工具（Lodash）→ 表现层（Animate.css）→ 数据交互（Axios/MockJS）→ 数据处理（Moment）→ 可视化（ECharts），大致是从「页面本身」往「页面背后的数据与展示」延展，每个库都是一个能立刻落地的工具。

### 主线一句话总结

**按前端常见场景逐个掌握对应明星库（DOM/工具/动画/请求/模拟/日期/图表）**，目标是建立「遇到某类问题知道用哪个库、怎么用 CDN 引入」的工具箱意识。

## 3. 使用场景等元信息沉淀

- **学完能用在哪**：日常开发按场景选库、用 CDN 快速引入、用 Axios 发请求、用 MockJS 造数据、用 ECharts 画图、用 Moment/Lodash 处理数据与日期。
- **和实际项目的关系**：Axios + MockJS + ECharts + Lodash 几乎是中后台项目的标准组合；jQuery 主要用于维护老项目。
- **和其他课程的关系**：Axios 是《网络》的工程化升级；ECharts 与独立的《ECharts 详细版》互补，本课偏入门；Moment/Lodash 等也在渡一的《JS 工具库》课程里有更深讲解。
- **面试/教学高频场景**：Axios 封装、ECharts 基础图表、Lodash 防抖节流/深拷贝是高频；jQuery 现多出现在老项目维护语境。

## 4. 临时补充

- 第 01 章的「常用第三方工具库.md」是一份很好的**全行业前端库速查表**，建议单独收藏——里面除了本课讲的 7 个，还列了 animejs、validate、date-fns、zepto、nprogress、qs 等进阶库。
- 这些库都可以通过 CDN（如 bootcdn）在线使用，课件专门科普了 CDN 概念，复习时一并记住。
- 第三方库更新很快（如 Moment 已官方推荐迁移到 date-fns/day.js），用的时候记得查最新情况，本课讲的是「这个库解决什么问题、怎么用」的通用思路。
