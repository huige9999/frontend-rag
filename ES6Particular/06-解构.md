# 第6章 解构

## 6-1. 对象解构

### 什么是解构

使用ES6的一种语法规则，将一个对象或数组的某个属性提取到某个变量中。

**重要特性：解构不会对被解构的目标造成任何影响**

### 基本对象解构

```js
const user = {
    name: "kevin",
    age: 11,
    sex: "男",
    address: {
        province: "四川",
        city: "成都"
    }
}

// 传统方式（注释）：
// let name, age, sex, address;
// name = user.name;
// age = user.age;
// sex = user.sex;
// address = user.address;

// 使用解构：先定义变量，然后从对象中读取同名属性赋值
let { name, age, sex, address, abc = 123 } = user

console.log(name, age, sex, address, abc)
// 输出: kevin 11 男 四川,成都 { province: '四川', city: '成都' } 123
```

### 在解构中使用默认值

```js
// 语法：{同名变量 = 默认值}
let { name, age, sex, address, abc = 123 } = user
```

当对象中不存在某个属性时，可以使用默认值。如果属性存在，则使用实际值；如果不存在，则使用默认值。

### 非同名属性解构

```js
const user = {
    name: "kevin",
    age: 11,
    sex: "男",
    address: {
        province: "四川",
        city: "成都"
    }
}

// 语法：{属性名:变量名}
// 从对象中读取sex属性，但赋值给gender变量，并设置默认值
let { name, age, sex: gender = 123, address } = user

console.log(name, age, gender, address)
// 输出: kevin 11 男 { province: '四川', city: '成都' }
```

### 嵌套解构

```js
const user = {
    name: "kevin",
    age: 11,
    sex: "男",
    address: {
        province: "四川",
        city: "成都"
    }
}

// 解构出user中的name和嵌套在address中的province
const { name, address: { province } } = user;

console.log(name, province)
// 输出: kevin 四川
```

### 对象剩余属性解构

```js
const user = {
    name: "kevin",
    age: 11,
    sex: "男",
    address: {
        province: "四川",
        city: "成都"
    }
}

// 解构出name，然后将剩余的所有属性放到一个新的对象中，变量名为obj
const { name, ...obj } = user;

console.log(name, obj)
// 输出: kevin { age: 11, sex: '男', address: { province: '四川', city: '成都' } }
```

### 对已声明变量进行解构

```js
const user = {
    name: "kevin",
    age: 11,
    sex: "男",
    address: {
        province: "四川",
        city: "成都"
    }
}

let name, age, sex, address;
// 注意：必须用圆括号包裹解构表达式
({ name, age, sex, address } = user);

console.log(name, age, sex, address)
```

## 6-2. 数组解构

### 基本数组解构

```js
const numbers = ["a", "b", "c", "d"];

// 传统方式（注释）：
// const {
//     0: n1,
//     1: n2
// } = numbers;

// 或先声明再解构：
// let n1, n2;
// ([n1, n2] = numbers);

// 直接解构：按位置对应
const [n1, n2] = numbers;

console.log(n1, n2)
// 输出: a b
```

### 跳过元素和默认值

```js
const numbers = ["a", "b", "c", "d"];

// 解构第一个和第四个元素，跳过中间的，n5使用默认值
const [n1, , , n4, n5 = 123] = numbers;

console.log(n1, n4, n5)
// 输出: a d 123
```

### 嵌套数组解构

```js
const numbers = ["a", "b", "c", "d", [1, 2, 3, 4]];

// 得到numbers下标为4的数组中的下标为2的数据，放到变量n中
const [, , , , [, , n]] = numbers;

console.log(n)
// 输出: 3
```

### 数组中包含对象的解构

```js
const numbers = ["a", "b", "c", "d", {
    a: 1,
    b: 2
}];

// 得到numbers下标为4的对象的属性a，赋值给变量A
// 方式1：直接在数组解构中解构对象
// const [, , , , { a: A }] = numbers;

// 方式2：先获取对象，再解构
const { a: A } = numbers[4];

console.log(A)
// 输出: 1
```

### 数组剩余元素解构

```js
const numbers = [324, 7, 23, 5, 3243];

// 得到数组前两项，分别放到变量a和b中，然后剩余的所有数据放到数组nums
const [a, b, ...nums] = numbers;

// 等价于传统方式：
// const a = numbers[0], b = numbers[1], nums = numbers.slice(2);

console.log(a, b, nums);
// 输出: 324 7 [ 23, 5, 3243 ]
```

### 交换变量值

```js
let a = 1, b = 2;

// 使用数组解构交换变量值
[b, a] = [a, b]

console.log(a, b)
// 输出: 2 1
```

## 解构的应用场景

1. **从对象或数组中提取需要的数据**，避免重复写对象属性访问
2. **函数参数解构**，使函数调用更清晰（见6-3章节）
3. **交换变量值**，无需临时变量
4. **处理函数返回值**，直接提取需要的部分
5. **设置默认值**，确保变量总有值
6. **剩余属性/元素收集**，方便处理剩余数据

解构是ES6中非常实用的特性，能够显著提升代码的可读性和简洁性。
