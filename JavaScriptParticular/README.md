## 🎯 学习建议

1. **前置知识**：需具备 HTML & CSS 基础
2. **学习周期**：每天8小时，约一个月完成全部课程（含练习和项目）
3. **课程版本**：基于 ES5（2009）标准，不含 ES6（2015）内容
4. **课程难度**：由浅入深，0→1，有难度但循序渐进
5. **学习方法**：细化目标、重视练习、编程属于工科，重视实践
