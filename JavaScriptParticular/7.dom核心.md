## 一、Web API 概述

### 1. 基本概念

- **标准库**：ECMAScript 中的对象和函数
- **Web API**：浏览器宿主环境中的对象和函数

Web API 的特点：

1. 知识繁杂
2. 成体系的知识
3. 程序思维：知识 + 程序思维 = 应用
4. 兼容性：了解，不记忆

Web API 分为两大类：

- **BOM**（Browser Object Model）：浏览器对象模型，控制浏览器本身
- **DOM**（Document Object Model）：文档对象模型，控制 HTML 文档

> ES 由 ECMAScript 规定，Web API 由 W3C（万维网联盟）制定

### 2. DOM 简介

DOM 版本发展：DOM 0 -> DOM 1 -> DOM 2 -> DOM 3 -> DOM 4（2015年）

**DOM 的核心理念**：将一个 HTML 或 XML 文档用对象模型表示，每个对象称之为 DOM 对象（又称为节点 Node）。

**节点类型**：

| 节点类型 | 说明 |
|---------|------|
| DocumentType | 文档类型节点 |
| Document | 文档节点，表示整个文档 |
| Comment | 注释节点 |
| Element | 元素节点 |
| Text | 文本节点 |
| Attribute | 属性节点 |
| DocumentFragment | 文档片段节点 |

**DOM 树**：文档中不同的节点形成的树形结构。

### 3. 示例：HTML 文档的 DOM 结构

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>标题</title>
</head>
<body>
    <!-- 注释 -->
    <p>lorem</p>
</body>
</html>
```

### 4. 示例：XML 文档同样有 DOM 结构

```xml
<User>
    <name>成哥</name>
    <age>50</age>
    <gender>男</gender>
    <img src="" alt="" />
</User>
```

## 二、获取 DOM 节点

> 全局对象 `window` 中有属性 `document`，代表的是整个文档节点。

### 1. 旧的获取元素节点的方式（DOM 0）

| 属性 | 说明 |
|------|------|
| `document.body` | 获取 body 元素节点 |
| `document.head` | 获取 head 元素节点 |
| `document.links` | 获取页面上所有的超链接元素节点，类数组 |
| `document.anchors` | 获取页面上所有的锚链接（具有 name 属性）元素节点 |
| `document.forms` | 获取页面中所有的 form 元素节点 |

### 2. 新的获取元素节点的方式

#### 通过方法获取

| 方法 | 说明 |
|------|------|
| `document.getElementById` | 通过 id 获取对应 id 的元素 |
| `document.getElementsByTagName` | 通过元素名称获取元素 |
| `document.getElementsByClassName` | 通过元素的类样式获取元素（IE9以下无效） |
| `document.getElementsByName` | 通过元素的 name 属性值获取元素 |
| `document.querySelector` | 通过 CSS 选择器获取元素，得到匹配的第一个（IE8以下无效） |
| `document.querySelectorAll` | 通过 CSS 选择器获取元素，得到所有匹配的结果（IE8以下无效） |
| `document.documentElement` | 获取根元素（html） |

**细节注意**：

1. 在所有得到类数组的方法中，除了 `querySelectorAll`，其他的方法都是**实时更新**的。
2. `getElementById` 得到元素执行效率最高。
3. 书写了 id 的元素，会自动成为 `window` 对象的属性。它是一个实时的单对象（事实上的标准，不推荐使用）。
4. `getElementsByTagName`、`getElementsByClassName`、`querySelector`、`querySelectorAll`，可以作为其他元素节点对象的方法使用。

#### 示例：多种获取元素方式

```html
<div name="n" id="score" class="abc">123</div>
<div id="mydiv2" class="abc">456</div>
<div name="n">asdfasdfasf</div>

<script>
    var div1 = document.getElementById("mydiv1");
    console.log(div1);

    var divabc = document.getElementsByClassName("abc");
    console.log(divabc);

    var divn = document.getElementsByName("n");
    console.log(divn);
</script>
```

#### 示例：在特定元素下使用查询方法

```html
<div class="container">
    <ul class="list">
        <li><a href="">Lorem ipsum dolor sit.</a></li>
        <li>
            <!-- 注释 -->
            啊手动阀手动阀 <a href="https://www.baidu.com" title="阿三发射点发 撒旦发生">Laboriosam commodi quis at!</a>
        </li>
        <!-- ... -->
    </ul>
</div>

<script>
    var a = document.getElementsByTagName("a")[1];
    var ul = document.querySelector(".list");
</script>
```

### 3. 根据节点关系获取节点

#### 获取所有节点（包括文本、注释等）

| 属性 | 说明 |
|------|------|
| `parentNode` | 获取父节点（元素、文档） |
| `previousSibling` | 获取上一个兄弟节点 |
| `nextSibling` | 获取下一个兄弟节点 |
| `childNodes` | 获取所有的子节点 |
| `firstChild` | 获取第一个子节点 |
| `lastChild` | 获取最后一个子节点 |
| `attributes` | 获取某个元素的属性节点 |

#### 仅获取元素节点

| 属性 | 说明 |
|------|------|
| `parentElement` | 获取父元素 |
| `previousElementSibling` | 获取上一个兄弟元素 |
| `nextElementSibling` | 获取下一个兄弟元素 |
| `children` | 获取子元素 |
| `firstElementChild` | 获取第一个子元素 |
| `lastElementChild` | 获取最后一个子元素 |

### 4. 获取节点信息

| 属性 | 说明 |
|------|------|
| `nodeName` | 获取节点名称 |
| `nodeValue` | 获取节点的值 |
| `nodeType` | 节点类型，是一个数字 |

## 三、[作业讲解] 获取 DOM 节点

### 作业1：获取两个列表中超链接的内容

**题目**：准备两个数组，分别存放 list1 和 list2 中所有超链接的内容。

**解答**：

```javascript
// 得到某个元素下面的所有a元素的内容数组
function getLinkContents(dom) {
    return Array.from(dom.getElementsByTagName("a")).map(function(a) {
        return a.firstChild.nodeValue;
    })
}

list1Links = getLinkContents(document.getElementById("list1"));
console.log(list1Links);
list2Links = getLinkContents(document.getElementById("list2"));
console.log(list2Links);
```

> 关键点：使用 `Array.from` 将类数组转为真正的数组，再使用 `map` 提取每个 `<a>` 元素的文本子节点的 `nodeValue`。

### 作业2：向上查找最近的 div 容器

**题目**：写一个函数，传入一个 DOM 对象，返回该对象的第一个 div 容器（向上查找）。

**解答**：

```javascript
function getDivContainer(dom) {
    for (var parent = dom.parentElement; parent && parent.nodeName !== "DIV"; parent = parent.parentElement) {}
    return parent;
}

var div = getDivContainer(test);
console.log(div);
```

> 关键点：利用 `for` 循环不断向上查找 `parentElement`，直到找到 `nodeName` 为 `"DIV"` 的元素为止。

## 四、DOM 元素操作

### 1. 初识元素事件

- **元素事件**：某个元素发生一件事（例如被点击 click）
- **事件处理程序**：是一个函数，发生了一件事后应该做什么
- **注册事件**：将事件处理程序与某个事件关联

> **this 关键字在事件处理程序中指代当前发生事件的元素**

```html
<button id="btn">点我</button>
<script>
    var btn = document.getElementById("btn");
    var count = 0;
    // 注册事件
    btn.onclick = function() {
        console.log(count++);
        console.log(this, this === btn); // this 指向 btn
    }
</script>
```

### 2. 获取和设置元素属性

#### 通用方式

- `getAttribute("属性名")`
- `setAttribute("属性名", "值")`

#### 可识别属性（正常的 HTML 属性）

推荐使用 `dom对象.属性名` 的方式。

**细节注意**：

1. 正常的属性即使没有赋值，也有默认值。
2. 布尔属性在 DOM 对象中，得到的是 boolean。
3. 某些表单元素可以获取到某些不存在的属性。
4. 某些属性与标识符冲突，此时需要更换属性名。

#### 示例：获取/设置属性与图片切换

```html
<input type="text" value="asdfasfd">
<input id="cbgender" type="checkbox" checked="checked">
<button id="btnGetTxt">获取文本框内容</button>
<button id="btnSetTxt">设置文本框内容</button>
<img src="1.jpg" alt="">

<script>
    var input = document.querySelector("input[type=text]");
    document.getElementById("btnGetTxt").onclick = function() {
        console.log(input.value, typeof input.value);
    }
    document.getElementById("btnSetTxt").onclick = function() {
        input.value = "请输入文本";
    }

    // 点击图片切换
    var i = 1;
    var img = document.querySelector("img");
    img.onclick = function() {
        i++;
        if (i === 3) {
            i = 1;
        }
        img.src = i + ".jpg";
    }
</script>
```

#### 自定义属性

HTML5 建议自定义属性使用 `data-` 作为前缀。如果遵从 HTML5 自定义属性规范，可以使用 `dom对象.dataset.属性名` 控制属性。

**删除自定义属性**：

- `removeAttribute("属性名")`
- `delete dom.dataset.属性名`

### 3. 获取和设置元素内容

| 属性 | 说明 |
|------|------|
| `innerHTML` | 获取和设置元素的内部 HTML 文本 |
| `innerText` | 获取和设置元素内部的纯文本，仅得到元素内部**显示出来**的文本 |
| `textContent` | 获取和设置元素内部的纯文本，得到的是内部**源代码中**的文本 |

### 4. 元素结构重构

| 方法 | 说明 |
|------|------|
| `父元素.appendChild(元素)` | 在某个元素末尾加入一个子元素 |
| `父元素.insertBefore(待插入的元素, 哪个元素之前)` | 在指定子元素前插入 |
| `父元素.replaceChild(替换的元素, 被替换的元素)` | 替换子元素 |

> 更改元素结构效率较低，尽量少用。

### 5. 创建和删除元素

#### 创建元素

- `document.createElement("元素名")`：创建一个元素对象
- `document.createTextNode("文本")`：创建文本节点
- `document.createDocumentFragment()`：创建文档片段
- `dom对象.cloneNode(是否深度克隆)`：复制一个新的 DOM 对象并返回

> `childNodes` 也是实时集合。

#### 示例：使用文档片段批量创建元素

```html
<ul id="ul1">
    <li>Lorem, ipsum.</li>
    <li>Quas, vero.</li>
    <!-- ... -->
</ul>

<script>
    var ul = document.getElementById("ul1");
    var frag = document.createDocumentFragment();
    for (var i = 1; i <= 100; i++) {
        var li = document.createElement("li");
        li.innerText = "选项" + i;
        frag.appendChild(li);
    }
    ul.appendChild(frag);
</script>
```

> 关键点：使用 `DocumentFragment` 可以减少 DOM 重排次数，提升性能。

#### 删除元素

- `removeChild`：父元素调用，传入子元素
- `remove`：把自己删除

### 6. 补充：可编辑的 style 标签

```html
<style contenteditable="true" style="display: block; white-space: pre;">
    .container {
        color: red;
    }
    .container a {
        color: inherit;
    }
</style>
<div class="container">
    <ul>
        <li><a href="">Lorem.</a></li>
        <!-- ... -->
    </ul>
</div>
```

> 技巧：给 `<style>` 标签加上 `contenteditable="true"` 并设置为块级显示，可以实现浏览器内实时编辑 CSS 的效果。

### 作业4：库存变更器

点击按钮增减库存，库存最小值为 0。

```html
<div>
    库存：
    <button data-plus="-1000">-1000</button>
    <button data-plus="-100">-100</button>
    <button data-plus="-10">-10</button>
    <button data-plus="-1">-</button>
    <span id="spanNumber">10</span>
    <button data-plus="1">+</button>
    <button data-plus="10">+10</button>
    <button data-plus="100">+100</button>
    <button data-plus="1000">+1000</button>
</div>

<script>
    var btns = document.querySelectorAll("button");
    var span = document.getElementById("spanNumber");
    for (var i = 0; i < btns.length; i++) {
        var b = btns[i];
        b.onclick = function() {
            var num = parseInt(span.innerText);
            var plus = parseInt(this.dataset.plus);
            var result = num + plus;
            if (result < 0) {
                result = 0;
            }
            span.innerHTML = result;
        }
    }
</script>
```

> 关键点：使用 `data-*` 自定义属性存储增减值，通过 `dataset.plus` 获取。注意 `this` 在事件处理函数中指向当前点击的按钮。

### 作业5：穿梭框

左右两个 `<select>` 列表，支持选中移动和全部移动。

```javascript
var selLeft = document.getElementById("sel1"),
    selRight = document.getElementById("sel2");

// 得到某个 select 元素内部被选中的 option 数组
function getSelectedOptions(sel) {
    return Array.from(sel.children).filter(function(item) {
        return item.selected;
    });
}

// 将 option 数组添加到指定的 select 中
function appendToSelect(opts, sel) {
    opts = Array.from(opts);
    var frag = document.createDocumentFragment();
    for (var i = 0; i < opts.length; i++) {
        opts[i].selected = false;
        frag.appendChild(opts[i]);
    }
    sel.appendChild(frag);
}

// 右移选中
btnRight.onclick = function() {
    var opts = getSelectedOptions(selLeft);
    appendToSelect(opts, selRight);
}

// 全部右移
btnRightAll.onclick = function() {
    appendToSelect(selLeft.children, selRight);
}
```

> 关键点：`appendChild` 会将一个已有的 DOM 节点从原来的位置移到新位置（移动而非复制）。使用 `DocumentFragment` 批量操作提升性能。

## 六、DOM 元素样式

### 1. 控制 DOM 元素的类样式

| 属性/方法 | 说明 |
|----------|------|
| `className` | 获取或设置元素的类名（字符串） |
| `classList` | DOM 4 的新属性，是一个用于控制元素类名的对象 |
| `classList.add("类名")` | 添加一个类名 |
| `classList.remove("类名")` | 移除一个类名 |
| `classList.contains("类名")` | 判断一个类名是否存在 |
| `classList.toggle("类名")` | 添加/移除一个类名（切换） |

#### 示例：使用 classList 切换类名

```html
<style>
    .item { width: 100px; height: 100px; }
    .red { background: red; }
    .green { background: green; }
</style>
<div class="item red"></div>

<script>
    var div = document.querySelector(".red");
    div.onclick = function() {
        if (this.classList.contains("red")) {
            this.classList.remove("red");
            this.classList.add("green");
        } else {
            this.classList.remove("green");
            this.classList.add("red");
        }
    }
</script>
```

#### 示例：使用 toggle 简化代码

```html
<style>
    .item { width: 100px; height: 100px; background: gray; }
    .red { background: red; }
</style>
<div class="item red"></div>

<script>
    var div = document.querySelector(".item");
    div.onclick = function() {
        this.classList.toggle("red");
    }
</script>
```

### 2. 获取样式

> CSS 的短横线命名，需要转换为小驼峰命名。例如：`background-color` -> `backgroundColor`

| 方式 | 说明 |
|------|------|
| `dom.style` | 得到**行内样式**对象 |
| `window.getComputedStyle(dom元素)` | 得到某个元素最终计算的样式 |
| `window.getComputedStyle(dom元素, "伪元素")` | 第二个参数用于获取某个伪元素的样式 |

#### 示例：获取最终计算样式

```html
<style>
    .item { width: 8em; height: 100px; background: gray; }
    .red { background: red !important; }
</style>
<div class="item red" style="background-color: green; width:auto"></div>

<script>
    var div = document.querySelector(".item");
    // 得到 div 最终计算的样式
    var style = getComputedStyle(div);
</script>
```

> 注意：`!important` 的优先级高于行内样式。`getComputedStyle` 返回的是最终生效的样式值。

#### 示例：获取伪元素样式

```html
<style>
    .item::before {
        content: "";
        display: block;
        width: 50px;
        height: 50px;
        background: #008c8c;
    }
</style>

<script>
    var div = document.querySelector(".item");
    // 得到伪元素的最终计算样式
    var style = getComputedStyle(div, "before");
</script>
```

### 3. 设置样式

```javascript
dom.style.样式名 = 值;
```

设置的是行内样式。

## 七、[作业讲解] DOM 元素样式

### 作业1：选中效果（导航栏 Tab 切换）

点击导航栏中的某一项，该项高亮（添加 active 类），其他项取消高亮。

```html
<header>
    <nav>
        <span>Lorem.</span>
        <span>Explicabo.</span>
        <span>Harum?</span>
        <span>Delectus?</span>
        <span>Commodi.</span>
        <span>Ullam!</span>
        <span>Tempora!</span>
    </nav>
</header>

<script>
    var spans = document.getElementsByTagName("span");
    Array.from(spans).forEach(function(sp) {
        sp.onclick = function() {
            // 去掉之前所有具有 active 的类样式
            var span = document.querySelector("span.active");
            if (span) {
                span.classList.remove("active");
            }
            this.classList.add("active");
        }
    })
</script>
```

> 关键点：先找到当前具有 `active` 类的元素并移除该类，再给当前点击的元素添加 `active` 类。

### 作业2：随机背景色

点击 div，设置 div 的背景色为随机颜色。

```html
<div class="container"></div>

<script>
    var div = document.querySelector(".container");
    div.onclick = function() {
        // 生成随机 RGB 颜色值 (0 - 255)
        this.style.background = `rgb(${getRandom(0, 255)},${getRandom(0, 255)},${getRandom(0, 255)})`;
    }

    function getRandom(min, max) {
        return Math.floor(Math.random() * (max - min + 1) + min);
    }
</script>
```

> 关键点：使用 `Math.random()` 生成 0-255 的随机数，拼接成 `rgb()` 颜色字符串。

## 八、[作业讲解] 拼图游戏

### 游戏需求

将一张图片分割为 3x3 的网格，随机打乱后，通过点击与空白块相邻的方块进行交换，最终恢复原图。

### 实现代码

```javascript
/**
 * 游戏配置
 */
var gameConfig = {
    width: 500,
    height: 500,
    rows: 3, // 行数
    cols: 3, // 列数
    isOver: false, // 游戏是否结束
    imgurl: "img/lol.png", // 图片路径
    dom: document.getElementById("game") // 游戏的 dom 对象
};
// 每一个小块的宽高
gameConfig.pieceWidth = gameConfig.width / gameConfig.cols;
gameConfig.pieceHeight = gameConfig.height / gameConfig.rows;
// 小块的数量
gameConfig.pieceNumber = gameConfig.rows * gameConfig.cols;

var blocks = []; // 包含小方块信息的数组

function isEqual(n1, n2) {
    return parseInt(n1) === parseInt(n2);
}

/**
 * 小方块的构造函数
 */
function Block(left, top, isVisible) {
    this.left = left;          // 当前的横坐标
    this.top = top;            // 当前的纵坐标
    this.correctLeft = this.left;  // 正确的横坐标
    this.correctTop = this.top;    // 正确的纵坐标
    this.isVisible = isVisible;    // 是否可见

    this.dom = document.createElement("div");
    this.dom.style.width = gameConfig.pieceWidth + "px";
    this.dom.style.height = gameConfig.pieceHeight + "px";
    this.dom.style.background = `url("${gameConfig.imgurl}") -${this.correctLeft}px -${this.correctTop}px`;
    this.dom.style.position = "absolute";
    this.dom.style.border = "1px solid #fff";
    this.dom.style.boxSizing = "border-box";
    this.dom.style.cursor = "pointer";

    if (!isVisible) {
        this.dom.style.display = "none";
    }
    gameConfig.dom.appendChild(this.dom);

    this.show = function() {
        this.dom.style.left = this.left + "px";
        this.dom.style.top = this.top + "px";
    }

    this.isCorrect = function() {
        return isEqual(this.left, this.correctLeft) && isEqual(this.top, this.correctTop);
    }

    this.show();
}
```

### 核心游戏逻辑

```javascript
function init() {
    initGameDom();     // 1. 初始化游戏容器
    initBlocksArray(); // 2. 初始化小方块数组
    shuffle();         // 3. 数组洗牌
    regEvent();        // 4. 注册点击事件

    function regEvent() {
        // 找到看不见的方块
        var inVisibleBlock = blocks.find(function(b) {
            return !b.isVisible;
        });
        blocks.forEach(function(b) {
            b.dom.onclick = function() {
                if (gameConfig.isOver) return;
                // 判断是否相邻（水平或垂直方向距离等于一个小块宽/高）
                if (b.top === inVisibleBlock.top &&
                    isEqual(Math.abs(b.left - inVisibleBlock.left), gameConfig.pieceWidth)
                    ||
                    b.left === inVisibleBlock.left &&
                    isEqual(Math.abs(b.top - inVisibleBlock.top), gameConfig.pieceHeight)) {
                    exchange(b, inVisibleBlock);
                    isWin();
                }
            }
        })
    }

    function isWin() {
        var wrongs = blocks.filter(function(item) {
            return !item.isCorrect();
        });
        if (wrongs.length === 0) {
            gameConfig.isOver = true;
            blocks.forEach(function(b) {
                b.dom.style.border = "none";
                b.dom.style.display = "block"; // 显示隐藏块
            });
        }
    }

    function exchange(b1, b2) {
        var temp = b1.left;
        b1.left = b2.left;
        b2.left = temp;
        temp = b1.top;
        b1.top = b2.top;
        b2.top = temp;
        b1.show();
        b2.show();
    }

    function shuffle() {
        for (var i = 0; i < blocks.length - 1; i++) {
            var index = getRandom(0, blocks.length - 2);
            exchange(blocks[i], blocks[index]);
        }
    }

    function initBlocksArray() {
        for (var i = 0; i < gameConfig.rows; i++) {
            for (var j = 0; j < gameConfig.cols; j++) {
                var isVisible = true;
                if (i === gameConfig.rows - 1 && j === gameConfig.cols - 1) {
                    isVisible = false; // 最后一个方块隐藏
                }
                var b = new Block(j * gameConfig.pieceWidth, i * gameConfig.pieceHeight, isVisible);
                blocks.push(b);
            }
        }
    }

    function initGameDom() {
        gameConfig.dom.style.width = gameConfig.width + "px";
        gameConfig.dom.style.height = gameConfig.height + "px";
        gameConfig.dom.style.border = "2px solid #ccc";
        gameConfig.dom.style.position = "relative";
    }
}

init();
```

### 拼图游戏要点总结

1. **背景图定位**：每个小方块使用 `background-position` 显示原图的对应区域。
2. **洗牌算法**：遍历数组，随机交换位置坐标。
3. **相邻判断**：通过计算点击方块与空白方块的水平/垂直距离判断是否相邻。
4. **胜利判定**：检查所有方块的当前位置是否与正确位置一致。
5. **构造函数模式**：使用构造函数 `Block` 封装每个方块的数据和行为，体现了面向对象的编程思想。
