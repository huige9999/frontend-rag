## 2. 章节逻辑说明

这门课是「HTML+CSS 语言基础」之后的**语言提升**阶段，主线非常清晰：**先用 CSS3 把页面「做得好看、动起来、布局现代」（01~04）→ 用网页简历做一次综合小练手（05）→ 再补 HTML5 的文档语义与原生 API（06~07）→ 最后用一个 H5 综合练习收尾（08）。** 整体是「CSS3 进阶在前、HTML5 在后、综合项目压轴」的递进。

### 先讲什么：CSS3 进阶四连（01~04）

前 4 章是 CSS3 的核心提升，按「视觉特效 → 动效 → 布局 → 杂项」的顺序铺开：

- **01 CSS3视觉**：边框圆角、阴影、渐变、变换等视觉层特性，解决「页面好不好看」。
- **02 CSS3过渡和动画**：transition / animation / keyframes，解决「页面动不动得起来」。
- **03 CSS3布局**：flex / 媒体查询等现代布局，解决「页面怎么排得稳、排得自适应」。
- **04 CSS3更多知识**：前 3 章没覆盖的 CSS3 杂项补充。

**为什么先讲 CSS3**：CSS3 是这一阶段「肉眼可见提升最大」的部分，先做视觉/动画/布局能快速建立成就感，也是后续网页简历能拿得出手的底气。

### 再讲什么：网页简历小练手（05）

- **05 网页简历**：用前面 CSS3 的能力做一个真实的单页面（个人简历），是从「零散知识点」走向「整页落地」的第一个综合练习。

### 然后讲什么：HTML5 文档与 API（06~07）

- **06 HTML5文档**：HTML5 的语义化标签、文档结构。
- **07 HTML5API**：HTML5 提供的原生浏览器 API（如本地存储、地理、多媒体、画布相关等）。

**为什么 HTML5 放在 CSS3 之后**：HTML5 更多是「结构 + 能力」，CSS3 是「表现」，先表现后结构能力，符合先把页面做漂亮、再拓展浏览器原生能力的节奏。

### 最后讲什么：H5 综合练习收尾（08）

- **08 H5综合练习**：把 CSS3 + HTML5 串起来做一个完整 H5 效果，作为整门课的总练手。

### 主线一句话总结

**CSS3 视觉/动画/布局进阶（01~04）→ 网页简历小练手（05）→ HTML5 文档与原生 API（06~07）→ H5 综合练习收尾（08）**，目标是从「会写静态页」进阶到「能做出好看、会动、布局现代、用到 H5 原生能力的完整页面」。

## 3. 使用场景等元信息沉淀

- **学完能用在哪**：独立完成带视觉特效、动画、flex 自适应布局、以及用到 H5 API 的完整静态/H5 页面；为后续 JS、框架、移动端课程打基础。
- **和实际项目的关系**：CSS3 动画与布局是几乎所有业务页面的日常；网页简历/H5 综合练习是个人作品集的常见素材。
- **和其他课程的关系**：属于「语言基础」提升课，上承《HTML+CSS 语言基础》，下接《JavaScript 语言提升》《程序思维训练》；与《现代CSS》（单独详细版课程）在 CSS 深度上互补，本课偏快速实战。
- **面试/教学高频场景**：flex 布局（03）、CSS3 动画与过渡（02）、HTML5 语义化与常见 API（06~07）。
