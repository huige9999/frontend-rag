# 05. 高级算法与经典问题

本章介绍三大类高级算法：动态规划、回溯算法和贪心算法，这些都是解决复杂问题的重要算法思想。

## 章节内容

### 一、动态规划（Dynamic Programming）

动态规划是一种将复杂问题分解为子问题来求解的方法，核心特点包括：
- **最优子结构**：原问题的最优解可由子问题的最优解组合而成
- **重叠子问题**：子问题会重复出现，通过存储避免重复计算
- **状态转移方程**：描述状态之间的递推关系

#### 1. 认识动态规划
- 斐波那契数列问题
- 表格法（自底向上）vs 记忆化搜索（自顶向下）

#### 2. 青蛙跳台阶
- 题目：青蛙一次可以跳1级或2级台阶，求跳到n级台阶的不同跳法数
- 状态转移方程：`dp[i] = dp[i-1] + dp[i-2]`

#### 3. 不同路径问题
- 题目：机器人从m×n网格左上角移动到右下角的不同路径数
- 状态转移方程：`dp[i][j] = dp[i-1][j] + dp[i][j-1]`

#### 4. 凑零钱问题
- 题目：给定不同面值硬币，凑出总金额所需的最少硬币数
- 状态转移方程：`dp[i] = min(dp[i-coin] + 1)`

#### 5. 打家劫舍
- 题目：小偷不能同时偷窃相邻房屋，求能偷到的最高金额
- 状态转移方程：`dp[i] = max(dp[i-1], dp[i-2] + nums[i])`

#### 6. 01背包问题
- 题目：在限定背包容量下，选择物品获得最大价值
- 状态转移方程：`dp[i][c] = max(dp[i-1][c], dp[i-1][c-wgt[i-1]] + val[i-1])`

### 二、回溯算法（Backtracking）

回溯算法本质上是一种暴力穷举算法，通过深度优先搜索遍历解空间树：
- **算法框架**：递归 + 回溯（撤销选择）
- **剪枝操作**：提前结束不可能的分支
- **适用问题**：组合、切割、子集、排列等

#### 1. 认识回溯算法
- 回溯算法的基本模板
- 树形结构的遍历思想

#### 2. 数字组合问题
- 题目：给定n和k，返回[1,n]中所有可能的k个数的组合
- 剪枝优化：提前终止不足的组合

#### 3. 电话号码字母组合
- 题目：给定数字字符串，返回所有可能的字母组合
- 数字到字母的映射关系

#### 4. 复原IP地址
- 题目：在数字字符串中插入点号，形成有效的IP地址
- IP地址有效性验证：范围[0,255]，无前导零

#### 5. 求数组子集
- 题目：返回给定数组的所有子集
- 每个元素的选与不选两种情况

### 三、贪心算法（Greedy）

贪心算法通过选择每一步的局部最优来达到全局最优：
- **核心思想**：每一步都做出当前看起来最好的选择
- **适用场景**：问题具有最优子结构和贪心选择性质
- **局限性**：不一定总能得到全局最优解

#### 1. 认识贪心算法
- 分发饼干问题：如何用饼干满足最多的小孩
- 拿钞票问题：每次选择面值最大的

#### 2. 柠檬水找零
- 题目：柠檬水摊找零问题
- 贪心策略：优先使用大面额钞票找零

#### 3. 摆动序列
- 题目：找出数组中最长的摆动子序列
- 贪心策略：统计差值正负交替的次数

#### 4. 分发糖果
- 题目：相邻孩子评分高的要多得糖果，求最少糖果数
- 两次遍历：左到右 + 右到左

#### 5. 跳跃游戏
- 题目：判断能否从数组起点跳到终点
- 贪心策略：维护最远可达位置

## 学习重点

1. **动态规划**：掌握状态转移方程的推导，理解空间优化技巧
2. **回溯算法**：熟练应用回溯模板，理解剪枝策略
3. **贪心算法**：理解贪心选择性质，知道何时可以使用贪心
