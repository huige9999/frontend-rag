# 4-23 场景 - 生成二维码

## 知识要点

- 使用 `qrcode` 库生成二维码
- `QRCode.toDataURL()` 将二维码转为 Base64 格式的 Data URL
- 可用于生成链接二维码、登录二维码等场景

## 安装

```shell
npm install qrcode
```

## 生成二维码

```js
const QRCode = require("qrcode");

QRCode.toDataURL("https://duyi.ke.qq.com/?tuin=a5d48d54", (err, url) => {
  if (err) {
    console.log(err);
  } else {
    console.log(url); // 输出 base64 格式的二维码图片
  }
});
```

## API 说明

| 方法 | 说明 |
|------|------|
| `QRCode.toDataURL(text, callback)` | 生成 Base64 Data URL |
| `QRCode.toString(text, options)` | 生成 SVG/终端字符串 |
| `QRCode.toFile(path, text, options)` | 保存为文件 |
| `QRCode.toCanvas(canvas, text)` | 绘制到 Canvas |

## 本节说明

本节代码以独立示例展示二维码生成，在实际项目中可将生成的二维码嵌入到页面或作为 API 响应返回。
