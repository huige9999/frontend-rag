# 1-10 https协议

## 知识要点

### HTTP 与 HTTPS 的区别

| 特性 | HTTP | HTTPS |
|------|------|-------|
| 端口 | 80 | 443 |
| 传输 | 明文传输 | 加密传输 |
| 安全性 | 不安全 | 安全 |
| 证书 | 不需要 | 需要 SSL/TLS 证书 |

### HTTPS 的工作原理

1. 客户端向服务器发起连接请求
2. 服务器返回 **SSL 证书**（包含公钥）
3. 客户端验证证书合法性
4. 双方协商生成**会话密钥**（对称加密）
5. 后续通信使用会话密钥加密

### 加密方式

- **对称加密**：加密和解密使用同一个密钥，速度快
- **非对称加密**：公钥加密、私钥解密，安全性高但速度慢
- HTTPS 结合两者：非对称加密协商密钥，对称加密传输数据
