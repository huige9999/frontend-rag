# 4-26 场景 - 富文本框

## 知识要点

- 富文本编辑器用于在网页中编辑带格式的文本内容
- 服务端需处理富文本提交的 HTML 内容
- 富文本内容通常包含 HTML 标签，服务端需注意 XSS 防护
- 本节新增 `/api/book` 路由处理书籍（富文本内容）的 CRUD

## 书籍路由

处理包含富文本内容的书籍数据：

```js
// routes/api/book.js
const express = require("express");
const router = express.Router();
const bookServ = require("../../services/bookService");
const { asyncHandler } = require("../getSendResult");

router.get("/", asyncHandler(async (req, res) => {
  const page = req.query.page || 1;
  const limit = req.query.limit || 10;
  return await bookServ.getBooks(page, limit);
}));

router.get("/:id", asyncHandler(async (req, res) => {
  return await bookServ.getBookById(req.params.id);
}));

router.post("/", asyncHandler(async (req, res, next) => {
  return await bookServ.addBook(req.body); // req.body 中包含富文本 HTML
}));

module.exports = router;
```

## 注册路由

```js
// routes/init.js 中新增
app.use("/api/book", require("./api/book"));
```

## 富文本处理注意事项

1. **XSS 防护**：富文本内容可能包含恶意脚本，需要对 HTML 进行过滤
2. **内容长度**：富文本 HTML 通常较长，需注意数据库字段大小
3. **图片上传**：富文本中的图片通常先上传到服务器，再插入 URL
4. **内容展示**：返回富文本时需设置正确的 Content-Type
