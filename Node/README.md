## 📂 课程结构

| 章节 | 课程数 | 主要内容 |
|------|--------|----------|
| 第一章 Node基础 | 12 | 全局对象、模块化、内置模块、文件IO/流、net/http/https、生命周期、EventEmitter |
| 第二章 MySQL | 4 | 增删改、单表查询、联表查询、函数和分组 |
| 第三章 Sequelize ORM | 12 | mysql驱动、模型定义、增删改查、数据验证、MD5加密、moment、日志 |
| 第四章 Express | 26 | Express基础、中间件、路由、Cookie/Session/JWT认证、CORS、各种实战场景 |
| 第五章 WebSocket与安全 | 7 | WebSocket原理、Socket.IO、在线聊天室、CSRF/XSS防御、Node组成原理、进程线程 |
