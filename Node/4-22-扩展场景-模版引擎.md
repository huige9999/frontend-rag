# 4-22 扩展场景 - 模版引擎

## 知识要点

- Express 支持多种模版引擎（如 EJS）
- 使用 `res.render()` 渲染模版并返回 HTML
- EJS 使用 `<%= %>` 输出转义内容，`<% %>` 执行 JS 代码
- 通过 `app.set("views", path)` 设置模版目录

## EJS 基本使用

### 安装

```shell
npm install ejs
```

### 独立使用

```js
const ejs = require("ejs");

// 渲染文件
ejs.renderFile("./test.ejs", { number: Math.random() })
  .then((result) => {
    console.log(result);
  });

// 渲染字符串
const str = `生成的数字是：<%= number %>`;
const result = ejs.render(str, { number: Math.random() });
```

## Express 中使用 EJS

### 设置模版目录

```js
app.set("views", path.resolve(__dirname, "./views"));
app.use("/student", require("./controller/student"));
```

### 控制器渲染模版

```js
// routes/controller/student.js
router.get("/", async (req, res) => {
  const result = await stuServ.getStudents(page, limit, sex, name);
  res.render("./students.ejs", {
    ...result,
    page,
    limit,
  });
});
```

### EJS 模版示例

```html
<!-- students.ejs -->
<h1>学生列表页</h1>
<p>
  学生总数： <%= total %> , 当前第<%= page %>页，每页<%= limit %>条数据
</p>
<ul>
  <% datas.forEach(stu => { %>
  <li><%= stu.name %></li>
  <% }) %>
</ul>
<a href="?page=<%=+page-1%>">上一页</a>
<a href="?page=<%=+page+1%>">下一页</a>
```

## EJS 语法

| 语法 | 说明 |
|------|------|
| `<%= %>` | 输出转义后的值 |
| `<%- %>` | 输出原始 HTML |
| `<% %>` | 执行 JS 代码 |
| `<%# %>` | 注释 |
| `<%- include('file') %>` | 包含其他模版 |
