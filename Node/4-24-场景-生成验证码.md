# 4-24 场景 - 生成验证码

## 知识要点

- 使用 `svg-captcha` 库生成 SVG 格式验证码
- 验证码文本存储在 session 中用于后续校验
- 基于请求频率判断是否需要验证码（防机器人）
- 验证码中间件集成到路由中

## 生成验证码

```js
// routes/captchaMid.js
var svgCaptcha = require("svg-captcha");

router.get("/captcha", (req, res) => {
  var captcha = svgCaptcha.create({
    size: 6,              // 验证码长度
    ignoreChars: "iIl10oO", // 排除易混淆字符
    noise: 6,             // 干扰线数量
    color: true,          // 彩色
  });
  req.session.captcha = captcha.text.toLowerCase(); // 存入 session
  res.type("svg");
  res.status(200).send(captcha.data);
});
```

## 验证码校验逻辑

根据请求频率判断是否需要验证码：

```js
function captchaHandler(req, res, next) {
  if (!req.session.records) {
    req.session.records = [];
  }
  const now = new Date().getTime();
  req.session.records.push(now);

  // 在 10 秒内请求达到 3 次则要求验证码
  const duration = 10000;
  const repeat = 3;

  req.session.records = req.session.records.filter(
    (time) => now - time <= duration
  );

  if (req.session.records.length >= repeat || "captcha" in req.body) {
    validateCaptcha(req, res, next);
  } else {
    next();
  }
}

function validateCaptcha(req, res, next) {
  const reqCaptcha = req.body.captcha ? req.body.captcha.toLowerCase() : "";
  if (reqCaptcha !== req.session.captcha) {
    res.send({ code: 401, msg: "验证码有问题" });
  } else {
    next();
  }
  req.session.captcha = ""; // 验证码使用后清空
}

router.post("*", captchaHandler);
router.put("*", captchaHandler);
```

## 防机器人策略

1. 记录每次 POST/PUT 请求的时间
2. 过滤掉 10 秒之前的记录
3. 如果 10 秒内请求超过 3 次，要求验证码
4. 如果请求体中包含 captcha 字段，也需要验证
