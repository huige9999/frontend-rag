# 1-2 全局对象

## 知识要点

Node.js 提供了一些全局对象，可以在任何模块中直接使用，无需 `require`。

### 常用全局对象

| 全局对象 | 说明 |
|---------|------|
| `__dirname` | 当前模块所在目录的绝对路径 |
| `__filename` | 当前模块文件的绝对路径 |
| `process` | 进程对象，提供进程相关信息和控制 |
| `setTimeout / setInterval` | 定时器函数 |
| `console` | 控制台输出 |
| `global` | Node 全局对象（类似浏览器的 window） |

### `process` 对象常用属性和方法

- `process.cwd()` — 获取当前命令行所在目录
- `process.argv` — 获取命令行参数数组
- `process.platform` — 获取操作系统平台
- `process.env` — 获取环境变量
- `process.exit()` — 退出进程
- `process.kill(pid)` — 杀死指定进程

---

## 代码示例

### 获取模块路径

```js
console.log(__dirname);
```

### 获取命令行目录

```js
console.log("当前命令行：", process.cwd());
```

### 定时器

```js
setTimeout(() => {
  console.log("abc");
}, 1000);
```

### 退出进程

```js
process.exit();
```

### 获取命令行参数

```js
console.log(process.argv);
```

### 获取系统平台

```js
console.log(process.platform);
```

### 杀死进程

```js
process.kill(10428);
```

### 读取环境变量

```js
console.log(process.env.ANDROID_HOME);
```
