# 4-18 场景 - 文件下载

## 知识要点

- 使用 `res.download()` 方法实现文件下载
- 文件下载路由独立配置
- 下载文件存放在 `resources` 目录中

## 文件下载路由

```js
// routes/api/download.js
const express = require("express");
const path = require("path");
const router = express.Router();

router.get("/:filename", (req, res) => {
  const absPath = path.resolve(
    __dirname,
    "../../resources",
    req.params.filename
  );
  res.download(absPath, req.params.filename);
});

module.exports = router;
```

## 注册下载路由

在 `routes/init.js` 中注册：

```js
app.use("/res", require("./api/download"));
```

访问 `/res/xxx.pdf` 即可下载对应的文件。

## res.download() 说明

- `res.download(path, filename)` -- 提供文件下载
- 第一个参数是服务器上的绝对路径
- 第二个参数是下载时显示的文件名
- 该方法会自动设置 `Content-Disposition` 响应头
