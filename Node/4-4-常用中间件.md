# 4-4 常用中间件

## 知识要点

- `express.static()` -- 静态资源映射中间件
- `express.urlencoded()` -- 解析 `application/x-www-form-urlencoded` 格式的请求体
- `express.json()` -- 解析 `application/json` 格式的请求体
- 了解 urlencoded 中间件的内部原理

## express.static 静态资源中间件

当请求时，根据请求路径从指定目录中寻找文件，如果存在则直接响应文件内容；如果不存在则移交给后续中间件。默认情况下映射目录会自动使用 index.html。

```js
const path = require("path");
const staticRoot = path.resolve(__dirname, "../public");
app.use(express.static(staticRoot));
```

## express.urlencoded 请求体解析中间件

用于解析 `application/x-www-form-urlencoded` 格式的请求体：

```js
app.use(express.urlencoded({ extended: true }));
```

## express.json JSON解析中间件

用于解析 `application/json` 格式的请求体：

```js
app.use(express.json());
```

## 手写 urlencoded 中间件

理解 express.urlencoded 的原理：

```js
const qs = require("querystring");

module.exports = (req, res, next) => {
  if (req.headers["content-type"] === "application/x-www-form-urlencoded") {
    let result = "";
    req.on("data", (chunk) => {
      result += chunk.toString("utf-8");
    });
    req.on("end", () => {
      const query = qs.parse(result);
      req.body = query;
      next();
    });
  } else {
    next();
  }
};

// 工厂函数版本
exports.urlencoded = (options = {}) => {
  options.type = options.type || "application/x-www-form-urlencoded";
  return (req, res, next) => {
    if (req.headers["content-type"] === options.type) {
      let result = "";
      req.on("data", (chunk) => {
        result += chunk.toString("utf-8");
      });
      req.on("end", () => {
        const query = qs.parse(result);
        req.body = query;
        next();
      });
    } else {
      next();
    }
  };
};
```

## 获取请求体数据

使用中间件后，可通过 `req.body` 获取解析后的请求体：

```js
app.post("/api/student", (req, res) => {
  console.log(req.body); // 已解析的请求体数据
});
```
