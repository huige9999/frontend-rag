# 美团省市级联选择器

## 章节概述

本章节通过实现一个类似美团的省市区三级联动选择器，学习如何处理复杂的DOM操作、事件绑定和数据联动逻辑。这是一个典型的前端交互案例，涵盖了数据获取、DOM操作、事件处理等多个核心概念。

## 学习目标

- 掌握异步数据获取和处理
- 理解DOM元素的操作和状态管理
- 学会处理复杂的级联逻辑
- 掌握事件委托和事件处理
- 培养模块化编程思维

## 核心概念

### 1. 级联选择器原理

级联选择器是指多个选择器之间存在依赖关系，前一个选择器的选择会影响后续选择器的可选内容。在本例中：
- 省份选择 → 影响可选城市
- 城市选择 → 影响可选地区
- 地区选择 → 最终结果

### 2. 数据结构设计

省市区数据通常采用树形结构：

```javascript
[
  {
    label: "北京市",
    children: [
      {
        label: "北京市",
        children: [
          { label: "东城区" },
          { label: "西城区" }
        ]
      }
    ]
  },
  {
    label: "浙江省",
    children: [
      {
        label: "杭州市",
        children: [
          { label: "西湖区" },
          { label: "滨江区" }
        ]
      }
    ]
  }
]
```

### JavaScript 逻辑

```javascript
(async () => {
  /**
   * 远程获取省市区数据
   * @returns {Promise<Array>} 返回省市区数据的Promise
   */
  async function getDatas() {
    return fetch('https://study.duyiedu.com/api/citylist')
      .then((resp) => resp.json())
      .then((resp) => resp.data);
  }

  // 获取DOM元素引用
  const doms = {
    selProvince: document.getElementById('selProvince'),
    selCity: document.getElementById('selCity'),
    selArea: document.getElementById('selArea'),
  };

  /* 
    初始化阶段：
    1. 远程获取省市区数据
    2. 填充省份选择器
    3. 城市和地区选择器初始为空
  */
  const datas = await getDatas(); // 远程获取数据
  console.log(datas);
  
  // 填充各个选择器
  fillSelect(doms.selProvince, datas);    // 填充省份
  fillSelect(doms.selCity, []);          // 城市初始为空
  fillSelect(doms.selArea, []);          // 地区初始为空

  /**
   * 填充下拉列表的通用函数
   * @param {HTMLElement} select - 要填充的下拉列表DOM元素
   * @param {Array} list - 被填充的数据数组，如果为空数组则表示禁用
   * 
   * 设计思路：
   * - 高度通用性，既可用于初始化也可用于事件触发后的更新
   * - 自动处理禁用状态
   * - 将数据绑定到DOM元素上，便于后续使用
   */
  function fillSelect(select, list) {
    // 根据数据是否为空设置禁用状态
    select.className = `select ${list.length ? '' : 'disabled'}`;
    
    // 获取提示文字（从data-name属性）
    const tipName = select.dataset.name;
    const span = select.querySelector('.title span');
    span.innerText = `请选择${tipName}`;

    // 将数据绑定到DOM元素，供事件处理使用
    select.datas = list;

    // 生成选项列表
    const ul = select.querySelector('.options');
    ul.innerHTML = list.map((obj) => `<li>${obj.label}</li>`).join('');
  }

  // 注册事件处理器
  regCommonEvent(doms.selProvince);  // 公共事件
  regCommonEvent(doms.selCity);       // 公共事件
  regCommonEvent(doms.selArea);       // 公共事件
  regProvinceEvent();                 // 省份特殊事件
  regCityEvent();                     // 城市特殊事件

  /**
   * 注册所有选择器的公共事件
   * @param {HTMLElement} select - 选择器DOM元素
   * 
   * 处理两个事件：
   * 1. 标题点击：展开/收起选项列表
   * 2. 选项点击：选中选项并更新显示
   */
  function regCommonEvent(select) {
    // 1. 标题点击事件
    const title = select.querySelector('.title');
    title.addEventListener('click', () => {
      // 禁用状态下不允许操作
      if (select.classList.contains('disabled')) {
        return;
      }
      
      // 关闭其他已展开的选择器（互斥行为）
      const expands = document.querySelectorAll('.select.expand');
      for (const sel of expands) {
        if (sel !== select) {
          sel.classList.remove('expand');
        }
      }
      
      // 切换当前选择器的展开状态
      select.classList.toggle('expand');
    });

    // 2. 选项列表点击事件
    const ul = select.querySelector('.options');
    ul.addEventListener('click', (e) => {
      // 只处理LI元素的点击
      if (e.target.tagName !== 'LI') {
        return;
      }
      
      // 移除之前的选中状态
      const beforeActiveLi = select.querySelector('li.active');
      beforeActiveLi && beforeActiveLi.classList.remove('active');
      
      // 设置新的选中状态
      const li = e.target;
      li.classList.add('active');
      
      // 更新标题显示
      const span = select.querySelector('.title span');
      span.innerText = li.innerText;
      
      // 收起选项列表
      select.classList.remove('expand');
    });
  }

  /**
   * 注册省份选择器的特殊事件
   * 
   * 省份变化时需要：
   * 1. 更新城市选择器
   * 2. 清空地区选择器
   */
  function regProvinceEvent() {
    const ul = doms.selProvince.querySelector('.options');
    ul.addEventListener('click', (e) => {
      if (e.target.tagName !== 'LI') {
        return;
      }
      
      const li = e.target;
      
      // 根据选中的省份找到对应的城市数据
      const pr = doms.selProvince.datas.find(
        (obj) => obj.label === li.innerText
      );
      
      // 填充城市选择器
      fillSelect(doms.selCity, pr.children);
      
      // 清空地区选择器
      fillSelect(doms.selArea, []);
    });
  }

  /**
   * 注册城市选择器的特殊事件
   * 
   * 城市变化时需要：
   * 1. 更新地区选择器
   */
  function regCityEvent() {
    const ul = doms.selCity.querySelector('.options');
    ul.addEventListener('click', (e) => {
      if (e.target.tagName !== 'LI') {
        return;
      }
      
      const li = e.target;
      
      // 根据选中的城市找到对应的地区数据
      const city = doms.selCity.datas.find((obj) => obj.label === li.innerText);
      
      // 填充地区选择器
      fillSelect(doms.selArea, city.children);
    });
  }
})();
```

## 核心功能解析

### 1. 数据获取与处理

```javascript
async function getDatas() {
  return fetch('https://study.duyiedu.com/api/citylist')
    .then((resp) => resp.json())
    .then((resp) => resp.data);
}
```

- 使用`fetch` API进行异步数据请求
- 返回Promise，支持async/await语法
- 数据获取后存储在全局变量中供后续使用

### 2. 通用填充函数

```javascript
function fillSelect(select, list) {
  // 状态管理
  select.className = `select ${list.length ? '' : 'disabled'}`;
  
  // 数据绑定
  select.datas = list;
  
  // DOM生成
  const ul = select.querySelector('.options');
  ul.innerHTML = list.map((obj) => `<li>${obj.label}</li>`).join('');
}
```

**设计亮点：**
- 状态和数据同步更新
- 数据绑定到DOM元素，避免全局变量污染
- 高度复用性，处理所有选择器的填充逻辑

### 3. 事件处理机制

#### 公共事件处理
- **展开/收起逻辑**：互斥展开，确保用户体验
- **选中处理**：状态管理、视觉反馈、数据更新

#### 级联事件处理
- **省份事件**：触发城市和地区更新
- **城市事件**：触发地区更新
- **地区事件**：无需级联处理

### 4. 级联逻辑实现

```
用户选择省份 → 查找对应城市数据 → 更新城市选择器 → 清空地区选择器
用户选择城市 → 查找对应地区数据 → 更新地区选择器
```

关键点：
- 通过`find`方法在绑定的数据中查找对应项
- 级联更新时重置下级选择器
- 保持数据流的单向性
