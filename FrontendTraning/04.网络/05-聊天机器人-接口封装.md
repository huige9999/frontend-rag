## 接口封装的概念

接口封装是指将网络请求相关的逻辑统一封装到一个模块中，提供简洁的 API 供其他模块使用。

### 为什么要封装接口？

1. **代码复用**：避免在多个地方重复编写相同的请求代码
2. **统一管理**：所有接口地址、配置集中管理，便于维护
3. **简化调用**：对外提供简单易用的方法，隐藏内部实现细节
4. **便于扩展**：需要添加拦截器、统一错误处理等功能时，只需修改封装模块

## 接口封装实现

### 基本结构

```javascript
var API = (function () {
  // 基础配置
  const BASE_URL = 'http://localhost:7001';
  const TOKEN_KEY = 'token';

  // 私有方法：封装 GET 请求
  function get(path) {
    const headers = {};
    const token = localStorage.getItem(TOKEN_KEY);
    if (token) {
      headers.authorization = `Bearer ${token}`;
    }
    return fetch(BASE_URL + path, { headers });
  }

  // 私有方法：封装 POST 请求
  function post(path, bodyObj) {
    const headers = {
      'Content-Type': 'application/json',
    };
    const token = localStorage.getItem(TOKEN_KEY);
    if (token) {
      headers.authorization = `Bearer ${token}`;
    }
    return fetch(BASE_URL + path, {
      headers,
      method: 'POST',
      body: JSON.stringify(bodyObj),
    });
  }

  // 公开的 API 方法
  return {
    reg,
    login,
    exists,
    profile,
    sendChat,
    getHistory,
    loginOut,
  };
})();
```

### 关键代码说明

#### 1. 使用 IIFE 创建模块

```javascript
var API = (function () {
  // 私有变量和方法
  return {
    // 公开的方法
  };
})();
```

使用立即执行函数表达式（IIFE）创建一个闭包，内部变量和方法对外不可见，实现真正的封装。

#### 2. 统一的请求头处理

```javascript
function get(path) {
  const headers = {};
  const token = localStorage.getItem(TOKEN_KEY);
  if (token) {
    headers.authorization = `Bearer ${token}`;
  }
  return fetch(BASE_URL + path, { headers });
}
```

自动从 localStorage 获取 token 并添加到请求头，避免每个请求都要手动处理认证。

#### 3. 登录成功保存 token

```javascript
async function login(loginInfo) {
  const resp = await post('/api/user/login', loginInfo);
  const result = await resp.json();
  if (result.code === 0) {
    // 登录成功
    // 将响应头中的token保存起来（localStorage）
    const token = resp.headers.get('authorization');
    localStorage.setItem(TOKEN_KEY, token);
  }
  return result;
}
```

登录成功后自动保存 token，后续请求会自动带上认证信息。

## 完整的 API 模块代码

```javascript
var API = (function () {
  // 基础 URL 配置
  const BASE_URL = 'http://localhost:7001';
  
  // token 在 localStorage 中的键名
  const TOKEN_KEY = 'token';

  // 封装 GET 请求
  // 自动处理认证头
  function get(path) {
    const headers = {};
    const token = localStorage.getItem(TOKEN_KEY);
    if (token) {
      headers.authorization = `Bearer ${token}`;
    }
    return fetch(BASE_URL + path, { headers });
  }

  // 封装 POST 请求
  // 自动处理认证头和 JSON 序列化
  function post(path, bodyObj) {
    const headers = {
      'Content-Type': 'application/json',
    };
    const token = localStorage.getItem(TOKEN_KEY);
    if (token) {
      headers.authorization = `Bearer ${token}`;
    }
    return fetch(BASE_URL + path, {
      headers,
      method: 'POST',
      body: JSON.stringify(bodyObj),
    });
  }

  // 用户注册
  async function reg(userInfo) {
    const resp = await post('/api/user/reg', userInfo);
    return await resp.json();
  }

  // 用户登录
  async function login(loginInfo) {
    const resp = await post('/api/user/login', loginInfo);
    const result = await resp.json();
    if (result.code === 0) {
      // 登录成功，保存 token
      const token = resp.headers.get('authorization');
      localStorage.setItem(TOKEN_KEY, token);
    }
    return result;
  }

  // 检查账号是否存在
  async function exists(loginId) {
    const resp = await get('/api/user/exists?loginId=' + loginId);
    return await resp.json();
  }

  // 获取用户信息
  async function profile() {
    const resp = await get('/api/user/profile');
    return await resp.json();
  }

  // 发送聊天消息
  async function sendChat(content) {
    const resp = await post('/api/chat', {
      content,
    });
    return await resp.json();
  }

  // 获取聊天历史
  async function getHistory() {
    const resp = await get('/api/chat/history');
    return await resp.json();
  }

  // 退出登录
  function loginOut() {
    localStorage.removeItem(TOKEN_KEY);
  }

  // 导出公开方法
  return {
    reg,
    login,
    exists,
    profile,
    sendChat,
    getHistory,
    loginOut,
  };
})();
```

## 封装的优势

### 1. 代码简化

**未封装时：**
```javascript
const token = localStorage.getItem('token');
const headers = {
  'Content-Type': 'application/json'
};
if (token) {
  headers.authorization = `Bearer ${token}`;
}

const resp = await fetch('http://localhost:7001/api/user/profile', {
  headers
});
const result = await resp.json();
```

**封装后：**
```javascript
const result = await API.profile();
```

### 2. 便于维护

当接口地址发生变化时，只需修改 `BASE_URL` 即可：

```javascript
const BASE_URL = 'https://api.example.com'; // 修改这里
```

### 3. 统一错误处理

可以在封装模块中添加统一的错误处理逻辑：

```javascript
async function handleRequest(requestFn) {
  try {
    const resp = await requestFn();
    const result = await resp.json();
    if (result.code !== 0) {
      throw new Error(result.msg);
    }
    return result;
  } catch (error) {
    console.error('请求失败：', error);
    throw error;
  }
}
```
