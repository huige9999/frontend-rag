# 01. 零碎的改动 - ES语言提升

## 课程概述

本章节介绍JavaScript语言在ES5和ES6版本中的重要改进，包括变量声明、字符串处理、数组操作、对象增强、函数语法等方面的提升。

## 1. 严格模式

**版本：ES5**

严格模式是ES5新增的一种运行模式，它使JavaScript在更严格的条件下运行。

### 主要特点

- 消除JavaScript语法的一些不合理、不严谨之处
- 消除代码运行的一些不安全之处
- 提高编译器效率，增加运行速度
- 为未来新版本的JavaScript做好铺垫

### 启用方式

```javascript
// 在脚本或函数开头添加
"use strict";

// 示例
function strictFunction() {
  "use strict";
  // 在严格模式下运行的代码
}
```

- [菜鸟教程 - JavaScript 严格模式](https://www.runoob.com/js/js-strict.html)
- [MDN - Strict mode](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Strict_mode)

## 2. let 和 const

**版本：ES6**

### 基本语法

ES6建议不再使用`var`定义变量，而使用`let`定义变量，`const`定义常量。

```javascript
// 使用 let 定义变量
let a = 1;
a = 2; // 变量的值是可修改的

// 使用 const 定义常量
const b = 1;
b = 2; // TypeError: Assignment to constant variable.
```

### 开发建议

> **均使用const，实在需要修改变量，再改为let**

### let 和 const 的特点

#### 1. 全局定义的变量不再作为属性添加到全局对象中

```javascript
// 使用 var
var globalVar = "var";
console.log(window.globalVar); // "var"

// 使用 let/const
let globalLet = "let";
const globalConst = "const";
console.log(window.globalLet); // undefined
console.log(window.globalConst); // undefined
```

#### 2. 在变量定义之前使用它会报错（暂时性死区）

```javascript
console.log(a); // ReferenceError: Cannot access 'a' before initialization
let a = 1;
```

#### 3. 不可重复定义同名变量

```javascript
// 使用 var 可以重复定义（不推荐）
var a = 1;
var a = 2;

// 使用 let/const 会报错
let b = 1;
let b = 2; // SyntaxError: Identifier 'b' has already been declared
```

#### 4. 使用const定义变量时，必须初始化

```javascript
const a; // SyntaxError: Missing initializer in const declaration
const b = 1; // 正确
```

#### 5. 变量具有块级作用域

```javascript
// 代码块示例
if (true) {
  let blockScoped = "块级作用域";
  console.log(blockScoped); // "块级作用域"
}
console.log(blockScoped); // ReferenceError: blockScoped is not defined
```

### for循环中的特殊处理

在for循环中使用let定义变量，JS会做特殊处理，让每次循环使用的都是一个独立的循环变量。

```javascript
// 问题演示：使用 var
for (var i = 0; i < 3; i++) {
  setTimeout(function () {
    console.log('var:', i);
  }, 1000);
}
// 输出：3 3 3

// 传统解决方案：IIFE（立即执行函数）
for (var i = 0; i < 3; i++) {
  (function (i) {
    setTimeout(function () {
      console.log('IIFE:', i);
    }, 1000);
  })(i);
}
// 输出：0 1 2

// 现代解决方案：使用 let
for (let i = 0; i < 3; i++) {
  setTimeout(function () {
    console.log('let:', i);
  }, 1000);
}
// 输出：0 1 2
```

## 4. 字符串新增API

**版本：ES5-ES11**

### 常用API列表

| API | 版本 | 含义 |
| --- | --- | --- |
| `String.prototype.includes(s)` | ES6 | 判断字符串中是否包含某个子串 |
| `String.prototype.trim()` | ES5 | 去掉字符串首尾空白字符 |
| `String.prototype.trimStart()` | ES10 | 去掉字符串开始的空白字符 |
| `String.prototype.trimEnd()` | ES10 | 去掉字符串末尾的空白字符 |
| `String.prototype.replaceAll(s, t)` | ES11 | 将字符串中**所有**的s替换为t |
| `String.prototype.startsWith(s)` | ES6 | 判断字符串是否以s开头 |
| `String.prototype.endsWith(s)` | ES6 | 判断字符串是否以s结尾 |

### 使用示例

```javascript
const str = "  Hello World  ";

// includes - 判断是否包含子串
str.includes("World"); // true
str.includes("world"); // false (大小写敏感)

// trim - 去除首尾空白
str.trim(); // "Hello World"

// trimStart 和 trimEnd
str.trimStart(); // "Hello World  "
str.trimEnd(); // "  Hello World"

// replaceAll - 替换所有匹配项
"aaabbbccc".replaceAll("a", "x"); // "xxxbbbccc"

// startsWith 和 endsWith
"Hello World".startsWith("Hello"); // true
"Hello World".endsWith("World"); // true
```

## 5. 模板字符串

**版本：ES6**

ES6提供了一种新的字符串字面量书写方式，使用反引号 `` ` `` 包裹。

### 基本语法

```javascript
`字符串内容`
```

### 特点

1. **轻松实现换行**
2. **便捷的变量拼接**（使用 `${}` 语法）
3. **支持表达式**

### 使用示例

#### 示例1：基础用法

```javascript
const user = { name: 'monica', age: 17 };

// 使用模板字符串
const s1 = `姓名：${user.name}，年龄：${user.age}
my name is ${user.name}`;

// 等同于传统字符串拼接
const s2 = '姓名：' + user.name + '，年龄：' + user.age + '\nmy name is ' + user.name;

console.log(s1);
/*
 * 姓名：monica，年龄：17
 * my name is monica
 */
```

#### 示例2：拼接HTML

```javascript
const user = { name: 'monica', age: 17 };

const html = `
<div>
  <p><span class="k">姓名</span><span class="v">${user.name}</span></p>
  <p><span class="k">年龄</span><span class="v">${user.age}</span></p>
</div>
`;

console.log(html);
/*
 * <div>
 *   <p><span class="k">姓名</span><span class="v">monica</span></p>
 *   <p><span class="k">年龄</span><span class="v">17</span></p>
 * </div>
 */
```

#### 示例3：支持表达式

```javascript
const a = 10, b = 20;

const result = `a + b = ${a + b}`; // "a + b = 30"
```

> 在需要换行或拼接时，模板字符串远胜于普通字符串

## 6. 数组增强

### 6.1 for-of 循环

**版本：ES6**

ES6提供了一种简洁的方式遍历数组和其他可迭代对象。

#### 示例1：遍历数组

```javascript
const arr = ['a', 'b', 'c'];

// 传统方式
for (let i = 0; i < arr.length; i++) {
  const item = arr[i];
  console.log(item);
}

// for-of 方式（更简洁）
for (const item of arr) {
  console.log(item);
}
```

#### 示例2：遍历伪数组（NodeList）

```javascript
const elements = document.querySelectorAll('.item');

// for-of 方式
for (const elem of elements) {
  // elem 为获取到的每一个元素
  console.log(elem);
}
```

### 6.2 数组新增API

| API | 版本 | 作用 |
| --- | --- | --- |
| `Array.isArray(target)` | ES5 | 判断target是否为一个数组 |
| `Array.from(source)` | ES6 | 将伪数组或可迭代对象转换为真数组 |
| `Array.prototype.fill(n)` | ES6 | 将数组的某些项设置为n |
| `Array.prototype.forEach(fn)` | ES5 | 遍历数组，对每项执行fn |
| `Array.prototype.map(fn)` | ES5 | 数组映射，返回新数组 |
| `Array.prototype.filter(fn)` | ES5 | 数组筛选，保留满足条件的项 |
| `Array.prototype.reduce(fn)` | ES5 | 数组聚合，按规则累加处理 |
| `Array.prototype.some(fn)` | ES5 | 判断是否有至少一项通过测试 |
| `Array.prototype.every(fn)` | ES5 | 判断是否所有项都通过测试 |
| `Array.prototype.find(fn)` | ES6 | 找到第一个通过测试的项 |
| `Array.prototype.includes(item)` | ES7 | 判断数组中是否存在item |

#### 使用示例

```javascript
// Array.from - 伪数组转真数组
const nodeList = document.querySelectorAll('div');
const arr = Array.from(nodeList); // 转换为真正的数组

// Array.prototype.fill
const arr1 = new Array(5).fill(0); // [0, 0, 0, 0, 0]

// Array.prototype.map
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(n => n * 2); // [2, 4, 6, 8]

// Array.prototype.filter
const evens = numbers.filter(n => n % 2 === 0); // [2, 4]

// Array.prototype.reduce
const sum = numbers.reduce((acc, n) => acc + n, 0); // 10

// Array.prototype.find
const found = numbers.find(n => n > 2); // 3

// Array.prototype.includes
numbers.includes(3); // true
```

## 7. 对象增强

### 7.1 对象成员速写

**版本：ES6**

在某些场景下，ES6提供了更简洁的对象成员书写方式。

#### 示例1：属性速写

```javascript
const name = 'monica', age = 17;
const sayHello = function () {
  console.log(`my name is ${this.name}`);
}

// 传统方式
const user1 = {
  name: name,
  age: age,
  sayHello: sayHello
};

// 速写方式
const user2 = {
  name,
  age,
  sayHello
};
```

#### 示例2：方法速写

```javascript
// 传统方式
const MyMath1 = {
  sum: function (a, b) {
    return a + b;
  },
  random: function (min, max) {
    return Math.random() * (max - min) + min;
  }
};

// 速写方式
const MyMath2 = {
  sum(a, b) {
    return a + b;
  },
  random(min, max) {
    return Math.random() * (max - min) + min;
  }
};
```

### 7.2 解构赋值

**版本：ES6**

ES6提供了一种特殊的语法，可以轻松地从数组或对象中提取想要的值。

#### 对象解构

```javascript
const user = {
  name: 'monica',
  age: 17,
  addr: {
    province: '黑龙江',
    city: '哈尔滨'
  }
};

// 基础解构
const { name, age } = user;
console.log(name, age); // "monica" 17

// 嵌套解构
const { addr: { city } } = user;
console.log(city); // "哈尔滨"

// 解构并重命名
const { name: userName, age: userAge } = user;
console.log(userName, userAge); // "monica" 17
```

#### 数组解构

```javascript
const arr = [1, 2, 3, 4];

// 基础解构
const [a, b, c, d] = arr;
console.log(a, b, c, d); // 1 2 3 4

// 跳过某些项
const [, x, y] = arr; // x=2, y=3
const [, m, , n] = arr; // m=2, n=4

// 剩余元素
const [first, second, ...rest] = arr;
console.log(first, second, rest); // 1 2 [3, 4]
```

#### 交换变量

```javascript
let a = 1, b = 2;
[b, a] = [a, b];
console.log(a, b); // 2 1
```

#### 函数参数解构

```javascript
// 对象参数解构
function method({ a, b }) {
  console.log(a, b);
}

const obj = { a: 1, b: 2, c: 3 };
method(obj); // 1 2

// 箭头函数参数解构
const method2 = ({ a, b }) => {
  console.log(a, b);
};

// 遍历时解构
const users = [
  { name: 'monica', age: 17 },
  { name: '邓哥', age: 70 }
];

for (const { name, age } of users) {
  console.log(name, age);
}
```

### 7.3 展开运算符

**版本：ES6**

#### 数组展开

```javascript
const arr = [3, 6, 1, 7, 2];

// 在函数调用中展开
Math.max(...arr); // 相当于：Math.max(3, 6, 1, 7, 2) → 7

// 合并数组
const arr2 = [0, ...arr, 10]; // [0, 3, 6, 1, 7, 2, 10]

// 复制数组
const arrCopy = [...arr];
```

#### 对象展开

```javascript
const o1 = {
  a: 1,
  b: 2,
};

const o2 = {
  a: 3,
  c: 4,
};

// 合并对象（后面的属性会覆盖前面的）
const o3 = {
  ...o1,
  ...o2
};
console.log(o3); // { a: 3, b: 2, c: 4 }

// 复制并修改对象
const user = { name: 'monica', age: 17 };
const user2 = {
  ...user,
  name: '邓哥'
};
console.log(user2); // { name: '邓哥', age: 17 }
```

### 7.4 属性描述符

**版本：ES5**

对于对象中的每个成员，JS使用属性描述符来描述它们的行为。

```javascript
const user = {
  name: 'monica',
  age: 17
};
```

上面的对象在JS内部被描述为：

```javascript
{
  // 属性 name 的描述符
  name: {
    value: 'monica',           // 属性值
    configurable: true,        // 描述符是否可被重新定义
    enumerable: true,          // 是否可被遍历（影响for-in循环）
    writable: true             // 是否可被修改
  },
  // 属性 age 的描述符
  age: {
    value: 17,
    configurable: true,
    enumerable: true,
    writable: true
  }
}
```

### 7.5 属性描述符API

#### Object.getOwnPropertyDescriptor

获取某个属性的描述符

```javascript
const user = {
  name: 'monica',
  age: 17
};

const descriptor = Object.getOwnPropertyDescriptor(user, 'name');
console.log(descriptor);
/*
{
  value: 'monica',
  writable: true,
  enumerable: true,
  configurable: true
}
*/
```

#### Object.defineProperty

定义或修改某个属性的描述符

```javascript
const user = {
  name: 'monica',
  age: 17
};

Object.defineProperty(user, 'name', {
  value: '邓哥',         // 修改值
  enumerable: false,     // 不可遍历
  writable: false        // 不可修改
});

user.name = '其他人';    // 无效（静默失败）
console.log(user.name);  // "邓哥"

for (const key in user) {
  console.log(key);      // 只输出 "age"，name不可遍历
}
```

#### getter 和 setter

属性描述符中的特殊配置，将属性的取值和赋值变为方法调用

```javascript
const obj = {};

Object.defineProperty(obj, 'a', {
  get() {
    // 读取属性a时，得到该方法的返回值
    return 1;
  },
  set(val) {
    // 设置属性a时，调用该方法
    console.log('设置了a的值为：', val);
  }
});

console.log(obj.a);    // 输出：1
obj.a = 3;             // 输出：设置了a的值为：3
console.log(obj.a);    // 输出：1
```

### 7.6 对象键值对API

| API | 版本 | 作用 |
| --- | --- | --- |
| `Object.keys(obj)` | ES5 | 获取对象的所有属性名组成的数组 |
| `Object.values(obj)` | ES8 | 获取对象的所有属性值组成的数组 |
| `Object.entries(obj)` | ES8 | 获取对象的所有[属性名,属性值]键值对数组 |
| `Object.fromEntries(entries)` | ES10 | 将键值对数组转换为对象 |

#### 使用示例

```javascript
const user = {
  name: 'monica',
  age: 17
};

Object.keys(user);    // ["name", "age"]
Object.values(user);  // ["monica", 17]
Object.entries(user); // [["name", "monica"], ["age", 17]]

const entries = [["name", "monica"], ["age", 17]];
Object.fromEntries(entries); // {name: 'monica', age: 17}
```

### 7.7 对象冻结

#### Object.freeze

冻结一个对象，使其所有属性均不可更改

```javascript
const obj = {
  a: 1,
  b: {
    c: 3
  }
};

Object.freeze(obj); // 冻结对象obj

obj.a = 2;          // 无效（静默失败）
obj.k = 4;          // 无效（静默失败）
delete obj.a;       // 无效（静默失败）
obj.b = 5;          // 无效（静默失败）

obj.b.c = 5;        // 有效！b对象没有被冻结

console.log(obj);   // {a: 1, b: {c: 5}}
```

#### Object.isFrozen

判断一个对象是否被冻结

```javascript
const obj = { a: 1 };
Object.freeze(obj);
Object.isFrozen(obj); // true
```

### 7.8 相同性判定

#### Object.is

判断两个值是否相同，与`===`基本一致，但有两个区别：

- `NaN` 和 `NaN` 相等
- `+0` 和 `-0` 不相等

```javascript
Object.is(1, 2);          // false
Object.is("1", 1);        // false
Object.is(NaN, NaN);      // true（===返回false）
Object.is(+0, -0);        // false（===返回true）
```

### 7.9 Set 和 Map

#### Set

**版本：ES6**

用于保存唯一值的序列

```javascript
const set = new Set([1, 2, 2, 3, 3, 3]);
console.log(set); // Set {1, 2, 3}

// 常用于数组去重
const arr = [1, 2, 2, 3, 3, 3];
const uniqueArr = [...new Set(arr)]; // [1, 2, 3]
```

#### Map

**版本：ES6**

用于保存键值对的映射，与对象的最大区别在于：对象的键只能是字符串或Symbol，而Map的键可以是任何类型。

```javascript
const map = new Map();

const objKey = { id: 1 };
map.set(objKey, '对象作为键');
map.set('string', '字符串作为键');
map.set(123, '数字作为键');

console.log(map.get(objKey)); // "对象作为键"
```

## 8. 函数增强

### 8.1 箭头函数

**版本：ES6**

所有使用**函数表达式**的位置，均可以替换为箭头函数。

#### 语法

```javascript
// 完整语法
(参数列表) => { 函数体 }

// 若有且仅有一个参数
参数 => { 函数体 }

// 若函数体有且仅有一条返回语句
(参数列表) => 返回语句
```

#### 使用示例

**示例1：普通函数转箭头函数**

```javascript
// 传统写法
const sum = function (a, b) {
  return a + b;
};

// 箭头函数写法
const sum = (a, b) => a + b;
```

**示例2：事件处理**

```javascript
// 传统写法
dom.onclick = function (e) {
  console.log(e);
};

// 箭头函数写法
dom.onclick = e => {
  console.log(e);
};
```

**示例3：回调函数**

```javascript
// 传统写法
setTimeout(function () {
  console.log('1秒后执行');
}, 1000);

// 箭头函数写法
setTimeout(() => {
  console.log('1秒后执行');
}, 1000);
```

#### 箭头函数的特点

1. **不能使用`new`调用**（不是构造函数）
2. **没有原型**（没有`prototype`属性）
3. **没有`arguments`**
4. **没有自己的`this`**（this来自定义时的外层作用域）

#### this 指向示例

```javascript
const counter = {
  count: 0,
  start: function () {
    // 这里的 this 指向 counter
    setInterval(() => {
      // 箭头函数没有自己的this，使用外层start函数的this
      this.count++;
      console.log(this.count);
    }, 1000);
  }
};

counter.start(); // 每秒输出：1, 2, 3, 4, ...
```

> 箭头函数特别适用于那些临时需要函数的位置

### 8.2 剩余参数

**版本：ES6**

ES6不建议再使用`arguments`，推荐使用剩余参数来收集未知数量的参数。

```javascript
// ...args 为剩余参数，必须放在最后
function method(a, b, ...args) {
  console.log('a:', a);      // 第一个参数
  console.log('b:', b);      // 第二个参数
  console.log('args:', args); // 剩余参数组成的数组
}

method(1, 2, 3, 4, 5, 6, 7);
// a: 1
// b: 2
// args: [3, 4, 5, 6, 7]

method(1, 2);
// a: 1
// b: 2
// args: []
```

### 8.3 参数默认值

**版本：ES6**

当参数没有传递或传递为`undefined`时，会使用默认值。

#### 示例1：基础用法

```javascript
function method(a, b = 1) {
  console.log('a:', a, 'b:', b);
}

method(1, 2);          // a: 1 b: 2
method(1);             // a: 1 b: 1（使用默认值）
method(1, undefined);  // a: 1 b: 1（使用默认值）
method(1, null);       // a: 1 b: null（不使用默认值）
```

#### 示例2：多个默认值

```javascript
const method = (a, b = 1, c = 2, d) => {
  console.log(a, b, c, d);
};

method(1, 2);                    // 1 2 2 undefined
method(1);                       // 1 1 2 undefined
method(1, undefined, undefined, 4); // 1 1 2 4
```

### 8.4 类语法

**版本：ES6**

ES6提供了一种全新的语法来书写构造函数，使代码更加清晰和面向对象。

#### 示例1：基本类语法

```javascript
// 传统写法
function User(firstName, lastName) {
  this.firstName = firstName;
  this.lastName = lastName;
  this.fullName = `${firstName} ${lastName}`;
}

User.isUser = function (u) {
  return !!u && !!u.fullName;
};

User.prototype.sayHello = function () {
  console.log(`Hello, my name is ${this.fullName}`);
};

// 新的类语法写法
class User {
  constructor(firstName, lastName) {
    this.firstName = firstName;
    this.lastName = lastName;
    this.fullName = `${firstName} ${lastName}`;
  }

  static isUser(u) {
    return !!u && !!u.fullName;
  }

  sayHello() {
    console.log(`Hello, my name is ${this.fullName}`);
  }
}
```

#### 示例2：继承

```javascript
// 传统写法
function Animal(type, name) {
  this.type = type;
  this.name = name;
}

Animal.prototype.intro = function () {
  console.log(`I am ${this.type}, my name is ${this.name}`);
};

function Dog(name) {
  Animal.call(this, '狗', name);
}

Dog.prototype = Object.create(Animal.prototype);

// 新的类语法写法
class Animal {
  constructor(type, name) {
    this.type = type;
    this.name = name;
  }

  intro() {
    console.log(`I am ${this.type}, my name is ${this.name}`);
  }
}

class Dog extends Animal {
  constructor(name) {
    super('狗', name); // 调用父类构造函数
  }
}

const dog = new Dog('旺财');
dog.intro(); // "I am 狗, my name is 旺财"
```

### 8.5 函数API

#### call、apply、bind

| API | 作用 |
| --- | --- |
| `Function.prototype.call(obj, ...args)` | 调用函数，绑定this为obj，参数以列表形式传递 |
| `Function.prototype.apply(obj, args)` | 调用函数，绑定this为obj，参数以数组形式传递 |
| `Function.prototype.bind(obj, ...args)` | 返回函数的拷贝，新函数的this绑定为obj，起始参数绑定为args |

#### 使用示例

```javascript
function greet(greeting, punctuation) {
  console.log(`${greeting}, ${this.name}${punctuation}`);
}

const person = { name: 'monica' };

// call - 参数列表传递
greet.call(person, 'Hello', '!'); // "Hello, monica!"

// apply - 参数数组传递
greet.apply(person, ['Hi', '.']); // "Hi, monica."

// bind - 返回新函数
const boundGreet = greet.bind(person, 'Hey');
boundGreet('~'); // "Hey, monica~"
```

### 练习1：let 和 const 的使用

**题目：下面的代码段是否有错误，错在哪里？**

```javascript
// 代码段1
let a;
const b;

// 代码段2
let a = 1;
const b = 2;
a++;
b++;

// 代码段3
console.log(a);
let a = 1;

// 代码段4
let a = 1;
let a = 2;
```

**参考答案：**

```javascript
// 代码段1
let a;
const b; // ❌ 报错：Missing initializer in const declaration

// 代码段2
let a = 1;
const b = 2;
a++;     // ✅ 正确
b++;     // ❌ 报错：Assignment to constant variable

// 代码段3
console.log(a); // ❌ 报错：Cannot access 'a' before initialization
let a = 1;

// 代码段4
let a = 1;
let a = 2; // ❌ 报错：Identifier 'a' has already been declared
```
