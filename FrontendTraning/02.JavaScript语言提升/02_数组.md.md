## 数组的创建和基本操作

### 创建数组

```javascript
// 数组字面量
const arr1 = [1, 2, 3, 4, 5];

// Array构造函数
const arr2 = new Array(1, 2, 3, 4, 5);

// 创建指定长度的空数组
const arr3 = new Array(5); // [empty × 5]
```

### 数组的基本属性和方法

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// 获取数组长度
console.log(arr.length); // 7

// 访问数组元素
console.log(arr[0]); // 45
console.log(arr[arr.length - 1]); // 53

// 修改数组元素
arr[0] = 100;
console.log(arr); // [100, 7, 2, 4, 1, 57, 53]

// 添加元素到末尾
arr.push(99);
console.log(arr); // [100, 7, 2, 4, 1, 57, 53, 99]

// 删除末尾元素
arr.pop();
console.log(arr); // [100, 7, 2, 4, 1, 57, 53]

// 添加元素到开头
arr.unshift(0);
console.log(arr); // [0, 100, 7, 2, 4, 1, 57, 53]

// 删除开头元素
arr.shift();
console.log(arr); // [100, 7, 2, 4, 1, 57, 53]
```

## 数组的高阶方法

### 1. forEach - 遍历数组

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// forEach: 遍历数组的每一项，不返回新数组
// 参数: item(当前项), index(索引), array(原数组)
arr.forEach(function (item, index, array) {
  console.log(item, index, array);
});

// 输出:
// 45 0 [45, 7, 2, 4, 1, 57, 53]
// 7 1 [45, 7, 2, 4, 1, 57, 53]
// 2 2 [45, 7, 2, 4, 1, 57, 53]
// ...
```

### 2. map - 映射数组

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// map: 对数组的每一项进行处理，返回新数组
// 参数: item(当前项), index(索引), array(原数组)
const newArr = arr.map(function (item) {
  return {
    n: item,
    doubleN: item * 2,
  };
});

console.log(newArr);
// 输出:
// [
//   { n: 45, doubleN: 90 },
//   { n: 7, doubleN: 14 },
//   { n: 2, doubleN: 4 },
//   { n: 4, doubleN: 8 },
//   { n: 1, doubleN: 2 },
//   { n: 57, doubleN: 114 },
//   { n: 53, doubleN: 106 }
// ]
```

### 3. filter - 过滤数组

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// filter: 根据条件过滤数组，返回符合条件的新数组
// 参数: item(当前项), index(索引), array(原数组)
const newArr = arr.filter(function (item) {
  // 返回 true 保留该元素，返回 false 过滤掉该元素
  return item % 2 !== 0; // 保留奇数
});

console.log(newArr); // [45, 7, 1, 57, 53]

// 实际应用示例：筛选大于等于50的数
const bigNumbers = arr.filter(function (item) {
  return item >= 50;
});
console.log(bigNumbers); // [45, 57, 53]
```

### 4. reduce - 归约数组

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// reduce: 将数组归约为一个值
// 参数: callback(累加器, 当前值), initialValue(初始值)
const sum = arr.reduce(function (accumulator, currentValue) {
  return accumulator + currentValue;
}, 0);

console.log(sum); // 169

// reduce 执行过程:
// 初始值: 0
// 第1次: 0 + 45 = 45
// 第2次: 45 + 7 = 52
// 第3次: 52 + 2 = 54
// ...
// 最终: 169
```

### 5. every - 全部满足

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// every: 检查数组中是否所有元素都满足条件
// 如果所有元素都返回 true，则返回 true；否则返回 false
const allLessThan60 = arr.every(function (item) {
  return item < 60;
});

console.log(allLessThan60); // true，因为所有数字都小于60

const allGreaterThan10 = arr.every(function (item) {
  return item > 10;
});

console.log(allGreaterThan10); // false，因为有些数字小于10
```

### 6. includes - 包含检查

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// includes: 检查数组是否包含某个元素
const result = arr.includes(22);
console.log(result); // false

const has45 = arr.includes(45);
console.log(has45); // true
```

### 7. find - 查找元素

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// find: 查找数组中第一个满足条件的元素
const found = arr.find(function (item) {
  return item > 50;
});

console.log(found); // 57，第一个大于50的元素
```

### 8. findIndex - 查找索引

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// findIndex: 查找数组中第一个满足条件的元素的索引
const index = arr.findIndex(function (item) {
  return item > 50;
});

console.log(index); // 5，57的索引
```

### 9. some - 部分满足

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// some: 检查数组中是否有元素满足条件
// 如果有一个元素返回 true，则返回 true；否则返回 false
const hasBigNumber = arr.some(function (item) {
  return item > 50;
});

console.log(hasBigNumber); // true，因为有57和53大于50

const hasNegative = arr.some(function (item) {
  return item < 0;
});

console.log(hasNegative); // false，因为没有负数
```

## for...of 遍历

```javascript
const arr = [45, 7, 2, 4, 1, 57, 53];

// for...of: 遍历数组的每一项
for (const item of arr) {
  console.log(item);
}

// 输出: 45, 7, 2, 4, 1, 57, 53
```

## 实际应用示例

### 示例1: 学生信息处理

```javascript
const students = [
  {
    id: 988985,
    name: '梁平',
    sex: '女',
    age: 15,
    address: '安徽省 淮南市',
    tel: '12957961008',
  },
  {
    id: 299422,
    name: '邱杰',
    sex: '男',
    age: 25,
    address: '辽宁省 本溪市',
    tel: '12685726676',
  },
  // ... 更多学生数据
];

// 1. 遍历输出学生的姓名
students.forEach(function (student) {
  console.log(student.name);
});

// 2. 得到所有女生
const femaleStudents = students.filter(function (student) {
  return student.sex === '女';
});

// 3. 得到所有年龄在25岁以下的女生
const youngFemaleStudents = students.filter(function (student) {
  return student.sex === '女' && student.age < 25;
});

// 4. 得到所有学生的姓名数组
const names = students.map(function (student) {
  return student.name;
});

// 5. 得到所有学生的姓名组成的字符串，以逗号分割
const namesString = students.map(function (student) {
  return student.name;
}).join(', ');

// 6. 把所有学生姓名以p元素包起来，形成一个html片段
const namesHtml = students.map(function (student) {
  return '<p>' + student.name + '</p>';
}).join('');

// 7. 得到所有女生的姓名数组
const femaleNames = students
  .filter(function (student) {
    return student.sex === '女';
  })
  .map(function (student) {
    return student.name;
  });

// 8. 得到所有女生的姓名和电话号码
const femaleInfo = students
  .filter(function (student) {
    return student.sex === '女';
  })
  .map(function (student) {
    return {
      name: student.name,
      tel: student.tel,
    };
  });

// 9. 得到所有学生的年龄的总和
const totalAge = students.reduce(function (sum, student) {
  return sum + student.age;
}, 0);

// 10. 得到所有学生的平均年龄
const averageAge = students.reduce(function (sum, student) {
  return sum + student.age;
}, 0) / students.length;

// 11. 得到一个对象： {name:['张三', '李四', ...], age: [17, 25, ...]}
const studentsObj = students.reduce(function (result, student) {
  result.name.push(student.name);
  result.age.push(student.age);
  return result;
}, { name: [], age: [] });

// 12. 找到id为796997的学生对象
const targetStudent = students.find(function (student) {
  return student.id === 796997;
});

// 13. 是否包含年龄大于28岁的男生
const hasOldMale = students.some(function (student) {
  return student.sex === '男' && student.age > 28;
});

// 14. 是否所有的女生年龄都在28岁以内
const allFemaleYoung = students
  .filter(function (student) {
    return student.sex === '女';
  })
  .every(function (student) {
    return student.age <= 28;
  });
```

### 示例2: 随机数数组生成

```javascript
// 得到一个随机数组成的数组，数组长度为10，随机数的范围在0-1之间
const randomArray1 = [];
for (let i = 0; i < 10; i++) {
  randomArray1.push(Math.random());
}
console.log(randomArray1); // [0.262, 0.167, 0.841, ...]

// 或使用更简洁的方式
const randomArray2 = Array.from({ length: 10 }, function () {
  return Math.random();
});
console.log(randomArray2);

// 得到一个随机数组成的数组，数组长度为10，随机数的范围在10-100之间
const randomArray3 = Array.from({ length: 10 }, function () {
  return Math.floor(Math.random() * 90) + 10;
});
console.log(randomArray3); // [35, 66, 45, ...]
```

### 示例3: 字符串判断

```javascript
// 判断某个字符串s是否为 .jpg、.png、.bmp、.gif 中的一个
function isImageExtension(s) {
  const extensions = ['.jpg', '.png', '.bmp', '.gif'];
  return extensions.includes(s);
}

// 或者使用数组的some方法
function isImageExtension2(s) {
  const extensions = ['.jpg', '.png', '.bmp', '.gif'];
  return extensions.some(function (ext) {
    return ext === s;
  });
}

console.log(isImageExtension('.jpg')); // true
console.log(isImageExtension('.pdf')); // false
```

## 数组方法的选择

| 方法 | 用途 | 是否修改原数组 | 返回值 |
|------|------|----------------|--------|
| forEach | 遍历数组 | 否 | undefined |
| map | 映射转换 | 否 | 新数组 |
| filter | 过滤 | 否 | 新数组 |
| reduce | 归约 | 否 | 单个值 |
| every | 全部满足条件检查 | 否 | 布尔值 |
| some | 部分满足条件检查 | 否 | 布尔值 |
| find | 查找元素 | 否 | 元素或undefined |
| findIndex | 查找索引 | 否 | 索引或-1 |
| includes | 包含检查 | 否 | 布尔值 |
| push | 添加到末尾 | 是 | 新长度 |
| pop | 删除末尾 | 是 | 被删除的元素 |
| shift | 删除开头 | 是 | 被删除的元素 |
| unshift | 添加到开头 | 是 | 新长度 |

## 常见错误

### 1. 忘记return

```javascript
// 错误示例
const newArr = arr.map(function (item) {
  item * 2; // 忘记return
});
console.log(newArr); // [undefined, undefined, ...]

// 正确示例
const newArr = arr.map(function (item) {
  return item * 2;
});
```

### 2. 混淆forEach和map

```javascript
// forEach不返回值
arr.forEach(function (item) {
  return item * 2; // 这个return没有意义
});

// map返回新数组
const doubled = arr.map(function (item) {
  return item * 2;
});
```

### 3. 修改原数组

```javascript
// filter、map等方法不会修改原数组
const original = [1, 2, 3, 4, 5];
const filtered = original.filter(function (item) {
  return item > 2;
});

console.log(original); // [1, 2, 3, 4, 5] - 原数组不变
console.log(filtered); // [3, 4, 5]
```
