## 核心知识点

### 1. 函数基础

函数是JavaScript中的重要概念，可以将代码封装成可重用的模块。

```javascript
// 函数声明
function add(a, b) {
    return a + b;
}

// 函数表达式
const multiply = function(a, b) {
    return a * b;
};

// 箭头函数
const divide = (a, b) => a / b;

// 调用函数
console.log(add(1, 2));      // 3
console.log(multiply(3, 4)); // 12
console.log(divide(8, 2));   // 4
```

### 2. 箭头函数

箭头函数是ES6引入的新语法，提供更简洁的函数写法，并且不绑定this。

```javascript
// 基本语法
const sum = (...args) => {
    // 使用reduce求和
    return args.reduce((acc, num) => acc + num, 0);
};

// 调用示例
console.log(sum());           // 0
console.log(sum(1));          // 1
console.log(sum(1, 2));       // 3
console.log(sum(1, 2, 3));    // 6

// 处理数组
const nums = [1, 2, 3, 4, 5, 6];
console.log(sum(...nums));    // 21
```

**箭头函数特点：**
- 语法更简洁
- 不绑定this，继承外层作用域的this
- 不能作为构造函数使用
- 没有arguments对象

### 3. 参数默认值和参数解构

函数可以设置参数默认值，支持参数解构，让函数调用更灵活。

```javascript
// 参数默认值
function getDatas(page = 1, limit = 10) {
    console.log(`获取第${page}页的数据，每页显示${limit}条`);
}

// 调用示例
getDatas();        // 输出：获取第1页的数据，每页显示10条
getDatas(2);       // 输出：获取第2页的数据，每页显示10条
getDatas(2, 30);   // 输出：获取第2页的数据，每页显示30条
```

```javascript
// 参数解构 + 默认值
function getDatas({ page = 1, limit = 10, keyword = '' } = {}) {
    console.log(`获取第${page}页的数据，每页显示${limit}条，查询关键字为${keyword || '空'}`);
}

// 调用示例
getDatas();                              // 输出：获取第1页的数据，每页显示10条，查询关键字为空
getDatas({ page: 2 });                   // 输出：获取第2页的数据，每页显示10条，查询关键字为空
getDatas({ page: 2, limit: 30 });        // 输出：获取第2页的数据，每页显示30条，查询关键字为空
getDatas({ keyword: 'js' });             // 输出：获取第1页的数据，每页显示10条，查询关键字为js
```

### 4. this指向

this的指向取决于函数的调用方式，这是JavaScript中的重要概念。

```javascript
// 方法中的this指向对象本身
const counter = {
    count: 0,
    startIncrease() {
        // 使用箭头函数保持this指向
        setInterval(() => {
            this.count++;
            console.log(this.count);
        }, 1000);
    }
};

counter.startIncrease(); // 每隔一秒输出递增的数字
```

**this指向规则：**
- 对象方法调用：this指向对象本身
- 普通函数调用：this指向全局对象（非严格模式）或undefined（严格模式）
- call/apply/bind：可以改变this指向
- 箭头函数：不绑定this，继承外层作用域

### 5. 高阶函数和闭包

高阶函数是可以接收函数作为参数或返回函数的函数，闭包则是函数访问其词法作用域外的变量。

```javascript
// 防抖函数示例
function debounce(fn, delay) {
    let timer = null;
    
    return function(...args) {
        // 清除之前的定时器
        if (timer) {
            clearTimeout(timer);
        }
        
        // 设置新的定时器
        timer = setTimeout(() => {
            fn.apply(this, args);
        }, delay);
    };
}

// 使用示例
const fn = (a, b) => {
    console.log(a + b);
};

const dFn = debounce(fn, 1000);
dFn(1, 2);  // 延迟输出：3
dFn(2, 3);  // 延迟输出：5
// 最终，1秒后输出：5
```

**高阶函数应用场景：**
- 函数防抖/节流
- 数组方法（map, filter, reduce等）
- 函数组合
- 柯里化

### 6. Class语法

ES6引入了class语法，提供更面向对象的编程方式。

```javascript
class User {
    constructor(account, password) {
        this.account = account;
        this.password = password;
    }
    
    sayHello() {
        console.log(`账号 —— ${this.account}，密码 —— ${this.password}`);
    }
}

// 使用示例
const u = new User('account', 'pwd123');
u.sayHello(); // 输出：账号 —— account，密码 —— pwd123
```

**Class特点：**
- 本质是语法糖，底层仍是原型链
- 支持继承extends
- 支持静态方法static
- 构造函数constructor

### 练习5：防抖函数

实现debounce函数，让函数执行推迟，如果在推迟期间再次调用，会进一步推迟。

```javascript
function debounce(fn, delay) {
    let timer = null;
    
    return function(...args) {
        if (timer) {
            clearTimeout(timer);
        }
        timer = setTimeout(() => {
            fn.apply(this, args);
        }, delay);
    };
}

// 测试
const fn = (a, b) => {
    console.log(a + b);
};

const dFn = debounce(fn, 1000);
dFn(1, 2);  // 1秒后输出：5（因为后续调用覆盖了前面的）
dFn(2, 3);
```

## 重点总结

### 函数定义方式
1. 函数声明：`function name() {}`
2. 函数表达式：`const name = function() {}`
3. 箭头函数：`const name = () => {}`

### this指向规则
1. 对象方法：指向对象本身
2. 普通函数：指向全局对象（非严格模式）
3. call/apply/bind：可以改变this指向
4. 箭头函数：继承外层作用域的this

### 参数处理技巧
1. 参数默认值：`function fn(a = 1) {}`
2. 参数解构：`function fn({a, b} = {}) {}`
3. 剩余参数：`function fn(...args) {}`

### 高阶函数应用
1. 防抖/节流
2. 数组方法链式调用
3. 函数组合
4. 闭包保存状态

### Class语法
1. 构造函数constructor
2. 实例方法
3. 静态方法static
4. 继承extends
