本章将通过实际练习，深入理解Vue组件的核心概念和开发技巧。我们将开发5个常用的通用组件，掌握组件的属性、事件、插槽、样式等核心知识点。

- 掌握组件属性（Props）的定义和使用

## 1. Avatar 头像组件

### 组件说明

Avatar组件用于显示用户头像，支持自定义图片URL和尺寸，具有圆形显示效果。

### 属性配置

| 属性名 | 含义               | 类型   | 必填 | 默认值 |
| ------ | ------------------ | ------ | ---- | ------ |
| url    | 头像图片路径       | String | 是   | 无     |
| size   | 头像尺寸，宽高相等 | Number | 否   | 150    |

### 核心代码

```vue
<template>
  <img
    class="avatar-container"
    :src="url"
    :style="{
      width: size + 'px',
      height: size + 'px',
    }"
  />
</template>

<script>
export default {
  props: {
    url: {
      type: String, // url属性必须是一个字符串
      required: true, // 属性必传
    },
    size: {
      type: Number,
      default: 150,
    },
  },
};
</script>

<style scoped>
/* 带有作用域的样式 */
.avatar-container {
  border-radius: 50%;
  object-fit: cover;
  display: block;
}
</style>
```

### 知识点说明

1. **Props验证**：通过`type`和`required`对属性进行类型验证和必填验证
2. **动态样式**：使用`:style`绑定动态计算尺寸
3. **作用域样式**：使用`scoped`确保样式只作用于当前组件
4. **object-fit**：CSS属性确保图片按比例填充容器

### 使用示例

```vue
<template>
  <div>
    <Avatar :url="avatarUrl" :size="100" />
    <Avatar :url="avatarUrl" :size="150" />
    <Avatar :url="avatarUrl" :size="200" />
  </div>
</template>

<script>
import Avatar from "./components/Avatar";
export default {
  components: { Avatar },
  data() {
    return {
      avatarUrl: "https://example.com/avatar.jpg"
    };
  }
};
</script>
```

3. **动态类名**：使用`:class`绑定计算属性返回的类名
4. **外部样式引入**：通过`@import`引入图标字体样式
5. **样式继承**：`color: inherit`和`font-size: inherit`使图标继承父元素样式

## 3. Pager 分页器组件

### 组件说明

Pager组件是一个完整的分页器，支持首页、末页、上一页、下一页以及页码跳转功能。

### 属性配置

| 属性名        | 含义       | 类型   | 必填 | 默认值 |
| ------------- | ---------- | ------ | ---- | ------ |
| current       | 当前页码   | Number | 否   | 1      |
| total         | 总数据量   | Number | 否   | 0      |
| limit         | 页容量     | Number | 否   | 10     |
| visibleNumber | 可见页码数 | Number | 否   | 10     |

### 事件配置

| 事件名     | 含义     | 事件参数 | 参数类型 |
| ---------- | -------- | -------- | -------- |
| pageChange | 页码变化 | 新的页码 | Number   |

### 核心代码

```vue
<template>
  <!-- 只有总页数大于1时才显示 -->
  <div class="pager-container" v-if="pageNumber > 1">
    <!-- 首页按钮 -->
    <a @click="handleClick(1)" :class="{ disabled: current === 1 }">
      |&lt;&lt;
    </a>
    <!-- 上一页按钮 -->
    <a @click="handleClick(current - 1)" :class="{ disabled: current === 1 }">
      &lt;&lt;
    </a>
    <!-- 页码数字 -->
    <a
      @click="handleClick(n)"
      v-for="(n, i) in numbers"
      :key="i"
      :class="{ active: n === current }"
    >
      {{ n }}
    </a>

    <!-- 下一页按钮 -->
    <a
      @click="handleClick(current + 1)"
      :class="{ disabled: current === pageNumber }"
    >
      &gt;&gt;
    </a>
    <!-- 末页按钮 -->
    <a
      @click="handleClick(pageNumber)"
      :class="{ disabled: current === pageNumber }"
    >
      &gt;&gt;|
    </a>
  </div>
</template>

<script>
export default {
  props: {
    current: {
      type: Number,
      default: 1,
    },
    total: {
      type: Number,
      default: 0,
    },
    limit: {
      type: Number,
      default: 10,
    },
    visibleNumber: {
      type: Number,
      default: 10,
    },
  },
  computed: {
    // 计算总页数：向上取整(总数 / 每页数量)
    pageNumber() {
      return Math.ceil(this.total / this.limit);
    },
    // 计算显示的最小页码
    visibleMin() {
      let min = this.current - Math.floor(this.visibleNumber / 2);
      if (min < 1) {
        min = 1;
      }
      return min;
    },
    // 计算显示的最大页码
    visibleMax() {
      let max = this.visibleMin + this.visibleNumber - 1;
      if (max > this.pageNumber) {
        max = this.pageNumber;
      }
      return max;
    },
    // 生成要显示的页码数组
    numbers() {
      let nums = [];
      for (let i = this.visibleMin; i <= this.visibleMax; i++) {
        nums.push(i);
      }
      return nums;
    },
  },
  methods: {
    // 处理页码点击
    handleClick(newPage) {
      // 边界检查
      if (newPage < 1) {
        newPage = 1;
      }
      if (newPage > this.pageNumber) {
        newPage = this.pageNumber;
      }
      // 如果点击的是当前页，不做处理
      if (newPage === this.current) {
        return;
      }
      // 向父组件抛出pageChange事件
      this.$emit("pageChange", newPage);
    },
  },
};
</script>

<style lang="less" scoped>
@import "~@/styles/var.less";
.pager-container {
  display: flex;
  justify-content: center;
  margin: 20px 0;
  a {
    color: @primary;
    margin: 0 6px;
    cursor: pointer;
    &.disabled {
      color: @lightWords;
      cursor: not-allowed;
    }
    &.active {
      color: @words;
      font-weight: bold;
      cursor: text;
    }
  }
}
</style>
```

### 知识点说明

1. **计算属性链式依赖**：`visibleMax`依赖`visibleMin`，`numbers`依赖两者
2. **条件渲染**：使用`v-if`控制分页器显示条件
3. **列表渲染**：使用`v-for`生成页码数组
4. **动态类名**：使用`:class`对象语法控制按钮状态
5. **事件抛出**：使用`$emit`向父组件传递事件
6. **边界处理**：在方法中处理页码的边界情况
7. **样式条件**：根据不同状态应用不同样式

### 使用示例

```vue
<template>
  <div>
    <Pager
      :total="total"
      :current="current"
      :limit="10"
      :visibleNumber="5"
      @pageChange="handlePageChange"
    />
  </div>
</template>

<script>
import Pager from "./components/Pager";
export default {
  components: { Pager },
  data() {
    return {
      current: 1,
      total: 302,
    };
  },
  methods: {
    handlePageChange(newPage) {
      this.current = newPage;
      console.log("加载第" + newPage + "页数据");
      // 这里可以调用API获取对应页码的数据
    },
  },
};
</script>
```

## 4. Empty 空状态组件

### 组件说明

Empty组件用于展示数据为空时的状态，包含图标和提示文字，通常需要在外层容器中居中显示。

### 属性配置

| 属性名 | 含义       | 类型   | 必填 | 默认值   |
| ------ | ---------- | ------ | ---- | -------- |
| text   | 显示的文字 | String | 否   | "无数据" |

### 核心代码

```vue
<template>
  <div class="empty-container">
    <Icon type="empty" />
    <p>{{ text }}</p>
  </div>
</template>

<script>
import Icon from "../Icon";
export default {
  components: {
    Icon,
  },
  props: {
    text: {
      type: String,
      default: "无数据",
    },
  },
};
</script>

<style scoped lang="less">
@import "~@/styles/mixin.less";
@import "~@/styles/var.less";
.empty-container {
  .self-center(); // 使用mixin实现居中
  color: @gray;
  text-align: center;
}
.icon-container {
  font-size: 72px;
}
</style>
```

### 知识点说明

1. **组件复用**：Empty组件复用了Icon组件
2. **Props默认值**：为text属性提供默认值
3. **Mixin混入**：使用`.self-center()` mixin实现居中布局
4. **样式变量**：使用LESS变量统一管理颜色
5. **样式模块化**：通过`@import`引入公共样式文件

### 使用示例

```vue
<template>
  <div class="container">
    <div v-if="hasData">
      <!-- 显示数据内容 -->
    </div>
    <Empty v-else text="暂无文章数据" />
  </div>
</template>

<script>
import Empty from "./components/Empty";
export default {
  components: { Empty },
  data() {
    return {
      hasData: false
    };
  }
};
</script>

<style>
.container {
  width: 100%;
  height: 400px;
  position: relative;
}
</style>
```

## 5. ImageLoader 图片渐进加载组件

### 组件说明

ImageLoader组件实现图片的渐进式加载效果，先显示模糊的占位图，加载完成后平滑过渡到高清原图。

### 属性配置

| 属性名      | 含义                                         | 类型   | 必填 | 默认值 |
| ----------- | -------------------------------------------- | ------ | ---- | ------ |
| src         | 原始图片的路径                               | String | 是   | 无     |
| placeholder | 原始图片加载完成前的占位图片                 | String | 是   | 无     |
| duration    | 原始图片加载完成后，切换到原始图经过的毫秒数 | Number | 否   | 500    |

### 事件配置

| 事件名 | 含义                   | 事件参数 | 参数类型 |
| ------ | ---------------------- | -------- | -------- |
| load   | 原始图片加载完成后触发 | 无       | 无       |

### 核心代码

```vue
<template>
  <div class="image-loader-container">
    <!-- 占位图：只在未完全加载时显示 -->
    <img v-if="!everythingDone" class="placeholder" :src="placeholder" alt="" />
    <!-- 原图：通过opacity控制显示 -->
    <img
      @load="handleLoad"
      :src="src"
      alt=""
      :style="{ opacity: originOpacity, transition: `${duration}ms` }"
    />
  </div>
</template>

<script>
export default {
  props: {
    src: {
      type: String,
      required: true,
    },
    placeholder: {
      type: String,
      required: true,
    },
    duration: {
      type: Number,
      default: 500,
    },
  },
  data() {
    return {
      originLoaded: false,    // 原图是否加载完成
      everythingDone: false,  // 是否一切都尘埃落定了（用于移除占位图）
    };
  },
  computed: {
    // 根据原图加载状态计算透明度
    originOpacity() {
      return this.originLoaded ? 1 : 0;
    },
  },
  methods: {
    // 处理原图加载完成事件
    handleLoad() {
      this.originLoaded = true;
      // 等待过渡动画完成后，移除占位图并触发load事件
      setTimeout(() => {
        this.everythingDone = true;
        this.$emit("load");
      }, this.duration);
    },
  },
};
</script>

<style scoped lang="less">
@import "~@/styles/mixin.less";
.image-loader-container {
  width: 100%;
  height: 100%;
  position: relative;
  overflow: hidden;
  img {
    .self-fill(); // 使用mixin让图片填充容器
    object-fit: cover;
  }

  .placeholder {
    filter: blur(2vw); // 对占位图应用模糊效果
  }
}
</style>
```

### 知识点说明

1. **状态管理**：使用两个状态变量控制组件的不同阶段
2. **条件渲染**：使用`v-if`控制占位图的显示和移除
3. **原生事件**：监听`load`事件判断图片加载完成
4. **计算属性**：根据状态动态计算透明度
5. **CSS过渡**：使用`transition`实现平滑过渡效果
6. **异步处理**：使用`setTimeout`等待过渡完成后再执行清理操作
7. **事件触发时机**：在过渡完成后触发自定义事件
8. **CSS滤镜**：使用`filter: blur()`对占位图应用模糊效果

### 使用示例

```vue
<template>
  <div class="test-container">
    <ImageLoader
      src="https://images.pexels.com/photos/33109/fall-autumn-red-season.jpg?fit=crop&crop=entropy&w=3456&h=2304"
      placeholder="https://images.pexels.com/photos/33109/fall-autumn-red-season.jpg?w=100"
      :duration="500"
      @load="handleLoaded"
    />
  </div>
</template>

<script>
import ImageLoader from "./components/ImageLoader";
export default {
  components: {
    ImageLoader,
  },
  methods: {
    handleLoaded() {
      console.log("图片加载完成");
    },
  },
};
</script>

<style scoped>
.test-container {
  width: 500px;
  height: 400px;
  border: 2px solid;
  margin: 0 auto;
}
</style>
```

### 实现原理图解

```
阶段1: 初始状态
┌─────────────────┐
│  [占位图模糊]   │ ← opacity: 1
│  [原图透明]     │ ← opacity: 0
└─────────────────┘

阶段2: 原图加载完成
┌─────────────────┐
│  [占位图模糊]   │ ← opacity: 1
│  [原图渐现]     │ ← opacity: 0 → 1 (500ms过渡)
└─────────────────┘

阶段3: 过渡完成
┌─────────────────┐
│  [原图清晰]     │ ← opacity: 1, 占位图移除
└─────────────────┘
```

## 组件开发最佳实践

### 1. 组件命名规范

- 组件文件名使用PascalCase（大驼峰）：`Avatar.vue`
- 组件注册时使用PascalCase：`components: { Avatar }`
- 模板中使用kebab-case：`<avatar />`或PascalCase：`<Avatar />`

### 2. Props定义规范

```javascript
props: {
  // 基础类型检查
  propA: Number,
  
  // 多个可能的类型
  propB: [String, Number],
  
  // 必填的字符串
  propC: {
    type: String,
    required: true
  },
  
  // 带默认值的数字
  propD: {
    type: Number,
    default: 100
  },
  
  // 带默认值的对象
  propE: {
    type: Object,
    default: function() {
      return { message: 'hello' }
    }
  }
}
```

### 3. 事件命名规范

- 事件名使用kebab-case（短横线命名）：`page-change`
- 使用`$emit`触发事件：`this.$emit('page-change', newPage)`
- 父组件监听事件：`@page-change="handler"`

### 4. 样式封装策略

```vue
<style scoped>
/* 作用域样式：只影响当前组件 */
.component {
  color: red;
}
</style>

<style>
/* 全局样式：影响所有地方 */
.global-helper {
  /* ... */
}
</style>

<style scoped lang="less">
/* 使用预处理器 */
@import "~@/styles/var.less";
.component {
  color: @primary;
}
</style>
```

### 5. 组件测试方法

为每个组件创建对应的`test.vue`文件：

```vue
<template>
  <div class="test-container">
    <ComponentName
      :prop1="value1"
      :prop2="value2"
      @event1="handleEvent1"
    />
  </div>
</template>

<script>
import ComponentName from "./";
export default {
  components: { ComponentName },
  data() {
    return {
      value1: "test",
      value2: 100
    };
  },
  methods: {
    handleEvent1(payload) {
      console.log("Event triggered:", payload);
    }
  }
};
</script>

<style scoped>
.test-container {
  width: 500px;
  height: 400px;
  border: 2px solid;
  margin: 0 auto;
}
</style>
```

## 总结

### 核心知识点回顾

1. **组件Props**：父组件向子组件传递数据
2. **组件Events**：子组件向父组件传递消息
3. **计算属性**：基于响应式数据进行计算
4. **条件渲染**：`v-if`、`v-show`控制显示
5. **列表渲染**：`v-for`遍历数组或对象
6. **样式绑定**：`:class`、`:style`动态绑定
7. **事件处理**：`@click`、`$emit`处理事件
8. **生命周期**：组件创建、挂载、更新、销毁
9. **作用域样式**：`scoped`隔离组件样式
10. **组件通信**：父子组件、兄弟组件通信

- 组件插槽（Slots）
- 动态组件和异步组件
- 组件混入（Mixins）
- 自定义指令
- 组件的provide/inject
- Vuex状态管理
- Vue Router路由
