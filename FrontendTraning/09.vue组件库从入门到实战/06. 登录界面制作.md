# 06. 登录界面制作

## 课程概述

本节课程主要制作博客管理系统的登录界面，包括界面设计、表单验证、验证码功能等，使用 Element UI 组件库快速构建美观实用的登录页面。

## 登录界面设计

### 1. 整体布局

登录界面采用居中布局，包含以下元素：

- 页面标题："个人空间后台系统"
- 管理员账号输入框
- 密码输入框（带显示/隐藏功能）
- 验证码输入框
- 验证码图片（点击刷新）
- 7天内免登录复选框
- 登录按钮

### 2. 界面风格

- **背景色**：深色背景 (#2d3a4b)
- **输入框**：半透明背景，白色文字
- **按钮**：Element UI 主题色
- **响应式**：适配不同屏幕尺寸

## 核心代码实现

### 1. 模板结构

```vue
<template>
  <div class="login-container">
    <el-form
      ref="loginForm"
      :model="loginForm"
      :rules="loginRules"
      class="login-form"
      auto-complete="on"
      label-position="left"
    >
      <!-- 页面标题 -->
      <div class="title-container">
        <h3 class="title">个人空间后台系统</h3>
      </div>

      <!-- 管理员账号 -->
      <el-form-item prop="loginId">
        <span class="svg-container">
          <svg-icon icon-class="user" />
        </span>
        <el-input
          ref="loginId"
          v-model="loginForm.loginId"
          placeholder="请输入管理员账号"
          name="loginId"
          type="text"
          tabindex="1"
          auto-complete="on"
        />
      </el-form-item>

      <!-- 密码输入框 -->
      <el-form-item prop="loginPwd">
        <span class="svg-container">
          <svg-icon icon-class="password" />
        </span>
        <el-input
          :key="passwordType"
          ref="password"
          v-model="loginForm.loginPwd"
          :type="passwordType"
          placeholder="请输入管理员密码"
          name="password"
          tabindex="2"
          auto-complete="on"
          @keyup.enter.native="handleLogin"
        />
        <span class="show-pwd" @click="showPwd">
          <svg-icon
            :icon-class="passwordType === 'password' ? 'eye' : 'eye-open'"
          />
        </span>
      </el-form-item>

      <!-- 验证码 -->
      <div class="captchaContainer">
        <el-form-item prop="captcha" class="captchaInputer">
          <span class="svg-container">
            <svg-icon icon-class="nested" />
          </span>
          <el-input
            ref="captcha"
            v-model="loginForm.captcha"
            placeholder="请输入验证码"
            name="captcha"
            type="text"
            tabindex="3"
            auto-complete="on"
          />
        </el-form-item>
        <div
          class="captchaImg"
          v-html="svg"
          @click="getCaptchaFunc"
        ></div>
      </div>

      <!-- 7 天内免登录 -->
      <div style="margin-bottom:15px">
        <el-checkbox v-model="loginForm.checked">7 天内免登录</el-checkbox>
      </div>

      <!-- 登录按钮 -->
      <el-button
        :loading="loading"
        type="primary"
        style="width: 100%; margin-bottom: 30px"
        @click.native.prevent="handleLogin"
        >登录</el-button
      >
    </el-form>
  </div>
</template>
```

### 2. 数据结构

```javascript
data() {
  return {
    svg: '',                    // 验证码 SVG
    loginForm: {
      loginId: '',             // 管理员账号
      loginPwd: '',            // 登录密码
      captcha: '',             // 验证码
      checked: true           // 7天内免登录
    },
    loginRules: {
      // 表单验证规则
      loginId: [{
        required: true,
        trigger: 'blur',
        message: '请输入管理员账号'
      }],
      loginPwd: [{
        required: true,
        trigger: 'blur',
        message: '请输入管理员密码'
      }],
      captcha: [{
        required: true,
        trigger: 'blur',
        message: '请输入验证码'
      }]
    },
    passwordType: 'password',   // 密码显示类型
    loading: false             // 登录加载状态
  };
}
```

### 3. 验证码功能

#### API 接口封装

`src/api/captcha.js`：

```javascript
import request from '@/utils/request'

// 获取验证码
export function getCaptcha() {
  return request({
    url: '/res/captcha',
    method: 'get'
  })
}
```

#### 获取验证码方法

```javascript
methods: {
  // 获取验证码
  getCaptchaFunc() {
    getCaptcha().then(res => {
      this.svg = res;  // 直接使用 SVG 字符串
    })
  }
}
```

#### 组件挂载时获取验证码

```javascript
created() {
  // 页面加载时获取验证码
  this.getCaptchaFunc();
}
```

### 4. 密码显示/隐藏功能

```javascript
methods: {
  showPwd() {
    if (this.passwordType === "password") {
      this.passwordType = "";      // 显示密码
    } else {
      this.passwordType = "password"; // 隐藏密码
    }
    this.$nextTick(() => {
      this.$refs.password.focus();  // 重新聚焦
    });
  }
}
```

### 5. 表单验证

```javascript
methods: {
  handleLogin() {
    this.$refs.loginForm.validate((valid) => {
      if (valid) {
        // 验证通过，执行登录逻辑
        console.log("表单验证通过");
      } else {
        // 验证失败
        console.log("表单验证失败");
        return false;
      }
    });
  }
}
```

## 关键功能点

### 1. 表单验证

- **必填验证**：所有字段都设置为必填
- **触发时机**：失去焦点时触发验证
- **错误提示**：自定义错误提示信息

### 2. 验证码功能

- **自动获取**：页面加载时自动获取验证码
- **点击刷新**：点击验证码图片可刷新
- **SVG 渲染**：使用 v-html 指令渲染 SVG

### 3. 密码显示

- **图标切换**：根据密码类型显示不同图标
- **自动聚焦**：切换后重新聚焦输入框
- **用户体验**：提供更好的输入体验

### 4. 回车登录

- **全局回车**：密码输入框支持回车登录
- **阻止默认**：使用 .prevent 修饰符

## 用户交互流程

### 1. 页面加载

1. 页面加载完成
2. 自动获取验证码
3. 验证码显示在右侧

### 2. 用户输入

1. 用户输入账号
2. 用户输入密码（可切换显示/隐藏）
3. 用户输入验证码
4. 可选择是否7天内免登录

### 3. 表单验证

1. 点击登录按钮
2. 触发表单验证
3. 检查所有必填字段
4. 显示验证错误信息

### 4. 验证码刷新

1. 用户点击验证码图片
2. 重新请求验证码
3. 更新验证码显示

## 常见问题

### 1. 验证码不显示

- 检查接口是否正常返回
- 检查 SVG 字符串格式
- 检查 v-html 渲染

### 2. 样式错乱

- 检查 Element UI 版本
- 检查样式覆盖顺序
- 检查浏览器兼容性

### 3. 表单验证不生效

- 检查 prop 属性是否正确
- 检查 rules 配置是否正确
- 检查验证时机设置
