# 第7章：首页标语 part1（页面渲染）

## 本章概述

本章将学习如何实现首页标语管理功能的页面渲染部分，包括数据获取、表格展示、图片预览等核心功能。

## 学习目标

- 掌握 Element UI 表格组件的使用
- 学会处理图片路径拼接
- 理解组件间数据传递
- 实现数据列表的渲染展示

## 核心内容

### 1. 创建首页标语页面

首先在 `src/views/banner/` 目录下创建 `index.vue` 文件：

```vue
<template>
  <div class="app-container">
    <el-table :data="data" style="width: 100%" border>
      <!-- 序号列 -->
      <el-table-column prop="date" label="序号" width="60" align="center">
        <template slot-scope="scope">{{ scope.$index + 1 }}</template>
      </el-table-column>
      
      <!-- 标题列 -->
      <el-table-column prop="title" label="标题" width="150" align="center">
        <template slot-scope="scope">{{ scope.row.title }}</template>
      </el-table-column>
      
      <!-- 描述列 -->
      <el-table-column prop="description" label="描述">
        <template slot-scope="scope">{{ scope.row.description }}</template>
      </el-table-column>
      
      <!-- 中图预览列 -->
      <el-table-column label="中图预览" align="center">
        <template slot-scope="scope">
          <el-image
            style="width: 100px;"
            :src="scope.row.midImg2"
            fit="fill">
          </el-image>
        </template>
      </el-table-column>
      
      <!-- 大图预览列 -->
      <el-table-column label="大图预览" align="center">
        <template slot-scope="scope">
          <el-image
            style="width: 100px;"
            :src="scope.row.bigImg2"
            fit="fill">
          </el-image>
        </template>
      </el-table-column>
      
      <!-- 操作列 -->
      <el-table-column label="操作" align="center">
        <template slot-scope="scope">
          <el-tooltip
            class="item"
            effect="dark"
            content="编辑"
            placement="top"
            :hide-after="2000"
          >
            <el-button
              type="primary"
              icon="el-icon-edit"
              circle
              size="mini"
            ></el-button>
          </el-tooltip>
        </template>
      </el-table-column>
    </el-table>
  </div>
</template>

<script>
import { getBanner } from "@/api/banner.js";
import { server_URL } from '@/urlConfig.js';

export default {
  data() {
    return {
      data: [], // 存储数据
    };
  },
  created() {
    this.fetchData();
  },
  methods: {
    fetchData() {
      getBanner().then((res) => {
        this.data = res.data;
        // 处理图片路径，拼接服务器地址
        for(var item of this.data){
          item.midImg2 = server_URL + item.midImg;
          item.bigImg2 = server_URL + item.bigImg;
        }
      });
    },
  },
};
</script>
```

### 2. API 接口封装

在 `src/api/banner.js` 中封装获取首页标语的接口：

```javascript
import request from '@/utils/request'

// 获取首页标语
export function getBanner() {
  return request({
    url: '/api/banner',
    method: 'get',
  })
}
```

### 3. 技术要点解析

#### 3.1 Element UI 表格组件

- **el-table**：表格容器，通过 `:data` 绑定数据
- **el-table-column**：表格列，定义每一列的显示内容
- **slot-scope**：作用域插槽，用于访问当前行的数据

#### 3.2 图片显示

- **el-image**：Element UI 的图片组件
- **fit 属性**：控制图片的填充方式
- **路径拼接**：将服务器地址与相对路径拼接成完整的图片URL

#### 3.3 序号显示

使用 `scope.$index + 1` 来显示行号，从1开始计数。

#### 3.4 操作按钮

- **el-tooltip**：工具提示组件，鼠标悬停时显示提示文字
- **el-button**：按钮组件，设置圆形、图标和尺寸

## 服务器配置

在 `src/urlConfig.js` 中配置服务器地址：

```javascript
export const server_URL = "http://localhost:7001";
```

```javascript
}
```
