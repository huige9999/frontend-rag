# 第17章 关于我、设置和个人信息模块

## 章节概述

本章主要讲解三个重要的配置模块：关于我、网站设置和个人信息设置。这些模块用于管理博主个人信息、网站基本信息和管理员账户信息。

## 核心内容

### 1. 模块功能概述

#### 1.1 关于我模块

用于设置博主的"关于我"页面内容，通常是一个介绍性的 URL 或文本内容。

#### 1.2 网站设置模块

用于配置网站的各项基本信息：
- 网站标题：显示在浏览器标签栏
- 邮箱：联系邮箱
- 备案号：网站备案信息
- 网站头像：网站 Logo
- 网站图标：Favicon 图标
- GitHub 信息：GitHub 名称和地址
- QQ 信息：QQ 号码和二维码
- 微信信息：微信号和二维码

#### 1.3 个人信息设置模块

用于管理员修改个人信息和密码：
- 用户名：管理员登录名称
- 旧密码：当前密码验证
- 新密码：设置新密码
- 密码确认：确认新密码

### 2. 关于我页面

#### 2.1 页面功能

- 显示当前的"关于我"内容（URL 或文本）
- 编辑模式：修改内容
- 只读模式：查看内容
- 保存修改：更新服务器数据

#### 2.2 完整代码实现

```vue
<template>
  <div class="app-container">
    <div class="block">关于我</div>
    <el-input
      v-model="url"
      placeholder="请输入内容"
      :disabled="isDisabled"
    ></el-input>
    <el-button type="primary" style="margin-top: 15px" @click="clickHandle">
      {{btnContent}}
    </el-button>
  </div>
</template>

<script>
import { getAbout, editAbout } from "@/api/about.js";

export default {
  data() {
    return {
      url: "",                // 关于我的内容
      isDisabled: true,      // 输入框是否禁用
      btnContent: '编辑'      // 按钮文本
    };
  },
  created() {
    this.fetchData();
  },
  methods: {
    // 获取数据
    fetchData() {
      getAbout().then(({ data }) => {
        this.url = data;
      });
    },
    
    // 按钮点击处理
    clickHandle() {
      if(this.btnContent == '编辑') {
        // 进入编辑模式
        this.btnContent = '完成';
        this.isDisabled = !this.isDisabled;
      } else {
        // 完成编辑，保存内容
        if(this.url) {
          this.btnContent = '编辑';
          this.isDisabled = !this.isDisabled;
          editAbout({url: this.url}).then(res => {
            console.log(res);
          });
          this.$message({
            message: '更改成功',
            type: 'success'
          });
        } else {
          this.$message({
            message: '输入框不能为空',
            type: 'warning'
          });
        }
      }
    }
  },
};
</script>

<style scoped>
.block {
  margin: 15px 0;
  font-weight: 100;
}
</style>
```

### 4. 个人信息设置页面

#### 4.1 页面功能

个人信息设置页面用于管理员修改：
- 用户名：显示和修改用户名
- 旧密码：验证当前密码
- 新密码：设置新密码
- 密码确认：确认新密码输入正确

#### 4.2 完整代码实现

```vue
<template>
  <div class="app-container">
    <el-form
      :model="adminInfo"
      :rules="setRules"
      ref="ruleForm"
      label-width="100px"
      width="500px"
    >
      <el-form-item label="用户名">
        <el-input
          v-model="adminInfo.name"
          placeholder="请输入用户名"
        ></el-input>
      </el-form-item>
      
      <el-form-item label="旧密码" prop="oldLoginPwd">
        <el-input
          v-model="adminInfo.oldLoginPwd"
          placeholder="请输入旧密码"
          type="password"
        ></el-input>
      </el-form-item>

      <el-form-item label="新密码" prop="loginPwd">
        <el-input
          v-model="adminInfo.loginPwd"
          placeholder="请输入新密码"
          type="password"
        ></el-input>
      </el-form-item>

      <el-form-item label="新密码确认" prop="loginPwdConfirm">
        <el-input
          v-model="adminInfo.loginPwdConfirm"
          placeholder="请确认新密码"
          type="password"
        ></el-input>
      </el-form-item>

      <el-button type="primary" style="margin-top: 15px" @click="handleClick">
        修改
      </el-button>
      <el-button type="danger" style="margin-top: 15px" @click="handleBack">
        返回
      </el-button>
    </el-form>
  </div>
</template>

<script>
import { getInfo, setUser } from "@/api/user.js";

export default {
  data() {
    // 自定义验证规则：验证两次密码是否一致
    var validatePass2 = (rule, value, callback) => {
      if (value === "") {
        callback(new Error("请再次输入新密码"));
      } else if (value !== this.adminInfo.loginPwd) {
        callback(new Error("两次输入密码不一致!"));
      } else {
        callback();
      }
    };
    
    return {
      adminInfo: {
        id: "",
        loginId: "",         // 登录 ID
        name: "",            // 用户名
        oldLoginPwd: "",     // 旧密码
        loginPwd: "",        // 新密码
        loginPwdConfirm: "", // 密码确认
      },
      // 表单验证规则
      setRules: {
        oldLoginPwd: [
          { required: true, trigger: "blur", message: "请输入旧的密码" },
        ],
        loginPwd: [
          { required: true, trigger: "blur", message: "请输入新的密码" },
        ],
        loginPwdConfirm: [
          { required: true, trigger: "blur", message: "请再次输入新密码" },
          { validator: validatePass2, trigger: "blur" },
        ],
      },
    };
  },
  created() {
    this.fetchData();
  },
  methods: {
    // 获取管理员信息
    fetchData() {
      getInfo().then(res => {
        this.adminInfo = res.data;
      })
    },
    
    // 返回首页
    handleBack() {
      this.$router.push('/');
    },
    
    // 修改处理
    handleClick() {
      // 验证必填项
      if(this.adminInfo.name && this.adminInfo.loginPwd && this.adminInfo.oldLoginPwd) {
        setUser(this.adminInfo).then(res => {
          // 处理响应
          if(typeof res === 'string') {
            const resp = JSON.parse(res);
            this.$message.error(resp.msg);
          } else {
            // 修改成功，退出登录
            this.$message.success('修改密码成功');
            this.$store.dispatch('user/logout').then(() => {
              this.$router.push(`/login?redirect=${this.$route.fullPath}`);
            })
          }
        })
      } else {
        this.$message.error('请填写所有的项目');
      }
    },
  },
};
</script>

<style scoped>
.app-container {
  width: 500px;
}
</style>
```

## 重点知识点

### 1. 输入框状态切换

通过 `disabled` 属性控制输入框的可编辑状态：

```vue
<el-input
  v-model="url"
  :disabled="isDisabled"
></el-input>
```

### 2. 按钮文本动态切换

根据当前状态显示不同的按钮文本：

```javascript
if(this.btnContent == '编辑') {
  this.btnContent = '完成';
} else {
  this.btnContent = '编辑';
}
```

### 3. 分割线组件

使用 `el-divider` 组件分割内容区域：

```vue
<el-divider></el-divider>
```

### 4. 图片预览

使用 `el-image` 组件显示图片：

```vue
<el-image
  style="width: 100px; height: 100px"
  :src="form.avatar"
></el-image>
```

### 5. 表单验证

使用 Element UI 的表单验证：

```javascript
// 验证规则
setRules: {
  oldLoginPwd: [
    { required: true, trigger: "blur", message: "请输入旧的密码" },
  ],
  loginPwd: [
    { required: true, trigger: "blur", message: "请输入新的密码" },
  ],
  loginPwdConfirm: [
    { required: true, trigger: "blur", message: "请再次输入新密码" },
    { validator: validatePass2, trigger: "blur" },
  ],
}
```

### 6. 自定义验证器

创建自定义验证规则：

```javascript
var validatePass2 = (rule, value, callback) => {
  if (value === "") {
    callback(new Error("请再次输入新密码"));
  } else if (value !== this.adminInfo.loginPwd) {
    callback(new Error("两次输入密码不一致!"));
  } else {
    callback();
  }
};
```

### 7. 双数据对象

使用两个数据对象分别处理显示和编辑：

```javascript
// 显示数据
form: { ... },

// 编辑数据
form2: { ... },

// 编辑时复制数据
this.form2 = {...this.form};
```

## 技术要点

1. **状态管理**：通过数据属性管理页面状态
2. **表单验证**：使用 Element UI 的验证机制
3. **图片上传**：集成 Upload 组件
4. **数据同步**：保持显示和编辑数据的一致性
5. **用户反馈**：操作后的消息提示
6. **安全退出**：修改密码后强制重新登录

## 注意事项

1. 修改密码后需要重新登录
2. 表单验证要考虑所有必填项
3. 图片上传需要使用 Upload 组件
4. 编辑和显示数据要分开处理
5. 网站设置信息较多，要注意分组展示
6. 对话框可以使用全屏模式提高编辑体验

## 最佳实践

1. **用户体验**：使用分割线和分组提高可读性
2. **数据验证**：前端和后端都要进行验证
3. **状态管理**：清晰的状态切换逻辑
4. **错误处理**：友好的错误提示
5. **安全考虑**：敏感操作需要验证旧密码
