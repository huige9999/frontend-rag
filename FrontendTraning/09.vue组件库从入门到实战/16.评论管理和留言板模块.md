# 第16章 评论管理和留言板模块

## 章节概述

本章主要讲解评论管理和留言板模块的实现，包括评论列表展示、留言列表展示、分页功能、删除功能等内容。这两个模块是博客系统中与用户交互的重要组成部分。

## 核心内容

### 1. 模块功能概述

#### 1.1 评论管理模块

评论管理模块主要功能：
- 评论列表展示：显示所有用户对文章的评论
- 评论者信息：显示头像、昵称等
- 评论内容：显示评论文本内容
- 评论文章：显示评论所属文章
- 评论时间：显示评论创建时间
- 删除评论：管理员可删除不当评论

#### 1.2 留言板模块

留言板模块主要功能：
- 留言列表展示：显示所有用户留言
- 留言者信息：显示头像、昵称等
- 留言内容：显示留言文本内容
- 留言时间：显示留言创建时间
- 删除留言：管理员可删除不当留言

### 2. 评论管理页面

#### 2.1 页面结构

评论管理页面包含以下元素：
- 数据表格：展示评论信息
- 序号列：显示评论序号
- 头像列：显示评论者头像
- 昵称列：显示评论者昵称
- 评论文章列：显示被评论的文章
- 评论内容列：显示评论文本
- 评论日期列：显示评论时间
- 操作列：删除按钮
- 分页组件：实现分页功能

#### 2.2 完整代码实现

```vue
<template>
  <div class="app-container">
    <!-- 表格 -->
    <el-table
      v-loading="listLoading"
      :data="data"
      element-loading-text="加载数据中"
      border
      fit
      highlight-current-row
    >
      <el-table-column align="center" label="序号" width="60">
        <template slot-scope="scope">{{
          scope.$index + (currentPage - 1) * eachPage + 1
        }}</template>
      </el-table-column>
      
      <el-table-column label="头像" align="center" width="80">
        <template slot-scope="scope">
           <el-avatar shape="square" size="small" :src="scope.row.avatar"></el-avatar>
        </template>
      </el-table-column>
      
      <el-table-column label="昵称" align="center" width="150">
        <template slot-scope="scope">{{ scope.row.nickname }}</template>
      </el-table-column>
      
       <el-table-column label="评论文章" align="center" width="150">
        <template slot-scope="scope">{{ scope.row.blog.title }}</template>
      </el-table-column>
      
      <el-table-column label="评论内容" align="center">
        <template slot-scope="scope">{{ scope.row.content }}</template>
      </el-table-column>
      
      <el-table-column label="评论日期" align="center" width="230">
        <template slot-scope="scope">{{ scope.row.createDate }}</template>
      </el-table-column>
      
      <el-table-column
        label="操作"
        align="center"
        width="160"
        class-name="small-padding fixed-width"
      >
        <template slot-scope="scope">
          <el-tooltip
            class="item"
            effect="dark"
            content="删除"
            placement="top"
            :hide-after="1500"
          >
            <el-button
              type="danger"
              icon="el-icon-delete"
              circle
              size="mini"
              @click="deleteComment(scope.row)"
            ></el-button>
          </el-tooltip>
        </template>
      </el-table-column>
    </el-table>
    
    <!-- 分页 -->
    <div class="block">
      <el-pagination
        style="margin-top: 30px"
        background
        :page-size="eachPage"
        :page-sizes="[5, 10, 15, 20]"
        layout="prev, pager, next, total, ->,sizes, jumper"
        :total="count"
        :current-page.sync="pagerCurrentPage"
        @prev-click="prevClickHandle"
        @next-click="nextClickHandle"
        @current-change="currentChangeHandle"
        @size-change="handleSizeChange"
      >
      </el-pagination>
    </div>
  </div>
</template>

<script>
import { getComment, delComment } from "@/api/comment.js";
import { formatDate } from "@/utils/tools";

export default {
  data() {
    return {
      listLoading: false,
      currentPage: 1,          // 当前页码
      eachPage: 5,            // 每页显示条数
      totalPage: 0,           // 总页数
      count: 0,               // 数据总条数
      data: [],               // 数据详情
      pagerCurrentPage: 1,    // 分页栏当前页码
    };
  },
  created() {
    this.fetchData();
  },
  methods: {
    // 获取评论数据
    fetchData() {
      this.listLoading = true;
      getComment(this.currentPage, this.eachPage).then(({ data }) => {
        this.listLoading = false;
        this.data = data.rows;
        // 处理时间格式
        for(let i of this.data){
          i.createDate = formatDate(i.createDate);
        }
        this.count = data.total;
        this.totalPage = Math.ceil(this.count / this.eachPage);
        
        // 处理边界情况
        if (this.currentPage > this.totalPage) {
          this.currentPage = this.totalPage;
          this.fetchData();
        }
      });
    },
    
    // 删除评论
    deleteComment({ id }) {
      this.$confirm("是否删除此条评论信息?", "提示", {
        confirmButtonText: "确定",
        cancelButtonText: "取消",
        type: "warning",
      })
        .then(() => {
          this.$message({
            type: "success",
            message: "删除成功!",
          });
          delComment(id).then((res) => {
            this.fetchData();
          });
        })
        .catch(() => {
          this.$message({
            type: "info",
            message: "已取消删除",
          });
        });
    },
    
    // 点击上一页
    prevClickHandle() {
      this.currentPage -= 1;
    },
    
    // 点击下一页
    nextClickHandle() {
      this.currentPage += 1;
    },
    
    // 点击页码
    currentChangeHandle(pageNum) {
      this.currentPage = ~~pageNum;
      this.fetchData();
    },
    
    // 改变每页显示条数
    handleSizeChange(pagerNum) {
      this.currentPage = 1;
      this.pagerCurrentPage = 1;
      this.eachPage = ~~pagerNum;
      this.fetchData();
    },
  },
};
</script>
```

### 4. API 接口封装

#### 4.1 评论 API

```javascript
// src/api/comment.js
import request from '@/utils/request'

// 分页获取评论
export function getComment(page = 1, limit = 10) {
  return request({
    url: '/api/comment',
    method: 'GET',
    params: {
      page,
      limit
    }
  })
}

// 删除评论
export function delComment(id) {
  return request.delete(`/api/comment/${id}`);
}
```

#### 4.2 留言 API

```javascript
// src/api/message.js
import request from '@/utils/request'

// 分页获取留言
export function getMessages(page = 1, limit = 10) {

    params: {
      page,
      limit,
    },
  });
}

// 删除留言
export function delMessage(id) {
  return request.delete(`/api/message/${id}`);
```

## 重点知识点

### 1. 头像组件使用

使用 Element UI 的 Avatar 组件显示用户头像：

```vue
<el-avatar 
  shape="square" 
  size="small" 
  :src="scope.row.avatar">
</el-avatar>
```

- `shape`：头像形状，可选 `circle`（圆形）、`square`（方形）
- `size`：头像大小
- `:src`：图片地址

### 2. 序号计算

正确的序号计算公式：

```vue
{{ scope.$index + (currentPage - 1) * eachPage + 1 }}
```

这个公式确保：
- 第一页序号：1, 2, 3, 4, 5
- 第二页序号：6, 7, 8, 9, 10
- 以此类推...

### 3. 分页组件配置

完整的分页组件配置：

```vue
<el-pagination
  background                    // 背景颜色
  :page-size="eachPage"        // 每页显示条数
  :page-sizes="[5, 10, 15, 20]" // 可选的每页条数
  layout="prev, pager, next, total, ->,sizes, jumper" // 布局
  :total="count"               // 总条数
  :current-page.sync="pagerCurrentPage" // 当前页码
  @prev-click="prevClickHandle"     // 上一页事件
  @next-click="nextClickHandle"     // 下一页事件
  @current-change="currentChangeHandle" // 页码改变事件
  @size-change="handleSizeChange"   // 每页条数改变事件
>
</el-pagination>
```

### 4. 时间格式化

使用工具函数格式化时间：

```javascript
import { formatDate } from "@/utils/tools";

// 格式化每条数据的时间
for(let i of this.data){
  i.createDate = formatDate(i.createDate);
}
```

### 5. 边界情况处理

处理当前页超过总页数的情况：

```javascript
if (this.currentPage > this.totalPage) {
  this.currentPage = this.totalPage;
  this.fetchData();
}
```

### 6. 分页逻辑

完整的分页逻辑：

```javascript
// 改变每页显示条数
handleSizeChange(pagerNum) {
  this.currentPage = 1;           // 重置到第一页
  this.pagerCurrentPage = 1;      // 重置分页组件
  this.eachPage = ~~pagerNum;     // 更新每页条数
  this.fetchData();               // 重新获取数据
}
```

## 技术要点

1. **头像显示**：使用 Avatar 组件展示用户头像
2. **分页功能**：完整的分页逻辑实现
3. **数据格式化**：时间戳转换为可读格式
4. **删除确认**：防止误删的确认对话框
5. **边界处理**：处理分页的边界情况
6. **工具提示**：使用 Tooltip 组件提供按钮说明

## 注意事项

1. 分页组件需要同步两个页码变量
2. 删除操作需要用户确认
3. 时间格式要统一处理
4. 序号计算要考虑分页偏移
5. 头像路径可能需要拼接服务器地址
6. 每页条数改变时需要重置页码

## 最佳实践

1. **统一样式**：评论和留言页面保持一致的样式
2. **复用逻辑**：提取公共的分页逻辑
3. **用户体验**：添加加载状态和操作反馈
4. **错误处理**：处理网络错误和边界情况
5. **代码复用**：考虑将分页逻辑提取为 mixin
