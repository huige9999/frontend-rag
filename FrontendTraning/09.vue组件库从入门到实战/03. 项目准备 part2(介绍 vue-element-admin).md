# 03. 项目准备 part2 - 介绍 vue-element-admin

## 课程概述

本节课程主要介绍 vue-element-admin 这一优秀的后台管理系统模板，了解其特点、优势以及如何使用它来快速搭建现代化的后台管理系统。

## 什么是 vue-element-admin？

vue-element-admin 是一个基于 Vue.js 2.0 和 Element UI 的后台管理系统模板，它提供了一套完整的解决方案，包括登录、权限管理、多语言、主题切换等功能。

### 主要特点

- **技术栈先进**：基于 Vue 2.0 + Vue Router + Vuex + Element UI
- **功能完整**：提供了后台管理系统常用的功能模块
- **代码规范**：遵循 ESLint 代码规范，便于团队协作
- **响应式设计**：支持多种设备和屏幕尺寸
- **主题定制**：支持多主题切换
- **国际化**：内置多语言支持

### 官方推荐项目

1. **vue-element-admin** - 完整版，功能丰富
2. **vue-admin-template** - 简化版，适合二次开发
3. **electron-vue-admin** - 桌面应用版本

## 课程资源

本节课程提供了两个重要的项目模板：

### 1. vue-admin-template-master.zip

这是一个简化版本的后台管理系统模板，特点：

- 移除了很多不必要的功能
- 保留了核心功能
- 适合作为项目起点
- 易于二次开发

### 2. vue-element-admin-master.zip

这是完整的后台管理系统模板，特点：

- 功能完整丰富
- 包含各种常见业务场景
- 适合学习和参考
- 可直接使用或选择性集成

## 如何选择？

### 选择 vue-admin-template 的情况

- 你需要快速开始一个新项目
- 你只想保留核心功能
- 你希望有更大的定制空间
- 项目需求相对简单

### 选择 vue-element-admin 的情况

- 你需要完整的功能参考
- 你想学习最佳实践
- 项目需求复杂且全面
- 需要快速演示
