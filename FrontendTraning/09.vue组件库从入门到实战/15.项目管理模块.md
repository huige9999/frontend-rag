# 第15章 项目管理模块

## 章节概述

本章主要讲解项目管理模块的实现，包括项目列表展示、项目添加、项目编辑和项目删除等功能。项目管理是展示个人作品和技能的重要模块。

## 核心内容

### 1. 项目管理模块功能概述

项目管理模块包含以下主要功能：
- 项目列表展示：展示所有项目的基本信息
- 项目添加：添加新的项目到系统
- 项目编辑：修改现有项目的信息
- 项目删除：删除不需要的项目
- 图片预览：查看项目预览图
- 外部链接：跳转到项目演示和 GitHub 仓库

### 2. 项目列表页面

#### 2.1 页面结构

项目列表页面包含以下元素：
- 数据表格：展示项目信息
- 序号列：显示项目序号
- 项目名称：可点击跳转到项目演示
- 项目描述：显示项目的主要特点
- 预览图：展示项目截图
- 操作按钮：GitHub、编辑、删除

```vue
<template>
  <div class="app-container">
    <!-- 数据表格 -->
    <el-table
      v-loading="listLoading"
      :data="data"
      element-loading-text="加载数据中"
      border

              size="mini"

      title="请编辑该项目信息"

        <!-- 项目名称 -->
        <el-form-item label="项目名称">

          </el-select>
        </el-form-item>

      <div slot="footer" class="dialog-footer">

          >确 定</el-button
        >
      </div>
    </el-dialog>
  </div>

<script>
import { getProject, setProject, delProject } from "@/api/project.js";
import { server_URL } from "@/urlConfig.js";
import Upload from "@/components/Upload.vue";

export default {
  data() {
    return {
      data: [],                // 存储数据
      listLoading: false,
      srcList: [],             // 图片预览列表
      dialogFormVisible: false, // 编辑对话框显示状态
      form: {
        name: "",
        description: "",
        url: "",
        github: "",
        thumb: "",
        order: 1,
      },
    };
  },
  components: {
    Upload,
  },
  created() {
    this.fetchData();
  },
  methods: {
    // 获取项目列表数据
    fetchData() {
      this.listLoading = true;
      getProject().then(({ data }) => {
        this.listLoading = false;
        this.data = data;
        // 处理图片路径
        for (let i of this.data) {
          i.thumb2 = server_URL + i.thumb;
          this.srcList.push(i.thumb2);
        }
      });
    },
    
    // 打开 GitHub 链接
    openGitHubHandle(projectInfo) {
      window.open(projectInfo.github);
    },
    
    // 编辑项目
    editProjectHandle(projectInfo) {
      this.dialogFormVisible = true;
      this.form = {...projectInfo, description: projectInfo.description.toString()}
    },
    
    // 删除项目
    deleteProjectHandle(projectInfo) {
      this.$confirm("确定要删除此项目吗？", "提示", {
        confirmButtonText: "确定",
        cancelButtonText: "取消",
        type: "warning",
      })
        .then(() => {
          delProject(projectInfo.id).then(() => {
            this.fetchData();
            this.$message({
              type: "success",
              message: "删除成功!",
            });
          });
        })
        .catch(() => {
          this.$message({
            type: "info",
            message: "已取消删除",
          });
        });
    },
    
    // 确认修改
    confirmEditProjectHandle() {
      let obj = {...this.form};
      obj.description = this.form.description.split(',');
      obj.order = parseInt(this.form.order);

      setProject(obj.id, obj).then(()=>{
        this.dialogFormVisible = false;
        this.fetchData();
        this.$message.success('修改成功');
      })
    },
  },
};
</script>

}
</style>
```

### 4. 项目 API 接口

#### 4.1 API 接口封装

```javascript
// src/api/project.js
import request from '@/utils/request'

// 新增项目
export function addProject(data) {
  return request({
    url: '/api/project',
    method: 'post',
    data
  })
}

// 获取项目
export function getProject() {
  return request.get("/api/project");
}

// 修改项目
export function setProject(id, data) {
  return request({
    url: `/api/project/${id}`,
    method: 'put',
    data
  })
}

// 删除项目
export function delProject(id) {
  return request({
    url: `/api/project/${id}`,
    method: 'DELETE',
  })
}
```

## 技术要点

1. **表格展示**：使用 Element UI 的 Table 组件展示项目列表
2. **图片预览**：使用 Image 组件实现图片点击预览
3. **对话框编辑**：使用 Dialog 组件实现项目编辑
4. **图标字体**：使用 iconfont 实现自定义图标
5. **工具提示**：使用 Tooltip 组件提供按钮说明
6. **数据转换**：正确处理数组与字符串的转换

## 注意事项

1. 项目描述要以英文逗号分隔
2. 图片路径需要拼接服务器地址
3. 删除操作需要二次确认
4. 编辑时要注意数据类型转换
5. GitHub 图标需要引入图标字体文件
6. 图片预览需要构建预览列表

## 最佳实践

1. **数据格式化**：在获取数据后立即进行格式化处理
2. **用户体验**：添加加载状态和操作反馈
3. **数据验证**：提交前验证必填项
4. **错误处理**：添加适当的错误提示
5. **代码复用**：提取公共逻辑和组件
