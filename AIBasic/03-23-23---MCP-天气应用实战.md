# [MCP]天气应用实战

## 课程概述

本课程介绍如何使用 Model Context Protocol (MCP) 构建一个天气查询应用，通过集成和风天气API，实现对指定城市和日期的天气查询功能。

## 核心概念

### MCP (Model Context Protocol)
- 允许AI模型与外部工具和服务进行交互
- 通过标准化的工具注册和调用机制实现扩展能力
- 使用stdio传输通道进行通信

## 核心代码解析

### 1. MCP服务器设置 (server.js)

```javascript
import { McpServer } from "@modelcontextprotocol/sdk/server/mcp.js";
import { StdioServerTransport } from "@modelcontextprotocol/sdk/server/stdio.js";
import { z } from "zod";
import { getWeather } from "./wetherHandler.js";

// 创建MCP服务器实例
const server = new McpServer({
  name: "wether-mcp-server",
  version: "1.0.0",
  description: "一个提供天气查询的MCP服务器",
});

// 注册天气查询工具
server.registerTool(
  "getWether",
  {
    title: "查询天气",
    description: "查询指定城市和日期的天气情况",
    inputSchema: {
      city: z.string().describe("城市名，例如：成都、上海、北京"),
      date: z.string().describe("日期的描述，例如 今天、明天、2025-7-20"),
    },
  },
  async ({ city, date }) => {
    try {
      const result = await getWeather(city, date);
      return {
        content: [
          {
            type: "text",
            text: result,
          },
        ],
      };
    } catch (err) {
      console.error(err);
      return {
        content: [
          {
            type: "text",
            text: err.message,
          },
        ],
      };
    }
  }
);

// 创建stdio传输通道并连接
const transport = new StdioServerTransport();
server.connect(transport);
```

### 2. 天气查询业务逻辑 (wetherHandler.js)

```javascript
const HEFENG_API_KEY = "52999ff62e2747febfd29b6db97b7bd7";

// 日期格式化工具
export function formatDate(text) {
  const today = new Date();

  if (text.includes("今天")) return today.toISOString().split("T")[0];
  if (text.includes("明天")) {
    const tomorrow = new Date(today.getTime() + 86400000);
    return tomorrow.toISOString().split("T")[0];
  }
  if (text.includes("后天")) {
    const dayAfter = new Date(today.getTime() + 2 * 86400000);
    return dayAfter.toISOString().split("T")[0];
  }

  if (/^\d{4}-\d{2}-\d{2}$/.test(text)) return text;

  return null;
}

// 获取城市ID
export async function getCityLocation(city) {
  const url = `https://geoapi.qweather.com/v2/city/lookup?location=${encodeURIComponent(
    city
  )}&key=${HEFENG_API_KEY}`;

  const res = await fetch(url);
  const data = await res.json();

  if (data.code === "200" && data.location?.length > 0) {
    return data.location[0].id;
  }

  return null;
}

// 获取天气信息
export async function getWeather(city, date) {
  const dateStr = formatDate(date);
  if (!dateStr) {
    throw new Error(`无法识别日期格式："${date}"`);
  }

  const locationId = await getCityLocation(city);
  if (!locationId) {
    throw new Error(`无法识别城市："${city}"`);
  }

  const url = `https://devapi.qweather.com/v7/weather/7d?location=${locationId}&key=${HEFENG_API_KEY}`;
  const res = await fetch(url);
  const data = await res.json();

  if (data.code !== "200") {
    throw new Error(`天气 API 返回错误：${data.code}`);
  }

  const match = data.daily.find((d) => d.fxDate === dateStr);
  if (!match) {
    throw new Error(`暂无 ${dateStr} 的天气数据`);
  }

  return `📍 ${city}（${dateStr}）天气：${match.textDay}，气温 ${match.tempMin}°C ~ ${match.tempMax}°C`;
}
```

## 技术要点

### 1. MCP服务器核心组件
- **McpServer**: 服务器实例，负责管理工具和资源
- **StdioServerTransport**: 使用标准输入输出进行通信
- **registerTool**: 注册可被AI调用的工具

### 2. 工具注册模式
```javascript
server.registerTool(
  "toolName",                    // 工具标识符
  {                             // 工具元数据
    title: "工具标题",
    description: "工具描述",
    inputSchema: {              // 输入参数schema
      paramName: z.string().describe("参数说明")
    }
  },
  async (params) => {           // 工具执行函数
    // 业务逻辑
    return { content: [...] };
  }
);
```

### 3. 和风天气API集成
- **城市查询**: `geoapi.qweather.com/v2/city/lookup`
- **天气查询**: `devapi.qweather.com/v7/weather/7d`
- 需要API密钥认证

### 4. 日期处理逻辑
- 支持自然语言：今天、明天、后天
- 支持标准格式：YYYY-MM-DD
- 使用Date对象进行日期计算

### 5. 错误处理
- 日期格式识别失败
- 城市名称识别失败
- API调用失败
- 数据缺失处理
