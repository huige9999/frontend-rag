# @命令

## 课程概述

@命令是Cursor中一个强大的上下文增强工具，通过引用不同的资源来为AI聊天提供更具体的上下文信息，从而提高AI的响应准确性和实用性。

## 核心概念

在Chat窗口输入@时，系统会显示多个可用选项，使用方法如下：

1. 在输入框中输入 @
2. 使用上下键选择想要的命令
3. 按Enter键确认

## 主要@命令类型

### 1. Files&Folders

**功能**：引用项目中的特定文件与文件夹作为上下文

**使用技巧**：
- 直接将文件或文件夹拖拽到Chat窗口最为方便
- 支持同时引用多个文件
- 可以为AI提供完整的文件内容上下文

### 2. Code

**功能**：引用代码片段

**最佳实践**：
- 在编辑器中选择代码后按`Command + i`直接添加到Chat窗口
- 比使用@Code命令更直观、更方便
- 支持多选代码片段

### 3. Docs

**功能**：访问官方文档或第三方文档信息

**使用场景**：
- 编写Next.js项目时参考官方文档
- 查询特定API的用法
- 获取框架或库的最新文档

**配置方法**：
1. 通过`Cursor设置` -> `Features` -> `@Docs` -> `Add new doc`添加自定义文档
2. Cursor会自动抓取和索引第三方文档
3. 在Chat窗口中输入`@Docs`和问题进行查询

### 4. Git

**功能**：分析Git变更情况、比较分支内容、查看文件状态

#### @Commit

**功能**：引用当前工作状态与最近一次提交之间的未提交变更

**包含内容**：
- 新增的文件
- 修改过的文件
- 已删除的文件

**使用示例**：
```
请帮我检查一下 @Commit 中的改动有没有潜在的问题，比如逻辑错误或代码风格不统一。
```

**AI反馈示例**：
- "auth.js中的错误处理逻辑存在重复"
- "login.js的登录判断条件可能导致空密码通过验证"

#### @Branch

**功能**：对比当前分支与主分支的差异

**提供信息**：
- 当前分支新增的提交
- 与主分支不同的代码变更

**使用示例**：
```
请帮我总结一下 @Branch 中相对于 main 分支的所有功能改动，方便我写 PR 描述。
```

**AI反馈示例**：
- "新增了 SignupForm.vue 组件"
- "修改了 API 调用逻辑"
- "引入了新的依赖（如 yup 用于表单验证）"

### 5. Terminals

**功能**：引用终端输出结果或状态信息

**主要用途**：
- 解释错误日志
- 分析命令执行结果
- 获取下一步操作建议

**使用示例**：
```
@Terminal 出现了这些报错，能不能帮我分析原因并给出修复建议？
```

**典型应用场景**：
- npm run build 报错分析
- git pull 后的冲突处理
- 测试失败的原因分析

**与传统Chat模式的区别**：

| 模式        | 读取终端输出能力     | 生成命令准确性       |
| ----------- | ---------------------- | ------------------------ |
| 普通聊天    | ❌ 需要复制粘贴内容 | ✅ 能生成，但不知上下文   |
| @Terminal | ✅ 自动引用终端上下文 | ✅ 更准确、带环境感知能力   |

### 6. Lint Errors

**功能**：访问和引用代码中的Lint报错与警告信息

**优点**：
- 自动捕获当前激活文件中的所有lint问题
- 无需手动复制报错内容
- AI可以自动修复或解释问题

**使用示例**：
```
请根据 @Lint Errors 修复当前文件中所有的 ESLint 报错
```

**适用场景**：
- 代码规范检查
- 未使用变量检测
- 格式错误修复
- 语法不一致问题

### 7. Web

**功能**：自动搜索互联网获取相关资料

**特点**：
- 实时搜索最新信息
- 获取技术文档和教程
- 查找解决方案和最佳实践

## 实际应用示例

### 示例代码

```javascript
// 冒泡排序实现
function bubbleSort(arr) {
  let n = arr.length;
  for (let i = 0; i < n - 1; i++) {
    for (let j = 0; j < n - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        // 交换元素
        let temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
      }
    }
  }
  return arr;
}

// 示例
const arr = [5, 2, 9, 1, 5, 6];
console.log('排序前：', arr);
console.log('排序后：', bubbleSort(arr));
```

### 综合应用场景

1. **代码审查**：结合@Commit和@Lint Errors进行全面的代码检查
2. **文档查询**：使用@Docs查询特定API或框架的最新文档
3. **问题诊断**：通过@Terminal分析构建或运行时错误
4. **分支管理**：使用@Branch了解代码变更历史
