# 🤖 AIBasic - AI基础课程

> 面向前端开发者的AI基础课程，从工具使用到原理实践

## 📚 课程介绍

本课程专为前端开发者设计，旨在帮助大家快速掌握AI时代的核心技能。课程分为四个主要部分：

- **第一章：课程导读** - 了解AI对前端的意义和课程整体安排
- **第二章：AI提效工具** - 掌握Cursor、Copilot等AI编码工具的使用
- **第三章：LLM基础知识** - 深入理解大语言模型的核心原理和实践
- **第四章：课程总结** - 回顾核心知识点和学习建议

## 🎯 学习目标

✨ 理解AI技术对前端开发的赋能作用  
🛠️ 熟练使用主流AI编程工具提升开发效率  
🧠 掌握大语言模型的核心概念和基本原理  
🚀 具备构建AI应用的基础能力

## 📖 前置知识

- HTML/CSS/JavaScript 基础
- Vue.js 框架经验
- Node.js 基础知识
