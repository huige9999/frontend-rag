# LLM发展历史

## 课程概述

本课程介绍大语言模型（LLM）的发展历史，从AI的基本概念到NLP技术的演进，帮助学生理解LLM在AI领域中的地位和发展脉络。

## 学习目标

- 理解LLM与AI的关系
- 掌握深度学习发展的关键节点
- 了解NLP技术的四个演进阶段
- 认识Transformer架构的重要性

## 核心内容

### 1. LLM和AI的关系

**LLM（Large Language Model，大语言模型）**仅仅是AI的一个分支。

**AI（Artificial Intelligence，人工智能）**：任何能够让机器模仿人类行为的技术
- 交通程序导航
- 游戏中的NPC
- 自动驾驶
- 机器人
- 语言模型
- 图像模型
- 语音识别

**机器学习（Machine Learning）**：AI的子集，人类不直接给出决策规则，通过算法让系统根据示例自主学习并做出决策。

**深度学习（Deep Learning）**：机器学习的分支，受人类神经系统启发，通过多层神经网络自动从数据中学习特征和模式，特别擅长处理图像、语音、文本等高维复杂数据。

### 2. 深度学习架构发展

深度学习的主要架构类型：

1. **CNN（卷积神经网络）**：主要用于图像任务
2. **RNN/LSTM（循环神经网络）**：主要用于序列任务
3. **GAN（生成对抗网络）**：主要用于生成类任务
4. **Transformer**：多模态/语言模型/NLP为主（现已统治深度学习领域）

#### 深度学习发展时间线

| 时间 | 模型名 | 类型 | 应用领域 | 备注 |
|------|--------|------|----------|------|
| 1986 | MLP + 反向传播 | 通用神经网络 | 初代神经网络 | Hinton提出反向传播算法 |
| 1990s | RNN | 循环网络 | 时间序列、语言建模 | 可记忆序列，但存在梯度消失问题 |
| 1997 | LSTM | RNN变体 | 更长序列建模 | 解决RNN梯度问题 |
| 1998 | LeNet-5 | CNN | 图像识别（手写数字） | 深度学习在图像上的首次成功 |
| 2012 | AlexNet | CNN | 图像识别 | ImageNet比赛震撼业界，DL大爆发 |
| 2014 | GAN | 生成模型 | 图像生成、风格迁移 | 生成对抗学习首次提出 |
| 2014 | Seq2Seq + Attention | RNN + 注意力 | 机器翻译 | 注意力机制前身，奠基Transformer |
| 2017 | Transformer | 自注意力 | NLP，现扩展至图像/音频/多模态 | 无需循环/卷积，端到端强大架构 |
| 2018+ | BERT, GPT, T5, etc. | LLM | 自然语言处理 | Transformer进入爆发期 |
| 2021+ | Diffusion Models | 生成模型 | 图像生成（如Stable Diffusion） | GAN之外的新方向 |

#### Transformer的统治地位

| 模型 | 是否支持序列输入 | 是否并行计算 | 主要用于 | 是否已被Transformer替代 |
|------|------------------|--------------|----------|--------------------------|
| RNN | ✅ | ❌（顺序依赖） | 文本、语音 | ✅ 大多数已被Transformer替代 |
| LSTM/GRU | ✅ | ❌ | 长序列建模 | ✅ 被Transformer替代 |
| CNN | ❌ | ✅ | 图像 | 部分视觉任务仍使用 |
| Transformer | ✅ | ✅ | 文本、图像、音频、代码、视频 | ✅ 当前主流通用架构 |

### 3. LLM发展史

#### AI技术两大核心应用方向

1. **计算机视觉（Computer Vision，CV）**：让计算机看懂图片和视频，模仿人类的眼睛
2. **自然语言处理（Natural Language Processing，NLP）**：让计算机能够理解人类语言，甚至可以进行交流

#### 语言的本质

**语言是信息的载体**，包含：
- 声音（口头话语）
- 文字（书面文字）

**NLP的核心任务**：让人类语言进行编码和解码，使计算机能够理解人类的语言。

### 4. NLP演进史的四个阶段

#### 阶段1：起源（1950s）

NLP的起源可追溯到阿兰·图灵在20世纪50年代提出的图灵测试。基本思想：如果一个计算机程序能在自然语言对话中表现得像一个人，那么我们可以说它具有智能。

#### 阶段2：基于规则（1950s-1970s）

通过基于语法和语义规则的方法解决NLP问题，类似人类学习新语言的思路。

**局限性**：
- 语言规则非常繁多且复杂，几乎无法基于语法规则来建模
- 语言是变化的艺术

#### 阶段3：基于统计（1970s-2010s）

1970年以后，IBM科学家采用基于统计的方法解决语音识别问题，将基于规则的问题转换成数学问题，使NLP任务准确率有质的提升。

**语言模型定义**：一种捕捉自然语言中词汇、短语和句子概率分布的统计模型。

| 模型/方法 | 简介 |
|----------|------|
| Bag-of-Words（BoW） | 将文本看作词的集合，忽略语序，仅统计词频或词是否出现。广泛用于文本分类和信息检索。 |
| N-Gram模型 | 最具代表性的统计语言模型之一，利用前N−1个词预测下一个词，体现了语言的局部依赖性。 |
| Hidden Markov Model（HMM） | 一种生成式模型，广泛用于语音识别、词性标注等任务，结合状态转移概率和观测概率进行序列建模。 |
| Maximum Entropy Model（最大熵模型） | 判别式模型，可灵活建模特征对输出的影响，常用于命名实体识别、文本分类等任务。 |
| IBM模型（1~5） | IBM提出的统计机器翻译模型，专注于词对齐、翻译概率和句子结构建模，对早期翻译系统影响深远。 |
| BLEU指标 | 用于评估机器翻译结果的指标，衡量候选翻译与参考翻译的n-gram重合度。 |

#### 阶段4：深度学习和大数据驱动（2013-至今）

以深度学习和大数据为代表的新范式逐步取代传统统计模型，分为两个浪潮：

- **第一波（2013～2018）**：深度学习技术开始应用于NLP，词向量（word embedding）等方法显著提升模型表示能力
- **第二波（2018～至今）**：以Transformer为核心架构的大规模预训练模型兴起，依托海量语料数据，实现更强的语言理解与生成能力

**重要语言模型列表**：

| 模型 | 简介 |
|------|------|
| Word2Vec（2013） | Google提出，通过上下文学习词向量，标志着词嵌入（embedding）时代的开始。 |
| GloVe（2014） | Stanford提出，基于共现矩阵的全局词嵌入方法。 |
| ELMo（2018） | 使用双向LSTM，能根据上下文动态生成词表示，是上下文词向量的开端。 |
| BERT（2018） | Google提出，使用Transformer编码器，预训练-微调范式奠定现代NLP基础。 |
| GPT系列（2018–至今） | OpenAI提出，基于Transformer解码器，擅长自然语言生成，GPT-3、GPT-4代表当今LLM顶尖水平。 |
| T5（2019） | Google提出，统一NLP任务为"文本到文本"的框架。 |
| RoBERTa、XLNet、ERNIE等 | 各大公司在BERT基础上优化推出的改进版本。 |
| ChatGPT（2022）、GPT-4（2023） | 标志着LLM进入应用时代，表现出对话、编程、写作等通用能力。 |

### 5. NLP发展历史总表

| 阶段 | 时间 | 方法/模型 | 类型 | 主要用途 | 是否考虑词序/语义 |
|------|------|-----------|------|----------|------------------|

## 重点概念

- **LLM与AI的关系**：LLM是AI→机器学习→深度学习→NLP→LLM这一技术链条的末端
- **Transformer的革命性**：首次实现序列处理的高效并行化，成为现代AI的通用架构
- **预训练-微调范式**：BERT确立的范式，先在大规模语料上预训练，再在特定任务上微调
- **统计方法的胜利**：从基于规则到基于统计的转变是NLP发展的关键转折点

## 学习建议

1. 理清技术发展脉络，从规则→统计→深度学习的演进逻辑
2. 重点理解Transformer架构的优势和革命性意义
3. 关注预训练-微调这一现代NLP的核心范式
4. 通过实际使用ChatGPT等产品体验LLM的能力边界
