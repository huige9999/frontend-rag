# Transformer内部结构

## 课程概述

本课程深入讲解大语言模型(LLM)的核心架构——Transformer的内部结构和工作原理，帮助学员理解模型如何处理和生成文本。

## 核心内容

### 1. LLM发展历程
- 神经概率语言模型
- 卷积神经网络模型
- 循环神经网络以及长短期记忆网络
- Transformer架构（当前主流）

### 2. 输入输出机制

#### 2.1 提示词输入
- 将LLM看作黑盒，输入称为提示词(prompt)
- 模型采用自回归方式，一次生成一个词元
- 每次生成基于当前输入序列进行前向传播

#### 2.2 前向传播过程
1. 输入嵌入
2. 多个神经网络层处理
3. 输出所有词元的概率分布
4. 选择一个词生成

#### 2.3 自回归特性
- 新生成的词元被追加到输入序列末尾
- 作为下一步生成的上下文
- Transformer是典型的自回归模型

### 3. LLM内部四大组件

#### 3.1 分词器(Tokenizer)
- 将自然语言输入转换为词元ID
- 示例：
  ```
  输入："我今天很开心"
  分词：['我', '今天', '很', '开', '心']
  ID映射：[1, 354, 2764, 77, 199]
  ```

#### 3.2 嵌入层(Embedding Layer)
- 将整数ID映射为向量表示
- 每个词元ID映射为固定维度向量（768维或4096维）
- 这些向量是模型可学习参数，在训练中不断优化
- 示例：
  ```
  输入ID：[1, 354, 2764, 77, 199]
  输出向量：
  [
   [0.12, -0.87, ...,  0.34],   # token 1
   [0.02,  0.45, ..., -0.11],   # token 354
   ...
   [0.76, -0.01, ...,  0.08]    # token 199
  ]
  ```

#### 3.3 堆叠的Transformer块(Stacked Transformer Blocks)
- **核心模块**：理解上下文、捕捉语言结构和语义关系
- 两个主要子模块：
  1. **自注意力机制(Self-Attention)**
     - 让每个词元"看见"前面的所有词元
     - 理解上下文和语义关系
     - 为不同词元分配不同权重
  
  2. **前馈网络(Feed-Forward Network)**
     - 对每个位置的词元向量做非线性变换
     - 提高模型表达能力

- **堆叠结构**：
  - 多层Transformer块串联
  - 每层接收上一层输出并进一步处理
  - 层数越多，理解能力越强

#### 3.4 语言建模头(Language Modeling Head)
- 将高维向量转换为下一个词元的预测
- 在自回归生成中，只用最后一个词元位置的向量

**为什么只用最后一个词？**
- Transformer是上下文感知结构
- 最后词元的向量已通过自注意力机制融合了所有前面词元的信息

**工作流程**：
1. 假设词表大小V=50,000，向量维度d=768
2. 语言建模头是[768×50000]的线性变换矩阵W
3. 计算logits：`logits = z · W`（z为最后一个词元向量）
4. 得到50,000维的得分向量
5. 通过softmax转换为概率分布

**Softmax计算**：
```
softmax(logits_i) = e^(logits_i) / Σ(e^(logits_j))
```
- 先对每个值取指数（确保正数）
- 再除以总和（确保总和为1）

### 4. 完整处理流程示例

**输入**："我今天很"

**步骤1：分词器处理**
```
分词：["我", "今天", "很"]
ID映射：[1, 354, 2764]
```

**步骤2：嵌入层处理**
```
"我"     → [0.12, -0.87, ..., 0.34]  
"今天"   → [-0.11, 0.45, ..., 0.90]  
"很"     → [0.20, 0.14, ..., -0.06]  
形成[3, 768]维度向量数组
```

**步骤3：Transformer块处理**
```
输出：[3, 768]维度向量数组
取最后一个：z = [0.12, -0.45, ..., 0.33]
```

**步骤4：计算logits**
```
logits = z × W → [1 × 50000]向量
logits = [0.9, -1.3, 2.1, ..., 5.7]
```

**步骤5：softmax归一化**
```
{
  开心: 0.61,
  累: 0.12,
  忙: 0.08,
  郁闷: 0.04,
  美丽: 0.02,
  ...
}
```

## 关键概念总结

1. **自回归模型**：基于前期预测进行后续预测
2. **前向传播**：数据从输入层通过各层神经网络产生输出
3. **词元嵌入**：将离散ID转换为连续向量表示
4. **自注意力机制**：让模型理解上下文关系
5. **概率分布**：通过softmax将得分转换为概率

## 技术要点

- Transformer采用堆叠结构，层数影响模型理解能力
- 语言建模头是参数量最多的层之一
- 整个过程是可微分的，支持端到端训练
- 自注意力机制是Transformer区别于传统RNN的关键

## 实际应用价值

理解Transformer内部结构有助于：
- 优化模型使用策略
- 理解模型生成过程
- 进行模型调试和改进
- 为后续学习打下基础
