# 03-09-09 本地部署大模型

## 概述

本课程介绍大语言模型的本地部署方法，主要使用 Ollama 平台来部署和运行开源大模型。

## 大模型使用方式

### 1. 远程 API 调用
- 模型由特定组织开发和托管（如 OpenAI 的 GPT-4、Anthropic 的 Claude）
- 用户通过 API 接口访问

**优点：**
- 功能强大
- 无须拥有强大的 GPU 也能使用强大的 LLM

**缺点：**
- 需要支付费用
- 存在隐私问题

### 2. 本地部署
- 通过工具将模型部署到用户自己的设备上
- 通常部署**开源模型**，如 Meta 的 Llama、微软的 Phi、阿里的千问、DeepSeek 等
- 适合学习阶段使用

## Ollama 平台

### 简介
[Ollama](https://ollama.com/) 是一个开源的大语言模型**运行平台**，提供简洁易用的 **命令行工具** 和 **REST API**，帮助用户在本地轻松下载、部署和运行各种开源的 LLM。

### 主要特点
1. 开源免费
2. 即装即用
3. 支持多种模型：Llama、DeepSeek、Qwen 等
4. 本地运行
5. 灵活扩展
6. **提供 REST API**：本地启动 Web 服务，监听 11434 端口

### 常用命令

| 命令 | 说明 |
|------|------|
| `ollama list` | 查看已下载的模型 |
| `ollama show <模型名称>` | 显示模型信息 |
| `ollama pull <模型名称>` | 拉取模型 |
| `ollama run <模型名称>` | 拉取并运行模型（存在则直接运行，不存在则先拉取） |
| `ollama run <模型名称> --verbose` | 查看每次对话后模型执行的效率细节 |

### 对话模式

**一次性对话：**
```bash
ollama run <模型名称> "提示词"
```

**多行对话：**
```bash
ollama run llama3.2
>"""
>提示词内容
>提示词内容
>提示词内容
>"""
```

## Ollama 模型库

[官网模型库](https://ollama.com/search) 提供了丰富的模型，主要分为两大类：

### 1. 语言处理类大模型（LLMs）

专注于自然语言生成与理解，用于文本生成、问答、代码编写、翻译、总结等任务。

| 模型名称 | 来源机构 | 规模（参数量） | 主要特点 |
|---------|---------|--------------|---------|
| **LLaMA 2 / 3** | Meta | 7B / 13B / 70B | 通用语言理解与生成，支持中文、对话能力强 |
| **Mistral** | Mistral AI | 7B | 小而强的模型，推理速度快，多语言支持 |
| **Gemma** | Google DeepMind | 2B / 7B | 开源许可宽松，轻量且效率高，适合嵌入式应用 |
| **Phi** | Microsoft | 2.7B | 适合教育和低算力环境，训练数据精细设计 |
| **Code LLaMA** | Meta | 7B+ | 专注于编程任务，如代码补全、代码解释等 |
| **Neural Chat** | Intel | 多种参数量 | 聊天风格自然，适合交互式对话 |

**说明：**
- 参数量以 B 为单位，B 来自于英语单词 billion（十亿）
- 例如 7B 表示参数量为 70 亿
- Ollama 也支持微调版本模型（如 `llama2-chinese`、`mistral-instruct`、`orca-mini` 等）

### 2. 图像处理类大模型（Vision / Multimodal Models）

支持 **图像 + 文本联合建模** 的多模态模型。

| 模型名称 | 来源机构 | 模态类型 | 说明 |
|---------|---------|---------|-----|
| **llava** | LLaVA 项目 | 图文联合（image+text） | 支持图像描述、图像问答、OCR 等任务 |
| **bakllava** | 社区开源 | 多模态 | 类似 LLaVA，支持多图问答 |
| **clip** | OpenAI | 图像 + 文本 | 图文相似度匹配，适合搜索、推荐系统 |

**应用场景：**
- AI 看图说话
- 图片问答助手
- 图像内容搜索

**注意：** 图像处理模型目前需配合本地 API 接口使用，CLI 中支持有限。

## 小结

- 本地部署大模型适合学习阶段使用
- Ollama 提供了简单易用的本地部署方案
- 可根据需求选择不同规模的开源模型
- 支持 CLI 和 REST API 两种使用方式
