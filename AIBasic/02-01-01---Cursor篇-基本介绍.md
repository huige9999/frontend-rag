# Cursor篇 - 基本介绍

## 课程概述

Cursor 是一款**基于 VS Code** 打造的 AI 编程增强编辑器，它集成了 GPT-4、Claude 等主流大模型，具备强大的 **代码自动补全、对话生成、智能重构** 等能力。无论你是初学者还是经验丰富的开发者，Cursor 都能成为你得力的"编程助手"。

## Cursor 核心特点

### 1. 对新手友好
- 不懂代码也能通过描述需求，让 AI 帮你构建基本的项目骨架
- 快速搭建 MVP（Minimum Viable Product，最小可行产品）
- 降低编程入门门槛

### 2. 对开发者高效
- 支持 Chat 模式进行自然语言交互
- 提供 Prompt 模板快速生成代码
- 支持函数解释、重构建议等智能功能
- 显著提升开发效率

### 3. 支持 AI 驱动协作开发
- 结合"结对编程"（Pair Programming）理念
- Cursor 成为随时在线、逻辑清晰、不知疲倦的搭档
- 适用于核心模块开发、新人培训、复杂问题 Debug

## Cursor 主界面

Cursor 在结构布局上与 VSCode 十分类似，但增加了面向 AI 编程的核心区域。

### 界面布局
- **左侧**：资源管理区
- **中间**：主编辑区域
- **下方**：终端
- **右侧**：AI 聊天面板

### AI 聊天面板功能
- **New chat 区域**：可直接发起 AI 对话
- **@ Add context**：绑定当前文件上下文（代码自动传入对话）
- **输入框**：支持提问、代码生成、bug 解释等自然语言操作
- **Agent 下拉**：选择对话模式

### 状态栏信息
- **底部状态栏右下角**：显示 Cursor Tab 与模型状态
- 显示当前是否连接到 AI 服务
- 显示当前模型类型
- 网络或模型切换时的提示信息

## 基础配置

### 左侧菜单设置项
- **General**：通用设置
- **Chat**：配置 AI 对话相关设置（例如上下文保留、模型选择）
- **Tab**：多标签页或工作区配置
- **Models**：选择和配置使用的模型（如 GPT-4、Claude、Gemini 等）
- **Background Agents**：管理运行中的智能 Agent
- **Tools & Integrations**：集成 Git、终端、测试框架等工具
- **Rules / Indexing & Docs / Network / Beta / Docs**：其他开发者相关设置和文档入口

### General 通用设置详解

#### 1. Manage Account
- **功能**：打开账户管理页面（跳转到官网管理后台）
- **可配置内容**：
  - 订阅套餐管理
  - 账单信息查看
  - 更换信用卡
  - 取消订阅等

#### 2. Editor Settings
- **功能**：配置编辑器的行为和界面
- **支持的配置**：
  - 字体（字体类型、大小、行高）
  - 主题（明亮/黑暗）
  - 自动格式化、自动补全
  - 显示 minimap（小地图）、行号、折叠标记
  - Tab 缩进大小、空格 vs Tab
  - 光标样式等

#### 3. Keyboard Shortcuts
- **功能**：配置键盘快捷键
- **操作方式**：
  - 打开命令面板：`Cmd/Ctrl + Shift + P`
  - 输入并选择：Preferences: Open Keyboard Shortcuts (JSON)

#### 4. Import Settings from VS Code
- **功能**：将 VS Code 的设置、扩展、快捷键等导入 Cursor
- **适用人群**：VS Code 老用户快速迁移到 Cursor 环境

#### 5. Reset "Don't Ask Again" Dialogs
- **功能**：重置之前选中"不再提示"的对话框
- **用途**：恢复提醒（比如保存前提示、执行高危命令的确认框等）

### 隐私设置

#### Data Sharing Enabled
- **功能**：控制是否将代码、提示词、编辑行为等匿名数据分享给 Cursor 用于产品改进
- **当前状态**：默认为「开启」
- **操作方式**：可点击「Share Data」按钮切换为关闭状态
