# 完成新增文档功能

## 接口

- 新增文档：http://localhost:7001/api/doc [POST]
- 查询所有文档：http://localhost:7001/api/doc [GET]
- 某用户是否具备修改文档权限：http://localhost:7001/api/doc/checkAccess/docId?username=xxx [GET]

## 新增文档流程

1. 出现对话框，填写新文档的标题和简介
2. 填写完成后，点击确认，服务器端就会立即创建新文档，并且返回新文档的 id
3. 根据文档 id 跳转到编辑器
