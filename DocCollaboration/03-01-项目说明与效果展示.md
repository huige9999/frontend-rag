# 项目

完成文档系统中 **协同文档编辑** 功能模块。



**功能需求**

- 新建共享文档
- 编辑共享文档
- 保存共享文档
- 添加协作者：意味着非协作者无法进行编辑



**项目架构**

**技术栈**

- 前端：React19 + Nextjs + heroui + yjs + zod（表单验证的） + codemirror（编辑器）
- 后端：Node.js + Express + MongoDB
