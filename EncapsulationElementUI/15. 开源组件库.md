# 15. 开源组件库

## 概述

代码的开源是指将源码托管到公开的平台，方便其他开发者查看和学习。常见的代码托管平台：

- **GitHub**（国外的平台）
- **Gitee**（码云，国内的平台）

如果是国内环境，建议使用 Gitee 来进行托管。

## 创建仓库

1. 注册 Gitee 账号
2. 点击右上角的 **+** 号，选择【新建仓库】
3. 填写仓库的相关信息（仓库名称、仓库介绍等），点击【创建】

## 对项目进行 Git 配置

### 1. 初始化 Git

在项目根目录下执行：

```bash
git init
```

### 2. 配置 .gitignore

并非所有文件都需要提交到远程仓库（如 `node_modules`），Vue 项目默认自带 `.gitignore` 文件，用于记录不需要提交的文件。

### 3. 生成 SSH 公钥和私钥

建议使用 SSH 方式推送代码。通过以下命令生成 SSH 密钥：

```bash
ssh-keygen -t rsa -C "youremail@example.com"
```

生成后，将公钥添加到 Gitee 的 SSH 钥匙管理面板。

## 推送代码

### 1. 添加远程仓库

```bash
git remote add origin git@gitee.com:xj89959853/duyi-plus.git
```

### 2. 推送代码到远程仓库

```bash
git push -u origin "master"
```

> 注意：使用 `vue@latest` 创建的项目，默认分支为 `main`。

## 添加 LICENSE

仓库在托管源码时，要求仓库有一个 LICENSE，一般选择 **MIT** 即可。在 `package.json` 中已经配置了：

```json
"license": "MIT"
```

```markdown

This template should help get you started developing with Vue 3 in Vite.

pnpm install

### Compile and Hot-Reload for Development

pnpm dev

pnpm build

```

在实际开源时，应将 README.md 补充为组件库的使用文档，包括安装方式、组件列表、使用示例等。
