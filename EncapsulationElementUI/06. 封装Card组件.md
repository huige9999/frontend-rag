# 封装 Card 组件

## 用户如何使用

```html
<!-- 卡片1：仅传入图片和摘要 -->
<Duyi-Card img-src="web.png" summary="前端主流布局系统进阶与实战，轻松解决页面布局难题" />

<!-- 卡片2：传入图片、摘要，并通过 footer 插槽自定义底部内容 -->
<Duyi-Card img-src="go.png" summary="Google资深工程师深度讲解Go语言 由浅入深掌握Go语言">
  <!-- 插槽内部的内容样式由用户自己决定 -->
  <template #footer>
    <div class="footer">
      <div class="level">中级 · 5225人报名</div>
      <div class="price">￥399.00</div>
    </div>
  </template>
</Duyi-Card>

<!-- 卡片3：通过默认插槽自定义摘要内容，通过属性控制宽度和图片高度 -->
<Duyi-Card img-src="java.png" :width="368" :img-height="90">
  <div>路线吃透面试学问，伴随身为Java工程师的你一路成长，高效充分准备面试拿offer。</div>
  <template #footer>
    <div class="footer-java">
      <div class="level">3步骤 · 3门课</div>
      <div class="level">10982收藏</div>
    </div>
  </template>
</Duyi-Card>
```

---

## 支持的属性

| 属性名    | 作用     | 类型   | 是否必须 | 默认值 |
| --------- | -------- | ------ | -------- | ------ |
| width     | 卡片宽度 | Number | 否       | 无     |
| imgSrc    | 卡片链接 | String | 是       | 无     |
| imgHeight | 图片高度 | Number | 否       | 无     |
| summary   | 卡片摘要 | String | 否       | 无     |

---

## 关键代码实现

### props.js

```js
export default {
  // 图片链接，必填属性
  imgSrc: {
    type: String,
    required: true
  },
  // 卡片的描述信息
  summary: {
    type: String
  },
  // 卡片的宽度
  width: {
    type: Number
  },
  // 卡片的高度
  imgHeight: {
    type: Number
  }
}
```

### Card.vue

```vue
<template>
  <!-- 这是最外层容器，会有一个默认的宽度 270px，但是如果用户传递的宽度，那么就使用用户传递的 -->
  <div class="duyi-card" :style="width ? { width: width + 'px' } : {}">
    <!-- 1. 卡片图片 -->
    <div class="duyi-card-img" :style="imgHeight ? { height: imgHeight + 'px' } : {}">
      <img :src="imgSrc" alt="img" />
    </div>
    <!-- 2. 卡片描述 -->
    <div v-if="summary" class="duyi-card-summary">
      {{ summary }}
    </div>
    <div v-else class="duyi-card-summary">
      <slot />
    </div>
    <!-- 3. 卡片的 footer -->
    <!-- 最后的 footer 仅仅提供一个插槽，至于内容以及样式由用户自己来决定 -->
    <div class="duyi-card-footer" v-if="$slots.footer">
      <slot name="footer" />
    </div>
  </div>
</template>

<script setup>
import propObj from './props'
defineOptions({
  name: 'Duyi-Card'
})

defineProps(propObj)
</script>
```

**实现要点：**

- 外层容器通过 `:style` 动态绑定用户传入的 `width`，未传则使用默认宽度
- 图片区域通过 `imgHeight` 属性控制图片高度，未传则使用默认高度
- 摘要区域支持两种方式：传入 `summary` 属性直接显示文本；不传则使用默认插槽渲染自定义内容
- footer 区域使用具名插槽 `#footer`，仅在用户传入时才渲染，样式完全由用户自定义
