# 17. 课程总结

## 课程知识点回顾

### 工程搭建

- 基于 Vite 搭建工程

### 组件封装

- 使用 Vue3 来封装各种组件
  - 在封装组件时，首先需要对组件进行设计
  - 先思考用户是如何使用这个组件
  - 能够传递哪些 props
  - 有哪些事件
  - 是否支持插槽
    - 默认插槽
    - 具名插槽
    - 通过 `$slots.default` 拿到插槽内容
  - 掌握 Vue3 封装组件后，基于 React 封装组件也是相同的思路

### 图标组件

- 基于 FontAwesome 进行图标组件的二次封装

### 二次封装

- 如何基于自己的组件再次进行二次封装

### CSS 变量

- 使用 CSS 变量
  - 在挂上某个类的时候，改变 CSS 变量的值，从而达到改变样式的目的，这是一种非常灵活的方式

### 组件间通信

- 父子组件使用 props、emits
- 非父子组件使用依赖注入模式（provide、inject）

### 打包与发布

- 如何在 Vite 环境打包代码，需要进行一些配置
- 如何开源和发布自己的组件库

## 学习效果

学习完这门课程，你应该能够达到的效果：

- 能够独立地封装并且发布一个组件库
- 能够针对公司内部的一些组件根据业务需求进行二次封装

## 进阶方向

架构课的组件库课程的聚焦点实际上是在如何搭建工程上面，架构课是从零进行搭建项目，而非直接基于 Vite，架构课会涉及到：

- **monorepo**：多包管理架构
- **TypeScript**：类型安全的开发
- **Rollup**：更底层的打包工具
- **单元测试**：保障组件质量
- 其他工程化实践
