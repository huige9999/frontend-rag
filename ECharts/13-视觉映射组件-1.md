# 13-视觉映射组件-1

## 简介
本节课学习 visualMap（视觉映射）组件的基础用法，主要讲解 continuous（连续型）视觉映射的配置，结合中国地图展示数据到颜色的映射关系。

## 关键代码

### continuous 连续型视觉映射（完整配置）

```javascript
var myChart = echarts.init(document.getElementById('chart'));
myChart.setOption({
    visualMap: [
        {
            type: 'continuous',       // 连续型视觉映射
            min: 0,                   // 最小值
            max: 1000,                // 最大值
            range: [300, 800],        // 指定选中范围
            calculable: true,         // 是否显示拖拽手柄（可计算）
            realtime: false,          // 拖拽时是否实时更新
            inverse: false,           // 是否反转
            orient: 'vertical',       // 方向：vertical / horizontal
            precision: 2,             // 数据精度（小数位数）
            itemWidth: 40,            // 图形宽度
            itemHeight: 200,          // 图形高度

            // align: 'right',        // 对齐方式：auto / left / right

            text: ['hight', 'low'],   // 两端文本
            textGap: 20,              // 两端文本与图例的距离
            show: true,               // 是否显示

            dimension: 0,             // 指定映射的数据维度

            hoverLink: true,          // 鼠标悬停时高亮关联数据

            // 方式1：共同定义
            // inRange: { color: ['pink'] },
            // outOfRange: { color: ['green'] },

            // 方式2：分别定义 target 和 controller
            // target: {
            //     inRange: { color: ['pink'] },
            //     outOfRange: { color: ['green'] }
            // },
            // controller: {
            //     inRange: { color: ['pink'] },
            //     outOfRange: { color: ['green'] }
            // }

            // 方式3：先全部定义，再单独覆盖 controller
            inRange: {
                color: ['teal', 'teal'],
                symbol: ['circle', 'rect'],
            },
            controller: {
                inRange: {
                    color: ['red', 'green', 'pink'],
                },
            }
        }
    ]
});
```

### 代码说明
- `type: 'continuous'` - 连续型视觉映射，数据值在一个连续区间上映射到视觉通道
- `min / max` - 指定视觉映射的最小值和最大值
- `range` - 定义当前选中的数据范围，用具体数值指定
- `calculable` - 是否显示拖拽用的手柄，允许用户调整映射范围
- `realtime` - 拖拽手柄时是否实时更新图表
- `inverse` - 是否反转 visualMap 组件的方向
- `orient` - visualMap 的方向，支持 vertical（垂直）和 horizontal（水平）
- `precision` - 数据展示精度，即小数点后位数
- `text` - 两端的文本标签，如 ['高', '低']
- `dimension` - 指定映射到数据的哪个维度
- `hoverLink` - 鼠标悬停在 visualMap 上时，是否高亮图表中对应的数据
- `inRange` - 定义在选定范围中的视觉元素（颜色、符号等）
- `outOfRange` - 定义超出选定范围的视觉元素
- `target / controller` - 分别定义图表目标和控制器（图例）的视觉映射
- `china.js` - 中国地图的 GeoJSON 数据文件，用于 map 类型图表
