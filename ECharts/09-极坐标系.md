# 09-极坐标系

## 简介
本节课学习 ECharts 的 polar（极坐标系）配置，以及 radiusAxis（径向轴）和 angleAxis（角度轴）的详细属性，通过散点图在极坐标系上展示 GitHub 打卡数据。

## 关键代码

### 极坐标系配置

```javascript
var hours = ['12a', '1a', '2a', '3a', '4a', '5a', '6a',
    '7a', '8a', '9a', '10a', '11a',
    '12p', '1p', '2p', '3p', '4p', '5p',
    '6p', '7p', '8p', '9p', '10p', '11p'];
var days = ['Saturday', 'Friday', 'Thursday',
    'Wednesday', 'Tuesday', 'Monday', 'Sunday'];

var data = [[0,0,5],[0,1,1],[0,2,0], ... ];  // [星期索引, 小时索引, 数值]

var myChart = echarts.init(document.getElementById('chart1'));

myChart.setOption({
    title: {
        text: 'GitHub上打卡',
    },
    legend: {
        data: ['打卡'],
        left: 'right',
    },
    polar: {
        center: ['50%', '50%'],   // 极坐标系圆心坐标
        radius: '80%',            // 极坐标系半径
    },
    radiusAxis: {    // 极坐标系的径向轴
        type: 'category',
        data: days,
        name: '星期',
        nameLocation: 'end',
        nameTextStyle: {
            color: 'blue',
            fontSize: 16,
        },
        nameGap: 20,
        nameRotate: 45,
        inverse: false,
        boundaryGap: true,

        axisLine: {
            lineStyle: {
                color: 'blue',
                width: 1
            }
        },
        axisTick: {
            length: 10,
            lineStyle: {
                color: '#000',
            }
        },
        axisLabel: {
            interval: 0,
        },
        splitLine: {
            show: true,
            lineStyle: {
                color: 'green',
            }
        },
        splitArea: {
            show: true,
            areaStyle: {}
        }
    },
    angleAxis: {    // 极坐标系的角度轴
        startAngle: 90,        // 起始角度
        clockwise: false,      // 是否顺时针
        type: 'category',
        data: hours,

        axisLine: {
            lineStyle: {
                color: 'teal',
            }
        },
        splitLine: {
            show: true,
        }
    },
    series: {
        name: '打卡',
        type: 'scatter',
        coordinateSystem: 'polar',    // 指定使用极坐标系
        data: data,
    }
});
```

### 代码说明
- `polar` - 极坐标系配置，`center` 设置圆心位置，`radius` 设置半径
- `radiusAxis` - 极坐标系的径向轴（从圆心向外），配置方式类似 xAxis
  - `type: 'category'` - 径向轴为类目型
  - `data` - 径向轴的类目数据
  - `axisLine` / `axisTick` / `axisLabel` / `splitLine` / `splitArea` - 与直角坐标轴配置一致
- `angleAxis` - 极坐标系的角度轴（围绕圆心）
  - `startAngle` - 起始角度（0 为 3 点钟方向，90 为 12 点钟方向）
  - `clockwise` - 是否顺时针排列刻度
  - `data` - 角度轴的类目数据
- `coordinateSystem: 'polar'` - series 中指定使用极坐标系，使散点图绘制在极坐标上
