# 12-区域缩放2

## 简介
本节课深入学习 dataZoom 区域缩放组件，涵盖 inside 类型（内置缩放）和 slider 类型（滑动条缩放）的详细配置，包括滑块样式、数据背景、手柄图标、标签格式化等属性。

## 关键代码

### dataZoom inside 类型 - X轴内置缩放

```javascript
var myChart = echarts.init(document.getElementById('chart1'));
myChart.setOption({
    animation: false,  // 不开启动画
    legend: {},
    xAxis: {},
    yAxis: {},
    dataZoom: [
        {
            type: 'inside',
            disabled: false,
            xAxisIndex: 0,
            // start/end: 按百分比控制缩放范围
            // start: 30,
            // end: 50,
            startValue: 2,   // 按数据值控制缩放起始
            endValue: 5,     // 按数据值控制缩放结束
            // rangeMode: ['value', 'percent'],  // 范围模式
            orient: 'vertical',
            // zoomLock: false,  // 锁定缩放大小
            zoomOnMouseWheel: true,     // 鼠标滚轮缩放
            moveOnMouseMove: true,      // 鼠标移动平移
            moveOnMouseWheel: 'alt',    // 按住alt+滚轮平移
            preventDefaultMouseMove: true,  // 阻止默认鼠标移动行为
        }
    ]
});
```

### dataZoom slider 类型 - 滑动条缩放（完整配置）

```javascript
{
    type: 'slider',
    backgroundColor: 'green',       // 滑动条背景色
    dataBackground: {                // 数据背景样式
        lineStyle: {
            opacity: 1,
            color: 'red',
        },
        areaStyle: {
            opacity: 1,
            color: 'pink',
            shadowColor: 'rgba(0, 255, 255, 1)',
            shadowBlur: 1,
            shadowOffsetX: 10,
        }
    },
    fillerColor: 'teal',            // 选中范围填充色
    borderColor: 'red',             // 边框颜色
    handleIcon: 'image://data:image/gif;base64,...',  // 手柄图标（支持图片URL）
    handleSize: '70%',              // 手柄大小
    handleStyle: {                   // 手柄样式
        color: 'green',
        borderColor: 'green',
        borderWidth: 2,
    },
    labelPrecision: 0,              // 标签精度
    labelFormatter: '{value} ~',    // 标签格式化（字符串模板）
    // labelFormatter 也可以是函数
    labelFormatter: function(value) {
        return value.toFixed(1) + ' ~';
    },
    showDetail: true,               // 拖拽时显示详情
    showDataShadow: true,           // 显示数据阴影
    realtime: false,                // 拖拽时是否实时更新图表
    textStyle: {                     // 文本样式
        color: 'blue',
        fontSize: 16,
    },
    // orient: 'vertical',          // 方向（水平/垂直）
    left: 'center'                  // 定位
}
```

### 代码说明
- `type: 'inside'` - 内置型数据区域缩放组件，通过鼠标/触摸操作缩放
- `type: 'slider'` - 滑动条型数据区域缩放组件，有独立的滑动条 UI
- `xAxisIndex / yAxisIndex` - 指定控制哪个轴
- `start / end` - 按百分比设定缩放范围（0~100）
- `startValue / endValue` - 按数据值设定缩放范围
- `zoomOnMouseWheel` - 控制鼠标滚轮是否触发缩放
- `moveOnMouseMove` - 控制鼠标移动是否触发平移
- `dataBackground` - 滑动条中的数据背景样式（线样式和面积样式）
- `fillerColor` - 缩放区域填充颜色
- `handleIcon / handleSize / handleStyle` - 滑动手柄的图标、大小和样式配置
- `labelFormatter` - 滑动条标签格式化，支持字符串模板和函数
- `realtime` - 拖拽时是否实时更新图表视图
