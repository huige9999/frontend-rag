# 30-数据集2-映射

## 简介
本节课学习 ECharts 数据集（dataset）的数据映射机制，掌握通过 `seriesLayoutBy` 控制维度行列方向，以及通过 `encode` 将任意维度映射到图表的指定视觉通道。

## 关键代码

### seriesLayoutBy - 维度行列映射

```javascript
option = {
  legend: {},
  tooltip: {},
  xAxis: [
    { type: 'category', gridIndex: 0 },
    { type: 'category', gridIndex: 1 },
  ],
  yAxis: [
    { gridIndex: 0 },
    { gridIndex: 1 },
  ],
  grid: [
    { bottom: '55%' },
    { top: '55%' },
  ],
  dataset: {
    source: [
      ['product', '2014', '2015', '2016'],
      ['iphone4', 43.3, 85.8, 93.7],
      ['iphone5', 83.1, 73.4, 55.1],
      ['iphone6', 86.4, 65.2, 82.5],
      ['iphone7', 72.4, 53.9, 39.1]
    ]
  },
  series: [
    // 维度为行：每行数据作为一个系列（按产品分组）
    { type: 'bar', seriesLayoutBy: 'row' },
    { type: 'bar', seriesLayoutBy: 'row' },
    { type: 'bar', seriesLayoutBy: 'row' },
    { type: 'bar', seriesLayoutBy: 'row' },

    // 维度为列（默认）：每列数据作为一个系列（按年份分组）
    { type: 'bar', xAxisIndex: 1, yAxisIndex: 1 },
    { type: 'bar', xAxisIndex: 1, yAxisIndex: 1 },
    { type: 'bar', xAxisIndex: 1, yAxisIndex: 1 },
  ]
};
```

### encode - 数据到图形的映射

```javascript
option = {
  dataset: {
    source: [
      ['score', 'amount', 'product'],
      [89.3, 58212, 'Matcha Latte'],
      [57.1, 78254, 'Milk Tea'],
      [74.4, 41032, 'Cheese Cocoa'],
      [50.1, 12755, 'Cheese Brownie'],
      [89.7, 20145, 'Matcha Cocoa'],
      [68.1, 79146, 'Tea'],
      [19.6, 91852, 'Orange Juice'],
      [10.6, 101852, 'Lemon Juice'],
      [32.7, 20112, 'Walnut Brownie']
    ]
  },
  grid: { containLabel: true },
  xAxis: { name: 'amount' },
  yAxis: { type: 'category' },
  visualMap: {
    orient: 'horizontal',
    left: 'center',
    min: 10,
    max: 100,
    text: ['High Score', 'Low Score'],
    dimension: 0,              // 指定映射到第0维度（score）
    inRange: {
      color: ['#D7DA8B', '#E15457']
    }
  },
  series: [
    {
      type: 'bar',
      encode: {
        x: 'amount',           // 将 amount 列映射到 X 轴
        y: 'product'           // 将 product 列映射到 Y 轴
      }
    }
  ]
};
```

### 代码说明
- `seriesLayoutBy` - 控制数据集的行列方向映射：
  - `'column'`（默认）：每一列对应一个系列
  - `'row'`：每一行对应一个系列
- `encode` - 数据到图形通道的映射，将指定维度映射到图表的指定位置
- `encode.x` / `encode.y` - 将某个维度（用维度名或索引）映射到 X/Y 轴
- `encode.tooltip` / `encode.label` - 将维度映射到提示框或标签
- `visualMap.dimension` - 指定视觉映射作用于哪个维度
- 数据映射的两个层面：
  1. **数据到系列**的映射：由 `seriesLayoutBy` 决定行列方向
  2. **数据到图形**的映射：由 `encode` 决定哪个维度映射到哪个视觉通道
- 通过 `gridIndex` / `xAxisIndex` / `yAxisIndex` 可以将不同系列绑定到不同的坐标系
