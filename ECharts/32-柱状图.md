# 32-柱状图

## 简介
本节课学习 ECharts 柱状图（bar）的基本配置，包括多系列柱状图、柱体样式、标签显示、高亮效果以及柱体宽度和间距的调整。

## 关键代码

### 柱状图基础配置

```javascript
option = {
  legend: {},
  xAxis: {
    data: ['篮球', '足球', '排球', '网球'],
    axisTick: {
      show: false,
    }
  },
  yAxis: {
    splitLine: {
      show: false,
    }
  },
  series: [
    {
      name: '昨日销量',
      type: 'bar',
      data: [60, 40, 72, 33],

      legendHoverLink: false,
      coordinateSystem: 'cartesian2d',
      label: {
        show: true,
        position: 'top',
      },
      itemStyle: {
        color: 'green',
      },
      emphasis: {
        label: {
          fontSize: 20,
        },
        itemStyle: {
          opacity: 0.5,
        }
      },
      barWidth: '50%',
      barMaxWidth: 50,
      barMinHeight: 500,
      barCategoryGap: '20%',
    },
    {
      name: '今日销量',
      type: 'bar',
      data: [45, 60, 13, 25]
    },
  ]
};
```

### 代码说明
- `type: 'bar'` - 指定系列类型为柱状图
- `legendHoverLink` - 控制图例悬浮时是否联动高亮对应系列，设为 false 则不联动
- `coordinateSystem: 'cartesian2d'` - 使用二维直角坐标系（默认值）
- `label` - 数据标签配置，`show: true` 显示标签，`position: 'top'` 标签显示在柱体顶部
- `itemStyle.color` - 设置柱体填充颜色
- `emphasis` - 高亮状态配置，鼠标悬浮时标签放大、柱体半透明
- `barWidth` - 柱体宽度，支持百分比和像素值
- `barMaxWidth` - 柱体最大宽度限制
- `barMinHeight` - 柱体最小高度，数据值较小时也能保证可见
- `barCategoryGap` - 同一类目下不同系列的柱间距，默认为 20%
