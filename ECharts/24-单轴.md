# 24-单轴

## 简介
本节课学习 ECharts 单轴（singleAxis）组件的使用方法，通过一周每天 24 小时的散点图案例，掌握单轴坐标系与散点图的组合以及动态生成多个轴和系列的技巧。

## 关键代码

### 单条轴的基础配置（注释示例）

```javascript
// 单条轴配置示例
option = {
  title: {
    text: 'Saturday',
  },
  singleAxis: {
    type: 'category',
    data: hours,
    boundaryGap: false,
    axisLabel: {
      interval: 2,
    }
  },
  tooltip: {},
  series: {
    type: 'scatter',
    coordinateSystem: 'singleAxis',
    data: random(),
    symbolSize: function (val) {
      return val[1] * 4;
    }
  }
};
```

### 多条单轴动态生成

```javascript
var hours = ['12a', '1a', '2a', '3a', '4a', '5a', '6a',
  '7a', '8a', '9a', '10a', '11a',
  '12p', '1p', '2p', '3p', '4p', '5p',
  '6p', '7p', '8p', '9p', '10p', '11p'];
var days = ['Saturday', 'Friday', 'Thursday',
  'Wednesday', 'Tuesday', 'Monday', 'Sunday'];

// 随机生成数据
function random() {
  var result = [];
  for (var i = 0; i < 24; i++) {
    result.push([i, Math.round(Math.random() * 10)]);
  }
  return result;
}

// 动态构建 option
var option = {
  title: [],
  tooltip: {},
  singleAxis: [],
  series: []
};

days.forEach(function (item, index) {
  option.title.push({
    text: item,
    top: index * 100 / 7 + 5 + '%',  // 按百分比定位每个标题
  });

  option.singleAxis.push({
    type: 'category',
    data: hours,
    boundaryGap: false,
    axisLabel: {
      interval: 2,
    },
    top: index * 100 / 7 + 5 + '%',   // 按百分比定位每条轴
    height: 100 / 7 - 10 + '%',       // 每条轴的高度
    left: 150,
  });

  option.series.push({
    singleAxisIndex: index,            // 绑定对应的单轴
    type: 'scatter',
    coordinateSystem: 'singleAxis',
    data: random(),
    symbolSize: function (val) {
      return val[1] * 4;
    }
  });
});

myChart.setOption(option);
```

### 代码说明
- `singleAxis` - 单轴组件，提供一维坐标轴，常配合散点图使用
- `singleAxis.type: 'category'` - 轴类型为类目轴，展示离散数据（如小时）
- `singleAxis.boundaryGap` - 是否在两端留白边界
- `singleAxis.axisLabel.interval` - 标签显示间隔，避免密集重叠
- `singleAxis.top` / `singleAxis.height` - 通过百分比精确定位每条轴在容器中的位置和高度
- `series.coordinateSystem: 'singleAxis'` - 散点图绑定到单轴坐标系
- `series.singleAxisIndex` - 指定系列使用哪条单轴，实现多轴多系列联动
- 动态生成技巧：通过 `forEach` 循环遍历天数数组，按百分比计算每条轴的垂直位置，实现一周七天每行一条轴的布局
- `symbolSize` - 根据数据值动态计算散点大小
