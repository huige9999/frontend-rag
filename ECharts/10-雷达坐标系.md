# 10-雷达坐标系

## 简介
本节课学习 ECharts 的 radar（雷达图坐标系）配置，包括指示器（indicator）定义、雷达图形状、名称格式化、分隔线和分隔区域等属性。

## 关键代码

### 雷达坐标系配置

```javascript
var myChart1 = echarts.init(document.getElementById('chart1'));

myChart1.setOption({
    radar: {
        indicator: [               // 雷达图的指示器，定义各维度的名称和范围
            {name: '得分', max: 150, min: 60, color: 'red'},
            {name: '篮板'},
            {name: '助攻'},
            {name: '抢断'},
            {name: '盖帽'},
            {name: '投篮'},
            {name: '三分'},
        ],
        center: ['50%', '50%'],   // 雷达图中心坐标
        radius: '75%',            // 雷达图半径
        startAngle: 90,           // 起始角度
        name: {
            show: true,
            formatter: function (value, indicator) {
                return '【' + value + '】';
            },
        },
        nameGap: 25,              // 指示器名称与轴线的距离
        splitNumber: 10,          // 雷达图分隔份数
        shape: 'polygon',         // 雷达图形状：'polygon' 多边形 / 'circle' 圆形

        axisLine: {               // 坐标轴轴线样式
            lineStyle: {}
        },
        splitLine: {              // 分隔线样式
            show: true,
            lineStyle: {},
        },
        splitArea: {              // 分隔区域样式
            show: true,
            areaStyle: {}
        }
    },
    series: {
        type: 'radar',
        data: [
            {
                name: '火箭',
                value: [109, 38, 29, 6, 2, 60, 17]
            },
            {
                name: '勇士',
                value: [118, 36, 25, 6, 6, 77, 13]
            }
        ]
    }
});
```

### 代码说明
- `indicator` - 雷达图的指示器数组，每个指示器定义一个维度
  - `name` - 指示器名称
  - `max` / `min` - 该维度的最大值/最小值范围
  - `color` - 指示器名称的颜色
- `center` - 雷达图中心位置 `[x, y]`
- `radius` - 雷达图半径
- `startAngle` - 起始角度（与极坐标系类似）
- `name.formatter` - 指示器名称格式化函数，接收 `value`（名称文本）和 `indicator`（指示器对象）
- `nameGap` - 指示器名称与轴线的距离
- `splitNumber` - 雷达图圆环分隔的段数
- `shape` - 雷达图形状，`'polygon'`（多边形）或 `'circle'`（圆形）
- `axisLine` - 坐标轴轴线（从中心向外的射线）样式
- `splitLine` - 分隔线（同心圆/多边形）样式
- `splitArea` - 分隔区域（交替填充区域）样式
- `series.type: 'radar'` - 系列类型为雷达图，`data` 中每项的 `value` 数组对应各维度的数值
