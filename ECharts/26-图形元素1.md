# 26-图形元素1

## 简介
本节课学习 ECharts 图形元素（graphic）配置项的基础用法，掌握在图表上添加图片（image）、文本（text）、矩形（rect）、圆形（circle）等自定义图形元素的方法，以及拖拽交互和事件处理。

## 关键代码

### 图片图形元素 - image

```javascript
graphic: [
  {
    type: 'image',
    id: 'favicon',              // 唯一标识，用于后续操作
    position: [0, 0],
    origin: [75, 75],           // 旋转和缩放的中心点
    scale: [1, 1],
    left: 1250,
    info: 'Hello Kaivon!',      // 自定义属性
    silent: false,              // 是否响应交互
    invisible: false,           // 是否隐藏
    ignore: false,              // 是否完全忽略
    cursor: 'crosshair',       // 鼠标样式
    draggable: true,            // 允许拖拽

    style: {
      image: 'http://echarts.baidu.com/images/favicon.png',
      width: 150,
      height: 150,
      x: 200,
      y: 200,
      fill: 'blue',
      shadowBlur: 10,
      shadowOffsetX: 10,
      shadowOffsetY: 10,
      shadowColor: '#000'
    },

    // 交互事件
    onclick: function () {
      console.log(1);
    },
    ondrag: function () {
      console.log('拖拽按下并拖动了！');
    },
    ondragstart: function () {
      console.log('拖拽开始了（鼠标按下）！');
    },
    ondragend: function () {
      console.log('拖拽结束了（鼠标抬起）！');
    },
    ondragenter: function () {
      console.log('碰到我了！');
    },
    ondragleave: function () {
      console.log('碰撞离开了！');
    },
    ondrop: function () {
      console.log('拖拽元素拖到我身上了，并在我身上抬起了鼠标！');
    }
  },
  {
    type: 'image',
    draggable: true,
    style: {
      image: 'http://echarts.baidu.com/images/favicon.png',
      width: 50,
      height: 50,
    },
  }
]
```

### 基础几何图形 - rect、circle、ring

```javascript
// 矩形
{
  type: 'rect',
  shape: {
    width: 100,
    height: 100,
    x: 100,
    y: 200,
  },
  style: {
    fill: 'rgba(15,186,100,0.5)',
    stroke: 'blue',
    lineWidth: 5,
    shadowBlur: 10,
    shadowOffsetX: 10,
    shadowOffsetY: 10,
    shadowColor: '#000',
  }
},
// 圆形
{
  type: 'circle',
  z: 5,
  shape: {
    cx: 330,
    cy: 250,
    r: 100,
  },
  style: {
    fill: 'rgba(15,186,100,0.5)',
    stroke: 'blue',
    lineWidth: 5,
    shadowBlur: 10,
    shadowColor: '#000',
  }
},
// 圆环
{
  type: 'ring',
  shape: {
    cx: 170,
    cy: 450,
    r: 100,        // 外半径
    r0: 50,        // 内半径
  },
  style: {
    fill: 'rgba(15,186,100,0.5)',
    stroke: 'blue',
    lineWidth: 5,
  }
}
```

### 动态操作图形元素（通过 $action）

```javascript
// 按钮点击后移除指定 id 的图形元素
var btn = document.querySelector('button');
btn.onclick = function () {
  mychart.setOption({
    graphic: [
      {
        id: 'favicon',
        $action: 'remove',   // 操作类型：remove / merge / replace
        style: {
          width: 300,
          height: 300,
          image: 'http://echarts.baidu.com/images/favicon.png',
        }
      }
    ]
  });
};
```

### 代码说明
- `graphic` - 图形元素配置，是一个数组，可同时添加多个图形
- `type` - 图形类型：`'image'`（图片）、`'text'`（文本）、`'rect'`（矩形）、`'circle'`（圆）、`'ring'`（圆环）等
- `id` - 图形元素的唯一标识，用于后续通过 `$action` 进行动态操作
- `draggable` - 是否允许拖拽
- `silent` - 是否静默（不响应鼠标事件）
- `invisible` - 是否隐藏但保留占位
- `ignore` - 是否完全忽略（不渲染也不交互）
- `cursor` - 鼠标悬停时的光标样式
- `style` - 图形样式，包括 `fill`（填充色）、`stroke`（描边）、`shadowBlur`（阴影）等
- `shape` - 图形形状参数，不同类型有不同的属性（如圆的 `cx/cy/r`，矩形的 `x/y/width/height`）
- `ondrag` / `onclick` / `ondragstart` / `ondragend` 等事件 - 图形元素的交互事件
- `$action` - 动态操作类型：`'remove'`（移除）、`'merge'`（合并）、`'replace'`（替换）
- `z` - 控制图形的层叠顺序
