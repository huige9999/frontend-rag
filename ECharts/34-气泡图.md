# 34-气泡图

## 简介
本节课学习 ECharts 气泡图（scatter）的高级应用，结合时间轴（timeline）展示各国人均收入、寿命与人口的关系演变，使用对数轴、visualMap 映射国家颜色，以及自定义 symbolSize 函数实现气泡大小映射。

## 关键代码

### 气泡图与时间轴配置

```javascript
// 气泡大小映射函数
function sizeFunction(x) {
  var y = Math.sqrt(x / 5e8) + 0.1;
  return y * 80;
}

var option = {
  baseOption: {
    backgroundColor: '#404a59',
    title: [
      {
        text: data.timeline[0],  // 年份标题
        left: '60%',
        top: '55%',
        textStyle: {
          fontSize: 100,
          color: 'rgba(255,255,255,0.7)'
        }
      },
      {
        text: '全国人均寿命与GDP关系演变',
        left: 'center',
        top: 10,
        textStyle: {
          color: '#aaa',
          fontWeight: 'normal',
          fontSize: 20,
        }
      }
    ],
    grid: {
      top: 100,
      left: 30,
      right: 100,
      containLabel: true,
    },
    xAxis: {
      name: "人均收入",
      nameLocation: 'middle',
      type: 'log',          // 对数轴
      min: 300,
      max: 100000,
      axisLabel: {
        formatter: '{value} $',
      }
    },
    yAxis: {
      max: 100,
      name: '平均寿命',
      axisLabel: {
        formatter: '{value} 岁',
      }
    },
    timeline: {
      axisType: 'category',
      orient: 'vertical',
      right: 0,
      top: 20,
      bottom: 20,
      width: 55,
      autoPlay: true,
      playInterval: 1000,
      // ... 其他样式配置
    },
    series: {
      type: 'scatter',
      itemStyle: {
        opacity: 0.8,
        shadowBlur: 10,
        shadowOffsetX: 0,
        shadowOffsetY: 0,
        shadowColor: 'rgba(0,0,0,0.5)',
      },
      symbolSize: function (val) {
        return sizeFunction(val[2]);  // val[2] 为总人口
      },
    },
    visualMap: {
      show: false,
      categories: data.counties,
      dimension: 3,    // 按第4个维度（国家名称）映射颜色
      inRange: {
        color: (function () {
          var colors = ['#bcd3bb', '#e88f70', '#edc1a5', '#9dc5c8',
                        '#e1e8c8', '#7b7c68', '#e5b5b5', '#f0b489',
                        '#928ea8', '#bda29a'];
          return colors.concat(colors);
        })()
      }
    },
    tooltip: {
      formatter: function (obj) {
        var value = obj.value;
        return '国家：' + value[3] + '<br>'
          + '人均寿命：' + value[1] + '岁 <br>'
          + '人均收入：' + value[0] + '美元 <br>'
          + '总人口：' + value[2] + '<br>';
      }
    }
  },
  options: []  // 各年份动态数据
};

// 动态填充时间轴数据
for (var i = 0; i < data.timeline.length; i++) {
  option.baseOption.timeline.data.push(data.timeline[i]);
  option.options.push({
    title: { text: data.timeline[i] },
    series: { data: data.series[i] }
  });
}
```

### 代码说明
- `type: 'log'` - X 轴使用对数类型，适合跨度大的数据（300 ~ 100000）
- `symbolSize` 函数 - 根据数据第三个维度（总人口）动态计算气泡大小
- `visualMap.categories` + `dimension: 3` - 按国家名称（第4维）分类映射颜色
- `timeline` - 时间轴组件，`autoPlay: true` 自动播放，`playInterval` 播放间隔（毫秒）
- `baseOption` + `options` - 使用增量配置模式，baseOption 为公共配置，options 数组存放各时间点差异配置
- `shadowBlur / shadowColor` - 气泡阴影效果，增加视觉层次
- 数据格式为 `[人均收入, 人均寿命, 总人口, 国家名, 年份]`
