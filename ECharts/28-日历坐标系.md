# 28-日历坐标系

## 简介
本节课学习 ECharts 日历坐标系（calendar）的配置方法，通过某人 2016 年每天步数的热力图案例，掌握日历坐标系的范围设置、单元格样式、日期标签定制等核心配置。

## 关键代码

### 数据生成函数

```javascript
// 生成指定年份每天的随机步数数据
// 数据格式：['2016-1-1', 步数值]
function getData(year = 2017) {
  function getEndDay(year, month) {
    return new Date(year, month, 0).getDate();  // 获取某月最后一天
  }

  var data = [];
  var days;
  for (var i = 1; i <= 12; i++) {
    days = getEndDay(year, i);
    for (var j = 0; j < days; j++) {
      data.push([year + '-' + i + '-' + (j + 1), Math.floor(Math.random() * 10000)]);
    }
  }
  return data;
}
```

### calendar 日历坐标系完整配置

```javascript
option = {
  title: {
    text: '2016年某人每天的步数',
    left: 'center',
    top: 30,
    textStyle: {
      fontSize: 20,
    }
  },
  tooltip: {},
  visualMap: {
    type: 'piecewise',       // 分段型视觉映射
    min: 0,
    max: 10000,
    orient: 'horizontal',
    left: 'center',
    top: 65,
    textStyle: {
      color: '#000',
      fontSize: 14
    }
  },
  series: {
    type: 'heatmap',
    coordinateSystem: 'calendar',  // 使用日历坐标系
    data: getData(2016),
  },
  calendar: {
    top: 120,
    left: 100,
    right: 100,
    range: 2016,                   // 日历范围：2016全年
    cellSize: [30, 40],            // 单元格大小 [宽, 高]
    orient: 'horizontal',          // 水平排列
    splitLine: {
      show: true,
      lineStyle: {
        color: 'blue',
        width: 2,
        type: 'dotted',
      }
    },
    itemStyle: {
      color: 'green',             // 默认单元格颜色
      borderColor: 'pink',
      borderWidth: 2,
    },
    dayLabel: {
      firstDay: 1,                // 一周从周一开始
      margin: 30,
      position: 'end',            // 标签位置
      nameMap: 'cn',              // 中文星期名
      color: 'teal',
      fontSize: 16,
    },
    monthLabel: {
      align: 'left',
      margin: 10,
      position: 'end',
      nameMap: 'cn',              // 中文月份名
      formatter: function (param) {
        return param.yyyy + ' 年 ' + param.M + ' 月';
      }
    },
    yearLabel: {
      margin: 40,
      position: 'bottom',
      formatter: '{end}'          // 显示结束年份
    }
  }
};
```

### 代码说明
- `calendar` - 日历坐标系组件，常与热力图（heatmap）配合使用
- `calendar.range` - 日历范围，支持年（`2016`）、月（`'2016-02'`）、日期区间（`['2016-01-02', '2017-02-23']`）三种格式
- `calendar.cellSize` - 单元格大小，可以是数字（正方形）或 `[宽, 高]` 数组，也支持 `'auto'`
- `calendar.orient` - 排列方向，`'horizontal'`（水平）或 `'vertical'`（垂直）
- `calendar.splitLine` - 月份分割线样式
- `calendar.itemStyle` - 单元格默认样式（未覆盖区域）
- `calendar.dayLabel` - 星期标签配置，`firstDay: 1` 表示从周一开始
- `calendar.dayLabel.nameMap` - 设为 `'cn'` 使用中文星期名
- `calendar.monthLabel` - 月份标签配置，`formatter` 支持函数自定义显示格式
- `calendar.yearLabel` - 年份标签，`formatter: '{end}'` 显示结束年份
- `series.coordinateSystem: 'calendar'` - 热力图绑定到日历坐标系
- `visualMap.type: 'piecewise'` - 分段型视觉映射，配合热力图使用效果更佳
- `getEndDay()` 函数利用 `new Date(year, month, 0).getDate()` 巧妙获取任意月份的天数
