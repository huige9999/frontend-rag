# 48-漏斗图

## 简介
本节课学习 ECharts 漏斗图（funnel）系列的使用，通过"预期"与"实际"两组漏斗数据的对比，展示漏斗图的各种配置和标签样式定制。

## 关键代码

```javascript
option = {
    title: {
        text: '漏斗图',
        subtext: '纯属虚构',
    },
    legend: {
        data: ['访问', '咨询', '订单', '点击', '展现'],
    },
    tooltip: {
        formatter: '{a}<br>{b} : {c}%',
    },
    toolbox: {
        feature: {
            dataView: {},
            restore: {},
            saveAsImage: {}
        }
    },
    series: [
        {
            name: '预期',
            type: 'funnel',
            min: 0,
            max: 100,
            minSize: '0%',
            maxSize: '100%',
            sort: 'descending',
            gap: 0,
            width: '80%',
            label: {
                show: true,
                fontSize: 26,
                position: 'right',
                formatter: '{b}预期量'
            },
            labelLine: {
                show: true,
                length: 50,
                lineStyle: {
                    color: '#000',
                    width: 3,
                }
            },
            itemStyle: {
                opacity: 0.7
            },
            emphasis: {
                label: {
                    textBorderColor: '#000',
                    textBorderWidth: 1,
                    textShadowColor: '#ccc',
                    textShadowBlur: 10,
                    formatter: '{b}预期量：{c}%',
                },
                labelLine: {
                    lineStyle: { color: 'red' }
                }
            },
            data: [
                { name: '访问', value: 60 },
                { name: '咨询', value: 40 },
                { name: '订单', value: 20 },
                { name: '点击', value: 80 },
                { name: '展现', value: 100 },
            ]
        },
        {
            name: '实际',
            type: 'funnel',
            width: '80%',
            maxSize: '80%',
            label: {
                position: 'inside',
                formatter: '{c}%',
                textStyle: { fontSize: 20 }
            },
            itemStyle: {
                opacity: 0.5,
                borderWidth: 2,
            },
            emphasis: {
                label: {
                    formatter: '{b}实际量：{c}%'
                }
            },
            data: [
                { name: '访问', value: 30 },
                { name: '咨询', value: 10 },
                { name: '订单', value: 5 },
                { name: '点击', value: 50 },
                { name: '展现', value: 80 },
            ]
        }
    ]
};
```

### 代码说明
- `type: 'funnel'` - 指定系列类型为漏斗图
- `min` / `max` - 数据的最小值和最大值范围
- `minSize: '0%'` / `maxSize: '100%'` - 漏斗最小/最大宽度的映射百分比
- `sort: 'descending'` - 数据排序方式，`'descending'` 为从大到小（默认），也可选 `'ascending'` 或 `'none'`
- `gap: 0` - 漏斗层之间的间隔
- `width: '80%'` - 漏斗图的整体宽度
- `label.position` - 标签位置，`'right'` 在右侧外部，`'inside'` 在漏斗内部
- `labelLine` - 标签引导线配置，`length` 控制线长
- `itemStyle.opacity` - 漏斗块的透明度，多组漏斗叠加时通过不同透明度区分
- `emphasis` - 鼠标悬浮时的高亮样式，可单独定制 `label` 和 `labelLine`
- 两个 series 叠加实现了"预期 vs 实际"的对比漏斗图
- `tooltip.formatter: '{a}<br>{b} : {c}%'` - `{a}` 系列名、`{b}` 数据名、`{c}` 数据值
