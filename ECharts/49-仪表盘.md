# 49-仪表盘

## 简介
本节课学习 ECharts 仪表盘（gauge）系列的使用，通过模拟汽车仪表盘（速度表、转速表、油表、水温表），展示仪表盘的各种样式配置和动态数据更新。

## 关键代码

```javascript
var option = {
    backgroundColor: '#1b1b1b',
    tooltip: {
        formatter: '{a}<br>{c} {b}'
    },
    toolbox: {
        feature: {
            saveAsImage: {}
        }
    },
    series: [
        {
            // 速度表
            name: '速度',
            type: 'gauge',
            data: [{ name: 'km/h', value: 40 }],

            endAngle: -45,
            clockwise: true,
            min: 0,
            max: 220,

            axisLine: {
                show: true,

```

### 代码说明
- `type: 'gauge'` - 指定系列类型为仪表盘
- `radius` - 仪表盘半径，支持百分比或像素值
- `center` - 仪表盘中心位置，`[x, y]` 格式，用于在同一个图表中放置多个仪表盘
- `startAngle` / `endAngle` - 仪表盘的起始和结束角度（度数）
- `clockwise: true` - 刻度是否顺时针增长
- `min` / `max` - 刻度范围
- `splitNumber` - 分割段数
- `axisLine.lineStyle.color` - 轴线颜色，使用二维数组 `[[比例, 颜色], ...]` 实现分段颜色（如绿-蓝-红代表安全-正常-危险）
- `splitLine` - 分割线样式（主刻度线）
- `axisTick` - 刻度线样式（小刻度），`splitNumber` 指定两个主刻度间的小刻度数
- `pointer` - 指针配置，`length` 和 `width` 控制指针大小
- `title` - 仪表盘标题（单位名称），`fontSize` 和 `fontStyle` 控制文字样式
- `detail` - 仪表盘详情（数值显示），`offsetCenter` 控制偏移位置
- `axisLabel.formatter` - 通过函数自定义刻度标签文本（如油表的 E/Gas/F）
- `setInterval` - 使用定时器每2秒更新数据，模拟实时动态仪表盘效果
