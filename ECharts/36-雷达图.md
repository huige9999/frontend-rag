# 36-雷达图

## 简介
本节课学习 ECharts 雷达图（radar）的配置，包括雷达坐标系 indicator 指标定义、形状设置、多系列数据对比、区域样式以及图例单选模式切换不同城市数据。

## 关键代码

### 雷达图配置

```javascript
option = {
  backgroundColor: '#161627',
  title: {
    text: 'AQI - 雷达图',
    left: 'center',
    top: 10,
    textStyle: {
      color: '#eee',
    }
  },
  legend: {
    data: ['北京', '上海', '广州'],
    bottom: 5,
    itemGap: 20,
    textStyle: {
      color: '#fff',
      fontSize: 14,
    },
    selectedMode: 'single',  // 单选模式，一次只显示一个城市
  },
  radar: {
    indicator: [
      { name: 'AQI', max: 300 },
      { name: 'PM2.5', max: 250 },
      { name: 'PM10', max: 300 },
      { name: 'CO', max: 5 },
      { name: 'NO2', max: 200 },
      { name: 'SO2', max: 100 },
    ],
    shape: 'circle',
    name: {
      color: 'rgb(238, 197, 102)'
    },
    splitArea: {
      show: false,
    },
    splitLine: {
      lineStyle: {
        color: [
          'rgba(238, 197, 102, 1)',
          'rgba(238, 197, 102, 0.8)',
          'rgba(238, 197, 102, 0.6)',
          'rgba(238, 197, 102, 0.4)',
          'rgba(238, 197, 102, 0.2)',
          'rgba(238, 197, 102, 0.1)',
        ]
      }
    },
    axisLine: {
      lineStyle: {
        color: 'rgba(238, 197, 102, 0.5)',
      }
    }
  },
  series: [
    {
      name: '北京',
      data: dataBJ,
      type: 'radar',
      symbol: 'none',
      symbolSize: 10,
      itemStyle: {
        color: '#F9713C',
        borderWidth: 2,
        borderColor: '#F9713C'
      },
      lineStyle: {
        color: '#F9713C',
        width: 1,
        opacity: 0.5
      },
      areaStyle: {
        color: '#F9713C',
        opacity: 0.1
      },
      emphasis: {
        lineStyle: { opacity: 0.7 },
        areaStyle: { opacity: 0.5 },
      }
    },
    // 上海、广州系列配置结构同上
  ]
};
```

### 代码说明
- `radar.indicator` - 雷达坐标系的指标列表，每项需指定 `name` 和 `max`（最大值）
- `radar.shape` - 雷达图形状，`'circle'` 为圆形，默认为 `'polygon'` 多边形
- `selectedMode: 'single'` - 图例单选模式，点击一个城市时自动隐藏其他城市
- `type: 'radar'` - 系列类型为雷达图
- `areaStyle` - 雷达区域填充样式，`opacity` 控制填充透明度
- `lineStyle` - 雷达连线的颜色、宽度和透明度
- `emphasis` - 高亮状态配置，提升连线和区域的透明度以突出显示
- 每个数据项为 7 天的 AQI 指标值数组，对应 indicator 的 6 个维度
