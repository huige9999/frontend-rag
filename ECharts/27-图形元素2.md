# 27-图形元素2

## 简介
本节课继续学习 ECharts 图形元素（graphic）的高级用法，掌握扇形（sector）、圆弧（arc）、多边形（polygon）、折线（polyline）、直线（line）、贝塞尔曲线（bezierCurve）以及分组（group）等图形元素的配置。

## 关键代码

### 扇形与圆弧 - sector、arc

```javascript
// 扇形
{
  type: 'sector',
  shape: {
    cx: 450,
    cy: 450,
    r: 100,               // 外半径
    r0: 50,               // 内半径（空心扇形）
    startAngle: Math.PI / 180 * 0,    // 起始角度（弧度）
    endAngle: Math.PI / 180 * 180,    // 结束角度（弧度）
    clockwise: true,      // 顺时针方向
  },
  style: {
    fill: 'rgba(15,186,100,0.5)',
    stroke: 'blue',
    lineWidth: 5,
    shadowBlur: 10,
    shadowOffsetX: 10,
    shadowOffsetY: 10,
    shadowColor: '#000',
  }
},
// 圆弧
{
  type: 'arc',
  shape: {
    cx: 600,
    cy: 340,
    r: 100,
    r0: 50,
    startAngle: Math.PI / 180 * 0,
    endAngle: Math.PI / 180 * 180,
    clockwise: true,
  },
  style: {
    fill: 'rgba(15,186,100,0.5)',
    stroke: 'blue',
    lineWidth: 5,
    shadowBlur: 10,
    shadowColor: '#000',
  }
}
```

### 多边形与折线 - polygon、polyline

```javascript
// 多边形
{
  type: 'polygon',
  shape: {
    points: [[230, 630], [130, 780], [230, 780]],  // 顶点坐标
    smooth: 'spline',  // 平滑曲线，也可设为数字如 0.4
  },
  style: {
    fill: 'rgba(15,186,100,0.5)',
    stroke: 'blue',
    lineWidth: 5,
    shadowBlur: 10,
    shadowColor: '#000',
  }
},
// 折线
{
  type: 'polyline',
  z: 5,
  shape: {
    points: [[430, 630], [330, 780], [430, 780]],
    smooth: 'spline',  // 平滑值设为 1
    smooth: 1,
  },
  style: {
    stroke: 'blue',     // 折线通常只设描边
    lineWidth: 10,
    shadowBlur: 10,
    shadowColor: '#000',
  }
}
```

### 直线与贝塞尔曲线 - line、bezierCurve

```javascript
// 直线
{
  type: 'line',
  z: 5,
  shape: {
    x1: 930, y1: 40,    // 起点坐标
    x2: 660, y2: 270,   // 终点坐标
    percent: 0.5,        // 绘制百分比，0.5 表示只画一半
  },
  style: {
    stroke: 'blue',
    lineWidth: 3,
    shadowBlur: 10,
    shadowColor: '#000',
  }
},
// 贝塞尔曲线
{
  type: 'bezierCurve',
  z: 5,
  shape: {
    x1: 0, y1: 0,       // 起点
    x2: 1120, y2: 460,  // 终点
    cpx1: 260, cpy1: 450,  // 控制点1
    cpx2: 860, cpy2: 50,   // 控制点2
  },
  style: {
    stroke: 'pink',
    lineWidth: 5,
  }
}
```

### 分组 - group

```javascript
{
  type: 'group',
  rotation: Math.PI / 180 * 45,  // 整组旋转45度
  right: 220,
  bottom: 220,
  bounding: 'raw',    // 定位方式

  children: [
    {
      // 半透明背景方块
      type: 'rect',
      left: 'center',
      top: 'center',
      z: 100,
      shape: {
        width: 800,
        height: 100,
      },
      style: {
        fill: 'rgba(0,0,0,0.3)',
      }
    },
    {
      // 文字
      type: 'text',
      left: 'center',
      top: 'center',
      z: 100,
      style: {
        text: '作者：陈学辉',
        fill: '#fff',
        font: 'bold 30px Microsoft YaHei',
      }
    }
  ]
}
```

### 代码说明
- `sector` - 扇形，通过 `r`/`r0` 控制内外半径实现空心效果，`startAngle`/`endAngle` 控制弧度范围
- `arc` - 圆弧，参数与扇形类似，但只绘制弧线不填充
- `polygon` - 多边形，通过 `points` 数组定义各顶点坐标
- `polyline` - 折线，通过 `points` 定义折线经过的点
- `smooth` - 平滑设置，`'spline'` 表示样条平滑，也可用数字（0~1）控制平滑程度
- `line` - 直线，`x1/y1` 为起点，`x2/y2` 为终点，`percent` 控制绘制进度（可用于动画）
- `bezierCurve` - 贝塞尔曲线，`cpx1/cpy1` 和 `cpx2/cpy2` 分别为两个控制点
- `group` - 图形分组，可以将多个图形元素组合在一起统一管理
- `group.rotation` - 整组旋转角度（弧度制）
- `group.bounding` - 定位计算方式，`'raw'` 表示直接定位
- `group.children` - 分组内的子图形元素数组
- `Math.PI / 180 * 角度` - 将角度转换为弧度的标准公式
