# 25-时间轴

## 简介
本节课学习 ECharts 时间轴（timeline）组件的配置方法，通过全国各省份 2002-2011 年 GDP 数据的动态切换案例，掌握 `baseOption` + `options` 的组合模式以及时间轴的播放、样式等核心配置。

## 关键代码

### 数据格式化函数

```javascript
// 将原始数组数据转换为 {name, value} 格式，并计算每年总和
function dataFormatter(obj) {
  var pList = ['北京', '天津', '河北', '山西', '内蒙古', '辽宁', '吉林',
    '黑龙江', '上海', '江苏', '浙江', '安徽', '福建', '江西', '山东',
    '河南', '湖北', '湖南', '广东', '广西', '海南', '重庆', '四川',
    '贵州', '云南', '西藏', '陕西', '甘肃', '青海', '宁夏', '新疆'];

  for (var year = 2002; year <= 2011; year++) {
    var sum = 0;
    temp = obj[year];
    for (var i = 0; i < temp.length; i++) {
      sum += temp[i];
      obj[year][i] = {
        name: pList[i],
        value: temp[i]
      };
    }
    obj[year + 'sum'] = sum;  // 存储年总和，如 obj['2002sum']
  }
  return obj;
}
```

### 代码说明
- `timeline` - 时间轴组件，用于在多个数据状态之间切换
- `baseOption` - 存放所有时间点共享的配置（坐标轴、系列类型、时间轴本身等）
- `options` - 数组，每个元素对应一个时间点的差异化配置（标题、数据等）
- `timeline.type: 'slider'` - 滑动型时间轴，也可设为 `'point'`（点状）
- `timeline.axisType: 'category'` - 时间轴类型为类目型
- `timeline.data` - 时间轴节点数据，支持字符串或对象（对象可单独配置样式和提示）
- `timeline.autoPlay` - 自动轮播切换
- `timeline.playInterval` - 切换间隔时间（毫秒）
- `timeline.loop` / `timeline.rewind` - 循环播放和反向播放控制
- `timeline.currentIndex` - 默认选中的时间节点索引
- `timeline.checkpointStyle` - 当前选中节点的样式（如用 pin 标记）
- `timeline.controlStyle` - 播放控件样式（播放按钮、前进后退等）
- `baseOption + options` 模式是 timeline 的核心用法：baseOption 定义不变的框架，options 数组中的每一项与 `timeline.data` 一一对应
