# 41-盒须图

## 简介
本节课学习 ECharts 盒须图（boxplot）的配置，结合 dataTool 数据工具将原始数据转换为盒须图数据格式，展示数据分布的五个统计量（最小值、Q1、中位数、Q3、最大值），并配合散点图显示离群值。

## 关键代码

### 数据预处理与盒须图配置

```javascript
// 原始数据：5 组实验数据
var data = [
  [850, 740, 900, 1070, 930, 850, 950, 980, 980, 880, 1000, 980, 930, 650, 760, 810, 1000, 1000, 960, 960],
  [960, 940, 960, 940, 880, 800, 850, 880, 900, 840, 830, 790, 810, 880, 880, 830, 800, 790, 760, 800],
  [880, 880, 880, 860, 720, 720, 620, 860, 970, 950, 880, 910, 850, 870, 840, 840, 850, 840, 840, 840],
  [890, 810, 810, 820, 800, 770, 760, 740, 750, 760, 910, 920, 890, 860, 880, 720, 840, 850, 850, 780],
  [890, 840, 780, 810, 760, 810, 790, 810, 820, 850, 870, 870, 810, 740, 810, 940, 950, 800, 810, 870]
];

// 使用 dataTool 转换数据
data = echarts.dataTool.prepareBoxplotData(data);

option = {
  title: [
    {
      text: 'Michelson-Morley Experiment',
      left: 'center',
    },
    {
      text: 'upper: Q3 + 1.5 * IQR \nlower: Q1 - 1.5 * IQR',
      borderWidth: 1,
      borderColor: '#999',
      textStyle: { fontSize: 14 },
      left: '10%',
      top: '90%',
    }
  ],
  grid: {
    left: '10%',
    right: '10%',
    bottom: '15%',
  },
  xAxis: {
    data: data.axisData,
    axisLabel: {
      formatter: 'expr {value}',
    }
  },
  yAxis: {
    name: 'km/s minus 299,000',
    splitArea: { show: true }
  },
  tooltip: {},
  series: [
    {
      name: 'boxplot',
      type: 'boxplot',
      data: data.boxData,
      tooltip: {
        formatter: function (param) {
          return [
            'Experiment ' + param.name + ': ',
            'upper: ' + param.data[5],
            'Q3: ' + param.data[4],
            'median: ' + param.data[3],
            'Q1: ' + param.data[2],
            'lower: ' + param.data[1],
          ].join('<br/>');
        }
      }
    },
    {
      name: 'outlier',
      type: 'scatter',
      data: data.outliers
    }
  ]
};
```

### 代码说明
- `echarts.dataTool.prepareBoxplotData()` - 数据转换工具，将原始数组自动计算为盒须图所需的五个统计量
  - 返回对象包含 `boxData`（盒须图数据）、`outliers`（离群值）和 `axisData`（类目轴数据）
- `type: 'boxplot'` - 指定系列类型为盒须图
- `data.boxData` - 转换后的盒须图数据，每项包含 `[下界, Q1, 中位数, Q3, 上界]`
- `data.outliers` - 离群值数据，用散点图（scatter）单独显示
- `param.data` 数组索引：`[0]` 下界、`[1]` Q1、`[2]` 中位数、`[3]` Q3、`[4]` 上界、`[5]` 上下界值
- `tooltip.formatter` - 自定义提示框，显示完整的五个统计量
- IQR = Q3 - Q1，上界 = Q3 + 1.5 * IQR，下界 = Q1 - 1.5 * IQR
- 需要引入 `dataTool.js` 插件文件
