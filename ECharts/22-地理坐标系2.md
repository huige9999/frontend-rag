# 22-地理坐标系2

## 简介
本节课学习 ECharts 地理坐标系（geo）配置项的详细用法，通过全国主要城市 PM2.5 数据的散点图案例，掌握 geo 组件的区域样式、标注、漫游、缩放限制以及 regions 区域定制等核心配置。

## 关键代码

### 数据准备 - 城市坐标与PM数据

```javascript
// 城市的地理坐标映射
var geoCoordMap = {
  "海门": [121.15, 31.89],
  "鄂尔多斯": [109.781327, 39.608266],
  // ... 更多城市坐标
  "北京": [116.46, 39.92],
  "上海": [121.48, 31.22],
  "广州": [113.23, 23.16],
};

// 每个城市的PM2.5数据
var pmData = [
  { name: "海门", value: 9 },
  { name: "鄂尔多斯", value: 12 },
  // ... 更多城市数据
];

// 数据格式转换函数：将坐标与PM值合并为 [经度, 纬度, pm值]
function converData() {
  var res = [];
  for (var i = 0; i < pmData.length; i++) {
    var geoData = geoCoordMap[pmData[i].name];
    res.push({
      name: pmData[i].name,
      value: geoData.concat(pmData[i].value),
    });
  }
  return res;
}
```

### geo 配置项 - 地理坐标系完整配置

```javascript
option = {
  backgroundColor: '#404a59',
  title: {
    text: '全国主要城市 PM2.5',
    left: 'center',
    textStyle: {
      color: '#fff',
    }
  },
  tooltip: {
    trigger: 'item',
    formatter: function (params) {
      return params.name + ' : ' + params.value[2];
    }
  },
  visualMap: {
    type: 'continuous',
    min: 0,
    max: 200,
    calculable: true,
    inRange: {
      color: ['#50a3ba', '#eac736', '#d94e5d']
    },
    textStyle: {
      color: '#fff'
    }
  },
  geo: {
    show: true,
    map: 'china',           // 使用中国地图（需引入 china.js）
    roam: true,             // 开启鼠标漫游（缩放和平移）
    aspectScale: 0.75,      // 地图长宽比
    scaleLimit: {
      min: 1,               // 最小缩放倍数
      max: 15,              // 最大缩放倍数
    },
    selectedMode: 'multiple', // 选中模式：多选
    label: {
      show: true,           // 显示省份名称
      color: 'red',
      fontSize: 16,
    },
    itemStyle: {
      areaColor: 'grey',    // 区域填充色
      borderColor: '#fff',  // 边界线颜色
      borderWidth: 2,
    },
    emphasis: {
      label: {
        color: 'blue',
        fontWeight: 'bold',
        fontSize: 30,
      },
      itemStyle: {
        areaColor: 'teal',
        borderColor: 'darkblue'
      }
    },
    layoutCenter: ['50%', '50%'],  // 地图中心位置
    layoutSize: '100%',            // 地图大小
    regions: [                     // 特定区域的定制样式
      {
        name: '陕西',
        selected: false,
        itemStyle: {
          areaColor: 'brown',
        },
        label: {
          color: 'black',
        },
        emphasis: {
          itemStyle: {
            areaColor: 'green',
            borderColor: 'gold'
          },
          label: {
            color: '#fff',
            fontSize: 20,
          },
        }
      }
    ]
  },
  series: [
    {
      name: 'pm2.5',
      type: 'scatter',
      coordinateSystem: 'geo',  // 使用地理坐标系
      data: converData(),
      symbolSize: function (val) {
        return Math.max(val[2] / 10, 8);  // 散点大小由PM值决定
      }
    }
  ],
};
```

### 代码说明
- `geo.map` - 指定地图类型，需提前通过 `china.js` 或 `word.js` 注册地图数据
- `geo.roam` - 开启鼠标漫游，支持缩放和平移操作
- `geo.aspectScale` - 地图的纵横比，默认 0.75
- `geo.scaleLimit` - 控制缩放的上下限范围
- `geo.selectedMode` - 区域选中模式，支持 `'single'` 和 `'multiple'`
- `geo.label` - 地图上省份/区域的文本标注配置
- `geo.itemStyle` - 地图区域的默认样式（填充色、边框等）
- `geo.emphasis` - 鼠标悬停时的高亮样式
- `geo.layoutCenter` / `geo.layoutSize` - 精确控制地图在容器中的位置和大小
- `geo.regions` - 对特定区域单独定制样式，如对陕西省设置不同的颜色和高亮效果
- `series.coordinateSystem: 'geo'` - 散点图系列绑定到地理坐标系
- `geoCoordMap` - 城市名称到经纬度坐标的映射对象，是地理散点图的核心数据结构
- `converData()` - 将城市坐标与PM值合并为 `[经度, 纬度, 数值]` 格式
