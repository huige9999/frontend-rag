# 47-桑基图

## 简介
本节课学习 ECharts 桑基图（sankey）系列的使用，通过可视化流量流向展示世界杯球队之间的对阵晋级关系。

## 关键代码

```javascript
option = {
    animation: false,
    title: {
        text: '世界杯',
    },
    series: [
        {
            type: 'sankey',
            nodeWidth: 20,
            nodeGap: 8,
            nodeAlign: 'right',
            orient: 'horizontal',
            draggable: true,
            focusNodeAdjacency: false,
            label: {
                color: 'green',
                fontSize: 22,
            },
            itemStyle: {
                shadowBlur: 10,
                shadowColor: 'rgba(0,0,0,0.6)',
            },
            lineStyle: {
                curveness: 0.5
            },
            data: data.nodes,   // 节点数据（来自 sankeyData.js）
            links: data.links,  // 边数据（来自 sankeyData.js）
        }
    ]
};
```

### 代码说明
- `type: 'sankey'` - 指定系列类型为桑基图
- `nodeWidth: 20` - 桑基图中每个节点的宽度（像素）
- `nodeGap: 8` - 节点之间的垂直间距
- `nodeAlign: 'right'` - 节点对齐方式，可选 `'left'`、`'right'`、`'justify'`
- `orient: 'horizontal'` - 桑基图的方向，可选 `'horizontal'`（水平）或 `'vertical'`（垂直）
- `draggable: true` - 允许拖拽节点改变位置
- `focusNodeAdjacency: false` - 是否高亮相邻节点，设为 false 时使用全局 emphasis 样式
- `label` - 节点标签配置，`color` 和 `fontSize` 控制文本颜色和大小
- `itemStyle.shadowBlur` / `shadowColor` - 节点的阴影效果
- `lineStyle.curveness: 0.5` - 边的弯曲度，0为直线，1为最大弯曲
- `data` - 节点数据数组，每个节点需有 `name` 属性，可选 `depth` 指定层级
- `links` - 边数据数组，每条边需有 `source`（起点名）、`target`（终点名）、`value`（流量值）
- `animation: false` - 关闭动画（数据量大时提升性能）
