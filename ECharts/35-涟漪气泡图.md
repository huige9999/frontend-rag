# 35-涟漪气泡图

## 简介
本节课学习 ECharts 散点图（scatter）与涟漪散点图（effectScatter）的组合使用，通过男女身高体重分布数据，演示 markPoint、markLine、markArea 标注功能，以及涟漪特效散点图的配置。

## 关键代码

### 散点图 + 涟漪气泡图

```javascript
option = {
  title: {
    text: '男性女性身高体重分布',
    subtext: '抽样调查来自: Heinz 2003',
  },
  legend: {
    left: 'center'
  },
  grid: {
    left: '3%',
    right: '7%',
    bottom: '3%',
    containLabel: true,
  },
  xAxis: {
    scale: true,
    splitLine: { show: false },
    axisLabel: {
      formatter: '{value} cm'
    }
  },
  yAxis: {
    scale: true,
    splitLine: { show: false },
    axisLabel: {
      formatter: '{value} kg'
    }
  },
  toolbox: {
    feature: {
      dataZoom: {}
    }
  },
  brush: {
    toolbox: ['rect', 'polygon', 'clear']
  },
  tooltip: {
    formatter: function (params) {
      return params.seriesName + '：<br>'
        + params.value[0] + 'cm '
        + params.value[1] + 'kg ';
    },
    axisPointer: {
      show: true,
      type: 'cross',
    }
  },
  series: [
    {
      name: '女性',
      type: 'scatter',
      data: female,
      markPoint: {
        data: [
          { type: 'max', name: '最大值', valueIndex: 0 },
          { type: 'min', name: '最小值', valueIndex: 0 },
        ]
      },
      markLine: {
        data: [
          { type: 'average', name: '平均值' },
          { xAxis: 160 }
        ],
        lineStyle: { type: 'solid' }
      },
      markArea: {
        silent: true,
        data: [
          [
            { name: '女性数据分布区域', xAxis: 'min', yAxis: 'min' },
            { xAxis: 'max', yAxis: 'max' }
          ]
        ],
        itemStyle: {
          color: 'transparent',
          borderWidth: 1,
          borderType: 'dashed'
        }
      }
    },
    {
      name: '男性',
      type: 'scatter',
      data: male,
      // 同上 markPoint / markLine / markArea 配置
    },
    {
      name: '身高 Top5',
      type: 'effectScatter',
      data: male.sort(function (a, b) {
        return b[0] - a[0];
      }).slice(0, 5),
      symbolSize: 20,

      effectType: 'ripple',
      showEffectOn: 'render',
      rippleEffect: {
        period: 4,
        scale: 2.5,
        brushType: 'fill',
      }
    }
  ]
};
```

### 代码说明
- `type: 'scatter'` - 普通散点图系列
- `type: 'effectScatter'` - 涟漪特效散点图，带动画效果
- `scale: true` - 坐标轴自动缩放，不强制从 0 开始
- `markPoint` - 标注点，支持 `max`、`min`、`average` 类型，`valueIndex` 指定维度
- `markLine` - 标线，可标记平均值、固定值等
- `markArea` - 标记区域，用虚线框标识数据分布范围
- `brush` - 区域选择组件，支持矩形、多边形框选
- `effectType: 'ripple'` - 涟漪效果类型
- `showEffectOn: 'render'` - 渲染时就显示特效（也可设为 `'emphasis'` 悬浮时才显示）
- `rippleEffect` - 涟漪参数：`period` 周期、`scale` 缩放倍数、`brushType` 填充方式
