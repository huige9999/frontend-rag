# 40-旭日图

## 简介
本节课学习 ECharts 旭日图（sunburst）的配置，包括多层级环形数据展示、层级样式配置、高亮/下钻策略、标签旋转方式以及点击下钻交互。数据来源于 sunburstData.js（世界咖啡研究感官词典数据）。

## 关键代码

### 旭日图配置

```javascript
option = {
  title: {
    text: 'WORLD COFFEE RESEARCH SENSORY LEXICON',
    subtext: 'Source: https://worldcoffeeresearch.org/work/sensory-lexicon/',
    left: 'center',
    sublink: 'https://worldcoffeeresearch.org/work/sensory-lexicon/'
  },
  series: {
    type: 'sunburst',
    data: data,

    center: ['50%', '50%'],
    radius: ['20%', '75%'],

    label: {
      rotate: 'radial',    // 标签沿径向旋转
      align: 'center',
      minAngle: 0,
    },

    itemStyle: {
      color: 'green',
    },

    highlightPolicy: 'descendant',
    emphasis: {
      itemStyle: {
        shadowColor: '#000',
        shadowBlur: 15,
      }
    },
    highlight: {
      itemStyle: {
        color: '#333',
      }
    },
    downplay: {
      itemStyle: {
        opacity: 0.3,
      }
    },

    nodeClick: 'rootToNode',
    sort: 'desc',
    renderLabelForZeroData: true,

    levels: [
      {
        // 下钻点的空白配置
        itemStyle: {
          color: 'pink',
        }
      },
      {
        // 第一层（最内侧）
        itemStyle: {
          borderWidth: 5,
        },
        label: {
          color: 'blue',
          fontSize: 16,
        }
      },
      {
        // 第二层
        label: {
          color: '#000',
          fontWeight: 'bold',
        }
      },
      {
        // 第三层
        r0: '70%',
        r: '72%',
        label: {
          position: 'outside',
          padding: 3,
        },
        itemStyle: {
          borderWidth: 3,
        }
      }
    ],
  }
};
```

### 代码说明
- `type: 'sunburst'` - 指定系列类型为旭日图
- `radius: ['20%', '75%']` - 内半径和外半径，形成环形展示
- `label.rotate: 'radial'` - 标签沿径向方向旋转，也可设为 `'tangential'`（切向）或角度值
- `highlightPolicy: 'descendant'` - 高亮策略，悬浮时高亮当前节点及其所有后代节点
- `emphasis` - 当前悬浮扇形块的样式（添加阴影效果）
- `highlight` - 相关扇形块（后代节点）的高亮样式
- `downplay` - 不相关扇形块的降低显示样式（降低透明度）
- `nodeClick: 'rootToNode'` - 点击节点时下钻到该节点并以该节点为根显示
- `sort: 'desc'` - 子节点按值降序排列
- `renderLabelForZeroData` - 是否渲染数据值为 0 的标签
- `levels` - 各层级样式配置，从内到外依次配置，可独立控制每层的颜色、边框、标签样式
- `levels` 中 `r0` 和 `r` 可单独调整某一层的内外半径
