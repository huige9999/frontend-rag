# 42-K线图

## 简介
本节课学习 ECharts K 线图（candlestick）的完整配置，包括股票 K 线数据格式、涨跌颜色设置、markPoint 标注极值、markLine 标线、移动平均线（MA）计算、dataZoom 区域缩放以及十字准星指示器。

## 关键代码

### 数据处理函数

```javascript
// 股票数据格式：[日期, 开盘价, 收盘价, 最低价, 最高价]
// 拆分类目数据和数值数据
function splitData(data) {
  var categoryData = [];
  var values = [];
  for (var i = 0; i < data.length; i++) {
    categoryData.push(data[i].splice(0, 1));
    values.push(data[i]);
  }
  return {
    categoryData: categoryData,
    values: values,
  }
}

// 计算移动平均线 MA
function getMAData(dayCount) {
  var result = [];
  for (var i = 0; i < data.values.length; i++) {
    if (i < dayCount) {
      result.push('-');
      continue;
    }
    var sum = 0;
    for (var j = 0; j < dayCount; j++) {
      sum += data.values[i - j][1];  // 收盘价下标为 1
    }
    result.push(sum / dayCount);
  }
  return result;
}
```

### K线图配置

```javascript
var upColor = '#ec0000',

      markPoint: {
```

### 代码说明
- `type: 'candlestick'` - 指定系列类型为 K 线图（蜡烛图）
- K 线数据格式：`[开盘价, 收盘价, 最低价, 最高价]`
- `color / color0` - 阳线（涨）和阴线（跌）的填充颜色
- `borderColor / borderColor0` - 阳线和阴线的边框颜色
- `valueDim` - 指定标注计算的维度：`'open'`、`'close'`、`'lowest'`、`'highest'`
- `markPoint` - 标注点，标记最高值、最低值、平均值等
- `markLine` - 标线，可连接最低到最高形成区间线
- `boundaryGap: false` - X 轴不留边距，K 线紧贴坐标轴
- `dataZoom` - 区域缩放组件，`'inside'` 支持鼠标滚轮缩放，`'slider'` 显示滑动条
- `tooltip.trigger: 'axis'` - 坐标轴触发提示框
- `axisPointer.type: 'cross'` - 十字准星指示器
- `getMAData(dayCount)` - 计算 N 日移动平均线，用折线图（line）叠加显示
