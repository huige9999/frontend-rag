# 37-树图

## 简介
本节课学习 ECharts 树图（tree）的配置，包括树形数据的层级结构、正交/径向布局方向、展开折叠交互、节点样式和连接线样式设置。数据来源于 threeData.js 文件。

## 关键代码

### 树图配置

```javascript
option = {
  tooltip: {},
  series: [
    {
      type: 'tree',
      data: [data],
      left: '2%',
      right: '2%',
      top: '8%',
      bottom: '20%',

      layout: 'orthogonal',  // 正交布局
      orient: 'LR',          // 从左到右

      symbol: 'emptyCircle',
      symbolSize: 7,

      roam: true,

      expandAndCollapse: true,
      initialTreeDepth: 2,

      itemStyle: {
        color: 'green',
      },
      label: {
        position: 'left',
        fontSize: 14,
      },
      lineStyle: {
        color: 'pink',
        width: 1,
        curveness: 0.5
      },
      emphasis: {
        label: {
          fontSize: 20,
        },
      },
      leaves: {
        label: {
          position: 'right',
        }
      }
    }
  ]
};
```

### 代码说明
- `type: 'tree'` - 指定系列类型为树图
- `data` - 树形数据格式为嵌套的 `{ name, children: [...] }` 结构
- `layout: 'orthogonal'` - 正交布局（直线连接），也可选 `'radial'`（径向布局）
- `orient: 'LR'` - 布局方向，从左到右（Left to Right），还支持 `'RL'`、`'TB'`、`'BT'`
- `symbol: 'emptyCircle'` - 节点图形为空心圆
- `roam: true` - 允许鼠标缩放和平移
- `expandAndCollapse` - 是否开启展开/折叠子树交互
- `initialTreeDepth: 2` - 初始展开的层级深度，超出部分折叠
- `lineStyle.curveness` - 连接线曲率，0 为直线，1 为最大弧度
- `leaves` - 叶子节点的特殊样式配置，此处将标签放在右侧
- `emphasis` - 高亮状态配置，放大叶子节点字体
