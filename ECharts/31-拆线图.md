# 31-拆线图

## 简介
本节课学习 ECharts 折线图（series-line）的详细配置，通过北京 AQI 空气质量指数的折线图案例，掌握折线样式、数据标注（markPoint）、标线（markLine）、标记区域（markArea）、视觉映射等核心功能。

## 关键代码

### 折线图完整配置

```javascript
option = {
  title: {
    text: 'Beijing AQI'
  },
  legend: {},
  tooltip: {
    trigger: 'axis',
  },
  xAxis: {
    data: data.map(function (item) {
      return item[0];     // 日期作为X轴（数据来自AQIData.js）
    }),
  },
  yAxis: {
    splitLine: {
      show: false
    }
  },
  toolbox: {
    right: '10%',
    feature: {
      dataZoom: {
        yAxisIndex: 'none'  // 数据缩放工具
      },
      restore: {},           // 还原工具
      saveAsImage: {}        // 保存图片工具
    }
  },
  dataZoom: [
    {
      type: 'inside',        // 内置型数据缩放
      start: 0,
      end: 10
    },
    {
      type: 'slider',        // 滑动型数据缩放
      start: 0,
      end: 10,
    }
  ],
  visualMap: {
    top: 10,
    right: 10,
    type: 'piecewise',       // 分段型视觉映射
    pieces: [
      { gt: 0, lte: 50, color: '#096' },        // 优
      { gt: 50, lte: 100, color: '#ffde33' },    // 良
      { gt: 100, lte: 150, color: '#ff9933' },   // 轻度污染
      { gt: 150, lte: 200, color: '#cc0033' },   // 中度污染
      { gt: 200, lte: 300, color: '#660099' },   // 重度污染
      { gt: 300, color: '#7e0023' }               // 严重污染
    ],
    outOfRange: {
      color: '#999'
    }
  },
  series: {
    type: 'line',
    name: 'Beijing AQI',
    coordinateSystem: 'cartesian2d',  // 直角坐标系

    data: data.map(function (item) {
      return item[1];       // AQI值作为Y轴
    }),

    // 折线样式
    symbol: 'emptyCircle',  // 标记形状：空心圆
    symbolSize: 10,
    showAllSymbol: 'auto',  // 显示所有标记
    hoverAnimation: true,   // 悬停动画
    legendHoverLink: false, // 图例悬停联动
    cursor: 'crossHair',    // 鼠标样式
    connectNulls: false,    // 是否连接空数据
    step: false,            // 是否为阶梯线图

    label: {
      show: true,
      position: 'right',    // 标签位置
      distance: 10,         // 距离标记点的距离
    },
    itemStyle: {
      color: 'green',
      borderColor: 'green',
    },
    lineStyle: {
      width: 0,             // 线宽为0则不显示折线，只显示面积/标记点
    },
    areaStyle: {},           // 面积样式，空对象使用默认
    smooth: true,            // 平滑曲线

    // 标注点 - markPoint
    markPoint: {
      data: [
        {
          type: 'max',       // 自动标注最大值
          coord: ['2000-06-14', 100],  // 通过坐标指定标注点
          value: 'k',
          symbol: 'pin'
        },
        {
          coord: ['2001-03-06', 338],
          value: 338,
          label: { color: 'yellow' },
          itemStyle: { color: 'green' },
          emphasis: {
            label: { color: '#fff' },
            itemStyle: { color: 'teal' },
          }
        }
      ],
      symbol: 'arrow',
      symbolSize: 50,
      label: { color: '#fff' },
      itemStyle: { color: 'blue' },
    },

    // 标线 - markLine
    markLine: {
      data: [
        [
          {
            name: '通过 x, y 属性指定起点坐标',
            x: 50, y: 50,   // 起点像素坐标
            symbol: 'roundRect',
            symbolSize: 20,
            lineStyle: { color: 'red' },
            label: { show: true, position: 'middle' }
          },
          {
            x: 400, y: 400,  // 终点像素坐标
          }
        ],
        [
          {
            name: '通过coord属性指定起点与终点的坐标',
            coord: [10, 20]  // 起点数据坐标
          },
          {
            coord: [480, 490] // 终点数据坐标
          }
        ],
        {
          name: '通过最大值与最小值来指定标记线',
          type: 'average',   // 平均值标线
        },
        {
          name: '通过x、y轴来直接指定标记线',
          xAxis: 100,        // 垂直线：在x=100处
        }
      ]
    },

    // 标记区域 - markArea
    markArea: {
      data: [
        [
          { name: 'y轴300-400的范围', yAxis: 300 },
          { yAxis: 400 }
        ],
        [
          {
            name: '两个坐标点之间的标注区域',
            coord: [10, 450],
            itemStyle: { color: 'pink' }
          },
          { coord: [100, 470] }
        ]
      ]
    }
  }
};
```

### 代码说明
- `series.type: 'line'` - 折线图系列
- `symbol` / `symbolSize` - 标记点的形状和大小，`emptyCircle` 为空心圆
- `smooth` - 是否开启平滑曲线（`true` / `false` / 数值）
- `connectNulls` - 是否连接空数据点
- `step` - 是否为阶梯线图（`true` / `'start'` / `'middle'` / `'end'`）
- `lineStyle.width: 0` - 线宽为0时不显示折线，配合 `areaStyle` 实现纯面积图
- `areaStyle` - 面积样式，开启后显示折线下方的填充区域
- `label` - 数据标签配置，`position` 控制标签位置
- `markPoint` - 标注点，用于标记特殊数据：
  - `type: 'max'` / `'min'` / `'average'` 自动标注极值或均值
  - `coord` 通过坐标精确指定标注位置
  - 可自定义标注的 `symbol`、`label`、`itemStyle`
- `markLine` - 标线，用于绘制参考线：
  - 通过 `x/y` 像素坐标或 `coord` 数据坐标指定起终点
  - `type: 'average'` 自动绘制均值线
  - `xAxis: 100` 在指定位置绘制垂直线
- `markArea` - 标记区域，用于高亮某个数据范围：
  - 通过 `yAxis` 指定 Y 轴范围
  - 通过 `coord` 指定两点之间的区域
- `dataZoom` - 数据缩放，`'inside'` 型支持鼠标滚轮缩放，`'slider'` 型提供滑动条
- `visualMap.type: 'piecewise'` - 分段型视觉映射，`pieces` 定义各区间及其颜色
- `toolbox` - 工具栏，提供数据缩放、还原、保存图片等辅助功能
