# 07-坐标轴1

## 简介
本节课学习 ECharts 中 xAxis（X 轴）坐标轴的多轴配置，包括多条 X 轴的设置、轴偏移（offset）等基础属性。

## 关键代码

### 多 X 轴配置

```javascript
var myChart1 = echarts.init(document.getElementById('chart1'));

myChart1.setOption({
    title: {
        text: '当月销售业绩',
    },
    legend: {
        data: ['今日销量', '昨日销量', '前日销量'],
    },
    xAxis: [
        {
            data: ['衬衫', '羊毛衫', '雪纺衫', '裤子', '高跟鞋', '袜子']
        },
        {
            data: ['衬衫1', '羊毛衫1', '雪纺衫1', '裤子1', '高跟鞋1', '袜子1']
        },
        {
            data: ['衬衫2', '羊毛衫2', '雪纺衫2', '裤子2', '高跟鞋2', '袜子2'],
            offset: -100,    // X轴偏移量，负值向上偏移
        }
    ],
    yAxis: {},
    series: [
        {
            name: '今日销量',
            type: 'bar',
            data: [5, 20, 36, 10, 19, 24]
        },
        {
            name: '昨日销量',
            type: 'bar',
            data: [5, 20, 36, 10, 19, 24]
        },
        {
            name: '前日销量',
            type: 'bar',
            data: [5, 20, 36, 10, 19, 24]
        }
    ]
});
```

### 代码说明
- `xAxis` 数组形式 - 当需要多个 X 轴时，使用数组配置，每个元素对应一条 X 轴
- `data` - 坐标轴的类目数据，设置坐标轴上显示的文字标签
- `offset` - 坐标轴相对于默认位置的偏移量，正值向下偏移，负值向上偏移，可用于实现多轴分层显示
- 多 X 轴常用于不同系列数据需要不同类目标注的场景
