# 46-关系图

## 简介
本节课学习 ECharts 关系图（graph）系列的使用，通过力导向布局展示人物关系网络图谱，支持节点拖拽、关系连线标注和分类高亮等交互。

## 关键代码

```javascript
// 创建类目（人物关系分组）
var categories = [];
for (var i = 0; i < 7; i++) {
    categories[i] = {
        name: '人物关系' + (i + 1)
    };
}

option = {
    title: {
        text: '人民的名义关系图谱',
        top: 'bottom',
        left: 'right',
        textStyle: { fontSize: 24 }
    },
    legend: {
        data: categories.map(function (item) {
            return item.name;
        }),
        top: 20,
        itemGap: 20,
    },
    tooltip: {
        formatter: function (param) {
            if (!param.data.des) {
                // 鼠标在边上
                return param.data.source + ' 与 ' + param.data.target + '的关系为：' + param.data.name;
            } else {
                // 鼠标在节点上
                return param.data.name + '<br>' + param.data.des;
            }
        }
    },
    series: {
        name: '人民的名义关系图谱',
        type: 'graph',
        coordinateSystem: null,
        hoverAnimation: true,
        layout: 'force',
        circular: {
            rotateLabel: false,
        },
        force: {
            repulsion: 500,
            gravity: 0.1,
            edgeLength: 30,
        },
        draggable: true,
        roam: true,
        nodeScaleRatio: 0.6,
        focusNodeAdjacency: true,
        symbol: 'circle',
        edgeSymbol: ['none', 'none'],
        itemStyle: {
            borderColor: '#fff',
            borderWidth: 1,
            shadowBlur: 10,
            shadowColor: 'rgba(0,0,0,0.6)',
        },
        lineStyle: {
            color: 'source',
            curveness: 0.3,
        },
        label: {
            show: false,
            textBorderColor: '#666',
            textBorderWidth: 2,
            fontSize: 14,
        },
        edgeLabel: {
            show: true,
            fontSize: 14,
            formatter: function (param) {
                return param.data.name;
            }
        },
        emphasis: {
            label: { fontSize: 20 },
            lineStyle: { width: 10 },
            edgeLabel: { color: 'pink' }
        },
        categories: categories,
        data: data.nodes,   // 节点数据（来自 graphData.js）
        links: data.links,  // 边数据（来自 graphData.js）
    }
};
```

### 代码说明
- `type: 'graph'` - 指定系列类型为关系图
- `layout: 'force'` - 使用力导向布局，节点会自动排列
- `force` - 力导向布局参数
  - `repulsion: 500` - 节点之间的斥力因子，值越大节点间距越大
  - `gravity: 0.1` - 节点受到的向心力，值越大节点越靠中心
  - `edgeLength: 30` - 边的理想长度
- `draggable: true` - 允许拖拽节点
- `roam: true` - 允许缩放和平移画布
- `focusNodeAdjacency: true` - 鼠标悬浮节点时高亮相邻节点和边
- `edgeSymbol` - 边两端的图形，`['none', 'none']` 表示两端均无特殊标记
- `lineStyle.color: 'source'` - 边颜色跟随起点节点的颜色
- `lineStyle.curveness: 0.3` - 边的弯曲度
- `edgeLabel` - 边上的文本标签，通过 `formatter` 显示关系名称
- `categories` - 节点分类，配合 `legend` 实现分类筛选
- `data` - 节点数据数组，每个节点包含 `name`、`symbolSize`、`category`、`des` 等属性
- `links` - 边数据数组，每条边包含 `source`（起点名）、`target`（终点名）、`name`（关系名）
- `tooltip.formatter` - 自定义提示框内容，根据鼠标在节点还是边上显示不同信息
