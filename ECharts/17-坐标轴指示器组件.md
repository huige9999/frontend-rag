# 17-坐标轴指示器组件

## 简介
本节课学习 axisPointer（坐标轴指示器）组件，它是 tooltip 的辅助组件，用于在坐标轴上显示指示线、阴影或十字线，支持全局配置、轴配置和 handle 拖拽手柄功能。

## 关键代码

### axisPointer 全局配置（handle 拖拽手柄）

```javascript
var myChart = echarts.init(document.getElementById('chart'));
var colors = ['#5793f3', '#d14a61', '#675bba'];

myChart.setOption({
    color: colors,
    legend: {},
    grid: {
        right: "20%",
    },
    axisPointer: {
        show: true,              // 全局显示坐标轴指示器
        // type: 'line',        // 指示器类型：line / shadow / cross / none

        value: 10,               // 初始指示值（需配合 status 使用）
        status: 'show',          // 初始状态：show / hide
        handle: {
            show: true,          // 显示拖拽手柄按钮
        },
        link: {},                // 不同轴的 axisPointer 联动
        triggerOn: 'click',     // 触发方式：click / mousemove
    },
    xAxis: {
        axisTick: { alignWithLabel: true },
        data: ['1月', '2月', '3月', '4月', '5月', '6月',
               '7月', '8月', '9月', '10月', '11月', '12月'],
    },
    yAxis: [
        {
            name: '蒸发量',
            position: 'right',
            min: 0, max: 250,
            axisLabel: { formatter: '{value} ml' },
            axisLine: { lineStyle: { color: colors[0] } },
            // axisPointer: { show: true },  // 也可以在轴级别单独配置
        },
        {
            name: '降水量',
            position: 'right',
            offset: 80,
            min: 0, max: 250,
            axisLabel: { formatter: '{value} ml' },
            axisLine: { lineStyle: { color: colors[1] } },
        },
        {
            name: '温度',
            min: 0, max: 25,
            axisLabel: { formatter: '{value} °C' },
            axisLine: { lineStyle: { color: colors[2] } },
        },
    ],
    series: [
        {
            name: '蒸发量', type: 'bar', yAxisIndex: 0,
            data: [2.0, 4.9, 7.0, 23.2, 25.6, 76.7, 135.6, 162.2, 32.6, 20.0, 6.4, 3.3]
        },
        {
            name: '降水量', type: 'bar', yAxisIndex: 1,
            data: [2.6, 5.9, 9.0, 26.4, 28.7, 70.7, 175.6, 182.2, 48.7, 18.8, 6.0, 2.3]
        },
        {
            name: '平均温度', type: 'line', yAxisIndex: 2,
            data: [2.0, 2.2, 3.3, 4.5, 6.3, 10.2, 20.3, 23.4, 23.0, 16.5, 12.0, 6.2]
        },
    ]
});
```

### tooltip 中的 axisPointer 配置（注释参考）

```javascript
tooltip: {
    trigger: 'axis',
    axisPointer: {
        type: 'shadow',      // 指示器类型：line / shadow / cross / none
        axis: 'auto',        // 指示的坐标轴：x / y / angle / radius / auto
        snap: false,         // 是否自动吸附到最近的数据点
        z: 3,                // 层级

        // label 配置
        label: {
            show: true,
            // formatter: '{value} ~',       // 字符串模板
            formatter: function(params) {    // 函数方式
                return params.value + ' ~';
            },
            margin: 20,        // 标签与轴的距离
        },

        // line 类型样式
        // type: 'line',
        lineStyle: {
            color: 'hotpink',
            width: 5,
            type: 'dashed',
        },

        // shadow 类型样式
        // type: 'shadow',
        shadowStyle: {
            color: 'rgba(250,150,150,0.3)',
            shadowColor: 'rgba(0,0,0,0.5)',
            shadowBlur: 10,
        },

        // cross 类型样式
        // type: 'cross',
        crossStyle: {
            color: 'blue',
            width: 5,
        },

        animation: true,     // 是否开启动画
    }
}
```

### 代码说明
- `axisPointer` 可以在三个层级配置：全局（option.axisPointer）、tooltip 内（tooltip.axisPointer）、轴级别（xAxis.axisPointer / yAxis.axisPointer）
- `type` - 指示器类型：'line'（直线）、'shadow'（阴影）、'cross'（十字准星）、'none'
- `handle.show` - 显示拖拽手柄按钮，用户可拖拽手柄来移动 axisPointer
- `value + status` - 设置 axisPointer 的初始位置和显示状态，`value` 指定初始值，`status: 'show'` 使其初始可见
- `link` - 多个轴的 axisPointer 联动，一个轴的指针移动时，其他轴的指针同步移动
- `triggerOn` - 触发方式，'click' 点击触发，'mousemove' 鼠标移动触发
- `snap` - 是否自动吸附到最近的数据点
- `label` - 坐标轴指示器的文本标签，支持 formatter 格式化
- `lineStyle` - line 类型指示器的线条样式
- `shadowStyle` - shadow 类型指示器的阴影样式
- `crossStyle` - cross 类型指示器的十字线样式
