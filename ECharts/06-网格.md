# 06-网格

## 简介
本节课学习 ECharts 的 grid（网格）组件配置，用于控制直角坐标系绘图区域的位置、大小、背景和边框等属性。

## 关键代码

### grid 网格配置

```javascript
var myChart = echarts.init(document.getElementById('chart1'));

myChart.setOption({
    grid: {
        show: true,               // 是否显示网格
        left: '20%',              // 网格左侧距离
        width: '50%',             // 网格宽度
        containLabel: false,      // 是否包含坐标轴刻度标签
        backgroundColor: 'gray',  // 网格背景色
        borderColor: '#f00'       // 网格边框颜色
    },
    xAxis: {
        splitLine: {    // 坐标轴上的分隔线
            show: false,
        },
    },
    yAxis: {
        splitLine: {    // 坐标轴上的分隔线
            show: false,
        },
    },
    series: {
        name: '销量',
        type: 'bar',
        data: [5, 20, 36, 10, 19, 24]
    }
});
```

### 代码说明
- `show` - 控制网格是否显示，默认 `false`
- `left` / `right` / `top` / `bottom` - 网格距离容器各侧的距离，支持像素值和百分比
- `width` / `height` - 网格的宽度和高度，支持像素值和百分比
- `containLabel` - 是否将坐标轴的刻度标签包含在网格区域内，设为 `true` 可防止标签溢出
- `backgroundColor` - 网格区域的背景颜色
- `borderColor` / `borderWidth` - 网格边框的颜色和宽度
- `splitLine` - 坐标轴的分隔线配置，`show: false` 可隐藏分隔线以保持网格区域整洁
