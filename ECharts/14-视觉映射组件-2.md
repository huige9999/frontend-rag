# 14-视觉映射组件-2

## 简介
本节课继续学习 visualMap 组件，代码内容与第13课一致，重点在于深入理解 visualMap 的三种视觉通道定义方式：共同定义（inRange/outOfRange）、分别定义（target/controller）、以及混合定义方式。

## 关键代码

### visualMap 三种视觉通道定义方式

```javascript
var myChart = echarts.init(document.getElementById('chart'));
myChart.setOption({
    visualMap: [
        {
            type: 'continuous',
            min: 0,
            max: 1000,
            range: [300, 800],
            calculable: true,
            realtime: false,
            inverse: false,
            orient: 'vertical',
            precision: 2,
            itemWidth: 40,
            itemHeight: 200,

            text: ['hight', 'low'],
            textGap: 20,
            show: true,

            dimension: 0,
            hoverLink: true,

            // ========== 方式1：共同定义 ==========
            // inRange 和 outOfRange 同时作用于 target（图表数据）和 controller（图例控件）
            // inRange: { color: ['pink'] },
            // outOfRange: { color: ['green'] },

            // ========== 方式2：分别定义 ==========
            // target 控制图表数据的视觉映射，controller 控制图例控件的视觉映射
            // target: {
            //     inRange: { color: ['pink'] },
            //     outOfRange: { color: ['green'] }
            // },
            // controller: {
            //     inRange: { color: ['pink'] },
            //     outOfRange: { color: ['green'] }
            // }

            // ========== 方式3：先全部定义，再单独覆盖 ==========
            // 先通过 inRange 统一定义，再通过 controller.inRange 单独覆盖控件样式
            inRange: {
                color: ['teal', 'teal'],
                symbol: ['circle', 'rect'],
            },
            controller: {
                inRange: {
                    color: ['red', 'green', 'pink'],
                },
            }
        }
    ],
    series: [
        {
            type: 'map',
            mapType: 'china',
            roam: true,
            data: [
                { name: '北京', value: random() },
                { name: '天津', value: random() },
                { name: '上海', value: random() },
                { name: '重庆', value: random() },
                // ... 其他省份数据
            ]
        }
    ]
});
```

### 代码说明
- **方式1 - 共同定义**：`inRange` 和 `outOfRange` 直接写在 visualMap 项下，同时作用于图表数据和图例控件
- **方式2 - 分别定义**：使用 `target` 和 `controller` 两个子对象，分别控制图表数据的视觉映射和图例控件的视觉映射
- **方式3 - 混合定义**：先用顶层的 `inRange` 统一定义所有视觉通道，再用 `controller.inRange` 单独覆盖控件部分的样式
- `inRange.color` - 指定数据范围内的颜色渐变
- `inRange.symbol` - 指定数据范围内的图形类型变化
- `controller.inRange.color` - 仅控制 visualMap 图例控件上的颜色显示
