# 21-地理坐标系1

## 简介
本节课学习 geo（地理坐标系）组件的配置，结合中国地图和城市 PM2.5 数据，讲解地理坐标系的显示、缩放、标注、区域样式以及散点图在地理坐标系上的应用。

## 关键代码

### 数据处理函数

```javascript
// 城市地理坐标映射表
var geoCoordMap = {
    "海门": [121.15, 31.89],
    "鄂尔多斯": [109.781327, 39.608266],
    "上海": [121.48, 31.22],
    "北京": [116.46, 39.92],
    // ... 更多城市坐标
};

// 城市 PM2.5 数据
var pmData = [
    { name: "海门", value: 9 },
    { name: "上海", value: 25 },
    { name: "北京", value: 79 },
    { name: "武汉", value: 273 },
    { name: "大庆", value: 279 },
    // ... 更多城市数据
];

// 数据格式转换函数：将 pmData 转换为 geo 坐标系所需格式
// 输出格式：{ name: '城市名', value: [经度, 纬度, pm25值] }
function converData() {
    var res = [];
    for (var i = 0; i < pmData.length; i++) {
        var geoData = geoCoordMap[pmData[i].name];
        res.push({
            name: pmData[i].name,
            value: geoData.concat(pmData[i].value),
        });
    }
    return res;
}
```

### geo 地理坐标系配置

```javascript
var myChart = echarts.init(document.getElementById('chart'));
myChart.setOption({
    backgroundColor: '#404a59',          // 图表背景色
    title: {
        text: '全国主要城市 PM2.5',
        left: 'center',
        textStyle: { color: '#fff' },
    },
    tooltip: {
        trigger: 'item',
        formatter: function(params) {
            return params.name + ' : ' + params.value[2];
        }
    },
    legend: {
        textStyle: { color: '#fff' },
        right: 0,
        bottom: 0,
    },
    visualMap: {
        type: 'continuous',
        min: 0,
        max: 200,
        calculable: true,
        inRange: {
            color: ['#50a3ba', '#eac736', '#d94e5d'],   // 蓝 -> 黄 -> 红 渐变色
        },
        textStyle: { color: '#fff' },
    },
    geo: {
        show: true,
        map: 'china',                // 地图类型（需引入 china.js）
        roam: true,                  // 是否开启鼠标缩放和平移漫游
        // center: [116.40717, 39.90469],  // 中心点（经纬度）
        aspectScale: 0.75,           // 地图长宽比（0.75 为标准中国地图比例）

        // boundingCoords: [           // 限定地图的经纬度范围
        //     [118.79647, 32.05838],
        //     [121.47370, 31.23037]
        // ],

        // zoom: 2,                   // 当前视角的缩放比例
        scaleLimit: {
            min: 1,                   // 最小缩放值
            max: 15,                  // 最大缩放值
        },
        selectedMode: 'multiple',     // 选中模式：single / multiple / false

        label: {
            show: true,               // 是否显示地图省份名称
            // position: 'left',
            // rotate: 45,
            color: 'red',
            fontSize: 16,
        },
        itemStyle: {                  // 地图区域的默认样式
            areaColor: 'grey',
            borderColor: '#fff',
            borderWidth: 2,
        },
        emphasis: {                   // 高亮状态样式
            label: {
                color: 'blue',
                fontWeight: 'bold',
                fontSize: 30,
            },
            itemStyle: {
                areaColor: 'teal',
                borderColor: 'darkblue',
            }
        },
        layoutCenter: ['50%', '50%'],  // 地图中心在屏幕中的位置
        layoutSize: '100%',             // 地图的大小

        // 单独为某个省份设置样式
        regions: [
            {
                name: '陕西',
                selected: false,
                itemStyle: {
                    areaColor: 'brown',
                },
                label: {
                    color: 'black',
                },
                emphasis: {
                    itemStyle: {
                        areaColor: 'green',
                        borderColor: 'gold',
                    },
                    label: {
                        color: '#fff',
                        fontSize: 20,
                    },
                }
            }
        ]
    },
    series: [
        {
            name: 'pm2.5',
            type: 'scatter',
            coordinateSystem: 'geo',     // 使用地理坐标系
            data: converData(),
            symbolSize: function(val) {
                return Math.max(val[2] / 10, 8);   // 根据PM2.5值计算气泡大小
            }
        }
    ]
});
```

### 代码说明
- `geo` - 地理坐标系组件，用于地图的展示和交互
- `map: 'china'` - 使用中国地图，需要预先引入 `china.js` 地图数据文件
- `roam: true` - 开启鼠标缩放和平移漫游功能
- `aspectScale` - 地图的长宽比，0.75 是中国地图的标准比例
- `center` - 地图的中心点，使用经纬度坐标
- `zoom` - 当前视角的缩放比例
- `scaleLimit` - 缩放范围限制，控制用户缩放的最小和最大比例
- `selectedMode` - 地图区域的选中模式
- `label` - 地图上省份名称的显示配置
- `itemStyle` - 地图区域的默认样式（面积颜色、边框颜色和宽度）
- `emphasis` - 高亮（鼠标悬停）状态下的样式
- `layoutCenter / layoutSize` - 控制地图在容器中的位置和大小
- `regions` - 为特定区域（省份）单独设置样式，可覆盖默认样式
- `coordinateSystem: 'geo'` - 在 series 中指定使用 geo 地理坐标系
- `converData()` - 将 PM2.5 数据转换为 geo 坐标系所需格式 `[经度, 纬度, 值]`
- `symbolSize` 函数 - 根据数据值动态计算散点气泡大小，使用 `Math.max` 保证最小尺寸
- `china.js` - 中国地图 GeoJSON 数据，注册名为 'china' 的地图
- `word.js` - 世界地图 GeoJSON 数据（本课未使用，但已引入备用）
