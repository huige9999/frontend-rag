# 19-区域选择-1

## 简介
本节课学习 brush（区域选择）组件的基础配置，了解如何通过框选、线性选择等方式选取图表数据区域，以及如何监听 brushSelected 事件获取选中的数据。

## 关键代码

### brush 区域选择完整配置

### brushSelected 事件监听

```javascript
myChart.on('brushSelected', function(params) {
    var brushed = [];                        // 存储选中的数据对应的索引
    var selected = params.batch[0].selected; // 选中的所有系列

    for (var i = 0; i < selected.length; i++) {
        var dataIndex = selected[i].dataIndex;
        brushed.push('[Series ' + i + '] ' + dataIndex.join(', '));
    }

    // 动态更新标题显示选中数据
    myChart.setOption({
        title: {
            backgroundColor: '#333',
            text: '选中数据的索引：\n' + brushed.join('\n'),
            bottom: 0,
            right: 0,
            width: 100,
            textStyle: {
                fontSize: 12,
                color: '#fff'
            }
        }
    });
});
```

### 代码说明
- `brush` - 区域选择组件，允许用户在图表上框选数据区域
- `xAxisIndex / yAxisIndex` - 指定在哪个坐标系内生效
- `seriesIndex` - 指定哪些系列参与框选（不指定则所有系列参与）
- `brushType` - 默认刷选类型：'rect'（矩形框选）、'lineX'（横向选择）、'lineY'（纵向选择）、'polygon'（多边形框选）
- `brushMode` - 刷选模式：'single'（单次选择）、'multiple'（多次叠加选择）
- `transformable` - 已有选框是否可以被拖拽移动或调整大小
- `brushStyle` - 选框的视觉样式
- `throttleType` - 事件节流方式，'debounce'（防抖）或 'fixRate'（固定频率）
- `throttleDelay` - 节流的延迟时间
- `removeOnClick` - 在空白处点击时是否移除所有选框
- `inBrush` - 被选中区域的数据的视觉样式
- `outOfBrush` - 未被选中区域的数据的视觉样式
- `brushSelected` 事件 - 当框选变化时触发，通过 `params.batch[0].selected` 获取各系列被选中的数据索引
