# 11-区域缩放1

## 简介
本节课学习 ECharts 的 dataZoom（区域缩放）组件配置，包括 inside 类型（内置型，通过鼠标/触摸操作缩放）和 slider 类型（滑动条型）两种缩放方式的详细属性。

## 关键代码

### dataZoom 区域缩放配置

```javascript
// 生成散点数据
var data1 = [], data2 = [], data3 = [];
function random(max) {
    return (Math.random() * max).toFixed(3);
}
for (var i = 0; i < 500; i++) {
    data1.push([random(15), random(10), random(1)]);
    data2.push([random(10), random(10), random(1)]);
    data3.push([random(15), random(10), random(1)]);
}

var myChart = echarts.init(document.getElementById('chart1'));

myChart.setOption({
    animation: false,    // 不开启动画（数据量大时建议关闭）

    legend: {},
    xAxis: {},
    yAxis: {},
    dataZoom: [
        // inside 类型 - 通过鼠标滚轮/触摸缩放
        {
            type: 'inside',
            disabled: false,
            xAxisIndex: 0,              // 控制的 X 轴索引
            startValue: 2,              // 缩放起始值
            endValue: 5,                // 缩放结束值
            orient: 'vertical',
            zoomOnMouseWheel: true,     // 鼠标滚轮缩放
            moveOnMouseMove: true,      // 鼠标移动平移
            moveOnMouseWheel: 'alt',    // 按住 Alt 滚轮平移
            preventDefaultMouseMove: true,
        },
        // inside 类型 - 控制 Y 轴
        {
            type: 'inside',
            yAxisIndex: [0],
            start: 30,
            end: 50,
        },
        // slider 类型 - 滑动条缩放（X 轴）
        {
            type: 'slider',
            backgroundColor: 'green',
            dataBackground: {           // 数据窗口背景样式
                lineStyle: {
                    opacity: 1,
                    color: 'red',
                },
                areaStyle: {
                    opacity: 1,
                    color: 'pink',
                    shadowColor: 'rgba(0, 255, 255, 1)',
                    shadowBlur: 1,
                    shadowOffsetX: 10,
                }
            },
            fillerColor: 'teal',        // 选中范围填充颜色
            borderColor: 'red',
            handleIcon: 'image://...',   // 手柄图标
            handleSize: '70%',          // 手柄大小
            handleStyle: {
                color: 'green',
                borderColor: 'green',
                borderWidth: 2,
            },
            labelPrecision: 0,          // 标签数值精度
            labelFormatter: function (value) {
                return value.toFixed(1) + ' ~';
            },
            showDetail: true,           // 拖拽时显示详细信息
            showDataShadow: true,       // 显示数据阴影
            realtime: false,            // 是否实时更新图表
            textStyle: {
                color: 'blue',
                fontSize: 16,
            },
            left: 'center',
        },
        // slider 类型 - 滑动条缩放（Y 轴）
        {
            type: 'slider',
            yAxisIndex: 0,
            start: 30,
            end: 50,
        }
    ],
    series: [
        {
            name: '钢铁',
            type: 'scatter',
            data: data1,
            symbolSize: function (val) {
                return val[2] * 40;
            }
        },
        {
            name: '石油',
            type: 'scatter',
            data: data2,
            symbolSize: function (val) {
                return val[2] * 40;
            }
        },
        {
            name: '煤炭',
            type: 'scatter',
            data: data3,
            symbolSize: function (val) {
                return val[2] * 40;
            }
        },
    ]
});
```

### 代码说明
- `dataZoom` 数组 - 可同时配置多个 dataZoom 组件，分别控制不同坐标轴
- **inside 类型** - 内置型数据区域缩放，依赖鼠标/触摸操作
  - `xAxisIndex` / `yAxisIndex` - 控制的坐标轴索引
  - `start` / `end` - 缩放范围的百分比位置（0~100）
  - `startValue` / `endValue` - 缩放范围的具体数值
  - `zoomOnMouseWheel` - 是否开启鼠标滚轮缩放
  - `moveOnMouseMove` - 是否开启鼠标移动平移
  - `moveOnMouseWheel` - 设置附加按键条件下滚轮平移
- **slider 类型** - 滑动条型数据区域缩放
  - `backgroundColor` - 滑动条背景色
  - `dataBackground` - 数据窗口的背景样式，包括 `lineStyle` 和 `areaStyle`
  - `fillerColor` - 选中范围的填充颜色
  - `handleIcon` / `handleSize` / `handleStyle` - 两端手柄的图标、大小和样式
  - `labelPrecision` - 滑动条标签的数值精度
  - `labelFormatter` - 标签格式化函数
  - `showDetail` - 拖拽时是否显示详细数值信息
  - `showDataShadow` - 是否在滑动条中显示数据趋势阴影
  - `realtime` - 拖拽时是否实时更新图表（大数据量时建议设为 `false`）
