# 33-饼图

## 简介
本节课学习 ECharts 饼图（pie）的完整配置，包括饼图数据格式、南丁格尔玫瑰图模式、标签及引导线样式、选中交互、起始角度等核心属性。

## 关键代码

### 饼图配置

```javascript
option = {
  backgroundColor: '#2c343c',

  title: {
    text: '访问来源',
    left: 'center',
    top: 20,
    textStyle: {
      color: '#ccc',
      fontSize: 30
    }
  },
  legend: {
    textStyle: {
      color: '#ccc',
    },
    top: 20,
    right: 20,
  },
  tooltip: {
    formatter: '{a}<br>{b} : {c} ({d}%)',
  },
  series: [
    {
      name: '访问来源',
      type: 'pie',
      data: [
        { value: 335, name: '直接访问' },
        { value: 310, name: '邮件营销' },
        { value: 274, name: '联盟广告' },
        { value: 235, name: '视频广告' },
        { value: 400, name: '搜索引擎' }
      ].sort(function (a, b) { return a.value - b.value }),

      legendHoverLink: true,
      hoverAnimation: true,
      hoverOffset: 50,
      selectedMode: 'multiple',
      selectedOffset: 50,
      clockwise: true,
      startAngle: 90,
      minShowLabelAngle: 90,

      roseType: 'area',

      label: {
        show: true,
        position: 'outside',
        fontSize: 20,
      },
      labelLine: {
        show: true,
        length: 50,
        length2: 30,
        smooth: false,
        lineStyle: {
          width: 3,
          color: '#fff',
        }
      },
      itemStyle: {
        borderColor: 'green',
        borderWidth: 2,
      },
      emphasis: {
        label: {
          fontSize: 30,
        },
        labelLine: {
          lineStyle: {
            color: 'blue',
          }
        },
        itemStyle: {
          borderWidth: 5,
        }
      },

      center: ['50%', '50%'],
      radius: '75%',
      // 环形图配置
      radius: ['65%', '20%'],
      label: {
        position: 'inside',
      },
      roseType: false,
    }
  ]
};
```

### 代码说明
- `type: 'pie'` - 指定系列类型为饼图
- `data` - 饼图数据格式为 `{ value, name }` 数组，`.sort()` 按值从小到大排序
- `hoverAnimation` - 鼠标悬浮时是否开启扇区放大动画
- `hoverOffset` - 悬浮时扇区偏移距离
- `selectedMode` - 选中模式，支持 `single`（单选）和 `multiple`（多选）
- `selectedOffset` - 选中扇区偏移距离
- `clockwise` - 扇区是否顺时针排列
- `startAngle` - 起始角度，默认 90 度（从正上方开始）
- `minShowLabelAngle` - 最小显示标签角度，小于该角度的扇区不显示标签
- `roseType` - 南丁格尔玫瑰图模式，`'area'` 表示面积模式，`false` 关闭
- `center` - 饼图中心坐标，支持百分比和像素值
- `radius` - 饼图半径，字符串为实心饼图，数组 `[外半径, 内半径]` 为环形图
- `labelLine` - 标签引导线配置，`length` 为第一段长度，`length2` 为第二段长度
- `tooltip.formatter` - `{a}` 系列名，`{b}` 数据名，`{c}` 数据值，`{d}` 百分比
