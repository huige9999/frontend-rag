# 08-坐标轴2

## 简介
本节课深入学习 xAxis 坐标轴的详细配置，包括坐标轴位置、轴线样式、刻度、刻度标签、分隔线和分隔区域等高级属性。

## 关键代码

### 坐标轴高级配置

```javascript
var myChart1 = echarts.init(document.getElementById('chart1'));

myChart1.setOption({
    title: {
        text: '当月销售业绩',
    },
    legend: {
        data: ['今日销量', '昨日销量', '前日销量'],
    },
    xAxis: {
        show: true,
        gridIndex: 0,
        position: 'bottom',          // 坐标轴位置：'top' 或 'bottom'
        type: 'category',            // 坐标轴类型：'value' 数值 / 'category' 类目 / 'time' 时间 / 'log' 对数
        data: ['衬衫', '羊毛衫', '雪纺衫', '裤子', '高跟鞋', {
            value: '袜子',
            textStyle: {
                fontSize: 16          // 单独设置某项的文本样式
            }
        }],

        name: '种类',                 // 坐标轴名称
        nameLocation: 'end',         // 坐标轴名称显示位置：'start' / 'middle' / 'end'
        nameTextStyle: {},
        nameGap: 20,                 // 坐标轴名称与轴线的距离
        nameRotate: 45,              // 坐标轴名称旋转角度
        inverse: false,              // 是否反向坐标轴
        boundaryGap: true,           // 坐标轴两边是否留白

        // 坐标轴轴线相关设置
        axisLine: {
            show: true,
            onZero: true,            // X 轴或 Y 轴的轴线是否在另一个轴的 0 刻度上
            symbol: ['none', 'arrow'],         // 轴线两端箭头
            symbolSize: [30, 25],              // 箭头大小 [宽度, 高度]
            symbolOffset: [0, -30],
            lineStyle: {
                color: '#000',
            }
        },

        // 坐标轴刻度相关设置
        axisTick: {
            show: true,
            alignWithLabel: false,   // 刻度线与标签对齐
            interval: 0,
            inside: true,            // 刻度朝内
            length: 10,
            lineStyle: {
                type: 'dotted'       // 刻度线样式：'solid' / 'dashed' / 'dotted'
            }
        },

        // 坐标轴刻度标签的相关设置
        axisLabel: {
            show: true,
            interval: 0,             // 标签显示间隔，0 表示显示所有标签
            inside: false,
            rotate: 45,              // 标签旋转角度
            margin: 18,              // 标签与轴线之间的距离
            formatter: function (value, index) {
                return value + '好';
            },
            color: 'green',
        },

        // 分隔线的设置
        splitLine: {
            show: true,
            interval: 1,
            lineStyle: {
                color: '#000',
            }
        },

        // 分隔区域的设置
        splitArea: {
            show: true,
        }
    },
    yAxis: {},
    series: [
        {
            name: '今日销量',
            type: 'bar',
            data: [5, 20, 36, 10, 19, 24]
        },
        {
            name: '昨日销量',
            type: 'bar',
            data: [5, 20, 36, 10, 19, 24]
        },
        {
            name: '前日销量',
            type: 'bar',
            data: [5, 20, 36, 10, 19, 24]
        }
    ]
});
```

### 代码说明
- `position` - 坐标轴位置，X 轴支持 `'top'` / `'bottom'`
- `type` - 坐标轴类型：`'category'` 类目轴、`'value'` 数值轴、`'time'` 时间轴、`'log'` 对数轴
- `name` / `nameLocation` / `nameGap` / `nameRotate` - 坐标轴名称及其位置、间距、旋转角度
- `inverse` - 是否反向坐标轴
- `boundaryGap` - 坐标轴两边留白策略，`true` 表示类目两侧留白
- `axisLine` - 坐标轴轴线配置，包括箭头 `symbol`、箭头大小 `symbolSize`、线样式 `lineStyle`
- `axisTick` - 坐标轴刻度配置，包括 `alignWithLabel` 对齐、`inside` 朝向、`length` 长度
- `axisLabel` - 坐标轴标签配置，包括 `interval` 显示间隔、`rotate` 旋转、`margin` 间距、`formatter` 格式化
- `splitLine` - 坐标轴分隔线配置
- `splitArea` - 坐标轴分隔区域配置，可用交替颜色背景区分区域
