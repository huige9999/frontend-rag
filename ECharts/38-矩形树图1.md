# 38-矩形树图1

## 简介
本节课学习 ECharts 矩形树图（treemap）的配置，包括数据转换函数、多层级的 levels 配置、颜色映射策略、面包屑导航以及矩形块的交互效果。数据来源于 treeMapData.js（ECharts 配置项查询分布数据）。

## 关键代码

### 数据转换函数

```javascript
// 将扁平的配置项对象转换为树形结构数据
function convert(source) {
  return (function inner(source, target, basePath) {
    for (var key in source) {
      var path = basePath ? (basePath + '.' + key) : key;
      if (key.indexOf('$') == -1) {
        target.children = target.children || [];
        var child = { name: path };
        target.children.push(child);
        inner(source[key], child, path);
      }
    }
    if (!target.children) {
      target.value = source.$count || 1;
    } else {
      if (source.$count != undefined) {
        target.children.push({
          name: basePath,
          value: source.$count,
        });
      }
    }
    return target;
  })(source, [], '');
}
```

### 代码说明
- `type: 'treemap'` - 指定系列类型为矩形树图
- `squareRatio` - 期望的矩形长宽比，设为黄金比例使方块更美观
- `leafDepth` - 叶子节点显示深度，设为 1 表示一次只展开一层
- `drillDownIcon` - 下钻图标，显示在可展开的节点旁
- `nodeClick: 'zoomToNode'` - 点击节点时缩放到该节点
- `levels` - 多层级配置数组，每个元素对应一个层级的样式
- `colorMappingBy: 'index'` - 按索引映射颜色
- `colorSaturation` - 颜色饱和度范围，不同层级设置不同范围实现渐变效果
- `upperLabel` - 上层标签（父节点的标签），显示在矩形块顶部
- `gapWidth` - 同层级矩形块之间的间距
- `borderColorSaturation` - 边框颜色饱和度，影响边框颜色深浅
- `breadcrumb` - 面包屑导航，用于返回上层节点
- `convert()` 函数 - 将原始嵌套配置对象转换为 treemap 需要的 `{ name, value, children }` 格式
