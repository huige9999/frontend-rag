# 50-象形柱图

## 简介
本节课学习 ECharts 象形柱图（pictorialBar）系列的使用，通过自定义 SVG 路径图形替代传统柱形，实现更具视觉表现力的数据展示，包括图形定位、重复、裁剪等高级用法。

## 关键代码

### 示例一：基础象形柱图（符号定位与重复）

```javascript
var walk = 'path://M29.902,23.275c1.86,0...'; // 人物行走 SVG 路径

option = {
    legend: {
        itemHeight: 20,
        textStyle: { fontSize: 20 },
        itemGap: 50,
    },
    xAxis: {
        data: ['a', 'b', 'c', 'd', 'e'],
        axisTick: { show: false },
        axisLabel: { fontSize: 20 }
    },
    yAxis: {
        max: 80,
        splitLine: { show: false }
    },
    series: [
        {
            name: '参考柱子',
            type: 'bar',
            itemStyle: { color: '#ffd48a' },
            data: [60, 60, 60, 60, 60]
        },
        {
            name: '图形元素',
            type: 'pictorialBar',
            label: {
                show: true,
                fontSize: 24,
            },
            itemStyle: {
                color: 'green',
                shadowColor: 'rgba(0,0,0,0.5)',
                shadowBlur: 10,
            },
            z: 5,
            emphasis: {
                itemStyle: { color: 'pink' }
            },
            symbol: walk,
            symbolSize: [50, 120],
            data: [
                {
                    value: 60,
                    symbolPosition: 'start',    // 图形在柱子起点
                },
                {
                    value: 60,
                    symbolPosition: 'end',      // 图形在柱子终点
                },
                {
                    value: 60,
                    symbolPosition: 'center',   // 图形在柱子中间
                },
                {
                    value: 60,
                    symbolPosition: 'end',
                    symbolOffset: [-80, '-100%'], // 图形偏移
                    symbolRotate: 90,             // 图形旋转
                },
                {
                    value: 60,
                    symbolRepeat: true,           // 图形重复
                    symbolRepeatDirection: 'end', // 重复方向
                    animationDelay: function (dataIndex, params) {
                        return params.index * 100;
                    }
                }
            ]
        }
    ]
};
```

### 示例二：裁剪效果象形柱图

```javascript
var women = 'path://M36.7,102.84c-1.17...'; // 人物剪影 SVG 路径

option = {
    legend: {
        itemHeight: 20,
        textStyle: { fontSize: 20 },
        itemGap: 50,
    },
    xAxis: {
        data: ['a', 'b', 'c'],
        axisTick: { show: false },
        axisLabel: { fontSize: 20 }
    },
    yAxis: {
        max: 80,
        splitLine: { show: false }
    },
    series: [
        {
            name: '背景图',
            type: 'pictorialBar',
            symbol: women,
            symbolBoundingData: 80,
            label: {
                show: true,
                position: 'outside',
                formatter: 'value: {c}',
                textStyle: {
                    color: '#bb0004',
                    fontSize: 18,
                }
            },
            itemStyle: { color: '#ccc' },
            data: [
                { value: 40, animationDuration: 0 },
                {
                    value: 40,
                    symbolSize: ['50%', '80%'],
                    symbolRepeat: 'fixed',
                    animationDuration: 0,
                },
                {
                    value: 40,
                    symbolSize: ['50%', '80%'],
                    symbolRepeat: 'fixed',
                }
            ]
        },
        {
            name: '裁剪图',
            type: 'pictorialBar',
            symbol: women,
            symbolBoundingData: 80,
            z: 5,
            data: [
                { value: 40, symbolClip: true },
                {
                    value: 40,
                    symbolSize: ['50%', '80%'],
                    symbolRepeat: 'fixed',
                    symbolClip: true,
                },
                {
                    value: 40,
                    symbolSize: ['50%', '80%'],
                    symbolRepeat: true,
                    symbolClip: false,
                }
            ]
        }
    ]
};
```

### 代码说明
- `type: 'pictorialBar'` - 指定系列类型为象形柱图
- `symbol` - 指定图形元素，可使用 `'circle'`、`'rect'` 等内置形状，或 `'path://...'` 自定义 SVG 路径
- `symbolSize` - 图形尺寸，`[宽, 高]` 格式
- `symbolPosition` - 图形在柱子上的位置
  - `'start'` - 柱子起点
  - `'end'` - 柱子终点
  - `'center'` - 柱子中间
- `symbolOffset` - 图形的偏移量，`[x偏移, y偏移]`
- `symbolRotate` - 图形旋转角度
- `symbolRepeat: true` - 图形是否沿柱子方向重复
- `symbolRepeatDirection: 'end'` - 图形重复的方向
- `symbolBoundingData` - 图形的边界数据值，用于限制图形的绘制范围
- `symbolClip: true` - 是否裁剪图形，开启后图形只显示数据值对应高度的部分（实现"填充"效果）
- `symbolRepeat: 'fixed'` - 固定数量的重复模式
- `animationDuration: 0` - 设为0关闭动画，用于背景图形
- `z: 5` - 层叠顺序，确保裁剪图在背景图之上
- 两个 series 叠加：灰色背景图（完整图形）+ 彩色裁剪图（按数据值裁剪），形成"进度条"视觉效果
