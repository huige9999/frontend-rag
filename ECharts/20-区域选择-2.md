# 20-区域选择-2

## 简介
本节课是区域选择组件的继续实践，代码内容与第19课一致，重点在于巩固 brush 组件的配置以及 brushSelected 事件的使用，配合堆叠柱状图展示数据框选与视觉反馈效果。

## 关键代码

### brush 区域选择配置（堆叠柱状图示例）

```javascript
var myChart = echarts.init(document.getElementById('chart'));
myChart.setOption({
    legend: {
        left: 'left',
    },
    xAxis: {
        data: xAxisData,
        name: 'X 轴',
    },
    yAxis: {
        inverse: true,
    },
    grid: {
        left: 100,
    },
    tooltip: {},
    visualMap: {
        type: 'continuous',
        text: ['Hight', 'Low'],
        inverse: true,
        top: 60,
        left: 10,
        calculable: true,
        itemHeight: 200,
        min: -2,
        max: 6,
        inRange: {
            colorLightness: [0.4, 0.8],
        },
        outOfRange: {
            color: '#bbb',
        },
        controller: {
            inRange: { color: '#2f4554' }
        }
    },
    dataZoom: {
        type: 'inside',
    },
    brush: {
        xAxisIndex: 0,
        brushType: 'lineX',          // 横向线性选择
        brushMode: 'single',         // 单选模式
        transformable: false,        // 选框不可移动/调整
        brushStyle: {
            borderWidth: 2,
            color: 'rgba(0,0,0,0.3)',
            borderColor: 'rgba(0,0,0,0.5)',
        },
        throttleType: 'debounce',    // 防抖节流
        throttleDelay: 1000,         // 1秒延迟
        removeOnClick: false,        // 点击不删除选框

        inBrush: {
            color: '#bbb',           // 选中区域颜色
        },
        outOfBrush: {
            opacity: 0.1,            // 未选中区域透明度
        }
    },
    series: [
        { name: 'bar1', type: 'bar', stack: 'one', data: data1 },
        { name: 'bar2', type: 'bar', stack: 'one', data: data2 },
        { name: 'bar3', type: 'bar', stack: 'two', data: data3 },
        { name: 'bar4', type: 'bar', stack: 'two', data: data4 },
    ]
});
```

### brushSelected 事件处理

```javascript
myChart.on('brushSelected', function(params) {
    var brushed = [];
    var selected = params.batch[0].selected;

    for (var i = 0; i < selected.length; i++) {
        var dataIndex = selected[i].dataIndex;
        brushed.push('[Series ' + i + '] ' + dataIndex.join(', '));
    }

    myChart.setOption({
        title: {
            backgroundColor: '#333',
            text: '选中数据的索引：\n' + brushed.join('\n'),
            bottom: 0,
            right: 0,
            width: 100,
            textStyle: {
                fontSize: 12,
                color: '#fff'
            }
        }
    });
});
```

### 代码说明
- `brushType: 'lineX'` - 使用横向线性选择模式，适合柱状图按列选择数据
- `brushMode: 'single'` - 单选模式，每次只能选择一个区域
- `transformable: false` - 选框一旦创建就不能被拖动或缩放
- `throttleType: 'debounce'` - 使用防抖策略，避免频繁触发事件
- `inBrush / outOfBrush` - 通过视觉差异（颜色、透明度）区分选中和未选中的数据
- `brushSelected` 事件 - 监听框选变化，`params.batch[0].selected` 是数组，包含每个系列中被选中的数据索引 `dataIndex`
- 配合 `stack` 堆叠柱状图，brush 可以选择整个堆叠组的数据
