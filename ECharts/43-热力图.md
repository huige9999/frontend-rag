# 43-热力图

## 简介
本节课学习 ECharts 热力图（heatmap）系列的使用，包括基于笛卡尔坐标系的基础热力图和基于地理坐标系的地图热力图两种形式。

## 示例一：笛卡尔坐标系热力图

### 关键代码

```javascript
var hours = ['12a', '1a', '2a', '3a', '4a', '5a', '6a',
    '7a', '8a', '9a', '10a', '11a',
    '12p', '1p', '2p', '3p', '4p', '5p',
    '6p', '7p', '8p', '9p', '10p', '11p'];
var days = ['Saturday', 'Friday', 'Thursday',
    'Wednesday', 'Tuesday', 'Monday', 'Sunday'];

// 数据格式: [x索引, y索引, 值]
var data = [[0, 0, 5], [0, 1, 1], [0, 2, 0], ...];

data = data.map(function (item) {
    return [item[1], item[0], item[2] || '-'];
});

option = {
    title: {
        text: '热力图',
        left: 'center',
    },
    grid: {
        height: '50%',
        y: '10%'
    },
    xAxis: {
        data: hours,
        splitArea: {
            show: true,
        }
    },
    yAxis: {
        data: days,
        splitArea: {
            show: true,
        }
    },
    tooltip: {
        position: 'top',
    },
    visualMap: {
        min: 0,
        max: 10,
        calculable: true,
        orient: 'horizontal',
        left: 'center',
        bottom: '15%',
    },
    series: [
        {
            name: '打卡',
            type: 'heatmap',
            data: data,
            label: {
                show: true,
            },
            emphasis: {
                itemStyle: {
                    shadowBlur: 10,
                    shadowColor: 'rgba(0,0,0,0.5)'
                }
            }
        }
    ]
};
```

### 代码说明
- `type: 'heatmap'` - 指定系列类型为热力图
- `xAxis.data` / `yAxis.data` - 分别设置x轴（小时）和y轴（星期）的类目数据
- `splitArea.show: true` - 显示坐标轴分隔区域，使热力图格子更清晰
- `visualMap` - 视觉映射组件，控制热力图颜色的映射范围（min~max），`calculable: true` 允许拖拽手柄筛选范围
- `data` - 数据格式为 `[xIndex, yIndex, value]`，通过 map 交换了x/y索引
- `emphasis.itemStyle.shadowBlur` - 鼠标悬浮时的阴影效果

## 示例二：地理坐标系热力图（地图热力图）

### 关键代码

```javascript
// 城市地理坐标映射
var geoCoordMap = {
    "海门": [121.15, 31.89],
    "鄂尔多斯": [109.781327, 39.608266],
    // ... 更多城市坐标
};

// 每个城市的PM2.5数据
var pmData = [
    { name: "海门", value: 9 },
    { name: "鄂尔多斯", value: 12 },
    // ... 更多城市数据
];

// 数据转换函数：将城市名+数值转换为 [经度, 纬度, 值]
function converData() {
    var res = [];
    for (var i = 0; i < pmData.length; i++) {
        var geoData = geoCoordMap[pmData[i].name];
        res.push({
            name: pmData[i].name,
            value: geoData.concat(pmData[i].value),
        });
    }
    return res;
}

option = {
    backgroundColor: '#404a59',
    title: {
        text: '全国主要城市 PM2.5',
        left: 'center',
        textStyle: {
            color: '#fff',
        }
    },
    visualMap: {
        type: 'continuous',
        min: 0,
        max: 200,
        calculable: true,
        inRange: {
            color: ['#50a3ba', '#eac736', '#d94e5d']
        }
    },
    geo: {
        map: 'china',
        roam: true,
        label: {
            show: true,
            color: '#fff',
        },
        itemStyle: {
            areaColor: 'grey'
        },
        emphasis: {
            itemStyle: {
                areaColor: 'teal'
            }
        }
    },
    series: [
        {
            name: 'PM 2.5',
            type: 'heatmap',
            data: converData(),
            coordinateSystem: 'geo',
            blurSize: 20,
            minOpacity: 0,
            maxOpacity: 1
        }
    ]
};
```

### 代码说明
- `coordinateSystem: 'geo'` - 指定使用地理坐标系，而非默认的笛卡尔坐标系
- `geo.map: 'china'` - 使用中国地图（需引入 china.js）
- `geo.roam: true` - 允许地图缩放和平移
- `visualMap.inRange.color` - 定义颜色渐变范围，从蓝（低值）到黄到红（高值）
- `blurSize` - 热力图的模糊半径，值越大热力点越模糊扩散
- `minOpacity` / `maxOpacity` - 控制热力点最小/最大透明度
- `converData()` - 自定义数据转换函数，将城市名映射为经纬度坐标并拼接数值
